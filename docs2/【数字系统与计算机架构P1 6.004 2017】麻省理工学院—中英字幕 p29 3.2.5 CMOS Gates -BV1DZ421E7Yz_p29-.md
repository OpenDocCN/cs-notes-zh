Mr。 Blue is asking a good question。 Will this recipe work for any and all logic functions。



![](img/32c8c1cef87067e01521f6bd5ecbae9f_24.png)

The answer is no， let's see why。

![](img/32c8c1cef87067e01521f6bd5ecbae9f_26.png)

Using CMOOSs， a single gate， a circuit with one pull up network and one pull down network can only implement the so called inverting functions where rising inputs lead to falling outputs and vice versa。



![](img/32c8c1cef87067e01521f6bd5ecbae9f_28.png)

![](img/32c8c1cef87067e01521f6bd5ecbae9f_29.png)

To see why， consider what happens when one of the gate's inputs goes from 0 to 1。

Any unFt switches controlled by the rising input will go from off to on。



![](img/32c8c1cef87067e01521f6bd5ecbae9f_31.png)

![](img/32c8c1cef87067e01521f6bd5ecbae9f_32.png)

This may enable one or more paths between the gates output and ground。

And PFET switches controlled by the rising input will go from on to off。



![](img/32c8c1cef87067e01521f6bd5ecbae9f_34.png)

This may disable one or more paths between the gates output and VDD。



![](img/32c8c1cef87067e01521f6bd5ecbae9f_36.png)

So if the gates output changes as a result of the rising input。

 it must be because some pull down path was enabled and some pull up path was disabled。



![](img/32c8c1cef87067e01521f6bd5ecbae9f_38.png)

In other words， any change in the output voltage due to a rising input must be a falling transition from 1 to zero。

Similar reasoning tells us that falling inputs must lead to rising outputs。 In fact。

 for any non constant seamosgate， we know that its output must be one when all inputs are0。

 since all the n fats are off and all the PFs are on。



![](img/32c8c1cef87067e01521f6bd5ecbae9f_40.png)

![](img/32c8c1cef87067e01521f6bd5ecbae9f_41.png)

![](img/32c8c1cef87067e01521f6bd5ecbae9f_42.png)

In vice versa， if all the inputs are1， the gates output must be zero。



![](img/32c8c1cef87067e01521f6bd5ecbae9f_44.png)

This means that so called positive logic can't be implemented with a single CMmoss gate。



![](img/32c8c1cef87067e01521f6bd5ecbae9f_46.png)

![](img/32c8c1cef87067e01521f6bd5ecbae9f_49.png)

Furthermore， we can see that when A is1 and B rises from 0 to 1， the output rises instead of falls。



![](img/32c8c1cef87067e01521f6bd5ecbae9f_51.png)
