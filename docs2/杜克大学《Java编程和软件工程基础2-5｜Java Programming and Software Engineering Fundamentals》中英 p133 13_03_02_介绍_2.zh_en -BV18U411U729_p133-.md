Now， let's learn about sorting， taking a bunch of data and putting it in order so that you can analyze it more easily。



![](img/db54f1b1a43bf56fadedf0f0a62a9c0e_2.png)

Sorting has lots of applications， especially when you're dealing with very large data sets。

For some problems， sorting the data as a first step makes the problem much easier to solve。

 Imagine if you needed to find the middle element in an array or an array list。If the data is sorted。

 you could just look at the middle element directly。 If the data is unsorted。What could you do。

 What kind of algorithm could you create to solve that problem。

 Soting can also make some problems more efficient。

 We have not really dealt with efficiency very much in this course， just on writing code that works。

But if you are dealing with billions of pieces of data。

 efficiency can be the difference between seconds and years。
