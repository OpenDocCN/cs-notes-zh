Cmoss gates consist of an output node connected to a pull up circuit that contains only PFEts and a pull down circuit that contains only nphts。



![](img/8388d180ba87ee4a2d1b58dbb7cd1ce8_2.png)

![](img/8388d180ba87ee4a2d1b58dbb7cd1ce8_3.png)

In our example， our gate computes F of A， B，C， and D。 It is observed that f of 1，0，1，1 equals 1。



![](img/8388d180ba87ee4a2d1b58dbb7cd1ce8_5.png)

![](img/8388d180ba87ee4a2d1b58dbb7cd1ce8_6.png)

With that information， what can you say about the value of F of 1，0，0，0。



![](img/8388d180ba87ee4a2d1b58dbb7cd1ce8_8.png)

All seamo gates are inverting， which means that if all your inputs are0。

 then only the PFts in the pull up are on。 and so the output equals 1。



![](img/8388d180ba87ee4a2d1b58dbb7cd1ce8_10.png)

If all your inputs are1， then only the endFs in the pull down circuitry are on。

 and the output equals 0。

![](img/8388d180ba87ee4a2d1b58dbb7cd1ce8_12.png)

If we are told that f of 1，0，1，1 equals 1， and we want to find out what f of 1，0，00 is。

 we notice that the difference between the first set of inputs and the second is that the third and fourth inputs have been changed from 1 to 0。

 while the others remain unchanged。

![](img/8388d180ba87ee4a2d1b58dbb7cd1ce8_14.png)

![](img/8388d180ba87ee4a2d1b58dbb7cd1ce8_15.png)

![](img/8388d180ba87ee4a2d1b58dbb7cd1ce8_16.png)

Changing an input from1 to 0 will turn more Pfets on and more nFts off。

 which means that there are more possibilities of the pull up circuitry to pull the output up to high and fewer chances of the pull down circuitry。

 pulling the output down to 0。 Since we were already producing a high output with even fewer Pfets turned on。

 we are guaranteed that by turning more Pfets on and turning some nfets off。

 the only thing we can do is maintain our high output。 So F of 1，0，0，0 equals1。😊。
