If we don't have an upper bound on the number of states in the FSM。

This is because in a K state FSM， every reachable state can be reached in less than K transitions。

We say that two FSMs are equivalent if and only if every input sequence yields identical output sequences from both FSMs。

 so as engineers， if we have an FSM， we would like to find the simplest and hence the least expensive equivalent FSM。
