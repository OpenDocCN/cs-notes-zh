# Python 编程入门：01：理解变量与数据类型

在本节课中，我们将要学习 Python 编程中最基础的两个概念：变量和数据类型。它们是构建所有程序的基石。

## 概述

变量是用于存储信息的容器。数据类型则定义了容器中可以存放什么类型的信息，以及可以对信息进行哪些操作。理解这两者，是编写有效代码的第一步。

## 什么是变量？

你可以把变量想象成一个贴有标签的盒子。标签就是变量的名字，盒子里面存放的就是数据。在 Python 中，我们使用等号 `=` 来给变量赋值。

**代码示例：**
```python
message = "Hello, World!"
number = 42
```
在上面的代码中，我们创建了两个变量：`message` 和 `number`。`message` 这个“盒子”里存放了文本“Hello, World!”，而 `number` 这个“盒子”里存放了数字 42。

## 基本数据类型

Python 有多种内置的数据类型，用于处理不同类型的数据。以下是几种最常用的基本数据类型。

### 1. 整数 (int)
整数是不带小数点的数字，可以是正数、负数或零。
**代码示例：**
```python
age = 25
temperature = -10
count = 0
```

### 2. 浮点数 (float)
浮点数是带有小数点的数字，用于表示实数。
**代码示例：**
```python
price = 19.99
pi = 3.14159
```

### 3. 字符串 (str)
字符串是由零个或多个字符组成的文本序列，需要用单引号 `‘ ’` 或双引号 `“ ”` 括起来。
**代码示例：**
```python
name = "Alice"
greeting = 'Hello there!'
```

### 4. 布尔值 (bool)
布尔值只有两个：`True`（真）和 `False`（假）。它们通常用于逻辑判断。
**代码示例：**
```python
is_learning = True
task_completed = False
```

---

## 如何查看数据类型？

有时你需要知道一个变量是什么类型。可以使用 `type()` 函数来查看。

**代码示例：**
```python
print(type(age))     # 输出：<class 'int'>
print(type(price))   # 输出：<class 'float'>
print(type(name))    # 输出：<class 'str'>
```

## 变量的命名规则

为了写出清晰易懂的代码，给变量起一个好名字非常重要。以下是命名时需要遵守的规则和建议。

*   变量名只能包含字母、数字和下划线。
*   变量名不能以数字开头。
*   变量名是区分大小写的（例如，`myVar` 和 `myvar` 是两个不同的变量）。
*   应使用描述性的名称，让其他人（以及未来的你）一眼就能明白这个变量的用途。

**良好示例：**
```python
user_name = “Bob”
item_count = 5
is_logged_in = True
```

**不佳示例：**
```python
a = “Bob”      # 不具描述性
2nd_item = 5   # 以数字开头
is-logged-in = True # 包含了连字符“-”
```

## 总结

本节课中我们一起学习了 Python 编程的基础。我们了解到，**变量**是存储数据的命名容器，而**数据类型**（如整数、浮点数、字符串和布尔值）则定义了数据的种类和可执行的操作。我们还学习了如何使用 `type()` 函数检查数据类型，以及为变量命名的基本规则。掌握这些概念，将为后续学习更复杂的编程知识打下坚实的基础。
