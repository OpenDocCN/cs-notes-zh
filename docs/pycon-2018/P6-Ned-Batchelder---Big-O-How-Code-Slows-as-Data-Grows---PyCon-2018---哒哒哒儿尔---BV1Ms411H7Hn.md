你必须说明你是在讨论典型情况的复杂性还是最坏情况。这是一个例子，我创建了一个包含 50,000 个数字的集合，这些数字相差 47。我是在按 47 的步伐增加这些数字。将这个数字添加到集合中是一个 O(1) 操作。因此，整个集合的构建是 O(n)，并且大约花了 10 毫秒。
