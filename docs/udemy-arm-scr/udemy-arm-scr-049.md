# 049：下载STM32开发工具 🛠️

在本节课中，我们将学习如何下载由意法半导体（STMicroelectronics）发布的两款核心开发工具：STM32CubeMX和STM32CubeIDE。它们是进行STM32微控制器开发的基础软件。

## 下载STM32CubeMX

上一节我们介绍了课程目标，本节中我们来看看如何获取STM32CubeMX。STM32CubeMX是一个图形化配置工具，用于初始化C代码和设置硬件。

![](img/058c2ab2f287f582ff4f5529c346ced1_1.png)

![](img/058c2ab2f287f582ff4f5529c346ced1_3.png)

以下是下载STM32CubeMX的步骤：

1.  打开浏览器，访问搜索引擎。
2.  搜索关键词“Download STM32CubeMX”。
3.  在搜索结果中，找到并进入意法半导体的官方网站（st.com）。
4.  在网站上找到STM32CubeMX的下载页面。
5.  页面会显示最新版本（例如5.5版）。如果需要旧版本，也可以在此选择。
6.  点击“Get Software”按钮。
7.  阅读并接受弹出的软件许可协议。
8.  在接下来的页面中，输入您的电子邮箱地址和姓名。
9.  登录您的账户。
10. 登录后，页面将跳转回下载页，点击“Download”按钮开始下载。有时下载也会自动开始。

下载过程完成后，您就获得了STM32CubeMX的安装文件。

## 下载STM32CubeIDE

在成功下载了配置工具后，接下来我们需要获取集成开发环境（IDE）。STM32CubeIDE是意法半导体新发布的开发环境，它将代码编辑、编译和调试功能集成在一起。

![](img/058c2ab2f287f582ff4f5529c346ced1_5.png)

以下是下载STM32CubeIDE的步骤：

![](img/058c2ab2f287f582ff4f5529c346ced1_7.png)

![](img/058c2ab2f287f582ff4f5529c346ced1_8.png)

1.  在STM32CubeMX下载页面或通过搜索，找到STM32CubeIDE的下载入口。
2.  此IDE完全免费，没有代码大小限制。这与某些有32KB代码限制的试用版IDE不同。
3.  进入下载页面后，选择与您操作系统对应的版本（例如Windows安装程序）。
4.  点击“Get Software”按钮。
5.  再次阅读并接受软件许可协议。
6.  可能需要再次登录您的账户。
7.  登录后，下载将自动开始。该软件安装包较大（例如约665MB），下载可能需要一些时间。

![](img/058c2ab2f287f582ff4f5529c346ced1_10.png)

下载完成后，您就准备好了STM32CubeIDE的安装程序。

![](img/058c2ab2f287f582ff4f5529c346ced1_12.png)
