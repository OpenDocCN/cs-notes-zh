## 定时器初始化

在配置好 GPIO 之后，本节我们将重点实现定时器（Timer2）的初始化。

我们创建一个名为 `timer2_init` 的子程序。

![](img/494489777607460524688a3fc6df818a_26.png)

```assembly
timer2_init:
```

![](img/494489777607460524688a3fc6df818a_28.png)

### 1. 使能时钟访问

首先，我们需要使能 Timer2 的时钟访问。在 C 语言中，这通常通过设置 `RCC_APB1ENR` 寄存器的相应位来完成。

![](img/494489777607460524688a3fc6df818a_30.png)

以下是实现步骤：

1.  将 `RCC_APB1ENR` 寄存器的地址加载到寄存器 R0。
2.  将该地址处的值读入寄存器 R1。
3.  使用 OR 操作设置 `TIM2EN` 位（Timer2 使能位）。
4.  将修改后的值写回寄存器。

![](img/494489777607460524688a3fc6df818a_32.png)

```assembly
    ldr r0, =RCC_APB1ENR
    ldr r1, [r0]
    orr r1, r1, #(1 << 0) // 假设 TIM2EN 是位0
    str r1, [r0]
```

### 2. 设置预分频器

![](img/494489777607460524688a3fc6df818a_34.png)

![](img/494489777607460524688a3fc6df818a_36.png)

接下来，我们配置预分频器寄存器以设置定时器的计数频率。

在 C 语言中，这对应 `TIM2_PSC` 寄存器。我们将一个预定义的配置值写入该寄存器。

![](img/494489777607460524688a3fc6df818a_38.png)

```assembly
    ldr r0, =TIM2_PSC
    movw r1, #PSC_CNF // 假设 PSC_CNF 是预定义的16位值
    strh r1, [r0]
```

![](img/494489777607460524688a3fc6df818a_40.png)

### 3. 设置自动重装载寄存器

![](img/494489777607460524688a3fc6df818a_42.png)

然后，我们设置自动重装载寄存器，它决定了定时器的溢出周期。

![](img/494489777607460524688a3fc6df818a_43.png)

![](img/494489777607460524688a3fc6df818a_45.png)

在 C 语言中，这对应 `TIM2_ARR` 寄存器。我们将一个预定义的计数值（例如 10000）写入该寄存器。

```assembly
    ldr r0, =TIM2_ARR
    movw r1, #ARR_CNF // 假设 ARR_CNF 是预定义的16位值
    strh r1, [r0]
```

![](img/494489777607460524688a3fc6df818a_47.png)

### 4. 清零计数器

![](img/494489777607460524688a3fc6df818a_49.png)

在启动定时器前，需要将当前计数值清零。

![](img/494489777607460524688a3fc6df818a_51.png)

在 C 语言中，这对应 `TIM2_CNT` 寄存器。我们简单地将 0 写入该寄存器。

![](img/494489777607460524688a3fc6df818a_53.png)

```assembly
    ldr r0, =TIM2_CNT
    mov r1, #0
    str r1, [r0]
```

### 5. 使能定时器

![](img/494489777607460524688a3fc6df818a_55.png)

最后，我们通过配置控制寄存器 1 来启动定时器。

![](img/494489777607460524688a3fc6df818a_57.png)

在 C 语言中，这对应 `TIM2_CR1` 寄存器。我们设置相应的控制位（例如 `CEN` 位）来使能定时器。

```assembly
    ldr r0, =TIM2_CR1
    ldr r1, =CR1_CNF // 假设 CR1_CNF 包含了使能位的值
    str r1, [r0]
    bx lr // 返回
```

![](img/494489777607460524688a3fc6df818a_59.png)

至此，定时器初始化完成。

## 实现延时功能

定时器初始化后，我们需要一个方法来等待定时器溢出事件。本节我们将创建一个延时子程序。

我们创建一个名为 `wait` 的子程序，其功能类似于 `delay`。

![](img/494489777607460524688a3fc6df818a_63.png)

```assembly
wait:
```

该子程序的核心是轮询状态寄存器，等待更新中断标志被置位。

在 C 语言中，逻辑是：`while(!(TIM2_SR & TIM2_SR_UIF));`。

![](img/494489777607460524688a3fc6df818a_65.png)

以下是汇编实现步骤：

1.  将状态寄存器地址加载到 R1。
2.  循环读取该寄存器的值到 R2。
3.  检查 `UIF` 位是否被置位（即与 `TIM2_SR_UIF` 进行与操作后结果非零）。
4.  如果结果为 0，则跳回循环开始继续等待。

```assembly
    ldr r1, =TIM2_SR
wait_loop:
    ldr r2, [r1]
    ands r2, r2, #TIM2_SR_UIF
    beq wait_loop // 如果 Z 标志位为 1 (结果为0)，则继续循环
```

当循环退出，意味着定时器已溢出。此时，我们需要手动清除 `UIF` 标志位，以便下次判断。

在 C 语言中，通过向 `TIM2_SR` 寄存器的对应位写 0 来清除标志。

以下是清除标志的汇编代码：

```assembly
    ldr r3, [r1]        // 再次读取状态寄存器
    bic r3, r3, #TIM2_SR_UIF // 清除 UIF 位
    str r3, [r1]        // 写回状态寄存器
    bx lr               // 返回
```

## LED 闪烁测试程序

现在，我们将使用初始化好的定时器和延时功能来控制 LED 闪烁。

我们创建一个名为 `led_blink` 的子程序。

![](img/494489777607460524688a3fc6df818a_67.png)

![](img/494489777607460524688a3fc6df818a_69.png)

以下是 LED 闪烁的逻辑步骤：

![](img/494489777607460524688a3fc6df818a_71.png)

1.  加载 GPIO 端口位设置/清除寄存器的地址到 R4。
2.  将“复位”值（关闭 LED）写入寄存器，关闭 LED。
3.  调用 `wait` 子程序进行延时。
4.  将“置位”值（打开 LED）写入寄存器，打开 LED。
5.  再次调用 `wait` 子程序进行延时。
6.  跳回步骤 2，形成循环。

```assembly
led_blink:
    ldr r4, =GPIO_BSRR
    mov r5, #BSRR_RESET_VAL
    str r5, [r4]        // 关闭 LED
    bl wait
    mov r5, #BSRR_SET_VAL
    str r5, [r4]        // 打开 LED
    bl wait
    b led_blink         // 无限循环
```
