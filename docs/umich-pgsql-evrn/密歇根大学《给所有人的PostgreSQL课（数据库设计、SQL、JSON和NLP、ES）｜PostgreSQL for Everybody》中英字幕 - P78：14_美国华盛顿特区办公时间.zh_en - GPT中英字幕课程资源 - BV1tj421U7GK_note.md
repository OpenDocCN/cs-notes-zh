## 给新学生的学习建议

在认识了部分学生后，我们来看看经验丰富的学生给即将开始学习的新同学提供了哪些建议。

Sue被问及对即将入学的学生有何建议。

她建议学生尝试运用当前在课堂上学到的编程知识，去做一些有趣的事情。即使只学了一点东西，也可以动手实践。她分享了自己的经验：在学习了用户输入后，她编写了一些**用于小测验或猜数字游戏**的代码。

```python
# 示例：一个简单的猜数字游戏逻辑
secret_number = 42
guess = int(input("猜一个数字："))
if guess == secret_number:
    print("恭喜你，猜对了！")
else:
    print("猜错了，再试试吧。")
```

Dr. Chuck确认了Sue是之前完成“迈尔斯-布里格斯性格测试”作业的学生，并称赞她进度超前。
