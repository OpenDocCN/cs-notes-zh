## 概述

在本节课中，我们将学习超越基础CRUD（增删改查）和JOIN操作的一系列进阶PostgreSQL技术。这些内容是对第一门课程核心知识的补充与扩展。

欢迎来到第一门课程的尾声。你已经完成了基础知识的学习，掌握了入门所需的核心技能。我可能已经多次教授过**CRUD**（创建、读取、更新、删除）、数据建模和JOIN操作。我常说，你刚刚学会了SQL中最重要的60%。这部分内容我已经以多种方式教授过很多遍，对你们中的一些人可能也讲过好几次。

因此，总有一个问题：接下来我想讲什么主题？下一个主题是什么？于是就有了这一整套“下一个主题”的内容。这就是下一门课程的内容，它是一个大杂烩，包含了我无法纳入第一个基础总结的各种主题。

## 课程内容概览

以下是本进阶课程将涵盖的一系列实用技术。

*   **高级日期处理**：学习处理复杂日期和时间操作。
*   **GROUP BY与聚合函数**：使用`GROUP BY`子句，以及`MAX`、`MIN`、`COUNT`等聚合函数对数据进行分组和汇总。
*   **事务**：理解和使用事务来确保数据库操作的完整性和一致性。
*   **子查询**：在查询中嵌套另一个查询，以构建更复杂的数据检索逻辑。
*   **存储过程**：创建可重用的数据库程序单元。
*   **文本与正则表达式操作**：利用PostgreSQL强大的文本处理功能和正则表达式进行高级字符串操作。
*   **高级文本索引**：学习如何为文本材料建立更高效的索引，以提升查询性能。

这是一门很棒的课程。我们将逐一介绍这些技术，展示每一种方法。这些将是极其有用的PostgreSQL技能。

## 总结

本节课中，我们一起预览了PostgreSQL进阶课程的核心内容。从高级日期操作、数据聚合到事务管理和文本处理，这些技术将帮助你更高效、更深入地使用PostgreSQL数据库。
