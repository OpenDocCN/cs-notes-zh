## 课程回顾与展望

到目前为止，我们主要学习了名为SQL的语言，并教授了使用该语言可以完成的各种操作。

在下一门课程中，我们将探讨数据库架构。我们将讨论性能、历史以及云技术。

![](img/d674a1c56b18bf16cbc4f1ee95ce369a_1.png)

以下是下一阶段将涉及的核心主题：
*   NoSQL
*   数据库的不同部署方式

我认为这会很有趣，因为我们将更深入地探讨“为什么”和“怎么做”，而不仅仅是“是什么”。数据库可能是我们人类构建的最复杂、最精密的软件之一，或许仅次于互联网本身。

在下一门课程中，让我们进一步学习事物实际运作的原理。这样，在你继续前进时，能对正在发生的事情有更扎实的理解基础，从而在未来学习更多知识。

干杯。
