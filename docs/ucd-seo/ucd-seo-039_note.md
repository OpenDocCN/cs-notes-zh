# 039：站外SEO导论 🚀

在本节课中，我们将要学习站外SEO（Off-Page SEO）的基本概念。站外SEO指的是在网站外部进行的、旨在提升网站在搜索引擎中权威性和排名的所有活动。我们将探讨如何制定有效的策略来增强网站权威，并深入了解链接分析、外链配置以及社交媒体和品牌建设对SEO的影响。

你将学习如何制定有效的策略来提升网站的权威性。

## 核心技能：外链档案管理

你还将学习如何创建和编辑反向链接档案。

以下是管理外链档案的几个关键步骤：
*   识别并分析指向你网站的高质量链接。
*   使用工具（如 `Ahrefs`, `Majestic`）监控外链的增长与健康度。
*   有计划地移除或拒绝低质量或有害的链接。
*   持续寻找并获取来自权威网站的新链接。



## 社交媒体与品牌建设的作用

掌握了上述技能后，我们将探讨社交媒体如何影响页面权威，以及长期建立品牌认知如何帮助改善网站的搜索结果。
