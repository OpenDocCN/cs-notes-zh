## 精选摘要概述

上一节我们介绍了搜索结果页面的基本构成，本节中我们来看看谷歌如何通过特殊形式来优化用户体验。精选摘要是谷歌在自然搜索结果顶部展示的一个视频或文章片段，通常被称为“零号位”。其目的是立即回答用户的查询。

*   这些摘要也常常是语音搜索设备（如Google Home）上相关问题的答案。
*   过去，同时获得精选摘要和第一页自然排名被视为巨大优势。
*   然而，谷歌在2020年5月左右的更新中，禁止了同一网站同时出现在精选摘要和第一页普通自然结果中。

## 精选摘要的利弊

这一变化带来了利弊。研究表明，“零点击搜索”在增加，这意味着用户的查询直接在搜索结果页面上被回答，而无需点击进入具体网站。

*   这对谷歌有利，因为它获得了广告展示并满足了用户。
*   但对内容创作者和网站所有者可能不利，因为谷歌抓取了他们的内容来直接提供答案，从而阻止了用户访问其网站。

点击率是衡量用户点击搜索结果并访问你网站频率的指标，计算公式为：**`CTR = 点击次数 / 展示次数`**。

*   当存在精选摘要时，排名第一的结果平均获得约8.6%的点击。
*   而紧随其下的结果平均获得约19.6%的点击。
*   相比之下，没有精选摘要时，排名第一的结果能获得约26%的点击。

## 是否应优化精选摘要？

那么，是否应该针对精选摘要进行优化？答案并非绝对。

以下是需要考虑的因素：

1.  **评估查询**：首先确定你试图排名的关键词查询是否已存在精选摘要。但请注意，情况可能随时变化。
2.  **品牌与思想领导力**：如果品牌曝光和建立思想领导力是你的目标，那么获得精选摘要非常理想，因为它能将你的品牌置于最前沿。
3.  **流量与转化**：如果你的文章更依赖将用户吸引到网站以完成特定行动（如购买、注册），那么将流量导向该页面可能比获得精选摘要更重要。

## 富摘要及其优势

与精选摘要不同，富摘要总是有益的，你应该始终针对其进行优化。富摘要会改变普通自然结果在搜索中的外观，使其更突出。

*   它们通过吸引用户注意力来帮助提高点击率。
*   常见的富摘要类型包括带有评论星级、数据、库存水平、日期等信息的搜索结果。

## 如何优化富摘要

为了获得富摘要，你需要在网站上应用正确的结构化数据。

*   结构化数据本质上是一种对你现有内容的标记，它帮助搜索引擎更好地理解数据和内容，从而以更视觉化的方式呈现在搜索结果中。
*   几乎任何内容类型都有对应的结构化数据模式，你可以在 `schema.org` 上搜索合适的标记语言。
*   你可以在一个页面上使用多种结构化数据类型，以针对不同类型的查询获得富摘要结果。

最常见的实现方式是使用JSON-LD格式，代码如下所示：

```json
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "Recipe",
  "name": "经典巧克力蛋糕",
  "author": {
    "@type": "Person",
    "name": "简·多伊"
  },
  "aggregateRating": {
    "@type": "AggregateRating",
    "ratingValue": "4.8",
    "reviewCount": "125"
  }
}
</script>
```
