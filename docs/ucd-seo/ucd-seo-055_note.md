课程的前半部分，我们重点探讨了SEO中页面内因素与页面外因素的重要性。以下是这两类核心因素的简要概述：

*   **页面内优化**：指在网站自身进行的优化，例如优化标题标签、元描述、内容质量和内部链接结构。
*   **页面外优化**：指发生在网站外部的、能影响排名的因素，主要包括获取高质量、相关性的外部链接（即**反向链接**）。

同时，我们也学习了如何制定优化策略，以实现网站的持续增长与改进。

在课程的后半部分，我们转向了技术层面。上一节我们介绍了策略制定，本节中我们来看看技术基础的重要性。我们讨论了坚实的技术基础对于支撑SEO工作的必要性，并学习了识别常见的技术问题，例如网站速度、移动端适配、网站结构（**XML站点地图**）和索引问题。

此外，你现在已经能够构建关键词研究流程，并说明如何将这些关键词应用到网站中。关键词研究的基本流程可以概括为：发现关键词 -> 分析关键词（搜索量、竞争度）-> 映射关键词（分配到具体页面）。
