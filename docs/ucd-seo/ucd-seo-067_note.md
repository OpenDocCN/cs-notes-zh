# 067：为客户创建关键词地图 🗺️

在本节课中，我们将学习如何为客户或项目创建一份关键词地图。这份文档对于规划网站内容和未来的优化更新至关重要。我们将通过一个Excel模板的例子，来了解如何将关键词映射到现有页面或规划新页面。

上一节我们介绍了关键词研究的方法，本节中我们来看看如何将研究成果系统化地应用到网站结构中。

## 概述：什么是关键词地图？

关键词地图是一份规划文档，它清晰地展示了网站中每个页面应聚焦的核心关键词及相关关键词。无论你是与客户还是同事合作，创建这样一份文档都至关重要，因为它能让网站的规划和后续更新变得更容易。

以下是一个典型的关键词地图Excel模板示例。为了更清晰地讲解，建议你下载文档示例“keyword map”，其中展示了更详细的布局。

你的文档布局方式，取决于你是在优化一个已有页面的现有网站，还是为一个新网站规划页面。第一种情况更为常见。

### 处理现有网站

对于现有网站，其页面可能并不完全适合你发现的关键词。在这种情况下，可以尝试找到一个主题最接近的现有页面，并建议对页面内容进行小幅修改以匹配目标关键词。

*   **优先修改现有页面**：页面会随着时间积累权威度。因此，如果能对现有页面进行小幅优化，通常能更快地影响排名。
*   **建议创建新页面**：如果没有任何页面与目标关键词匹配，则可能需要建议创建全新的页面。

### 文档结构解析

在我的文档中，布局通常包含以下部分：

1.  **页面列表**：列出网站上已有的页面，或我们建议创建的新页面。
2.  **核心关键词**：为该页面确定的主要目标关键词。
3.  **次要/相关关键词**：通常是搜索量较低的同义词或紧密相关词。
4.  **排名列**：用于记录你检查排名时的日期。这样，在后续应用了优化更改后，你可以将新排名与之前的排名进行比较。
5.  **关键词搜索量**：记录关键词的月均搜索量。
6.  **备注列**：用于添加信息，说明我提出特定建议的原因。

## 关键词地图应用实例

在第二个页面示例中，我建议将“关于我们”页面的核心关键词定为 **`cheap textbook rentals`**。

当我们深入到具体的关键词使用建议时，我可以推荐在标题标签中使用类似 **`rent cheap textbooks + [公司名称]`** 的短语，这仍然符合“关于我们”页面的定位。

然而，是否要重新定位“关于我们”页面以突出特定关键词，最终取决于公司及其品牌信息。有些公司可能不希望这样做。

因此，在这个案例中，我包含了一个创建新页面作为第二选项，以及一个建议的URL（应包含那些关键词）。在与客户详细讨论后，我会通过高亮或删除其中一个选项来记录他们的偏好。

以下是其他页面优化建议的示例：

*   **客户评价页面**：我建议将焦点从“testimonials”转移到“reviews”，因为这可能会带来更多流量。
*   **租赁流程页面**：我建议更新他们已有的关于租赁流程的页面，以纳入更好的关键词。
*   **新页面建议**：基于“e textbooks”这个关键词机会，我建议创建一个全新的页面。

如果你最终是为一个新网站工作，这个过程非常相似，而且根据他们寻求你介入的阶段，往往更容易。
