# 113：受众培养方法 🧑‍🤝‍🧑

在本节课中，我们将学习如何从零开始，为你的内容营销项目建立一个真正关心你内容的、有参与度的受众群体。拥有自己的受众至关重要，这意味着有更多人可能转发你的内容或链接到你。

到目前为止，在本模块中，我一直在讨论内容营销的基础知识。这些都是你整体内容营销计划的关键基石。如果你拥有一群关注你并重视你在网上发布内容的受众，所有这些策略的效果才会最好。因此，本节课我将介绍如何建立你的受众基础。

在我们深入探讨的过程中，请务必记住，我们寻找的是**有参与度的受众**。这并非简单的数字游戏，并非订阅者或关注者越多就越好。关键在于这些订阅者或关注者是否真正关心你的内容，以及他们在你的市场领域内拥有多大的集体影响力。

## 从零开始：你的初始状态

当你刚开始在网上发布内容时，你的受众是这样的：或者说，你的帖子会遇到这种情况：本质上，你一无所有，一无所获。你没有受众，必须采取行动来建立它。

因此，你可以使用多种不同的方法来实现这一目标。

## 核心策略：借助他人受众

要建立自己的受众，你需要找到方法出现在**他人受众**面前。出现在他人受众面前非常有效，因为其中一些人可能会对你产生好奇。他们可能会在社交媒体上关注你，或者开始阅读你网站上的其他内容。简而言之，他们开始成为你受众的一部分。这是一个非常重要的概念。

请记住，你开始时没有受众。即使你是一个大品牌，拥有固有的受众，你还没有一个习惯于从你这里获取内容的受众群体，你仍然需要去建立它。

## 建立受众的具体方法

以下是几种开始建立受众的具体方法。

### 1. 参与意见领袖的讨论
如果你几乎没有受众，开始可能会很棘手。一种启动方式是开始关注你所在领域的意见领袖。当你对他们发布在博客或社交媒体上的内容有实质性价值可以补充时，可以发表评论。请注意，这只有在你的评论确实为同一主题的受众增加了价值时才有效。许多人会阅读意见领袖的帖子及相关评论，这是最初出现在意见领袖及其受众面前的方式。你的评论质量越高，这个策略就越有效。

### 2. 进行明智的客座发帖
另一个建立受众的策略是明智的客座发帖，正如我在上一课中提到的。**只在你的目标受众所在的地方进行客座发帖**。在我展示的这个例子中，是我在搜索引擎领域网站上的客座帖子，那里的目标受众与我的完全一致。这非常棒，因为该受众中的一部分人会在社交媒体上关注你，或访问你的网站查看，其中一些人将成为你受众的一部分。请注意，使用此策略的最佳方式之一是在一个拥有大量且与你的目标受众一致的受众的网站上，定期发布专栏文章。

### 3. 发布新闻稿
建立受众的另一个可能方法是发布新闻稿。如果你的内容足够有新闻价值，这实际上可能有一些价值。但请注意，这更像是我所说的**次要策略**而非主要策略。也就是说，我不会将此作为建立受众的主要方法。尽管如此，一旦你发布了新闻稿，如果你的新闻稿内容足够有趣，也许有人会在某个社交媒体上分享它，然后可能其他人会就此撰写文章。当他们这样做时，分享或文章就会出现在他们的受众面前。

### 4. 利用社交媒体分享
一旦你开始在社交媒体平台上建立自己的关注者，你自己的帖子可以帮助你出现在他人受众面前。例如，你在自己的网站上写了一篇文章，然后你在社交媒体上分享它，出现在你自己的受众面前。接着，你受众中的某个人（比如杰克·斯蒂克曼）分享了这篇文章，并称赞这是一篇很棒的文章。现在，你就出现在了杰克的受众面前。这些人可能会开始关注你，并在未来寻找你的内容。这样，你又开始建立自己的受众了。

### 5. 在会议上发言
让我将你的思维扩展到另一个方向。假设你获邀在会议上发言。当你发言时，你面对着一屋子的人（如果是线上会议，也有一群观众在观看和聆听你）。这些人也是他人受众，他们是会议本身的受众。这是一种极好的借助他人受众的形式，因为你可以帮助人们了解更多他们非常感兴趣的话题，这确立了你的权威地位。这些人中的一部分也会开始关注你。

## 重要提醒：质量胜过数量

但要小心，这**不是简单的数字游戏**。只有当人们因为对你制作的内容感兴趣而关注你时，更多的关注者才有意义。例如，不要陷入为社交媒体账户购买粉丝的陷阱，那将毫无价值。你需要的是关心你所说内容的、有参与度的关注者，而这些必须靠你自己赢得。
