# 搜索引擎优化（谷歌、SEO基础、优化网站、进阶、毕业项目）：106：链接价值评估

在本节课中，我们将学习搜索引擎如何评估链接的价值。我们将从基础的PageRank算法讲起，了解链接价值的基本原理，并介绍如何利用现有工具来测量链接的价值。理解这些概念对于制定有效的链接建设策略至关重要。

![](img/414f0734c33c6a423ac969856c14c3b4_1.png)

上一节我们初步了解了链接在搜索引擎排名中的作用。本节中，我们将更深入地探讨搜索引擎如何为链接赋值，并开始学习如何自行测量链接的价值。

## PageRank基础

首先，我想谈谈PageRank的基础知识，这是谷歌最初用来评估链接价值的方法。

![](img/414f0734c33c6a423ac969856c14c3b4_1.png)

请注意，谷歌已告知我们，原始的PageRank算法在今天的工作方式已与我本节课描述的不完全相同。它已被高度修改，并补充了全新的算法。然而，理解它最初的工作原理仍然有价值，因为它能帮助我们理解如何思考链接的价值。

为了理解其工作原理，我们假设一个页面拥有一定数量的PageRank值。该页面可以通过链接将一定百分比的PageRank值“投票”给其他页面，我们将其称为 **F(x)**，类似于代数函数。

其工作方式是，PageRank以某种方式在出站链接之间分配。在本例中，**F(x)** 表示它投票出去的PageRank，它可以平均分配给两个页面。实际上，基于链接在页面中的位置、链接相关性等因素，分配给每个链接的百分比可能并非精确的50%，但为了便于讨论，你可以想象它是完全相等的。

目前，情况看起来很简单。然而，从这里开始会变得稍微复杂一些。

## 链接网络的复杂性

![](img/414f0734c33c6a423ac969856c14c3b4_3.png)

想象一下，左边有一个页面J。它链接到右上角和右下角的页面B和C，目前这很直接。但现在页面B链接回页面A。计算立刻变得复杂，因为页面A投票给页面B一部分PageRank，页面B又投票一部分PageRank回页面A，这导致页面A的PageRank增加，然后页面A又将增加的部分PageRank投票回页面B，页面B又可以投票更多回页面A。这个过程变得非常迭代，要计算出最终的PageRank值需要进行复杂的计算。

![](img/414f0734c33c6a423ac969856c14c3b4_3.png)

这是搜索引擎一直在解决的问题。实际上，情况要复杂得多，因为真实的网络看起来更像这样。甚至我现在展示的这个图也相对简化了。真实的网络由数万亿个网页组成，它们以各种方式相互链接。在这个巨大的链接阵列中，谷歌和其他搜索引擎能够解决这些计算，并为他们索引的每个页面计算出PageRank值。

如果你假设每个页面初始PageRank值为1，然后随着其他页面的投票而获得更多PageRank，你会发现有些页面的PageRank值可能非常高。想象一下谷歌主页的PageRank。

请注意，PageRank是分配给网页的，而不是网站。不存在域名或网站的PageRank，尽管工具供应商提供了这样的指标，但谷歌表示他们不考虑这一点。这是一个需要记住的重要事项。

## 链接分析工具

现在，我们来看看如何获取数据来衡量链接价值。以下是几款能帮助你获取网页间链接数据的重要工具：

*   **Ahrefs**
*   **Majestic**
*   **SEMrush**
*   **Moz Link Explorer**

![](img/414f0734c33c6a423ac969856c14c3b4_5.png)

这些工具很有用，因为你可以看到哪些网站的页面愿意链接到他们认为重要的其他网页。

正如我已经讨论过的，这些工具还提供了一个分数，用于衡量链接网站的权威性，以及另一个衡量链接页面权威性的指标。

![](img/414f0734c33c6a423ac969856c14c3b4_5.png)

![](img/414f0734c33c6a423ac969856c14c3b4_7.png)

这些分数通常从0到100。然而，请注意这些分数采用对数刻度。例如，一个网站权威性分数（如Moz的域名权威性DA）为60的网站，其权威性实际上是一个分数为50的网站或页面的10倍。

因此，理论上，在相关性相同的情况下，获得一个DA为60的链接可能比获得5个DA为50的链接更好。

![](img/414f0734c33c6a423ac969856c14c3b4_9.png)

![](img/414f0734c33c6a423ac969856c14c3b4_7.png)

为了扩展我们的例子，由于一个DA 70的链接价值相当于10个DA 60的链接，也就相当于100个DA 50的链接，依此类推。理论上，一个DA 70的链接实际上相当于100，000个DA 20的链接。记得我之前说过，一个链接的价值可能是另一个的百万倍。这只是一个例证。你可以在本幻灯片的这个例子中看到同样的情况，其中两个DA 70的链接价值超过100个DA 50的链接。

![](img/414f0734c33c6a423ac969856c14c3b4_9.png)

## 工具指标的局限性

![](img/414f0734c33c6a423ac969856c14c3b4_11.png)

有许多工具可以收集页面或网站的反向链接数据，它们各自都有旨在衡量链接价值（即权威性）的指标，如当前幻灯片所示。工具供应商提供的指标只是对权威性的近似估算，而谷歌有自己的PageRank分数，我们无法获取这些分数。

此外，无论其PageRank分数如何，谷歌都会过滤掉大量链接。一个简单的例子是，带有 `nofollow` 属性的链接不会传递任何PageRank。例如，所有社交媒体网站上的链接都是如此。

![](img/414f0734c33c6a423ac969856c14c3b4_11.png)

## 评估链接价值的其他因素

关于链接还有两点需要了解，当你在评估链接价值时应该牢记：

![](img/414f0734c33c6a423ac969856c14c3b4_13.png)

1.  **相关性**：如果你的页面是关于福特野马的，而你从一个天气网站获得链接，这相关性就不高，价值也较低。相比之下，来自《Car and Driver》杂志的链接显然相关性高得多，你从页面主题的权威来源那里获得了一票。因此，这应该具有更大的权重，搜索引擎也理解这一点。
2.  **编辑完整性**：提供链接的网站的编辑完整性也很重要。如果网站出售链接或内容质量非常差，那么这些链接可能完全被阻止传递任何价值。请注意，我们并不确切知道谷歌如何评分或评估这类因素，但我们知道他们会忽略来自许多类型网站的链接。

![](img/414f0734c33c6a423ac969856c14c3b4_13.png)

## 整体策略

就你的整体策略而言，要着重了解在你的市场领域内哪些网站具有权威性。这里我指的是主题权威性。想象你有一个关于福特野马的网站，你可以从其他权威汽车网站吸引大量链接。这是在市场中建立影响力的好方法。即使搜索引擎不存在，你也会这么做。来自这些网站的链接可能也承载着更高的价值。相比之下，另一个没有这些相关权威链接的网站，在几乎所有相关性大致相同的查询中，你的排名都可能优于它。
