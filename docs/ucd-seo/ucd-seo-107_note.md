# 搜索引擎优化：107：谷歌不欢迎的链接 🔗

在本节课中，我们将探讨谷歌在评估链接时的一些核心关切，并学习如何避免使用搜索引擎不喜欢的链接建设策略。理解这些原则对于建立长期、健康的网站排名至关重要。

上一节我们介绍了如何理解链接的价值以及什么因素使一个链接比另一个更有价值。本节中，我们来看看谷歌对链接建设的基本态度，并提供一些实用的指导原则。

首先，我想分享前谷歌网络垃圾团队负责人马特·卡茨的一些观点。他在2012年的一次采访中谈到链接建设时表示：“它让你陷入一种思维定式，人们会专注于错误的事情。”他的意思是，谷歌不希望站长们为了提升排名而采取行动，即使这些行动从商业或营销角度看毫无意义。

谷歌致力于根据用户的搜索查询，尽可能展示最能满足其需求的结果。这意味着他们希望专注于寻找来自**高声誉、高权威、高诚信度网站**的高质量深度内容。为了帮助识别这些网站，谷歌非常重视来自**高声誉、高权威、高诚信度网站**的链接。吸引大量来自低质量网站的链接无助于实现这一目标，谷歌很可能会完全忽略这些链接，这意味着它们对你也没有帮助。

那么，你应该怎么做呢？你应该通过内容创作和营销努力，将自己打造成一个**高声誉、高权威、高诚信度**的网站。更进一步说，你必须做那些能让你的网站在市场中脱颖而出的工作。

你需要理解的第一点是，链接必须是有效的引用。为了说明这个概念，我展示了一篇真实学术研究论文的引用。想想你在学校时认识的做研究论文的教授，或者你被要求作为项目一部分撰写研究论文的经历。在论文末尾，你会包含你在研究过程中参考的其他文献的引用，这就是它们被称为“引用”的原因。它们的独特之处在于，获得这些引用无需支付任何费用，仅仅是受益者将其分享给他人。从链接的角度来看，这就是有效的引用。可以这样理解：你获得的链接必须是“编辑性给予”的，即给予者单纯因为认为你的链接能为其网页访客提供价值而给予。

如果你致力于为网站吸引链接，以下是一些需要思考的经验法则：

以下是判断链接价值的几个自检问题：
*   **如果谷歌不存在，你还会想要这个网站的链接吗？** 如果答案是否定的，那么这个链接可能没有价值。
*   **你会自豪地将这个链接展示给潜在客户吗？还是会感到尴尬？** 给予你链接的人应该是真心认可你。如果不是，那就有问题。
*   **如果你需要为“这是一个好链接”而争辩，那么你已经知道它不是了。** 一个好链接应该是显而易见的，无需争辩。

现在，我将从另一个角度给你一些额外的规则。

第一条是：**你不能为自己投票**。例如，每当你在博客或社交媒体网站上留言并包含返回自己网站的链接时，这些链接都会被谷歌忽略。任何由你撰写内容并链接到自己，且发布网站没有进行严格编辑审核的做法，都属于此类。

请记住，我之前将链接比作对你内容的“投票”。要知道，**你不能购买选票**。如果你付钱让他人在其网站上放置指向你网站的链接，这就不是“编辑性给予”。他们给你链接不是因为他们认为这对他们的用户有好处，而是因为你付了钱。至少，他们的意图因为你付费而受到了影响。

**你也不能交换选票**。例如，人们过去常常互相交换链接，有些网站甚至与成千上万的其他网站交换链接，只是为了获得更多指向自己网站的链接来提升搜索引擎排名。然而，搜索引擎不看重这种交换。它们能识别出这只是价值不大的互换。需要说明的是，我并非说你永远不应该交换链接，但你只应与那些对你的业务极其相关且有价值的网站交换链接，并且你愿意为此将你网站的一些访问者引导至他们的网站，因为这对他们有用，并且你的用户也确实能从中受益。

最后但同样重要的是，**链接不能被窃取**。确实存在一些方法可以做到这一点，例如有人曾利用黑客手段入侵他人网站，并在其网页源代码中注入链接。这显然是绝对禁止的行为。

本节课还要涵盖另一个重要概念：**人们为什么要在自己的网站上链接到其他网站？** 每当你设置一个链接，你都是在邀请你网站的用户离开。为什么会有人想这么做呢？基本上，链接到一个非自有网站的唯一原因是，你相信你的用户会在你链接的页面上发现巨大价值，并且在此过程中，他们对你的好感度会因此增加。你需要理解这一点，并相应地调整你的链接吸引策略。

以上是对谷歌不希望在你的链接中看到的行为的一个较高层次的概述，但理解这些核心概念非常重要。如果你能将这些原则内化，就可以将其作为指导方针，避免陷入陷阱，以从根本上与搜索引擎相悖的方式去追求链接。
