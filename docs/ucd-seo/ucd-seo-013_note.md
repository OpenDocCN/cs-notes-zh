# 013：企鹅算法与链接清理 🐧

在本节课中，我们将学习谷歌企鹅算法的核心内容。我们将了解哪些链接建设行为会被视为作弊，以及如何清理和规避低质量链接，从而避免网站受到搜索引擎的惩罚。

上一节我们讨论了熊猫算法，本节中我们来看看企鹅算法。企鹅算法延续了谷歌打击垃圾和不良操作网站的传统。你将能够描述另一类至今仍在被严厉打击的违规行为。我们还将简要提及负面SEO的概念，以及如何避免成为其目标。

## 企鹅算法概述

接下来要讨论的主要算法更新是企鹅算法。如前所述，链接对SEO非常重要，这催生了许多获取网站反向链接的“创意”方法。

在SEO领域很长一段时间里，反向链接被视为“圣杯”。许多SEO从业者和站长不惜采用任何必要的手段来获取链接。虽然来自权威网站的链接确实有益，但人们发现，仅仅从大量低质量网站创建链接，是提升特定关键词排名的非常有效的手段。

例如，在过去，如果我想为“San Francisco Attorney”这个关键词排名，我只需要从许多许多网站上建立大量链接，并在这些链接中使用“San Francisco attorney”作为锚文本。

2012年发布的企鹅算法，旨在制止SEO行业中广泛存在的垃圾链接建设策略。与熊猫算法一样，企鹅算法也是一个过滤器，这意味着它会定期重新运行。之前被惩罚的网站有机会逃脱惩罚，而新的网站则可能落入“法网”。

## 企鹅算法打击的链接作弊行为

企鹅算法主要针对那些大量采用操纵性链接建设手段的网站。以下是几种主要的操纵性链接行为：

**链接网络**
链接网络涉及创建一组相互链接的网站。有时这些网站托管在不同的服务器上，由不同的个人注册。这样做是为了试图向谷歌隐藏链接网络。

**链接交易**
链接交易包括常见的互惠链接，例如“如果你链接到我的网站，我也会链接到你的”。但也包括更难以发现的链接，例如拥有多个网站的人，只使用他们的一部分网站来链接到另一个网站。他们这样做是为了避免被轻易识别为互惠链接。

**评论垃圾**
这种行为至今仍很常见，尽管它毫无价值。这包括在新闻文章、博客和其他网站上的评论。评论中通常包含一个指向他们自己网站的链接。在博客上，这些评论通常看起来很奉承，评论者会谈论他们多么喜欢这个话题，以及多么期待阅读更多文章，然后顺便插入一个链接。99%的情况下，这些实际上是由机器人为了SEO目的而创建的自动评论。不幸的是，新的博客所有者常常会发布这些评论，误以为它们是真实的。有时，评论更直接，但仍然吸引读者点击链接，例如评论可能说“嘿，了解我如何在家工作赚了10万美元”，然后附上其网站的链接。

**激进的精确匹配锚文本**
我们之前简要讨论过这一点。例如，为你想要排名的关键词建立大量精确匹配的链接。谷歌开始惩罚这种行为，因为这显然不是人们因为觉得你的网站有用而自然链接时，会自然发生的锚文本现象。

然而，如果你的域名包含锚文本中使用的关键词，例如你拥有域名 `SanFranciscoAttorney.com`，那么大量带有“San Francisco attorney”锚文本的链接实际上很可能被认为是有效的。当然，这取决于你反向链接档案的其余部分以及你获得链接的网站类型。

**付费链接**
企鹅算法也打击付费链接。谷歌很难确定一个链接是否是付费的，但算法在寻找此类链接时会检查某些特征。例如，如果链接位于侧边栏并被广告包围，它很可能是一个付费链接。链接周围的文字也是如此。所以，如果他们在链接旁边看到“赞助商”或“赞助”之类的词，该链接很可能被认为是付费的。这也包括包含对某产品或企业评论的帖子，因为用户可能因该评论获得了某种形式的补偿，无论是否是金钱。

**低质量网站上的链接**
算法还会寻找低质量网站上的链接。如果网站与你的受众无关，例如，在一个关于插花的网站上出现指向律师的链接，那么该链接很可能是付费获得的或通过操纵手段获得的。

## 链接建设的风险与过时方法

过去，你可以通过将被惩罚的网站重定向到一个新域名来暂时逃避这些惩罚。但这些重定向也开始将惩罚“伤害”传递到新网站。

随着谷歌对垃圾链接建设策略的认识越来越深入，他们通过算法更新不断打击这些行为。链接建设变得更加复杂，风险也更高。过去SEO用来获取大量反向链接的一些具体方法也失效了，一些网站甚至因此受到惩罚。

因此，我们建议不要使用某些过去可能有效但现在已过时的特定类型的链接建设方法。这些包括：

以下是几种已被视为垃圾或已贬值的链接建设策略：

*   **从各种目录中获取链接。**
*   **创建用户可以下载的免费小部件，其中包含指向你网站的锚文本链接。** 我遇到过的一些例子是，站长可以添加到其网站作为有用工具的免费抵押贷款计算器，但该工具底部会包含一个链接，锚文本写着“抵押贷款”之类的内容。这被视为垃圾行为。
*   **为博客或WordPress创建免费模板或主题。** 这些模板的作者会在页脚或有时只是隐藏在代码中，放置一个指向他们自己网站的链接。
*   **在大量论坛中创建用户帐户，并在签名或个人资料中添加指向其网站的链接。** 他们通常会发一次帖，然后就离开。LinkedIn个人资料和签名链接的价值自那时起已被贬低，因此这样做并不值得。

我建议在谷歌支持中心阅读更多关于谷歌链接质量指南的内容（链接已提供）。此外，我还提供了一个指向搜索引擎大地（Search Engine Land）文章的链接，文章底部有一个视频，讨论了谷歌评估链接是否为付费链接的方式。这是一个非常有用的页面和视频，请花些时间查看。

## 负面SEO与链接拒绝工具

企鹅算法推出后，许多网站受到惩罚，流量大幅下降。清理垃圾链接的行动开始了，一些以前通过出售链接赚钱的网站改变了做法，开始向网站所有者收费以移除链接。

由于看到的影响，SEO从业者和站长开始担心负面SEO的可能性，即不道德的SEO从业者为竞争对手的网站创建大量垃圾反向链接，试图使其在搜索引擎中受到惩罚。

由于论坛上的许多评论和请求，谷歌创建了链接拒绝工具，该工具现已成为谷歌搜索控制台的一部分。链接拒绝工具允许网站所有者拒绝指向其网站的垃圾链接。

然而，在使用此工具之前，建议你检查以下几点：

以下是使用链接拒绝工具前的检查清单：

1.  **确认惩罚：** 确保你的网站确实受到了企鹅算法的惩罚。除非你知道自己有一些非常垃圾的链接，并且想主动避免受到惩罚，否则不要仅仅因为某些链接可能对你的网站有益就去移除所有链接。
2.  **评估链接质量：** 确保你拒绝的链接确实是低质量链接。
3.  **优先自行移除：** 在可能的情况下，尝试先自行联系移除链接。
4.  **检查链接变体：** 确保你同时移除指向你网站的 `www` 版本链接和任何非 `www` 版本的链接。
