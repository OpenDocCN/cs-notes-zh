# 115：社交媒体营销导论

在本模块中，我们将聚焦于社交媒体营销，特别是它在支持内容营销工作中所扮演的角色。社交媒体在帮助你建立自有受众方面可以发挥非常重要的作用，这能让内容营销的成功变得容易得多。此外，社交媒体能帮助你与有影响力的人建立关系，他们可以帮助加速整个进程。

本模块将涵盖指导你线下行为的基本社交互动礼仪，如何同样指导你的线上行为。我们将更详细地探讨如何建立你的受众。同时，也会讨论社交媒体中的链接在搜索引擎优化中扮演或不扮演的角色。最后，我们将提供一些企业利用社交媒体提升声誉和知名度的案例研究。
