# 051：选择合适的关键词 🔑

在本节课中，我们将学习关键词研究的重要性，并探讨如何选择和优化关键词，以更好地匹配目标受众的搜索意图，从而为网站吸引最相关的流量。

当用户在谷歌或其他搜索引擎中输入搜索词时，他们通常心中有一个特定的目标。你越能理解网站受众及其需求，就越能精准地定位那些能将受众引导至你网站的关键词。

## 理解搜索背后的心理学 🧠

为了进行有效的关键词研究，首先必须理解人们如何进行搜索以及他们为何选择特定的关键词。理解目标受众的心理将帮助你吸引并留住更多的访问者。

![](img/cc130e371105d23c21dd8112c9d70676_3.png)

![](img/cc130e371105d23c21dd8112c9d70676_4.png)

上一节我们介绍了关键词研究的重要性，本节中我们来看看用户搜索的具体原因和方式。

用户可能出于各种原因进行搜索，例如购物或寻找特定信息。这通常会影响到他们如何组织搜索查询。

![](img/cc130e371105d23c21dd8112c9d70676_6.png)

以下是用户搜索意图的主要类型：
*   **信息型**：寻求知识或答案（例如，“如何制作陶器”）。
*   **导航型**：寻找特定网站或页面（例如，“Facebook 登录”）。
*   **交易型**：意图进行购买或完成某项操作（例如，“购买 iPhone 15”）。
*   **商业调查型**：在购买前进行比较研究（例如，“最佳游戏笔记本电脑评测”）。

![](img/cc130e371105d23c21dd8112c9d70676_8.png)

## 关键词的广度与深度 📊

人们通常会使用大量不同的查询来寻找关于一个相似主题的信息。以学习制作陶器为例，用户可能使用“如何制作陶器”这样的宽泛关键词，也可能深入寻找制作过程中的特定环节，例如“初学者陶轮”、“陶器釉料”或“我需要窑炉吗”。

![](img/cc130e371105d23c21dd8112c9d70676_10.png)

![](img/cc130e371105d23c21dd8112c9d70676_11.png)

既然谷歌的职责是为搜索查询提供最相关的页面，那么这些网页就应该致力于回答用户当前提出的问题，并包含用户可能想了解的该主题相关信息。

因此，你的陶艺网站可以有一个关于“如何制作陶器”的主页，并从这个初始页面链接到其他专注于初学者友好主题的页面。

![](img/cc130e371105d23c21dd8112c9d70676_13.png)

一个良好的做法是围绕一个特定主题，提供一系列相关问题的答案。这使得你的网站既能专注于一个更宽泛的关键词短语（如“如何制作陶器”），同时也能针对并争取为非常具体的关键词排名。

![](img/cc130e371105d23c21dd8112c9d70676_15.png)

## 选择具体且相关的关键词 🎯

在构建网站和页面时，考虑选择恰当关键词的重要性至关重要。例如，如果你拥有一个PC游戏网站，并希望吸引寻找游戏信息的搜索者。

![](img/cc130e371105d23c21dd8112c9d70676_17.png)

你不应使用像“最佳游戏”这样的关键词。因为“游戏”一词非常宽泛，可能指棋盘游戏、派对游戏或主机游戏。更好的关键词是“最佳PC游戏”。

![](img/cc130e371105d23c21dd8112c9d70676_19.png)

但你还可以通过添加诸如“年度最佳”、“有史以来最佳”、“儿童”或“多人”等修饰语，使其更具针对性。关键词越具体，其竞争通常就越小，为这些相关查询排名的机会就越高。正因如此，用户会高度参与你网站的内容，因为你提供了他们正在寻找的确切信息。

![](img/cc130e371105d23c21dd8112c9d70676_21.png)
