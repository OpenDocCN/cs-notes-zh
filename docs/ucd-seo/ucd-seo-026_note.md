# 搜索引擎优化：026：创建理想买家画像 🎯

![](img/58e3ca29c29771b33188a25d2393914d_0.png)

![](img/58e3ca29c29771b33188a25d2393914d_1.png)

在本节课中，我们将学习如何利用收集到的受众信息，构建一个代表理想买家的“人物画像”。这个画像将帮助我们理解目标用户，并指导我们选择更精准的关键词。

上一节我们介绍了如何收集受众信息，本节中我们来看看如何将这些信息整合并加以利用。

## 构建买家画像

![](img/58e3ca29c29771b33188a25d2393914d_3.png)

一旦你掌握了人口统计数据，一个好方法是围绕你的理想买家提出几个问题，并据此构建人物画像。这能帮助你更好地理解你希望吸引到网站的用户类型。

你的目标应该是创建一个能与所构建人物画像“对话”的网站。这将使网站对用户更友好，并自然地在你撰写的内容中融入正确的关键词。

![](img/58e3ca29c29771b33188a25d2393914d_5.png)

![](img/58e3ca29c29771b33188a25d2393914d_7.png)

如果你试图吸引多种类型的买家，为每种类型分别构建不同的人物画像是完全可以接受的。这也能在你为不同画像撰写针对性内容时提供指导。

## 影响关键词选择的画像特征

![](img/58e3ca29c29771b33188a25d2393914d_13.png)

现在，让我们开始思考你的理想买家会使用哪些类型的关键词。为了更好地理解，我们可以向目标买家提出几个问题。

![](img/58e3ca29c29771b33188a25d2393914d_15.png)

以下是构建画像时需要考虑的几个关键方面：

![](img/58e3ca29c29771b33188a25d2393914d_17.png)

*   **年龄**：年轻人可能使用大量俚语，而年长者可能对技术不太熟悉，这会影响关键词的选择。你的学习材料中提供了一个相关研究的链接。
*   **地域**：询问他们来自哪里，是否存在地区性的词汇差异。例如，在加利福尼亚州，针织帽被称为“beanie”，而在俄克拉荷马州，许多人称之为“toboggan”（这个词在西海岸通常指一种雪橇）。另一个更广为人知的例子是“pop”与“soda”的区别。英式英语和美式英语的词汇也存在很大差异，例如“apartment”（公寓）在英式英语中是“flat”，“cookie”（饼干）是“biscuit”，“elevator”（电梯）是“lift”。更多关于英美术语差异的信息，请查看学习材料中提供的链接。
*   **性别**：买家的性别也可能对其词汇产生微小影响，但要注意避免将买家画像置于刻板的性别印象中。了解目标受众和画像的性别是一个好习惯，无论是否存在词汇差异。例如，我们曾与一家销售男士婚礼礼服的公司合作。根据多项研究，女性往往承担了大部分与婚礼相关的研究和规划工作。因此，构建一个买家画像来应对她可能进行的搜索查询也是个好主意。这类查询通常与男性的搜索内容不同。在博客文章等内容方面，我们可以专门为新郎设置话题，同时帮助解决新娘在婚礼策划过程中可能遇到的问题。

## 创建画像示例

![](img/58e3ca29c29771b33188a25d2393914d_19.png)

开始构建你的买家画像时，我个人喜欢从一张与我的买家年龄相仿的人物图片开始，这让他们感觉更真实。

![](img/58e3ca29c29771b33188a25d2393914d_21.png)

以“教科书租赁”为例，我们可以有一个男性画像、一个女性画像，或者各一个。在这个特定例子中，我将选择一个男性画像。

我的示例画像名叫查克，是一名大学新生。查克目前没有工作，因为他想先专注于学业，看看自己的课业负担如何。因此，他资金非常有限，非常注重预算。查克认为，租用教科书比购买更符合他的预算。他年轻且精通技术，经常阅读Kindle电子书，因此熟悉现在获取电子书是多么容易。鉴于此，他想知道是否可以在线租赁教科书，或者是否可以租赁该书的电子版本。查克可能会使用“online textbook rental”（在线教科书租赁）或“e-textbook rental”（电子教科书租赁）等关键词来寻找解决方案。

你甚至可以添加具体细节，比如他的研究领域，以提醒自己如何优化针对特定学习领域的博客文章等内容。虽然在本例中不适用，但添加其他信息供你参考也可能有用，例如，这是一个B2B（企业对企业）还是B2C（企业对消费者）画像。如果你是为B2B受众写作，你的问题可能会有所不同。研究其他类型的买家画像以获取他人包含哪些信息的良好范例，总是有益的。
