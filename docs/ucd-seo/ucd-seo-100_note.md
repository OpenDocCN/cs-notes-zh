# 搜索引擎优化：100：制定SMART项目目标 🎯

在本节课中，我们将学习如何为SEO项目制定清晰、可衡量的目标。我们将重点介绍SMART目标设定方法，并学习如何将宽泛的营销目标转化为具体、可追踪的指标。

## 课程概述

上一节我们讨论了客户获取、期望设定和关系管理。本节中，我们来看看如何通过设定明确的目标来持续证明你的价值。

SEO不仅仅是发现搜索机会。你还需要证明你所做的工作正在产生回报，并且客户正在看到成果。如果你在公司内部工作，你需要定期向管理层提供报告，展示你的工作内容和取得的成果类型。没有你的指导，客户可能会使用错误的信息作为改进指标。有些人仅仅依赖关键词排名来衡量你的努力是否成功。正如我们在过去的课程中讨论过的，这并不是衡量进展最可靠的方式。

## 制定SMART目标

以下是确定项目SMART目标的方法，以及如何将更笼统的营销目标转化为SMART目标，以便于轻松跟踪和评估。

虽然不同的营销活动有通用的指标可以追踪，但你需要根据每个客户的独特业务需求，为其制定具体的、需要达成的目标。你可以通过查看SMART目标方法来决定为客户追踪哪些目标最为合适。

**SMART** 代表 **具体**、**可衡量**、**可实现**、**相关** 和 **有时限**。

通常，你可以通过将你的努力与组织的营销目标保持一致来确定良好的SMART目标。

例如，一个目标可能是“从我们的网站产生更多潜在客户”。为了将其转化为SMART目标，我们会避免使用“更多”这类词语，以使其更具体和可衡量。

你还需要确保这些目标是可实现的。为此，请查看现有数据，以便更准确地预测你的努力可能实现什么。例如，你可能需要查看当前产生了多少潜在客户，哪些页面产生了这些潜在客户，然后将其与你对网站的计划进行比较，以得出一个估计的增长值。

这将使目标更加充实。例如，“从我们的网站产生更多潜在客户”现在可以变为“每月将我们网站产生的潜在客户增加25%”。

不过，我们还可以进一步改进。SMART目标中的“相关性”部分意味着我们要确保我们的目标与其他业务目标相关。它让我们能够解释特定指标如何有助于实现我们的目标。就SEO而言，如果我们添加我们正在生成的潜在客户类型及其来源，这个目标可能会变得更加相关。

因此，现在可以改为“每月通过自然搜索将我们网站产生的潜在客户增加25%”。

不过，我们不要忘记“时限性”。这指的是你应该在何时达成目标。同样，尽量使其尽可能具体和可实现。如果你说在SEO的第一个月内，你就要开始实现潜在客户月环比增长25%，我会怀疑这些潜在客户的质量。设定一个更现实的目标，比如三到五个月，以配合SEO建议的实施时间，会更合理。

最终，你可能会得到类似这样的目标：“在特定日期之前，每月通过自然搜索将我们网站产生的潜在客户增加25%”。

现在你知道，你应该追踪的指标之一是通过自然搜索产生的潜在客户。并且你应该提供一个显示月环比增长的图表。

## 应用与实践

你可以根据企业面临的营销挑战及其独特需求，制定个别的SMART目标。每个目标都将为你开始追踪的指标类型提供见解。

请记住，随着时间的推移，收益递减是常见现象。提供一个显示何时会趋于平稳的长期图表可能会有所帮助。

对于没有任何历史数据的新网站来说，设定SMART目标可能很困难。在这些情况下，我建议设定非常保守的估计，并附上说明，指出这些目标需要保持灵活性，尤其是在初始阶段。这允许你在必要时提高目标。
