# 搜索引擎优化：049：404页面最佳实践

![](img/a2e6c7c1d71c7e1081ee7b293457f2b0_0.png)

![](img/a2e6c7c1d71c7e1081ee7b293457f2b0_1.png)

在本节课中，我们将探讨技术性SEO策略的最后一个主题：设计和处理网站404页面的最佳实践。我们将分析哪些情况下应该对404错误页面进行重定向，并学习如何设计404页面，以减轻访客遇到错误时的挫败感和困惑。

## 何时重定向404页面

![](img/a2e6c7c1d71c7e1081ee7b293457f2b0_3.png)

上一节我们介绍了重定向的最佳实践，本节中我们来看看如何处理404页面。404页面是当用户或搜索引擎请求的页面不存在时显示的错误页面。一个常见的疑问是：是否应该将所有404页面都重定向到新页面？

存在一个误解，认为所有404错误页面都是有害的。实际上，搜索引擎能够识别并正确处理404页面。问题主要出现在网站存在大量“软404”错误，或由过期产品产生大量404页面时。理想情况下，应与网站所有者合作，尽量减少网站产生的404页面数量。

然而，在某些情况下，404页面可能是最佳选择。但如果你拥有的404页面包含大量反向链接和权重，或者曾经获得过高流量，那么遵循上一节讨论的重定向最佳实践，将它们重定向到更合适的页面，以保留这些页面的权重，是一个好主意。

![](img/a2e6c7c1d71c7e1081ee7b293457f2b0_5.png)

## 设计友好的404页面

![](img/a2e6c7c1d71c7e1081ee7b293457f2b0_7.png)

![](img/a2e6c7c1d71c7e1081ee7b293457f2b0_9.png)

当用户访问一个404页面时，他们常常会感到困惑，不知道下一步该去哪里或如何找到原本想访问的内容。设计一个友好的404页面有助于避免用户立即离开你的网站，从而减轻这种困惑。

以下是设计404页面时的一些最佳实践：

*   **提供导航选项**：当访客到达404页面时，为他们提供一些导航项，以便他们可以访问网站上的其他页面。从SEO的角度看，这也是一个很好的做法。这样，即使搜索引擎机器人跟随链接到达404页面，它也能找到出路，继续抓取网站上的其他内容。
*   **保持网站整体风格**：在显示错误页面时，保持网站的整体主题风格是一个值得遵循的最佳实践。这有助于让用户知道他们仍然在正确的网站上，同时也为他们提供了下一步的去向选项。
*   **显示清晰的错误信息**：向用户显示错误信息，告知他们页面不可用的原因，这一点很重要。信息可以很简单，例如“页面未找到”，也可以更详细。一般来说，此页面上的内容越有帮助，用户留下来的可能性就越大。
*   **包含明确的行动号召**：包含一个明确的行动号召，告知用户可以采取的下一步操作，这会很有帮助。例如，提供一个返回主页的链接，让他们可以重新开始搜索；或者提供一个搜索框，让他们可以轻松搜索网站上的其他相关页面。

## 如何查找404页面

![](img/a2e6c7c1d71c7e1081ee7b293457f2b0_11.png)

有多种工具可以帮助你查找网站内的404页面：

![](img/a2e6c7c1d71c7e1081ee7b293457f2b0_13.png)

*   **Google Search Console**：这个工具（前身为Google网站管理员工具）会提供它发现的所有404页面的报告。此报告可以下载，以便你审查单个URL或发现规律。
*   **网站爬虫工具**：你可以使用像Screaming Frog或Xenu这样的工具来爬取你的网站。这些工具会提供关于失效链接的详细报告。
*   **浏览器扩展**：也有一些浏览器扩展可以使用。例如，Google Chrome浏览器有一个名为“Check My Links”的扩展，它有助于发现导致404页面的内部失效链接。
