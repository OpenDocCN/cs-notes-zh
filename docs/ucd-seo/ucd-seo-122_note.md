在本节课中，我们将学习如何与行业内有影响力的人士建立关系。这是一个需要耐心和正确心态的复杂过程，我们将探讨其核心步骤和策略。

## 建立关系的正确心态

上一节我们介绍了关系建立的重要性，本节中我们来看看启动这一过程所需的基础。首先，你必须具备正确的心态。没有正确的心态，你的努力很可能会失败。有了正确的心态，你成功的几率会大大增加。简而言之，这是一个复杂的过程，你必须保持耐心。此外，你与每一位影响者互动的方式都会有所不同。

你需要认识到，建立关系涉及多个步骤。你必须按顺序完成这些步骤。如果你试图跳过某些步骤，很可能会损害你长期成功的机会。

## 关系建立的步骤与策略

关系建立并非一蹴而就。它不像你给某人发一封邮件或在Instagram上提一次名字，就能立刻建立起一段美好的关系。这需要时间和努力，并且必须采用正确的方法。

以下是建立关系的关键步骤：

1.  **深入了解对方**：我喜欢从研究他们分享的内容、撰写的文章以及他们的职业生活开始。这是了解他们是谁、关心什么的过程的开始。你需要花时间去了解他们，这不仅仅是了解他们的专业领域。了解他们喜欢谈论和分享的其他事情也很有用，例如他们可能喜欢葡萄酒、某个运动队或汽车等。虽然你们的对话不一定围绕这些展开，但这些元素可能会成为对话的一部分。

2.  **寻找连接契机**：你也可以为这个过程寻找一个“启动器”。例如，如果能通过他们认识且信任的人引荐，那就太好了。这会为你带来一些信任，这很棒。但不要操之过急，不要认为引荐给了你随意向他们提出请求的通行证。引荐有助于加速关系的发展和信任的建立，但你仍需运用判断力，不要试图推进得太快。

3.  **主动提供价值**：了解这些人关注什么、如何思考以及喜欢什么非常重要。一旦掌握了这些，你就可以采取下一步：开始为他们增加价值。帮助他们提升知名度是一个很好的开始。你可以通过转发他们的一些内容来实现这一点，但不要转发全部，那样会让你看起来像个跟踪者。此外，如果他们的社交动态接受评论，寻找机会添加有意义的评论。仅仅说“好帖子”并不是有意义的评论。添加有用的评论，如果他们不立即回复，也不要失望，这可能需要时间。

4.  **融入更广泛的社区**：仅仅与影响者互动是不够的。如果影响者或任何人在网上看到你只与他们互动，这行不通。从长远来看，这会让他们感到不安。事实上，让他们看到你正在为社区中的其他人增加价值，通过他们的动态或其他方式，并且你真正成为那个更大社区的一部分，这一点至关重要。我发现这对于建立成功的关系至关重要。实际上，一个很好的平衡方法是积极帮助影响者之外的其他人，并与这些人互动，多于你与影响者的互动。这可以包括在影响者的动态评论中帮助他人，但你也可以在那些动态之外这样做。如果你考虑正常的社交行为，情况往往就是如此。如果你只专注于一个人，这看起来和感觉起来都是不平衡的。

5.  **利用线下机会**：此外，我本人是参加会议的拥护者。我曾经通过参加会议建立关系，当一位影响者正在演讲时，我坐在前排，等待演讲结束。就在主持人说“感谢我们的演讲者”时，我离开座位，第一个上前与他们交谈。这样我就有机会与他们进行面对面的交流。不过，如果你打算这样做，一定要准备好说一些相关且有趣的话，例如分享对他们所说内容的共同观点，然后分享你自己的见解。关键在于，关系的价值越高，你应该愿意为建立它付出更多的努力。
