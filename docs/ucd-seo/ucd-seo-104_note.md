# 内容营销课程：104：课程导论

在本节课中，我们将学习内容营销的基础知识，了解它如何助力业务增长并提升自然搜索流量。

内容营销在业务增长中扮演着重要角色。当我担任Stone Temple Consulting的首席执行官时，我们利用内容营销建立了行业内最强大的SEO机构品牌之一，并同时实现了自然搜索流量的增长。最终，我将该机构出售给了一家名为Proficient的上市系统集成商。

关于我的背景，我创立、建设并出售了四家不同的公司，这些公司均以SEO为主要运营模式。其中三家公司将SEO作为其流量和收入的主要来源，另一家就是我刚才提到的机构。同时，我还是《SEO的艺术》一书的主要作者，该书目前已更新至第四版，并且我很荣幸在多年间获得了多项奖项。

在本课程中，你将学习内容营销的基础知识，以及如何利用它来有效推动业务增长，同时促进自然搜索流量的提升。此外，你还将学习如何通过社交媒体和影响力营销来增强你的内容营销计划。

完成本课程后，你将有能力在你的组织中实施内容营销。

在学习本课程时，你可以思考一个问题：如果你的搜索引擎流量增加50%，甚至翻倍，这将对你的业务带来多大的帮助？

让我们开始吧。
