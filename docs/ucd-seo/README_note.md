# UCD 搜索引擎优化笔记：1：SEO核心概念与基础

在本节课中，我们将要学习搜索引擎优化的核心概念与基础。理解这些基础是后续所有优化工作的前提。

## 概述

搜索引擎优化旨在提升网站在搜索引擎自然搜索结果中的可见度。其核心是理解搜索引擎如何工作，以及用户如何搜索。

## 搜索引擎工作原理

上一节我们介绍了SEO的目标，本节中我们来看看搜索引擎是如何工作的。搜索引擎的工作可以简化为三个主要步骤：抓取、索引和排名。

以下是搜索引擎工作的三个核心步骤：

1.  **抓取**：搜索引擎派出“蜘蛛”程序，在互联网上自动访问和发现网页内容。
2.  **索引**：搜索引擎将抓取到的网页内容进行分析和存储，建立庞大的数据库，即索引库。
3.  **排名**：当用户输入查询词时，搜索引擎从索引库中找出相关网页，并按照一系列算法规则进行排序，将最相关、质量最高的结果展示给用户。

## 核心优化目标

理解了搜索引擎的工作原理后，我们的优化工作就有了明确的方向。优化主要围绕两个核心目标展开：相关性和权威性。

以下是两个核心优化目标：

*   **相关性**：确保你的网页内容与用户的搜索意图高度匹配。这涉及到关键词的使用、内容的主题和质量。
*   **权威性**：向搜索引擎证明你的网站和内容是值得信赖的。这通常通过其他高质量网站的链接（外链）和网站自身的用户体验指标来体现。

搜索引擎的排名算法可以抽象为一个评估函数：

**排名得分 = f(相关性， 权威性)**

## 关键词研究基础

关键词是连接用户搜索意图与网站内容的桥梁。没有正确的关键词，优化工作将失去方向。

进行关键词研究，你需要找到用户真正在搜索的词组。以下是关键词的主要类型：

*   **核心关键词**：描述你的主要产品或服务的宽泛词汇，例如“跑步鞋”。
*   **长尾关键词**：更具体、搜索量较低但转化意向更强的词组，例如“2023年轻量级女性缓震跑步鞋”。
*   **商业意图关键词**：包含“购买”、“价格”、“评测”等词汇，表明用户处于购买决策阶段。
*   **信息意图关键词**：包含“是什么”、“如何”、“教程”等词汇，表明用户处于信息搜集阶段。

## 页面基础优化

确定了目标关键词后，接下来就需要在网页上合理地使用它们。页面上的特定位置对搜索引擎尤为重要。

以下是一些需要优化的关键页面元素：

1.  **标题标签**：HTML中的`<title>`标签，是搜索结果中显示的第一行可点击标题。应包含核心关键词。
    ```html
    <title>核心关键词 - 品牌名</title>
    ```
2.  **元描述**：HTML中的`<meta name="description">`标签，是搜索结果中标题下方的摘要文字。应简洁概括页面内容，吸引点击。
3.  **标题**：页面内容中的`<h1>`, `<h2>`等标题标签，用于组织内容结构。`<h1>`通常作为页面主标题。
4.  **正文内容**：提供高质量、原创、能解决用户问题的详细内容。自然地融入关键词。
5.  **URL结构**：网页地址应简洁、可读，包含关键词。
    ```
    佳实践：/category/keyword/
    不佳实践：/page?id=123
    ```
6.  **图片ALT属性**：为图片添加描述性文字，帮助搜索引擎理解图片内容，形式为`<img alt="描述文字">`。

## 总结

本节课中我们一起学习了搜索引擎优化的基础。我们了解了搜索引擎抓取、索引和排名的工作原理，明确了围绕相关性和权威性进行优化的核心目标，掌握了关键词研究的基本分类，并学习了标题、描述、内容等页面基础元素的优化方法。这些是构建有效SEO策略的基石。
