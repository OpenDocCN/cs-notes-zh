# 搜索引擎优化：093：SEO影响预测 第三部分 📊

在本节课中，我们将学习SEO预测的核心环节：如何获取和使用点击率数据，以及如何基于此进行网站整体优化和新项目（如新增内容板块）的流量预测。

上一节我们介绍了预测的基本框架和挑战，本节中我们来看看预测中至关重要的工具——点击率图表，以及如何将其应用于实际预测场景。

## 点击率图表的选择：自制 vs. 行业通用

进行SEO预测时，点击率图表是关键。你可以选择创建自己的图表，也可以使用网络上现成的版本。这两种方法各有利弊。

*   **自制图表**：能精确反映你所在行业和网站的用户行为。例如，某些行业的搜索结果页面可能包含更多YouTube视频或富媒体结果，这会影响点击率。
*   **行业通用图表**：可以节省大量时间和计算成本，尤其适合不熟悉Excel公式或BI工具的用户。

然而，拥有自己的图表在展示数据如何与你网站的实际点击率相关联时非常有用。建议至少偶尔创建一次自己的图表，并与流行的通用图表进行比较。如果两者接近，你可以节省时间，使用现成的图表；如果差异很大，则证明自制的图表更准确，未来可以沿用此方法，每年更新几次。

![](img/5b2f17d2527dda0c8549ec8a593315f3_1.png)

如果你决定使用现成图表，除了课程提供的示例，还可以参考“Advanced Web Ranking”这类网站，它们收集了按行业划分的点击率数据，会比标准图表更贴近你的行业。

但需要注意的是，即使是行业图表也存在准确性疑问。例如，“钓鱼”活动可能被归类为“体育”或“爱好”，前几名的点击率差异会导致预测的流量数字大不相同。此外，这类工具通常只能按月选择数据，如果你的业务季节性很强，可能需要导出逐月数据以了解网站长期的平均水平。

**建议**：查看你所在行业的通用图表，如果时间允许，计算自己的点击率并与之比较。

## 如何创建自定义点击率图表

计算自己的点击率是一个深入的过程。以下是需要遵循的高级步骤：

1.  **获取数据**：从Google Search Console下载数据。由于直接下载限制为100行，建议将其连接到Data Studio等工具以获取更多数据行。
2.  **清理数据**：手动过滤掉品牌词，这能更好地代表你行业的点击率。同时，过滤掉可能扭曲数据的异常值，例如展示次数极少但点击率极高的查询。
3.  **进行计算**：在Google Search Console数据的每个排名级别上，用**总点击量 / 总展示量**，最终得出自定义的点击率曲线。

这个过程非常复杂。为了帮助你，这里提供两个来自权威行业博客的详细指南链接，它们提供了逐步操作的示例：
*   **Sear博客**：提供了相关的视频教程。
*   **Ayma博客**：提供了带截图的详细步骤说明。

## 制定预测计划与路线图

现在你有了点击率图表，下一步是考虑在你预测的时间范围内计划完成哪些工作。例如，你是在做年度预测还是季度预测。

你应该制定一个包含计划项目及理想完成日期的路线图。这有助于预测你何时可能开始看到努力带来的价值。

*   **全站改进预测**：例如技术修复（如速度优化）、清除大量重复内容、改进内部链接等。

## 全站改进的预测方法

我个人发现，预测全站改进比预测新增内容板块更困难，因为后者可以围绕一组有数据可循的特定关键词进行。

预测全站改进的最佳方法如下：

1.  **导出关键词**：从Google Search Console下载你的核心关键词。这些词你已经获得排名和流量，全站改进很可能会使它们受益。
2.  **获取搜索量**：将这些关键词通过Google Ads关键字规划师等工具运行，以获取搜索量数据。
3.  **应用点击率**：使用你发现或自制的点击率图表，结合搜索量数据和排名位置，预测在特定位置可能获得的流量百分比。

公式可以简化为：`预测流量 = 关键词月搜索量 × 预估排名位置的点击率`

## 新增内容板块的预测示例

为网站新增板块（如博客）的预测过程与全站预测类似，但更侧重于一组新关键词。以下是一个详细示例：

假设你要创建一个教育板块，回答用户关于特定主题（例如钓鱼）的疑问。

1.  **收集关键词**：我找到了约201个与鱼类相关的长尾关键词（如“何时钓鲈鱼”或“钓鲈鱼最佳鱼饵是什么”）。
2.  **获取搜索量**：将这些词通过Google Ads关键字规划师运行，获取具体的搜索量数据。由于钓鱼有季节性，我获取了预测范围内（10个月）每个月的预估搜索量。
3.  **评估当前表现**：汇总每月的总搜索量，并根据当前排名（例如第1页底部或第2页顶部）估算当前获得的流量（通常很低）。
4.  **预测排名提升**：我知道我们将大幅改进这些词的内容。因此，我创建了一个预测点击率，从一个较低的百分比开始，随着一年中我们获得更多搜索引擎关注而逐渐上升。
5.  **估算位置**：我根据我们已排名良好的关键词的平均位置，以及基于竞争对手搜索结果和我们域名权重的排名可能性，来估算这些查询的潜在位置。
6.  **计算总收益**：基于此预测，在实施后的10个月期末，我预测我们将获得接近49万次额外访问。

然后，我可以将这些数据用于不同场景（激进、保守等），并纳入更大的全站预测中，或作为争取资源投入、优化特定关键词内容和工具的依据。

## 呈现预测结果

当你提供预测时，请确保：
*   **提供可视化图表**。
*   **将预测与收入关联**。
*   **说明实现目标所需的条件**，例如：“为获得此流量，我们需要在此日期前完成该项目。”
*   **包含不同情况下的多种场景**。
