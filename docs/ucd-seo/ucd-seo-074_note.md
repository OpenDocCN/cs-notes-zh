# 搜索引擎优化：074：内容组织与评估

在本节课中，我们将学习如何系统地组织和评估网站内容。通过内部内容审计，你可以发现改进网站内容的机会。本节将详细介绍评估内容的步骤，帮助你高效地识别和组织现有内容，以提升搜索引擎表现和用户体验。

上一节我们介绍了内部内容审计的价值，本节中我们来看看如何具体执行内容评估与组织工作。

## 内容评估的准备工作

首先，创建一个Excel电子表格，用于记录需要特别关注的页面和评估要点。接着，逐一查看分析工具中列出的每个页面，对页面进行分析，并将URL及相关笔记添加到Excel文件中，以便后续参考。

你可以采用任何有助于高效追踪内容的笔记方式，也可以参考我的评估风格。我喜欢记录页面的URL，以便日后需要时能快速返回该页面。此外，我还会记录季节性、内容类型等信息。

## 详细评估维度

以下是评估内容时需要关注的几个关键维度，了解这些有助于你理解追踪的目的。

### 1. 记录资源季节性
记录资源的季节性，有助于在一年中的特定时间更频繁地引用该内容。例如，如果拥有大量与特定节日相关的资源，可以在每个节日创建新的博客文章，引用过去与该节日相关的优质内容。这能持续为旧内容带来流量，并确保搜索引擎机器人重新抓取这些旧文章。同时，这也能帮助你识别过时的内容，以便更新或彻底删除。

### 2. 标注内容类型
标注你正在查看的内容类型。例如，是文本文章、信息图、视频还是其他形式？对于信息图这类资源，建议定期回顾，并联系可能感兴趣的新网站或博客，请求其重新发布信息图并链接回你的网站。如果是一篇没有图片的文本文章，你现在就知道可以返回该文章并添加图片等资源来改进内容。

### 3. 记录链接情况
记录页面是否包含内部链接和外部链接。这允许你后续根据需要添加链接。内部链接向搜索引擎表明你认为该内容足够重要，值得链接，同时也有助于机器人更频繁地抓取页面。外部链接为用户提供更多价值，使你的页面更有可能在搜索中获得良好排名。

### 4. 记录文章类型或类别
这与内容类型不同，这里记录的是资源所属的文章类型或类别（例如教程、评测、新闻）。例如，如果最终拥有大量“操作指南”类文章，你可能希望创建一个单独的页面来汇总所有“操作指南”主题。这将有助于内容的可发现性，并为这些资源带来更多流量。

### 5. 追踪目标受众方式
在我的表格中，“目标”列指的是我如何定位目标受众。这包括通过直接关键词（例如，关于租用教科书的文章）或间接方法（例如，一篇关于大学生预算生活的文章，它不直接与我的产品相关，但仍能吸引我寻找的受众类型）。

### 6. 检查行动号召
检查文章或资源是否包含行动号召。包含行动号召可以增加访问者执行该操作的可能性。即使只是一个简单的“注册新闻通讯”按钮，也可以被视为该页面的行动号召。每个页面都应包含一个有助于支持业务目标的行动号召。行动号召不一定每次都相同，也不必过于张扬；它可以以内容中有用提示的形式出现。检查内容并确保包含某种形式的行动号召，可以显著提高网站内的转化率。

## 执行改进与后续工作

根据初步评估结果，所有文章都需要添加行动号召。有些文章需要添加图片和更好的内部链接。我们可以进行这些更新，这也会更新内容，使其不再被搜索引擎视为陈旧内容。将这些更新后的文章添加到社交媒体日历中也是一个好主意，这样旧内容可以被用户重新发现和利用。

持续进行评估，直到表格中积累了足够数量的文章。对于大型网站，这是一个持续性的项目，你可能希望每月查看几个不同的页面。然后，可以对跳出率高的页面或（如果网站跟踪转化）转化率低的页面进行同样的评估，看看如何改进它们。随着发现更多内容，你可以查看自然流量页面之外的页面，看看哪些内容可能需要一点推动才能被搜索引擎发现。
