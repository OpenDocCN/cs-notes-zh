# 010：创建自定义类型 🧱

![](img/232260650b75fdbdafbf1665c1032719_0.png)

在本节课中，我们将学习如何创建自定义类型（类）。这是理解Python面向对象编程和更高级测试框架（如`unittest`）的重要基础。

## 概述

我们之前使用的`doctest`框架虽然能很好地记录函数，但当测试数量很多时，可能会使函数内部的代码变得难以阅读。存在其他不具此问题的测试框架，其中之一是Python自带的`unittest`模块。为了使用`unittest`，我们需要学习更多Python语言特性，特别是如何创建自定义类型。

你已经使用过许多类型：字符串、函数、模块、字典、列表、布尔值等。这些类型都是由Python软件基金会的程序员编写的。我们将从重新审视这些现有类型开始，然后学习如何创建我们自己的类型。

## 现有类型回顾

![](img/232260650b75fdbdafbf1665c1032719_4.png)

![](img/232260650b75fdbdafbf1665c1032719_5.png)

`math`模块包含许多函数。其中之一是`math.ceil`，它返回一个数字的“天花板值”，即大于或等于参数的最小整数。

我们这样调用它：`math.ceil(3.2)`。`math`模块也包含数据，特别是变量`e`和`pi`。

导入`math`会创建一个名为`math`的变量，它引用一个**命名空间**。命名空间是一块内存，其中列出了诸如`cos`、`ceil`等名称，每个名称都指向一个函数对象，而`e`和`pi`则指向浮点数。

如你所知，像`int`和`str`这样的类型包含一种特殊的函数，称为**方法**。

![](img/232260650b75fdbdafbf1665c1032719_7.png)

以下是调用`help`函数查看`str`类型时的输出。`str`是一个**类**，而类是Python定义类型的方式。`str`类中的一个方法是`count`。

![](img/232260650b75fdbdafbf1665c1032719_9.png)

字符串方法`count`用于计算一个字符串在另一个字符串中出现的次数。

因为`str`是内置类型，我们已经有了变量`str`。它引用一个类而非模块，但看起来非常相似，内部列出了诸如`center`、`count`等名称，每个名称都引用一个方法对象。

就像我们写`math.ceil(3.2)`来调用`math`模块中的`ceil`函数一样，我们写`str.count`并传入一个字符串和要搜索的子字符串。同样，由于方法的特殊性在于它们期望类的实例作为第一个参数，我们可以用面向对象的方式调用`s.count('y')`。这是调用方法的标准方式。

`str`类有许多有用的方法，但也缺少一些方法。接下来，我们将编写自己的类来定制`str`类。

## 创建自定义类

我们的类将拥有所有字符串方法，但会额外添加一个检查字符串是否以相同字母开头和结尾的方法。

以下是定义类的开始。第一行告诉Python我们正在定义一个名为`WordPlayStr`的新类，它基于`str`类。我们说`WordPlayStr`是`str`的一个**子类**。

![](img/232260650b75fdbdafbf1665c1032719_11.png)

![](img/232260650b75fdbdafbf1665c1032719_12.png)

它继承了`str`的所有特性。正如我们所说，我们的类是一个具有一些额外功能的字符串类。

让我们创建一个新的`WordPlayStr`对象。我们可以获取它的长度，调用字符串方法，甚至可以将其与常规字符串进行比较。到目前为止，它的行为与字符串完全相同。然而，它的类型不是`str`，而是`WordPlayStr`。

让我们创建一个文件来编写新的类定义，包括我们将要添加的方法。

以下是我们在交互式环境中输入的类定义。我们可以向类中添加方法。让我们添加一个方法，如果单词以相同字母开头和结尾则返回`True`，否则返回`False`。

遵循设计方法，第一步是构思示例。

我们将创建一个首尾字母相同的`WordPlayStr`对象。然后当我们调用正在编写的方法（或许命名为`same_start_and_end`）时，我们希望它返回`True`。如果我们创建一个首尾字母不同的`WordPlayStr`对象，那么调用该方法时，我们期望结果为`False`。

方法名`same_start_and_end`看起来不错。让我们填写方法头。有一个参数，我们称之为`self`，它是拥有此方法的`WordPlayStr`对象。类型是`WordPlayStr`，此方法返回一个布尔值。描述是：返回`self`是否以相同字母开头和结尾。

因为`WordPlayStr`类继承自`str`类，我们可以访问所有字符串功能。这意味着我们可以使用索引。我们可以查看索引`0`处的字符，并将其与索引`-1`处的字符进行比较。如果这两个字符相同，`same_start_and_end`应产生`True`，否则产生`False`。

这在空字符串上无效，因此我们将添加一个前置条件来禁止空字符串。

让我们在文件底部添加一个`if __name__ == '__main__':`部分来测试我们的代码。运行它，测试通过。

重要的是要注意，我们并没有改变`str`类。它不知道这些新方法。只有我们的子类`WordPlayStr`有能力检查其首尾字符是否相同。

在本课程的后面部分，我们将看到关于类的更深入解释，但现在你已经准备好学习Python的`unittest`模块了。
