在本节课中，我们将实现第三个算法，用于判断一个字符串是否为回文。



## 概述

上一节我们完成了函数设计流程的前四个步骤。本节中，我们将进入第五步：实现函数体。我们将学习一个通过比较字符对来判断回文的算法。



## 算法思路

该算法的核心思想是，从字符串的两端开始，逐对比较字符。

*   首先比较索引0的字符与最后一个索引的字符。
*   然后比较索引1的字符与倒数第二个索引的字符。
*   以此类推，直到到达字符串的中间。

如果发现任何一对字符不相等，则可以立即断定该字符串不是回文。



## 实现步骤

为了实现这个算法，我们需要使用两个变量来跟踪要比较的字符索引。

以下是初始设置：
*   变量 `i` 初始指向字符串的第一个索引（0）。
*   变量 `j` 初始指向字符串的最后一个索引（`len(s) - 1`）。

我们可以用一个矩形表示字符串。初始时，`i` 在索引0处，`j` 在最后一个索引处，这两个位置的字符是第一对要比较的。随着比较的进行，`i` 会递增，`j` 会递减，两者向字符串中间移动。



## 编写代码

我们将使用一个 `while` 循环来持续比较字符对，只要满足两个条件：
1.  索引 `i` 处的字符等于索引 `j` 处的字符。
2.  索引 `i` 仍然小于索引 `j`（即尚未到达或越过中点）。

以下是循环体的逻辑：
```python
while i < j and s[i] == s[j]:
    i = i + 1  # i 向右移动
    j = j - 1  # j 向左移动
```
循环结束后，我们需要返回判断结果。



## 确定返回值

循环结束有两个原因：
1.  **找到了不相等的字符对**：此时 `s[i] != s[j]`，循环因条件 `s[i] == s[j]` 为假而终止。这意味着字符串不是回文。
2.  **到达了字符串中点**：此时 `i >= j`，循环因条件 `i < j` 为假而终止。这意味着所有比较的字符对都相等，字符串是回文。

因此，我们可以通过检查是否 `j <= i` 来判断是否到达了中点。如果是，则返回 `True`（是回文）；否则返回 `False`。

最终的返回语句可以简化为：
```python
return j <= i
```
