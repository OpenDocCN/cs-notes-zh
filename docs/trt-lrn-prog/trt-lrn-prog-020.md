## 算法概述

解决排序问题的一种方法是使用插入排序算法。这正是本节课要讲解的内容。

回忆一下选择排序中，列表在开始时、经过若干轮迭代后以及结束时的状态。插入排序也是如此。在开始时，整个列表是未排序的，变量 `i` 指向索引 0。经过若干轮迭代后，列表的一部分已排序，另一部分未排序。`i` 指向列表未排序部分的第一个元素的索引。在结束时，整个列表已排序，`i` 指向列表的长度。

对于插入排序，算法的一轮迭代涉及将索引 `i` 处的元素纳入列表的已排序部分。

## 算法执行示例

让我们来看一个例子。假设我们有一个列表 `[3, 7, 2, 5]`。

**第一轮迭代 (`i = 1`):**
由于整个列表仍被视为未排序，将 `i` 处的元素纳入已排序部分很简单：我们只需递增 `i` 并进入下一轮。此时，已排序部分为 `[3]`，未排序部分为 `[7, 2, 5]`。

**第二轮迭代 (`i = 2`):**
现在，我们需要将索引 `i`（值为 `2`）的元素纳入已排序部分 `[3, 7]`。我们知道，在任何给定的迭代中，只有索引从 `0` 到 `i`（包括 `i`）的元素可能会改变。
1.  将 `2` 与 `7` 比较，`2` 小于 `7`，所以 `7` 需要右移。
2.  将 `2` 与 `3` 比较，`2` 小于 `3`，所以 `3` 也需要右移。
3.  现在，索引 `0` 的位置空出，将 `2` 插入此处。
完成此轮后，列表变为 `[2, 3, 7, 5]`，已排序部分为 `[2, 3, 7]`。

**第三轮迭代 (`i = 3`):**
现在，需要将索引 `i`（值为 `5`）的元素纳入已排序部分 `[2, 3, 7]`。
1.  将 `5` 与 `7` 比较，`5` 小于 `7`，所以 `7` 右移。
2.  将 `5` 与 `3` 比较，`5` 大于 `3`，因此停止比较。
3.  将 `5` 插入到 `3` 之后的位置（即原 `7` 的位置）。
完成此轮后，列表完全排序为 `[2, 3, 5, 7]`。

## 算法实现

现在，让我们来实现这个算法。对于算法的每一轮迭代，索引 `i` 处的元素被插入到列表的已排序部分。我们将编写一个辅助函数 `insert` 来完成核心的插入操作。

以下是实现步骤的分解：

**核心插入函数 `insert` 的逻辑：**
变量 `value` 将引用列表在索引 `i` 处的值，即要插入到已排序部分的元素。接下来，我们需要找到 `value` 应该插入的位置索引 `j`。我们还需要通过右移元素来为 `value` 腾出空间。一旦我们知道了 `value` 所属的索引 `j`，我们就将 `value` 放在那里。

找到索引并腾出空间是最复杂的部分。我们将使用一个 `while` 循环。暂时不指定循环条件，稍后再添加。在循环体中，一个元素将被向右移动。然后索引 `j` 减 1。

为了确定循环条件，让我们回顾一下我们的例子。最初，`j` 等于 `i`。
1.  `2` 与 `7` 比较，因为 `2` 小于 `7`，`7` 将右移，然后 `j` 的值减 1。
2.  接着，`2` 与 `3` 比较，`3` 右移，然后 `j` 的值再次减 1。
3.  此时，`j` 为 0，不可能再向左移动，我们找到了 `2` 应该插入的位置。

让我们将这个停止的原因添加到 `while` 循环条件中：当 `j` 等于 0 时，我们希望停止，所以我们希望只要 `j` 不为 0 就继续循环。

循环结束还有另一个原因，让我们考虑第二个例子（插入 `5` 时）：
1.  `5` 与 `7` 比较，`7` 右移，`j` 减 1。
2.  接着，`5` 与 `3` 比较，因为 `5` 大于 `3`，我们停止。

因此，我们的 `while` 循环应该继续的第二个条件是：`list[j-1]` 大于 `value`。

综合以上两点，循环继续的条件是：`j > 0` 且 `list[j-1] > value`。

以下是插入排序的完整 Python 实现代码：

```python
def insert(L, i):
    """将 L[i] 插入到 L[:i] 的已排序部分中。"""
    value = L[i]
    j = i
    # 寻找插入位置并右移元素
    while j > 0 and L[j - 1] > value:
        L[j] = L[j - 1]
        j -= 1
    # 插入值
    L[j] = value

def insertion_sort(L):
    """对列表 L 进行插入排序。"""
    for i in range(len(L)):
        insert(L, i)
```

## 测试与总结

现在我们已经完成了实现，让我们运行一个简单的测试。

```python

```
