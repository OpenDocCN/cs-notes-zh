在本节课中，我们将实现第一个用于判断字符串是否为回文的算法。

## 概述

上一节我们介绍了算法设计的基本思路。本节中，我们来看看如何具体实现一个回文判断算法。我们的第一个方案是：将字符串反转，然后比较反转后的字符串是否与原字符串相同。

## 实现字符串反转函数

由于Python的字符串类型没有内置的`reverse`方法，我们需要先编写一个辅助函数来反转字符串。

以下是反转函数的设计步骤：

**1. 设计示例**
我们首先构思一些测试用例。例如：
*   输入 `"hello"`，期望输出 `"olleh"`。
*   输入 `"a"`，期望输出 `"a"`。

**2. 确定类型契约与函数头**
函数接收一个字符串，并返回一个字符串。我们将其命名为 `reverse`，参数用 `s` 表示。
```python
def reverse(s):
```

**3. 编写描述**
函数描述为：返回字符串 `s` 的反转版本。

**4. 设计算法**
我们将采用累积的方法构建新字符串。初始化一个空字符串 `rev` 作为累加器。
与手动反转（从最后一个字符开始取）不同，Python的`for`循环遍历字符串是从第一个字符开始的。因此，我们的策略是：遍历原字符串的每个字符，并将其**添加**到累加器字符串的**开头**。
用自然语言描述算法：对于字符串中的每个字符，将其添加到累加器的开头。

**5. 转换为代码**
根据以上描述，算法可以直接翻译为Python代码。
```python
def reverse(s):
    rev = ""
    for ch in s:
        rev = ch + rev
    return rev
```

**6. 测试函数**
现在运行模块并测试我们之前设计的示例。
```python
>>> reverse("hello")
'olleh'
>>> reverse("a")
'a'
```
测试结果符合预期，反转函数工作正常。

## 实现回文判断函数

现在我们已经有了反转字符串的函数，接下来实现回文判断函数 `is_palindrome` 就非常简单了。

核心逻辑是：判断原字符串 `s` 是否等于其反转后的字符串。
用公式表示这个判断条件就是：`s == reverse(s)`

因此，函数实现如下：
```python
def is_palindrome(s):
    return s == reverse(s)
```

让我们测试这个函数。运行模块后，使用几个字符串进行测试。
```python
>>> is_palindrome("noon")
True
>>> is_palindrome("racecar")
True
>>> is_palindrome("dented")
False
```
测试结果显示，函数正确地判断了“noon”和“racecar”是回文，而“dented”不是回文。

## 总结

![](img/ebbc031109b04c5104a537ca625a937c_4.png)

本节课中我们一起学习了如何实现第一个回文判断算法。我们首先设计并实现了一个用于反转字符串的辅助函数 `reverse`，其核心是使用循环和字符串拼接从前往后构建反转结果。接着，我们利用这个辅助函数，通过简单的等式比较 `s == reverse(s)` 实现了回文判断函数 `is_palindrome`。这个算法逻辑清晰，易于理解，是解决问题的一个良好开端。
