在本节课中，我们将学习一种重要的证明方法——**分情况证明法**。我们将通过一个具体的例子来理解这种方法：证明一个整数的平方与其本身具有相同的奇偶性。

![](img/7e53b666130df33696522a3fb0d5b7ba_8.png)

## 概述与核心定理

![](img/7e53b666130df33696522a3fb0d5b7ba_10.png)

![](img/7e53b666130df33696522a3fb0d5b7ba_11.png)

![](img/7e53b666130df33696522a3fb0d5b7ba_12.png)

考虑以下定理：**一个整数的平方与其本身具有相同的奇偶性**。这里的“奇偶性”指的是一个数是偶数还是奇数。

![](img/7e53b666130df33696522a3fb0d5b7ba_14.png)

换句话说，如果初始数是偶数，那么它的平方也是偶数；如果初始数是奇数，那么它的平方也是奇数。

![](img/7e53b666130df33696522a3fb0d5b7ba_15.png)

![](img/7e53b666130df33696522a3fb0d5b7ba_16.png)

![](img/7e53b666130df33696522a3fb0d5b7ba_17.png)

![](img/7e53b666130df33696522a3fb0d5b7ba_19.png)

## 为什么需要分情况证明？

![](img/7e53b666130df33696522a3fb0d5b7ba_20.png)

![](img/7e53b666130df33696522a3fb0d5b7ba_22.png)

![](img/7e53b666130df33696522a3fb0d5b7ba_23.png)

如果我们直接观察一个整数 `n` 并计算其平方 `n²`，我们很难直接得出关于其奇偶性的结论。因为 `n` 本身可能是偶数，也可能是奇数，这两种情况下的代数表达式和性质是不同的。

![](img/7e53b666130df33696522a3fb0d5b7ba_25.png)

![](img/7e53b666130df33696522a3fb0d5b7ba_26.png)

因此，为了证明这个定理，我们需要将问题分解为两个独立的、互斥的情况来处理：
1.  **情况一**：`n` 是偶数。
2.  **情况二**：`n` 是奇数。

![](img/7e53b666130df33696522a3fb0d5b7ba_27.png)

![](img/7e53b666130df33696522a3fb0d5b7ba_29.png)

由于任何整数都必然是偶数或奇数，如果我们能分别证明在这两种情况下结论都成立，那么我们就证明了整个定理。

![](img/7e53b666130df33696522a3fb0d5b7ba_31.png)

![](img/7e53b666130df33696522a3fb0d5b7ba_32.png)

![](img/7e53b666130df33696522a3fb0d5b7ba_34.png)

## 情况一：证明偶数的平方是偶数

上一节我们介绍了分情况证明的思路，本节中我们来看看如何具体执行。首先，我们来证明第一种情况。

![](img/7e53b666130df33696522a3fb0d5b7ba_36.png)

![](img/7e53b666130df33696522a3fb0d5b7ba_37.png)

![](img/7e53b666130df33696522a3fb0d5b7ba_38.png)

**假设**：设 `n` 是一个偶数。

![](img/7e53b666130df33696522a3fb0d5b7ba_39.png)

![](img/7e53b666130df33696522a3fb0d5b7ba_40.png)

![](img/7e53b666130df33696522a3fb0d5b7ba_41.png)

![](img/7e53b666130df33696522a3fb0d5b7ba_43.png)

**目标**：证明 `n²` 是偶数。

![](img/7e53b666130df33696522a3fb0d5b7ba_45.png)

![](img/7e53b666130df33696522a3fb0d5b7ba_46.png)

以下是证明步骤：

![](img/7e53b666130df33696522a3fb0d5b7ba_48.png)

1.  **应用定义**：根据偶数的定义，存在某个整数 `k₁`，使得 `n = 2k₁`。
2.  **计算平方**：我们考察 `n²` 的表达式：
    `n² = (2k₁)²`
3.  **代数变换**：对上式进行展开和重组：
    `n² = 4k₁² = 2 * (2k₁²)`
4.  **构造新整数**：令 `k₂ = 2k₁²`。由于 `k₁` 是整数，`k₂` 也是整数。
5.  **得出结论**：因此，`n² = 2k₂`。根据偶数的定义，`n²` 是偶数。

![](img/7e53b666130df33696522a3fb0d5b7ba_50.png)

![](img/7e53b666130df33696522a3fb0d5b7ba_51.png)

![](img/7e53b666130df33696522a3fb0d5b7ba_52.png)

![](img/7e53b666130df33696522a3fb0d5b7ba_53.png)

**于是，我们证明了：如果 `n` 是偶数，则 `n²` 是偶数。**

![](img/7e53b666130df33696522a3fb0d5b7ba_54.png)

![](img/7e53b666130df33696522a3fb0d5b7ba_55.png)

## 情况二：证明奇数的平方是奇数

![](img/7e53b666130df33696522a3fb0d5b7ba_57.png)

![](img/7e53b666130df33696522a3fb0d5b7ba_58.png)

![](img/7e53b666130df33696522a3fb0d5b7ba_59.png)

类似地，我们需要处理第二种情况。其证明逻辑与情况一相似，但会用到奇数的定义。

![](img/7e53b666130df33696522a3fb0d5b7ba_61.png)

![](img/7e53b666130df33696522a3fb0d5b7ba_62.png)

**假设**：设 `n` 是一个奇数。

![](img/7e53b666130df33696522a3fb0d5b7ba_64.png)

![](img/7e53b666130df33696522a3fb0d5b7ba_65.png)

![](img/7e53b666130df33696522a3fb0d5b7ba_66.png)

**目标**：证明 `n²` 是奇数。

![](img/7e53b666130df33696522a3fb0d5b7ba_68.png)

![](img/7e53b666130df33696522a3fb0d5b7ba_69.png)

![](img/7e53b666130df33696522a3fb0d5b7ba_70.png)

以下是证明思路（作为练习）：
1.  根据奇数的定义，存在某个整数 `m`，使得 `n = 2m + 1`。
2.  计算 `n² = (2m + 1)²`。
3.  展开并整理表达式：`n² = 4m² + 4m + 1 = 2(2m² + 2m) + 1`。
4.  令 `p = 2m² + 2m`，则 `p` 是整数。
5.  因此，`n² = 2p + 1`。根据奇数的定义，`n²` 是奇数。

![](img/7e53b666130df33696522a3fb0d5b7ba_72.png)

**于是，我们证明了：如果 `n` 是奇数，则 `n²` 是奇数。**

![](img/7e53b666130df33696522a3fb0d5b7ba_75.png)

## 分情况证明法的原理总结

![](img/7e53b666130df33696522a3fb0d5b7ba_79.png)

![](img/7e53b666130df33696522a3fb0d5b7ba_80.png)

本节课中我们一起学习了分情况证明法。其核心思想可以总结如下：

当我们试图证明一个形如 **“若 (P 或 Q)，则 R”** 的命题时，我们可以将其分解为两个（或多个）子命题来分别证明：
1.  证明 **“若 P，则 R”**。
2.  证明 **“若 Q，则 R”**。

![](img/7e53b666130df33696522a3fb0d5b7ba_82.png)

![](img/7e53b666130df33696522a3fb0d5b7ba_83.png)

![](img/7e53b666130df33696522a3fb0d5b7ba_84.png)

![](img/7e53b666130df33696522a3fb0d5b7ba_85.png)

![](img/7e53b666130df33696522a3fb0d5b7ba_86.png)

因为前提 `P 或 Q` 为真，意味着至少 `P` 和 `Q` 中有一个为真。如果我们证明了在 `P` 为真时 `R` 为真，并且在 `Q` 为真时 `R` 也为真，那么无论实际情况是 `P` 还是 `Q`，结论 `R` 都必然为真。这就完成了整个命题的证明。

![](img/7e53b666130df33696522a3fb0d5b7ba_88.png)

![](img/7e53b666130df33696522a3fb0d5b7ba_89.png)

在我们的例子中：
*   `P` 代表 “`n` 是偶数”。
*   `Q` 代表 “`n` 是奇数”。
*   `R` 代表 “`n²` 与 `n` 奇偶性相同”。

![](img/7e53b666130df33696522a3fb0d5b7ba_91.png)

![](img/7e53b666130df33696522a3fb0d5b7ba_92.png)

![](img/7e53b666130df33696522a3fb0d5b7ba_93.png)

![](img/7e53b666130df33696522a3fb0d5b7ba_94.png)

通过分别证明 `P → R` 和 `Q → R`，我们最终证明了定理 `(P 或 Q) → R`。
