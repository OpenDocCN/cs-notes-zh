在本节课中，我们将要学习一种重要的证明方法——逆否证明法。我们将通过一个具体的数学定理来理解其逻辑基础、应用步骤以及为何在某些情况下它比直接证明更有效。

## 概述

![](img/082ac95bf7e01779a5f4eedc62d51b54_5.png)

![](img/082ac95bf7e01779a5f4eedc62d51b54_6.png)

![](img/082ac95bf7e01779a5f4eedc62d51b54_7.png)

在逻辑学中，一个蕴含式 `P ⇒ Q` 与其逆否命题 `¬Q ⇒ ¬P` 在逻辑上是完全等价的。这意味着，如果我们能够证明逆否命题成立，那么原命题也必然成立。本节课将深入探讨如何利用这一逻辑等价关系来证明数学命题，特别是当直接证明较为困难时。

## 逆否证明法的逻辑基础

![](img/082ac95bf7e01779a5f4eedc62d51b54_9.png)

上一节我们介绍了蕴含式的基本概念，本节中我们来看看如何利用其逻辑等价形式进行证明。

![](img/082ac95bf7e01779a5f4eedc62d51b54_10.png)

![](img/082ac95bf7e01779a5f4eedc62d51b54_11.png)

![](img/082ac95bf7e01779a5f4eedc62d51b54_12.png)

我们知道，一个标准的蕴含式 `P ⇒ Q` 在逻辑上等价于其逆否命题 `¬Q ⇒ ¬P`。这个等价关系是逆否证明法的核心。

![](img/082ac95bf7e01779a5f4eedc62d51b54_14.png)

![](img/082ac95bf7e01779a5f4eedc62d51b54_15.png)

因此，当我们的目标是证明一个形如“若P则Q”的命题时，我们可以转而尝试证明其逆否命题“若非Q则非P”。通过证明逆否命题，我们就能有效地证明原命题。

## 为何选择逆否证明法？一个挑战性的例子

![](img/082ac95bf7e01779a5f4eedc62d51b54_17.png)

![](img/082ac95bf7e01779a5f4eedc62d51b54_18.png)

![](img/082ac95bf7e01779a5f4eedc62d51b54_19.png)

让我们通过一个具体的定理来理解为何有时需要采用逆否证明法。

**定理**：若整数 `n` 的平方 `n²` 是偶数，则 `n` 本身也是偶数。

![](img/082ac95bf7e01779a5f4eedc62d51b54_21.png)

![](img/082ac95bf7e01779a5f4eedc62d51b54_22.png)

![](img/082ac95bf7e01779a5f4eedc62d51b54_23.png)

请注意，这个定理与我们之前见过的定理方向相反。它不是说“若n是偶数，则n²是偶数”，而是说“若n²是偶数，则n是偶数”。

首先，我们尝试直接证明，看看会遇到什么挑战。

![](img/082ac95bf7e01779a5f4eedc62d51b54_25.png)

![](img/082ac95bf7e01779a5f4eedc62d51b54_26.png)

![](img/082ac95bf7e01779a5f4eedc62d51b54_27.png)

**直接证明的尝试**：
1.  假设 `n²` 是偶数。根据定义，存在某个整数 `k`，使得 `n² = 2k`。
2.  我们想得出关于 `n` 的结论。一个自然的想法是对等式两边取平方根，得到 `n = √(2k) = √2 * √k`。
3.  这个表达式 `√2 * √k` 看起来完全不像一个偶数的形式，其中包含了无理数 `√2`，使得推导无法继续进行。

由此可见，对这个命题进行直接证明似乎颇具挑战性。因此，我们转向逆否证明法。

![](img/082ac95bf7e01779a5f4eedc62d51b54_29.png)

![](img/082ac95bf7e01779a5f4eedc62d51b54_30.png)

![](img/082ac95bf7e01779a5f4eedc62d51b54_31.png)

## 应用逆否证明法

![](img/082ac95bf7e01779a5f4eedc62d51b54_32.png)

既然直接证明不太奏效，让我们尝试证明其逆否命题。

原命题是：`n² 是偶数 ⇒ n 是偶数`。
其逆否命题是：`n 不是偶数 ⇒ n² 不是偶数`。

![](img/082ac95bf7e01779a5f4eedc62d51b54_34.png)

![](img/082ac95bf7e01779a5f4eedc62d51b54_35.png)

![](img/082ac95bf7e01779a5f4eedc62d51b54_36.png)

换句话说，我们假设 `n` 不是偶数（即 `n` 是奇数），然后试图推导出 `n²` 不是偶数（即 `n²` 是奇数）。

![](img/082ac95bf7e01779a5f4eedc62d51b54_37.png)

![](img/082ac95bf7e01779a5f4eedc62d51b54_38.png)

**逆否证明过程**：
1.  **假设**：`n` 是奇数。根据定义，存在某个整数 `m`，使得 `n = 2m + 1`。
2.  **计算平方**：`n² = (2m + 1)² = 4m² + 4m + 1 = 2(2m² + 2m) + 1`。
3.  **得出结论**：令 `k = 2m² + 2m`，则 `n² = 2k + 1`。这正是奇数的定义。
4.  因此，我们证明了：若 `n` 是奇数，则 `n²` 是奇数。

![](img/082ac95bf7e01779a5f4eedc62d51b54_40.png)

![](img/082ac95bf7e01779a5f4eedc62d51b54_41.png)

由于“`n` 是奇数 ⇒ `n²` 是奇数”在逻辑上等价于“`n²` 是偶数 ⇒ `n` 是偶数”，所以我们通过证明逆否命题，成功地证明了原定理。

![](img/082ac95bf7e01779a5f4eedc62d51b54_43.png)

![](img/082ac95bf7e01779a5f4eedc62d51b54_44.png)

## 方法总结与要点

![](img/082ac95bf7e01779a5f4eedc62d51b54_46.png)

以下是应用逆否证明法的关键步骤：

![](img/082ac95bf7e01779a5f4eedc62d51b54_47.png)

1.  **识别命题形式**：确认你要证明的命题是一个蕴含式 `P ⇒ Q`。
2.  **构造逆否命题**：写出其逻辑等价的逆否形式 `¬Q ⇒ ¬P`。
3.  **证明逆否命题**：使用直接证明或其他已知方法证明 `¬Q ⇒ ¬P` 成立。
4.  **得出结论**：由于逻辑等价，原命题 `P ⇒ Q` 得证。

逆否证明法的优势在于，有时逆否命题的假设和结论更易于处理或与我们已知的定理更接近。正如本例所示，证明“n是奇数 ⇒ n²是奇数”比直接证明原命题要简单直观得多。
