在本节课中，我们将要学习集合论中的一个核心概念：两个集合的交集。我们将通过定义、图示和具体例子来理解交集的含义，并学习如何用严谨的数学语言证明关于交集的结论。

![](img/1c6249e6d4f2e944b9efdaf80d3718cd_7.png)

上一节我们介绍了集合的并集，它关注的是属于至少一个集合的元素。本节中我们来看看交集，它关注的是同时属于两个集合的元素。

![](img/1c6249e6d4f2e944b9efdaf80d3718cd_8.png)

![](img/1c6249e6d4f2e944b9efdaf80d3718cd_9.png)

## 交集的定义与符号

![](img/1c6249e6d4f2e944b9efdaf80d3718cd_11.png)

![](img/1c6249e6d4f2e944b9efdaf80d3718cd_12.png)

两个集合 **A** 和 **B** 的交集，记作 **A ∩ B**，其定义是同时属于集合 **A** 和集合 **B** 的所有元素构成的集合。

![](img/1c6249e6d4f2e944b9efdaf80d3718cd_14.png)

用逻辑语言和集合构造器可以表示为：
**A ∩ B = { x | x ∈ A ∧ x ∈ B }**

![](img/1c6249e6d4f2e944b9efdaf80d3718cd_15.png)

![](img/1c6249e6d4f2e944b9efdaf80d3718cd_16.png)

你可以通过符号的形状来记忆：交集符号 **∩** 开口向下，而逻辑“与”符号 **∧** 看起来有点像字母 **A**。这有助于联想交集要求元素 **x** 既在 **A** 中，也在 **B** 中。

![](img/1c6249e6d4f2e944b9efdaf80d3718cd_18.png)

![](img/1c6249e6d4f2e944b9efdaf80d3718cd_19.png)

我们可以通过文氏图来直观地理解交集。

![](img/1c6249e6d4f2e944b9efdaf80d3718cd_25.png)

![](img/1c6249e6d4f2e944b9efdaf80d3718cd_26.png)

假设有两个集合 **A** 和 **B**，并且它们有重叠的部分。在图中，**A** 和 **B** 重叠的中间区域，就是它们的交集 **A ∩ B**。这个区域中的元素同时属于 **A** 和 **B**。

如果集合 **A** 和 **B** 完全没有重叠部分，那么它们的交集就是空集 **∅**，因为没有任何元素能同时属于这两个集合。

## 实例分析：偶数和3的倍数

![](img/1c6249e6d4f2e944b9efdaf80d3718cd_28.png)

让我们通过一个具体的例子来加深理解。

定义集合 **A** 为所有偶数的集合：
**A = { n ∈ ℤ | n = 2p, p ∈ ℤ }**

定义集合 **B** 为所有3的倍数的集合：
**B = { n ∈ ℤ | n = 3q, q ∈ ℤ }**

现在，我们想找出 **A ∩ B**，即同时是偶数和3的倍数的整数。

![](img/1c6249e6d4f2e944b9efdaf80d3718cd_30.png)

![](img/1c6249e6d4f2e944b9efdaf80d3718cd_31.png)

![](img/1c6249e6d4f2e944b9efdaf80d3718cd_32.png)

我们可以列出一些整数来观察：
*   偶数 **A**：0, 2, 4, 6, 8, 10, 12, ...
*   3的倍数 **B**：0, 3, 6, 9, 12, 15, ...

![](img/1c6249e6d4f2e944b9efdaf80d3718cd_33.png)

![](img/1c6249e6d4f2e944b9efdaf80d3718cd_34.png)

![](img/1c6249e6d4f2e944b9efdaf80d3718cd_35.png)

以下是同时出现在两个列表中的数：
*   0 同时出现在两个列表中。
*   6 同时出现在两个列表中。
*   12 同时出现在两个列表中。

![](img/1c6249e6d4f2e944b9efdaf80d3718cd_37.png)

观察可知，**A ∩ B** 中的元素是 0, 6, 12, ...，即所有6的倍数。这很合理，因为一个数要同时是2和3的倍数，它就必须是它们最小公倍数6的倍数。

![](img/1c6249e6d4f2e944b9efdaf80d3718cd_38.png)

![](img/1c6249e6d4f2e944b9efdaf80d3718cd_39.png)

因此，我们断言：**A ∩ B = { n ∈ ℤ | n = 6r, r ∈ ℤ }**。

![](img/1c6249e6d4f2e944b9efdaf80d3718cd_41.png)

![](img/1c6249e6d4f2e944b9efdaf80d3718cd_43.png)

## 形式化证明

![](img/1c6249e6d4f2e944b9efdaf80d3718cd_44.png)

![](img/1c6249e6d4f2e944b9efdaf80d3718cd_45.png)

为了严谨地证明上述断言，我们需要证明两个集合相等：**A ∩ B** 等于“6的倍数”这个集合。这需要证明两个方向：
1.  如果某个元素在 **A ∩ B** 中，那么它一定是6的倍数。
2.  如果某个元素是6的倍数，那么它一定在 **A ∩ B** 中。

![](img/1c6249e6d4f2e944b9efdaf80d3718cd_47.png)

以下是完整的证明过程：

**证明：** 我们欲证 **A ∩ B = { n ∈ ℤ | n = 6r, r ∈ ℤ }**。

![](img/1c6249e6d4f2e944b9efdaf80d3718cd_49.png)

**(⇒) 方向：** 假设 **n ∈ A ∩ B**。
1.  根据交集的定义，这意味着 **n ∈ A** 且 **n ∈ B**。
2.  根据集合 **A** 和 **B** 的定义，存在整数 **p** 和 **q**，使得 **n = 2p** 且 **n = 3q**。
3.  由 **n = 2p** 和 **n = 3q** 可得 **2p = 3q**。这意味着 **2p** 是3的倍数。
4.  由于2不是3的倍数，因此 **p** 本身必须是3的倍数。故存在整数 **r**，使得 **p = 3r**。
5.  将 **p = 3r** 代入 **n = 2p**，得到 **n = 2 * (3r) = 6r**。
6.  因此，**n** 是6的倍数，即 **n ∈ { n ∈ ℤ | n = 6r, r ∈ ℤ }**。

**(⇐) 方向：** 假设 **n** 是6的倍数，即存在整数 **r** 使得 **n = 6r**。
1.  我们可以将 **n** 写为 **n = 2 * (3r)**。令 **p = 3r**，则 **n = 2p**，且 **p** 是整数。故 **n ∈ A**。
2.  我们也可以将 **n** 写为 **n = 3 * (2r)**。令 **q = 2r**，则 **n = 3q**，且 **q** 是整数。故 **n ∈ B**。
3.  由于 **n ∈ A** 且 **n ∈ B**，根据交集定义，**n ∈ A ∩ B**。

综上，我们证明了 **A ∩ B** 与“6的倍数集合”互为子集，因此两者相等。∎

本节课中我们一起学习了集合交集的定义、文氏图表示法，并通过一个具体例子（偶数集与3的倍数集的交集）展示了如何从直观理解到形式化证明。关键在于掌握交集的核心是寻找同时满足两个集合性质的元素，并用严谨的逻辑步骤来证明关于交集的等式。
