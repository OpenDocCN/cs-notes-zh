# Python入门：01：变量与基础数据类型

在本节课中，我们将要学习Python编程中最基础的两个概念：变量和基础数据类型。理解这些是编写任何Python程序的第一步。

## 变量：数据的容器

上一节我们介绍了课程目标，本节中我们来看看什么是变量。你可以把变量想象成一个贴有标签的盒子，用于存放数据。在Python中，我们使用等号（`=`）来给变量赋值。

其基本语法是：
```python
变量名 = 值
```

例如：
```python
my_name = “小明”
age = 25
```

## 基础数据类型

变量可以存储不同类型的数据。Python有几种内置的基础数据类型，它们决定了你可以对数据做什么操作。

以下是几种最常用的基础数据类型：

*   **整数（int）**：表示没有小数部分的数字，如 `-5`, `0`, `100`。
*   **浮点数（float）**：表示带有小数点的数字，如 `3.14`, `2.0`, `-0.5`。
*   **字符串（str）**：表示文本信息，需要用单引号（`‘`）或双引号（`“`）括起来，如 `“你好世界”`, `‘Python’`。
*   **布尔值（bool）**：只有两个值，`True`（真）或 `False`（假），常用于逻辑判断。

## 查看与理解类型

在编程时，了解一个变量当前存储的是什么类型的数据非常重要。Python提供了一个内置函数 `type()` 来帮助我们查看。

其用法是：
```python
type(变量名或数据)
```

例如，执行 `type(10)` 会返回 `<class ‘int’>`，告诉我们10是一个整数类型。

## 一个简单的综合示例

让我们通过一个简单的例子，将以上概念结合起来。

```python
# 定义几个变量
name = “张三”      # 字符串类型
score = 95.5       # 浮点数类型
is_passed = True   # 布尔类型

# 打印这些变量
print(“学生姓名：”, name)
print(“考试分数：”, score)
print(“是否通过：”, is_passed)

# 查看它们的类型
print(“name的类型是：”, type(name))
print(“score的类型是：”, type(score))
```

运行这段代码，你将在屏幕上看到变量的值以及它们各自的数据类型。

本节课中我们一起学习了Python的变量与基础数据类型。我们知道了变量是存储数据的容器，并通过等号进行赋值。我们还认识了四种基础数据类型：用于整数的**int**、用于小数的**float**、用于文本的**str**以及用于逻辑判断的**bool**。最后，我们学会了使用 `type()` 函数来查看数据的类型。掌握这些基础知识，是后续学习更复杂操作的关键。
