# 计算机安全：CS161：课程概述与章节导航 📚

在本教程中，我们将一起概览加州大学伯克利分校（UCB）CS161《计算机安全》课程（2023年秋季）的完整内容结构。本课程从基础的安全原则出发，逐步深入到内存安全、密码学、Web安全、网络安全及高级主题，为初学者构建一个系统性的计算机安全知识框架。

*   **001：课程介绍与安全原则** 🛡️
*   **002：x86汇编与调用栈** 💻
*   **003：内存安全漏洞** 🚨
*   **004：更多内存安全漏洞** ⚠️
*   **005：缓解内存安全漏洞** 🛠️
*   **006：密码学导论** 🔐
*   **007：分组密码** 🧩
*   **008：分组密码工作模式** 🔄
*   **009：密码学哈希与消息认证码** 🔑
*   **010：伪随机数生成器与迪菲-赫尔曼密钥交换** 🔢
*   **011：公钥加密与数字签名** 📜
*   **012：证书、密码与案例研究** 📄
*   **013：Web技术导论** 🌐
*   **014：Cookie与跨站请求伪造** 🍪
*   **015：跨站脚本与用户界面攻击** ⚡
*   **016：SQL注入与验证码** 💉
*   **017：网络基础导论** 📡
*   **018：底层网络攻击** 🕵️
