# CSE12 基础数据结构和面向对象设计：001：课程介绍与大纲 📚

在本节课中，我们将学习CSE12课程的整体介绍、课程目标以及课程大纲。我们将了解这门课程将涵盖哪些核心主题，以及学习这些知识的重要性。

## 概述

CSE12是一门关于基础数据结构和面向对象设计的课程。课程旨在教授学生如何有效地组织和管理数据，以及如何使用面向对象的设计原则来构建健壮的软件系统。

## 课程目标

课程的主要目标是让学生掌握以下能力：

*   理解并实现基本的数据结构。
*   应用面向对象的设计原则。
*   分析算法的效率。
*   编写清晰、可维护的代码。

## 核心主题

以下是本课程将涵盖的核心主题列表：

1.  **数组与链表**：学习顺序存储和链式存储的基本结构。
2.  **栈与队列**：理解后进先出（LIFO）和先进先出（FIFO）的数据结构。
3.  **树与二叉树**：探索层次化数据组织方式，特别是二叉搜索树。
4.  **哈希表**：学习通过哈希函数实现快速数据访问。
5.  **排序与搜索算法**：掌握如快速排序、归并排序和二分查找等经典算法。
6.  **面向对象概念**：包括类、对象、继承、多态和封装。
7.  **算法复杂度分析**：使用大O符号（**O(n)**， **O(log n)**， **O(n²)**）评估算法性能。

## 课程结构

课程通常包括讲座、实验、编程作业和考试。学生需要通过实践来巩固理论知识。

## 学习资源

为了辅助学习，课程提供了多种资源：

*   官方课程网站和幻灯片。
*   教科书和参考书目。
*   在线论坛用于讨论问题。
*   由助教主持的讨论课和办公时间。

## 评估方式

学生的成绩将由以下几个部分构成：

*   编程作业（约40%）
*   期中考试（约25%）
*   期末考试（约30%）
*   课堂参与（约5%）
