## 回顾与目标 🎯

![](img/84e48ca68c32b2f24d02befb6faff7dd_10.png)

![](img/84e48ca68c32b2f24d02befb6faff7dd_12.png)

上一节我们介绍了多项式乘法的两种表示方法：系数表示法和值表示法。我们看到，在值表示法中，两个多项式的乘法可以在线性时间内完成。本节课的目标是设计一个算法，能够快速地将系数表示转换为值表示（评估），并在乘法完成后，快速地将值表示转换回系数表示（插值）。

我们的核心算法流程如下：
1.  **评估**：将多项式 `p` 和 `q` 从系数表示转换为在特定点（单位根）上的值表示。
2.  **乘法**：在值表示中，线性时间内计算乘积多项式 `r = p * q` 的值。
3.  **插值**：将乘积多项式 `r` 从值表示转换回系数表示。

我们将使用复数中的**单位根**来实现高效的评估和插值。

## 单位根的关键性质 🔑

![](img/84e48ca68c32b2f24d02befb6faff7dd_20.png)

单位根是方程 `x^n = 1` 的复数解。对于2的幂次 `m`，其 `m` 次单位根在复平面上均匀分布在单位圆上。

我们只需要记住关于单位根的两个核心事实：
1.  **生成性质**：所有单位根都可以由第一个单位根生成。即，若 `ω_1` 是第一个 `m` 次单位根，则 `ω_k = (ω_1)^k`。
2.  **平方减半性质**：将 `m` 次单位根平方，会得到 `m/2` 次单位根。这个性质是分治算法能够工作的关键。

![](img/84e48ca68c32b2f24d02befb6faff7dd_22.png)

## 快速评估：FFT算法 ⚡

我们的目标是：给定一个 `m-1` 次多项式 `p(x)`（`m` 是2的幂），快速计算它在所有 `m` 次单位根 `ω_0, ω_1, ..., ω_{m-1}` 上的值。

### 分治策略

FFT采用分而治之的策略：
*   **分**：将原问题分解为两个规模减半的子问题。
*   **治**：递归解决子问题。
*   **合**：将子问题的解合并，得到原问题的解。

以下是算法的递归关系，其中 `T(m)` 是处理规模为 `m` 的问题所需的时间：
`T(m) = 2 * T(m/2) + O(m)`
这个递归式的解是 `T(m) = O(m log m)`。

![](img/84e48ca68c32b2f24d02befb6faff7dd_26.png)

### 具体步骤

![](img/84e48ca68c32b2f24d02befb6faff7dd_28.png)

1.  **分解多项式**：
    将多项式 `p(x)` 按奇偶次项拆分为两个多项式：
    *   `p_even(x^2)`：包含所有偶次项（`p0, p2, p4, ...`）。
    *   `p_odd(x^2)`：包含所有奇次项（`p1, p3, p5, ...`）。
    可以表示为：
    `p(x) = p_even(x^2) + x * p_odd(x^2)`
    注意，`p_even` 和 `p_odd` 都是关于 `z = x^2` 的 `(m/2 - 1)` 次多项式。

2.  **递归求解**：
    我们需要计算 `p(x)` 在 `m` 个点 `ω_0, ..., ω_{m-1}` 上的值。根据平方减半性质，`(ω_k)^2` 对应的是 `m/2` 次单位根。
    因此，我们只需递归地计算 `p_even(z)` 和 `p_odd(z)` 在所有 `m/2` 次单位根上的值。

3.  **合并结果**：
    对于每个 `k`（`0 ≤ k < m`），利用递归结果合并：
    `p(ω_k) = p_even((ω_k)^2) + ω_k * p_odd((ω_k)^2)`
    由于 `(ω_k)^2` 是 `m/2` 次单位根，`p_even` 和 `p_odd` 在该点的值已经由递归步骤算出。因此，合并每个点只需常数时间，总共 `O(m)` 时间。

![](img/84e48ca68c32b2f24d02befb6faff7dd_30.png)

### 示例

![](img/84e48ca68c32b2f24d02befb6faff7dd_32.png)

![](img/84e48ca68c32b2f24d02befb6faff7dd_34.png)

考虑多项式 `p(x) = 1 + 3x - 4x^2 + 7x^3`，在4次单位根 `{1, -1, i, -i}` 上求值。
*   偶部：`p_even(z) = 1 - 4z`
*   奇部：`p_odd(z) = 3 + 7z`
*   递归计算 `p_even` 和 `p_odd` 在2次单位根 `{1, -1}` 上的值。
*   合并：
    *   `p(1) = p_even(1) + 1 * p_odd(1)`
    *   `p(-1) = p_even(1) + (-1) * p_odd(1)`
    *   `p(i) = p_even(-1) + i * p_odd(-1)`
    *   `p(-i) = p_even(-1) + (-i) * p_odd(-1)`

可以看到，计算 `p(1)` 和 `p(-1)` 时共享了 `p_even(1)` 和 `p_odd(1)` 的计算结果，这正是分治算法提高效率的原因。

![](img/84e48ca68c32b2f24d02befb6faff7dd_36.png)

## 快速插值：逆FFT算法 🔄

评估步骤将系数表示 `[p0, p1, ..., p_{m-1}]` 转换为了值表示 `[p(ω_0), p(ω_1), ..., p(ω_{m-1})]`。逆过程（插值）同样可以在 `O(m log m)` 时间内完成。

![](img/84e48ca68c32b2f24d02befb6faff7dd_38.png)

令人惊讶的是，**逆FFT（IFFT）可以通过对FFT算法进行微小修改来实现**。具体而言，给定值表示，要恢复系数表示，存在以下公式：

![](img/84e48ca68c32b2f24d02befb6faff7dd_40.png)

对于 `j` 从 `0` 到 `m-1`：
`p_j = (1/m) * Σ_{k=0}^{m-1} [ p(ω_k) * ω^{-j*k} ]`

![](img/84e48ca68c32b2f24d02befb6faff7dd_42.png)

![](img/84e48ca68c32b2f24d02befb6faff7dd_44.png)

观察这个公式，它相当于：
1.  将值向量 `[p(ω_0), ..., p(ω_{m-1})]` 视为某个新多项式 `q(x)` 的系数。
2.  计算 `q(x)` 在所有点 `ω^{-0}, ω^{-1}, ..., ω^{-(m-1)}`（它们也是单位根）上的值。
3.  将结果除以 `m`。

而**计算多项式在所有单位根上的值，正是FFT所做的事情**。因此，只需对FFT算法稍作调整（将单位根替换为其倒数，并在最后除以 `m`），就可以得到逆FFT算法。

## 矩阵视角 🧮

![](img/84e48ca68c32b2f24d02befb6faff7dd_46.png)

![](img/84e48ca68c32b2f24d02befb6faff7dd_48.png)

![](img/84e48ca68c32b2f24d02befb6faff7dd_50.png)

FFT和IFFT可以从矩阵乘法的角度来理解。

![](img/84e48ca68c32b2f24d02befb6faff7dd_52.png)

![](img/84e48ca68c32b2f24d02befb6faff7dd_54.png)

![](img/84e48ca68c32b2f24d02befb6faff7dd_56.png)

*   **FFT即矩阵乘法**：评估步骤 `[p(ω_0), ..., p(ω_{m-1})]^T` 可以看作是一个特定矩阵 `F`（傅里叶矩阵）乘以系数向量 `[p0, ..., p_{m-1}]^T`。
    `F` 的第 `j` 行第 `k` 列元素为 `ω^{j*k}`。
    朴素矩阵乘法需要 `O(m^2)` 时间，但FFT算法在 `O(m log m)` 时间内隐式地完成了 `F * p` 的计算。

*   **IFFT即逆矩阵乘法**：插值步骤是FFT的逆过程，对应于乘以傅里叶矩阵的逆矩阵 `F^{-1}`。
    `F^{-1}` 与 `F` 形式相似，只是元素变为 `ω^{-j*k}` 并多了 `1/m` 的缩放因子。
    同样，IFFT算法在 `O(m log m)` 时间内完成了 `F^{-1} * (值向量)` 的计算。

这种视角揭示了FFT的强大之处：它能够极其快速地与一个结构特殊的稠密矩阵进行乘法运算。

## 应用：互相关 🤝

![](img/84e48ca68c32b2f24d02befb6faff7dd_64.png)

多项式乘法有一个直接而重要的应用：计算两个序列的**互相关**。

给定两个数组 `a = [a0, a1, ..., a_{n-1}]` 和 `b = [b0, b1, ..., b_{n-1}]`，它们的互相关序列 `c` 定义为：
对于每个偏移量 `k`，`c_k = Σ_i a_i * b_{i+k}`（下标可能需要进行边界处理）。

![](img/84e48ca68c32b2f24d02befb6faff7dd_66.png)

**这正好对应于多项式乘积的系数**！如果将数组 `a` 和反转的数组 `b` 视为多项式的系数，那么它们的乘积多项式的第 `k` 项系数，就包含了所有满足 `i + j = k` 的 `a_i * b_j` 之和，这与互相关的计算模式一致。

![](img/84e48ca68c32b2f24d02befb6faff7dd_68.png)

因此，我们可以：
1.  用FFT将 `a` 和反转的 `b` 转换到值域。
2.  在值域对应点相乘，得到乘积多项式的值表示。
3.  用IFFT转换回系数表示，即得到互相关序列 `c`。

整个过程时间复杂度为 `O(n log n)`，远优于朴素的 `O(n^2)` 算法。互相关在信号处理、模式匹配等领域有广泛应用。

## 总结 📚

本节课我们一起学习了快速傅里叶变换的核心内容：

![](img/84e48ca68c32b2f24d02befb6faff7dd_76.png)

1.  **FFT算法**：通过巧妙的奇偶分治，利用单位根的平方减半性质，在 `O(n log n)` 时间内将多项式从系数表示转换为在单位根上的值表示。
2.  **IFFT算法**：逆变换几乎与FFT相同，实现了从值表示到系数表示的快速插值。
3.  **完整多项式乘法**：结合FFT（评估）、点乘（`O(n)`）、IFFT（插值）三步，在 `O(n log n)` 时间内完成多项式乘法。
4.  **矩阵理解**：FFT/IFFT本质上是与傅里叶矩阵及其逆矩阵进行快速乘法。
5.  **关键应用**：我们探讨了互相关问题，并展示了如何通过多项式乘法在 `O(n log n)` 时间内高效解决。
