# 4：快速傅里叶变换（第一部分）🎯

在本节课中，我们将要学习多项式乘法。我们将从多项式的基本运算开始，逐步探索如何利用复数和一种称为“值表示”的新方法来显著提升多项式乘法的计算速度。本节课的核心目标是理解快速傅里叶变换（FFT）算法的基本思想。

---

## 概述 📋

多项式是形如 `P(x) = p₀ + p₁x + p₂x² + ... + pₙ₋₁xⁿ⁻¹` 的函数。我们通常用其系数数组 `[p₀, p₁, ..., pₙ₋₁]` 来表示一个多项式，这称为**系数表示**。

本节课我们将重点关注三种多项式运算：
1.  **多项式加法**
2.  **多项式求值**（在给定点 `α` 计算 `P(α)`）
3.  **多项式乘法**

我们将看到，前两种运算可以有非常高效的算法，而多项式乘法在系数表示下的基础算法较慢。为了改进它，我们将引入**值表示**和**复数**的概念，为下一部分学习快速傅里叶变换（FFT）算法打下基础。

---

## 多项式加法 ➕

给定两个具有 `n` 个系数的多项式 `P(x)` 和 `Q(x)`，目标是计算它们的和 `P(x) + Q(x)`。

以下是计算多项式加法的步骤：
1.  对齐两个多项式的同类项（即相同次数的项）。
2.  将对应的系数两两相加。
3.  结果多项式的系数就是这些和。

**算法复杂度**：需要进行 `n` 次系数加法，因此时间复杂度为 **O(n)**。这比整数加法更简单，因为多项式加法没有“进位”操作。

---

## 多项式求值 🔢

给定一个多项式 `P(x)` 和一个实数 `α`，目标是计算 `P(α)` 的值。

### 算法一：朴素方法
直接计算 `P(α) = p₀ + p₁α + p₂α² + ... + pₙ₋₁αⁿ⁻¹`。计算每个 `αⁱ` 需要 `i` 次乘法，总乘法次数约为 `1+2+...+(n-1) = O(n²)`。

### 算法二：霍纳方法（Horner‘s Method）
通过重用中间计算结果来优化。我们按如下方式计算：
```
result = pₙ₋₁
for i from n-2 down to 0:
    result = result * α + pᵢ
```
这种方法只需要 `n` 次乘法和 `n` 次加法，因此时间复杂度为 **O(n)**。

---

## 多项式乘法 ✖️

给定两个具有 `n` 个系数的多项式 `P(x)` 和 `Q(x)`，目标是计算它们的乘积 `P(x) * Q(x)`。乘积多项式的次数最高为 `2n-2`。

### 基础算法
模仿手动乘法：将 `P(x)` 的每一项与 `Q(x)` 的每一项相乘，然后将结果合并同类项。

**算法复杂度**：需要进行约 `n * n = n²` 次系数乘法和加法，因此时间复杂度为 **O(n²)**。

上一节我们介绍了多项式乘法的朴素算法，本节中我们来看看如何利用一种新的多项式表示法来寻求更快的算法。

---

## 多项式的值表示 💡

一个 `n-1` 次多项式不仅由其 `n` 个系数唯一确定，也可以由其在 `n` 个不同点上的取值唯一确定。这种表示方式称为**值表示**。

例如，对于一次多项式（直线），只需两个点即可确定；对于二次多项式（抛物线），则需要三个点。

假设我们选择一组点 `{α₁, α₂, ..., αₘ}`，其中 `m ≥ n`。那么向量 `[P(α₁), P(α₂), ..., P(αₘ)]` 就是多项式 `P(x)` 的一个值表示。

---

## 值表示下的运算 ⚡

在值表示下，多项式的加法和乘法变得异常简单：

*   **加法**：若已知 `P(x)` 和 `Q(x)` 在相同点集上的值，则它们和的值表示就是对应点上的值相加：`[P(α₁)+Q(α₁), ..., P(αₘ)+Q(αₘ)]`。
*   **乘法**：同理，它们积的值表示就是对应点上的值相乘：`[P(α₁)*Q(α₁), ..., P(αₘ)*Q(αₘ)]`。

**算法复杂度**：这两种运算都只需要 `O(m)` 时间，当 `m` 与 `n` 同阶时，就是 **O(n)** 时间。这比系数表示下的乘法快得多！

这启发我们设计一个快速多项式乘法算法：
1.  **求值**：将输入多项式从系数表示转换为值表示。
2.  **逐点相乘**：在值表示下进行 `O(n)` 的乘法。
3.  **插值**：将结果从值表示转换回系数表示。

目前，步骤1（求值）和步骤3（插值）如果用朴素方法（对每个点进行 `O(n)` 的求值）需要 `O(n²)` 时间，抵消了优势。因此，关键在于如何快速地进行求值和插值。

---

## 复数的引入 🔮

为了加速求值和插值步骤，我们需要巧妙地选择用于求值的那组点 `{α₁, ..., αₘ}`。这个选择将引入**复数**。

复数形如 `a + bi`，其中 `i` 是虚数单位，满足 `i² = -1`。复数可以在复平面上表示，其乘法对应于幅角的相加。

### 单位根
我们特别关注一类复数，称为 **单位根**。`n` 次单位根是满足 `ωⁿ = 1` 的复数 `ω`。

在复平面上，`n` 次单位根均匀分布在单位圆上。例如：
*   `2` 次单位根：`+1`, `-1`
*   `4` 次单位根：`+1`, `-1`, `+i`, `-i`
*   `8` 次单位根：有 `8` 个，均匀分布在圆周上。

我们记 `ωₙ = e^(2πi/n)` 为**主 `n` 次单位根**，那么所有 `n` 次单位根可以表示为：`ωₙ⁰ = 1, ωₙ¹, ωₙ², ..., ωₙⁿ⁻¹`。

### 单位根的关键性质
单位根有两个对我们至关重要的性质：

1.  **生成性质**：所有单位根都可以由主单位根的幂次生成：`ωₙᵏ = (ωₙ¹)ᵏ`。
2.  **折半性质**：将 `n` 次单位根（`n` 为偶数）平方后，会得到 `n/2` 次单位根的集合。例如，`4` 次单位根 `{1, i, -1, -i}` 平方后得到 `{1, -1}`，这正是 `2` 次单位根。这个性质是分治策略能够应用的核心。

---

## 算法蓝图与展望 🗺️

现在，我们有了设计快速多项式乘法算法的所有要素：

1.  **选择点集**：选择 `m` 个点进行求值，其中 `m` 是大于 `2n` 的 `2` 的幂次（例如 `m=2^k`）。我们选择的点就是 `m` 次单位根：`ωₘ⁰, ωₘ¹, ..., ωₘᵐ⁻¹`。
2.  **快速求值**：利用单位根的折半性质，可以设计一个分治算法，在 `O(m log m)` 时间内将多项式从系数表示转换为在这些单位根上的值表示。这个算法就是**快速傅里叶变换（FFT）**。
3.  **快速插值**：有趣的是，从值表示（在单位根上）转换回系数表示的过程，与FFT几乎相同，也可以在 `O(m log m)` 时间内完成。
4.  **逐点相乘**：在值表示下进行 `O(m)` 的乘法。

由于 `m` 是 `O(n)` 的，整个算法的时间复杂度为 **O(n log n)**，远快于朴素的 `O(n²)` 算法。

---

## 总结 📚

本节课中我们一起学习了：
*   多项式运算（加法、求值、乘法）的基础算法及其复杂度。
*   多项式除了系数表示，还有**值表示**。在值表示下，加法和乘法是线性的。
*   为了加速多项式乘法，我们计划采用“系数→值表示→相乘→系数”的路径，其瓶颈在于求值和插值。
*   引入**复数**和**单位根**，它们具有关键的**折半性质**，为下一课实现 `O(n log n)` 的快速傅里叶变换（FFT）算法提供了可能。
