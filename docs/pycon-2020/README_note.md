# Python 入门：01：理解变量与数据类型

在本节课中，我们将要学习 Python 编程中最基础的两个概念：变量和数据类型。它们是构建所有程序的基石。

## 什么是变量？

变量是计算机内存中用于存储数据的“盒子”或“标签”。你可以把数据放进这个盒子里，并通过标签名称来使用它。

在 Python 中，创建一个变量并赋值非常简单，使用等号 `=` 即可。

```python
name = "Alice"
age = 25
```

上面的代码创建了两个变量：`name` 存储了字符串 “Alice”，`age` 存储了数字 25。

## 基本数据类型

Python 有多种内置的数据类型，用于表示不同种类的数据。理解它们至关重要。

以下是几种最常用的基本数据类型：

*   **整数 (int)**：表示没有小数部分的数字，如 `10`, `-5`, `0`。
*   **浮点数 (float)**：表示带有小数点的数字，如 `3.14`, `2.0`, `-0.5`。
*   **字符串 (str)**：表示文本，需要用单引号 `‘ ’` 或双引号 `“ ”` 括起来，如 `“Hello”`, `‘Python’`。
*   **布尔值 (bool)**：表示逻辑值，只有两个：`True` (真) 和 `False` (假)。

上一节我们介绍了如何创建变量，本节中我们来看看如何检查一个变量的数据类型。

你可以使用 `type()` 函数来查看任何变量或值的数据类型。

```python
height = 1.75
is_student = True
print(type(name))   # 输出：<class ‘str‘>
print(type(age))    # 输出：<class ‘int‘>
print(type(height)) # 输出：<class ‘float‘>
print(type(is_student)) # 输出：<class ‘bool‘>
```

## 变量的命名规则

为了写出清晰易懂的代码，为变量起一个好名字很重要。Python 有一些必须遵守的规则和推荐的惯例。

以下是变量命名的主要规则和惯例：

*   **规则**：变量名只能包含字母、数字和下划线 `_`，且不能以数字开头。
*   **规则**：变量名区分大小写（`age` 和 `Age` 是两个不同的变量）。
*   **惯例**：使用描述性的名称，例如用 `user_name` 而不是 `n`。
*   **惯例**：对于多个单词组成的变量名，通常使用小写字母和下划线，如 `first_name`。这种风格称为“蛇形命名法”。

## 一个简单的例子

现在，让我们把以上概念组合起来，写一个简单的程序。

```python
# 定义变量
product_name = “笔记本电脑”
price = 5999.99
quantity = 2
in_stock = True

# 进行计算
total_cost = price * quantity

# 打印信息
print(“产品：”, product_name)
print(“单价：”, price)
print(“数量：”, quantity)
print(“库存状态：”, in_stock)
print(“总价：”, total_cost)
```

运行这段代码，你会看到它输出了产品的相关信息并计算了总价。这展示了变量如何存储信息以及不同类型的数据如何被使用。

本节课中我们一起学习了 Python 的变量和基本数据类型。你学会了如何创建变量、了解了几种关键的数据类型（整数、浮点数、字符串、布尔值），并掌握了变量的命名规范。记住，变量是存储数据的容器，而数据类型定义了容器中数据的种类和可执行的操作。这是你 Python 编程之旅的第一步，扎实掌握这些基础将使后续的学习更加轻松。
