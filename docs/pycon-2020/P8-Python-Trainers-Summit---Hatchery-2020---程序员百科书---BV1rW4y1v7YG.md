他们基本上会考虑一个企业想要做什么。企业想要赚更多，花更少，培训可以帮助他们做到这一点。所以培训会帮助他们解决新问题，即赚更多。它会帮助他们更快地解决旧问题，即花更少。它甚至会鼓励员工留下来。

员工留任的成本非常高。如果他们能够提供培训，使员工了解最新技术或有用技术，员工就更愿意留下来。他们会将其视为一种不错的福利。事实上，我几年前在一家公司时，人力资源部的负责人在我开始课程之前发言，她说：“你们应该知道，每个人。”

“我们知道在接下来的几年里你们会离开我们，因为这在高科技行业中是常有的事。但我们希望这门课程能帮助你在这里待得更久，因为你会欣赏我们所做的事情，并且在你离开我们之后，这也会对你的职业发展有所帮助。”太棒了。因此，公司确实希望培训能帮助他们。

他们并不是出于善心在做这件事。他们这样做是因为他们想要帮助自己。这很好。培训经理，这是我对任何负责培训项目的人或至少是我的联络人的称呼。实际上没有公司称其为培训经理。这是我自己的术语。

他们可能称呼为首席学习官，或者在人才管理部门或人才发展部门工作的人员，这个词现在非常常见。人力资源。每家公司都有略微不同的术语和不同的结构。

但归根结底，他们是投资经理。他们被给了一笔资金，并被告知：“通过培训使我们的公司赚得更多，花得更少。”他们会根据这些课程来衡量你的成功。因此，培训经理总是四处寻找，“我在哪里可以找到一个课程，能够……”

“这将改善我公司的效率，让我们赚得更多，花得更少？”他们有自己的一组交易员在合作。但当然，我试图谈论的是你如何突破这一点或使用不同的技巧来做到这一点。培训经理在培训业务中就是你的客户。
