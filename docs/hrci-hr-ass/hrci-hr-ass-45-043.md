# HRCI人力资源助理课程：4-5：行为绩效评估方法 🧑‍💼

![](img/e3f4b84dbdeeb5f981ec3104a96f7e1f_0.png)

在本节课中，我们将学习绩效评估的一种核心方法——行为评估法。绩效评估，也称为绩效审查，是管理者与员工在固定时间会面，回顾员工在一段时间内工作表现的过程。评估数据可作为未来人事决策的依据。绩效评估共有四种方法，本节我们将重点介绍第一种：行为评估法。

行为评估法根据员工与工作相关的行为表现来对其进行评判。最常见的行为评估方法是**BARS方法**。

## 什么是BARS方法？ 📊

BARS是“行为锚定等级评价法”的缩写。传统的评级量表通常包含从1到5的数字评分。而BARS方法则不同，它首先分析员工的职位描述，以识别与该职位相关的具体任务。

以下是BARS方法的核心步骤：
1.  **分析职位**：明确与特定职位相关的关键任务和行为。
2.  **制定行为描述**：针对每项任务，创建一系列描述员工行为表现的具体陈述。
3.  **锚定评分**：每条行为陈述都对应一个该任务项下的具体数值评分。

## BARS方法应用实例 🍕

为了帮助理解，我们来看一个例子。假设杰伊是“薄片披萨店”的一名披萨厨师，她与老板山姆进行行为绩效评估面谈。

杰伊在以下任务类别上接受评估：沟通能力、可靠性、责任感、职业素养和**工作知识**。

以“工作知识”类别为例，其中一项评估标准是：**展现对食品安全准则的掌握**。

在这种情况下，BARS量表可能如下所示：
*   **1分（从不）**：从未遵守食品安全准则。
*   **2分（几乎不）**：几乎不遵守食品安全准则。
*   **3分（有时）**：有时遵守食品安全准则。
*   **4分（通常）**：通常遵守食品安全准则。
*   **5分（总是）**：总是遵守食品安全准则。

在面谈中，山姆回顾了过去三个月杰伊在BARS量表上的得分。由于杰伊在食品安全和清洁方面持续表现出熟练度，她因此在“工作知识”类别中获得了**5分（总是）**。

## 本节总结 ✨

本节课我们一起学习了行为绩效评估中的BARS方法。BARS是衡量员工工作相关行为最常见的方法，它使用一个锚定了具体行为描述的评级量表，来评估员工完成其职位相关特定任务的表现。
