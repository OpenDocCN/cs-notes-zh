## 🧾 薪资（Salary）的定义与特点  

首先，我们来看薪资的定义。  

**薪资（Salary）**是指按照固定标准定期支付的报酬，通常以年度金额表示。  

其核心特征可以用公式表示为：  

```
年度薪资 = 每年固定支付金额
```

薪资通常：  

- 以“年收入”形式表达  
- 按月或每两周发放  
- 常见于专业人员和白领岗位  

薪资的本质是**固定周期、固定金额的稳定收入模式**。  



---

## ⏱️ 工资（Wages）的定义与特点  

![](img/a9933adc013ebeeedbcf33e98d7fe8f8_2.png)

在了解薪资之后，我们来看另一种常见的报酬方式——工资。  

**工资（Wages）**是指基于工作时间或工作天数支付的报酬。  

其计算方式通常为：  

```
工资收入 = 每小时工资 × 工作小时数
```

或  

```
工资收入 = 每日工资 × 工作天数
```

工资通常：  

- 以“每小时金额”或“每日金额”表示  
- 根据实际工作时间计算  
- 常见于体力劳动或非技术性岗位  

工资的核心特征是**按工作时间计酬，收入浮动性较高**。  



---

## ➕ 差额薪酬（Differential Pay）的定义与应用  

在理解了基本薪资和工资之后，我们进一步了解差额薪酬。  

**差额薪酬（Differential Pay）**是指在基本工资或薪资之外额外支付的金额。  

其表达方式可以表示为：  

```
总收入 = 基础薪酬 + 差额薪酬
```

差额薪酬通常适用于以下情况：  

- 激励员工接受特定工作任务  
- 鼓励员工接受特定班次（如夜班）  
- 补偿特殊工作环境或岗位情况  

需要注意的是：  

- 除加班费外  
- 差额薪酬通常不受联邦法律强制规定  

差额薪酬的本质是**在特定情境下的额外激励或补偿机制**。  



---

## 🔍 三种薪酬方式的对比  

到目前为止，我们已经介绍了三种基本的薪酬形式。  

以下是三种方式的核心区别：  

- **薪资（Salary）**：固定年度金额，定期支付  
- **工资（Wages）**：按小时或天数计算，随工作时间变化  
- **差额薪酬（Differential Pay）**：在基础薪酬之上的额外支付  

这三种方式构成了薪酬体系的基础结构。
