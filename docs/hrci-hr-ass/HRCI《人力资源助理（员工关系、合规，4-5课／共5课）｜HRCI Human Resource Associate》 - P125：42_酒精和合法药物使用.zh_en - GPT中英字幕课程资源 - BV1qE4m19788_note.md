## 🍷 酒精使用与工作表现  



上一部分我们介绍了本节课的整体框架，本节我们首先来看**酒精在职场中的管理原则**。  



在本视频中，我们讨论工作场所中的酒精和合法药物使用问题。  

组织必须谨慎处理这些问题，以确保员工的身心健康与生产效率。  



通常情况下，只要员工饮酒**不影响工作表现**，雇主一般不会干预。  

但如果员工饮酒行为影响其工作表现，雇主有权采取行动。  



需要特别注意的是，联邦法律，例如 **《entity["legislation","Americans with Disabilities Act","us federal law 1990"]》**，保护患有酒精依赖症的员工不因其疾病而受到歧视。  



![](img/05c6266c698f287c238962e5f10142a4_2.png)

核心原则可以概括为：  



```
若 工作表现 = 合格
→ 不干预

若 工作表现 ≠ 合格
→ 可采取纪律措施
```



雇主不得仅因员工患有酒精依赖症而做出雇佣决定。  

但是，如果员工未达到与同事相同的绩效标准，雇主可以采取适当的纪律处分，包括解雇。

## 💊 合法药物使用与工作影响  



上一节我们讨论了酒精问题，接下来我们来看**合法药物的使用管理**。  



许多员工会使用处方药或非处方药，例如：  



以下是常见的合法药物类型：  



- 安眠药  
- 感冒药  
- 止痛药  



一般情况下，只要这些合法药物**不影响员工的安全或工作表现**，雇主通常不会干预。  



但某些药物可能产生副作用，例如：  



以下是可能影响工作的副作用：  



![](img/05c6266c698f287c238962e5f10142a4_4.png)

- 嗜睡  
- 认知能力下降  



这些副作用可能妨碍员工履行工作职责。  



核心判断逻辑如下：  



```
若 药物使用 → 不影响安全 & 不影响绩效
→ 不干预

若 药物使用 → 影响安全 或 影响绩效
→ 可管理或干预
```



![](img/05c6266c698f287c238962e5f10142a4_6.png)

## ⚖️ 残疾法律与合理便利  



在了解合法药物后，我们进一步探讨**残疾法律对雇主的限制**。  



州和联邦层面的残疾法律，可能限制雇主在处理药物相关问题时的选择。  

如果员工符合“合法残疾”的认定标准，雇主可能必须提供合理便利。  



其核心逻辑为：  



```
若 员工 = 合法残疾身份
→ 雇主需提供合理便利（Reasonable Accommodation）
```



这意味着在某些情况下，雇主不能简单禁止药物使用，而必须依法处理。

## 🌿 大麻与药物检测政策  



![](img/05c6266c698f287c238962e5f10142a4_8.png)

在理解法律限制后，我们来看**大麻与职场药物检测**。  



大麻是当今工作场所中最常见的违禁药物。  



许多雇主使用“五项检测”进行筛查。  



以下是五项检测通常包含的物质：  



- 大麻  
- 可卡因  
- 苯环利定（PCP）  
- 安非他命  
- 阿片类药物  



雇主可能在入职前或在职期间进行药物检测，以确保员工遵守工作场所规定。  



为了保证检测结果可靠，雇主可能会建立**证据保全链（Chain of Custody）**，用于记录样本从采集到检测全过程中的责任人。  



其流程可表示为：  



```
样本采集 → 记录责任人 → 运输 → 实验室检测 → 结果确认
```

## 🏛️ 联邦法律对药物政策的影响  



在了解检测机制后，我们进一步探讨**联邦法律对职场药物政策的影响**。  



某些岗位根据联邦法律必须制定特殊政策，目的是维护无毒工作环境。  

特别是涉及以下领域的岗位：  



- 员工安全  
- 他人安全  
- 数据安全  



以下联邦法律对企业政策产生重要影响：  



![](img/05c6266c698f287c238962e5f10142a4_10.png)

- **《entity["legislation","Controlled Substances Act","us federal law 1970"]》**  
- **《entity["legislation","Drug Free Workplace Act","us federal law 1988"]》**  



此外，理解各州关于大麻的法律至关重要。  

部分州禁止某些入职前大麻检测，或对阳性结果的解释有特殊规定。  



如果州法律缺失，雇主通常遵循联邦指导原则。  

部分组织还可能允许员工在工作时间以外合法使用医用大麻。



---



## 🪪 医用大麻的合规要求  



在州法律允许的前提下，企业需要确保工作环境不受影响。  



员工若申请使用医用大麻，必须提供有效的医用大麻卡。  

该卡证明其出于医疗目的的合法使用。  



这些卡通常由州公共卫生部门或人力资源部门签发，并需每年更新。  



员工还必须：  



- 提供证据证明大麻是最合适的治疗方式  
- 确认大麻使用不会影响工作  
- 确保不危及安全工作环境  



![](img/05c6266c698f287c238962e5f10142a4_12.png)

核心原则可以表达为：  



```
合法资格 + 不影响工作 + 不危害安全
= 可能获得批准
```
