# HRCI《人力资源助理（员工关系、合规）》：第4-5课：调查性骚扰 🔍



## 📘 课程概述  

![](img/5fa0ab57058376d4c8e7e96c95e058da_0.png)

在本节课中，我们将学习**当性骚扰事件发生时，如何进行调查**。  

你已经学习了如何预防性骚扰。本节将进一步讲解：一旦组织内出现性骚扰指控，应如何按照规范步骤进行调查与处理。  

我们将系统梳理调查流程、关键问题以及可能的处理结果，帮助你掌握完整的操作框架。



---

## ⚖️ 一、组织的法律责任与调查必要性  

上一节我们学习了如何预防性骚扰。本节首先明确组织在法律上的责任。  

![](img/5fa0ab57058376d4c8e7e96c95e058da_2.png)

当组织**知情却未立即采取纠正措施**时，将对性骚扰事件承担法律责任。  

其核心逻辑可以表示为：  

```text
组织责任 = 已知性骚扰情况 + 未及时采取纠正措施
```

即：

```text
Liability = Awareness + No Immediate Corrective Action
```

如果组织已经制定了防止性骚扰的政策，但事件仍然发生，那么相关投诉必须被调查并妥善解决。



---

## 🧭 二、性骚扰调查的六个步骤  

明确了责任之后，我们来看完整的调查流程。  

经理、主管或HR专业人员应遵循以下**六个步骤**：  

### 调查流程总览

```text
1. 接收投诉
2. 与投诉员工讨论
3. 面谈被指控员工
4. 访谈证人
5. 审查证据并判断
6. 做出处理决定
```



---

## 📥 三、步骤一：接收投诉  

首先是接收骚扰指控。  

当HR经理得知情况后，应立即开始调查，以判断投诉是否具有可信度。  

![](img/5fa0ab57058376d4c8e7e96c95e058da_4.png)

核心目标是：

```text
确定投诉是否具有初步可信性
```

接下来，HR需要与投诉员工进行详细讨论。



---

## 🗣 四、步骤二：与投诉员工面谈  

在启动调查后，HR经理应向员工提出一系列问题，并记录其陈述。  

以下是需要了解的关键信息：  

- 员工与被指控人之间是否存在关系  
- 哪些员工可能目睹或听闻相关行为  
- 是否有其他员工曾遭遇类似情况  
- 每起事件发生的时间与地点  
- 如果存在延迟报告，原因是什么  
- 员工认为雇主可以如何纠正问题  

可以用结构化记录方式表示：  

```text
记录内容 = 关系 + 证人 + 类似经历者 + 时间地点 + 延迟原因 + 解决建议
```



---

## 👤 五、步骤三：面谈被指控员工  

![](img/5fa0ab57058376d4c8e7e96c95e058da_6.png)

完成对投诉人的访谈后，下一步是面谈被指控员工。  

HR经理应向其提出一系列问题，并**完整、准确地记录回答**。  

同时，被指控员工应被要求：  

```text
不得与组织内其他员工讨论该事件
```

这样可以防止串供或影响调查公正性。



---

## 👥 六、步骤四：访谈证人  

在双方当事人陈述完成后，HR经理应访谈可能目睹事件的其他员工。  

在访谈过程中必须：  

```text
完整记录证人陈述
```

记录是调查有效性的核心依据。



---

## 📑 七、步骤五：审查证据并做出判断  

完成信息收集后，HR经理需要审查全部文件资料，以判断投诉是否成立。  

调查结果可能包括以下三种情况：  

- 投诉有效（确实发生性骚扰）  
- 投诉无效（未发生性骚扰）  
- 证据不足（调查结论不明确）  

判断逻辑可表示为：  

```text
结论 ∈ {成立, 不成立, 证据不足}
```

需要注意的是，不同的人可能对同一事件有不同的感知，因此在性骚扰案件中，确定客观事实往往具有一定难度。



---

## ⚖️ 八、步骤六：确定处理方案  

在得出结论后，最后一步是确定合适的解决方案。  

管理层将决定是否采取纪律处分措施。  

可能的处分方式包括：  

- 口头警告  
- 降职  
- 停职  
- 解雇  

可以表示为：  

```text
纪律处分 ∈ {警告, 降职, 停职, 解雇}
```

如果投诉员工对结果不满意，可以：  

- 向平等就业机会委员会（EEOC）提出投诉  
- 提起诉讼  

此外，必须强调：  

```text
报复行为 = 违法
```

对提出投诉的人进行报复，或对协助投诉或提供信息的人进行报复，均属于违法行为。

掌握规范流程不仅可以降低法律风险，也有助于维护公平的工作环境。
