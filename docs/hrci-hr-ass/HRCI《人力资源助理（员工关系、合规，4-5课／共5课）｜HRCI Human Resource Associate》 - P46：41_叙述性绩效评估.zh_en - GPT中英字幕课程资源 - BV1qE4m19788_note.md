# HRCI人力资源助理课程：4：叙述性绩效评估方法 📝

![](img/7460f70ccf1b3b8de8c943fc6f54b5a8_2.png)

在本节课中，我们将要学习绩效评估的最后一种方法——叙述性方法。之前我们已经介绍了行为法、比较法和评级法，本节我们将重点探讨叙述性方法的具体形式与应用。

叙述性方法要求管理者或人力资源代表撰写一份关于员工绩效的书面评估报告。这种方法侧重于通过文字描述来记录和评价员工的表现。

以下是三种主要的叙述性评估类型：

*   **关键事件评估**：管理者记录员工在特定事件中表现优异或欠佳的具体事例。这些事件将构成书面叙述的基础，并在绩效评估期间与员工进行讨论。
    *   **示例**：在Slicelice餐厅，厨师Blake最近被观察到在班前未洗手、披散头发且备餐时不戴手套。其经理Riley为此撰写了一份关键事件评估，并在会议中告知Blake，他需要重新参加公司的食品安全培训。若再发生类似事件，Blake将被解雇。

*   **短文评估**：管理者撰写一篇短文，描述员工在整个评估期内的表现。这种格式使管理者更容易识别出需要改进的领域。
    *   **示例**：Connective公司的经理们在年度绩效评估中使用此方法。他们利用非正式评估的数据为每位员工撰写短文评估，并在评估期间与员工分享。

![](img/7460f70ccf1b3b8de8c943fc6f54b5a8_6.png)

*   **实地评估**：由非员工直属经理或主管的人员进行评估。通常由组织或部门内的人力资源专员，或来自组织或部门外部的人员执行。
    *   **示例**：在Connective公司提拔某位员工至管理岗位前，该员工需要与一位人力资源代表进行一次实地评估。

本节课中我们一起学习了绩效评估的三种叙述性方法：关键事件评估、短文评估和实地评估。在你的HR职业生涯中，很可能会遇到其中的每一种方法。
