# HRCI《人力资源助理（员工关系、合规）》：第4-5课：叙述性绩效评估 📝  



在本节课中，我们将学习绩效评估的最后一种方法——**叙述性绩效评估方法**。  

![](img/7460f70ccf1b3b8de8c943fc6f54b5a8_0.png)

此前，我们已经学习了**行为法**、**比较法**和**评分法**。在本节中，我们将系统讲解叙述法的概念、类型及应用场景。  



## 一、什么是叙述性绩效评估 ✍️  



在介绍具体类型之前，我们先了解叙述法的基本定义。  

**叙述法（Narrative Method）**要求管理者或人力资源代表为每位员工撰写一份关于其绩效表现的书面评价。  



其核心表达形式可以概括为：  



![](img/7460f70ccf1b3b8de8c943fc6f54b5a8_2.png)

```text
员工绩效评估 = 管理者撰写的书面描述
```



这种方法强调文字说明，而非打分或排名。  



叙述性评估主要包括三种类型。以下是三种常见形式：  



- 关键事件评估（Critical Incident Review）  
- 论文式评估（Essay Review）  
- 现场评估（Field Review）  



接下来，我们分别进行讲解。  



## 二、关键事件评估 🧾  



首先，我们来看第一种形式——关键事件评估。  



关键事件评估是指管理者记录员工在工作中表现出**积极或消极行为的具体事件**。  



其基本逻辑可以表示为：  



```text
记录关键事件 → 整理成书面叙述 → 与员工面谈讨论
```



这些记录将成为绩效面谈时讨论的依据。  



### 示例说明  



例如，Blake 是一家餐厅的后厨员工。最近被观察到：  



- 上班前未洗手  
- 工作时头发未扎起  
- 准备食物时未佩戴手套  



Blake 的经理 Riley 为其撰写了一份关键事件评估报告。  

在会议中，经理告知 Blake 需要重新参加组织的食品安全培训。  

如果类似事件再次发生，Blake 的劳动关系将被终止。  



这个例子说明，关键事件评估通常用于记录**具体行为**，并与纪律管理直接相关。  



## 三、论文式评估 📄  



了解了关键事件评估之后，我们来看第二种形式——论文式评估。  



论文式评估是指管理者在评估周期结束时，为员工撰写一篇简短的文字总结，描述员工在整个评估期间的表现。  



其核心结构可以表示为：  



```text
评估周期整体表现 = 数据汇总 + 综合文字描述
```



这种方法有助于管理者更容易识别需要改进的领域。  



### 应用示例  



例如，某公司使用论文式评估进行年度绩效评估。  

管理者利用平时的非正式评估数据，整理形成每位员工的论文式评价。  

在正式绩效面谈时，将该评价分享给员工。  



这种方式适用于周期性总结性评估，例如年度评估。  



## 四、现场评估 🏢  



在了解前两种由直属经理主导的方式后，我们来看第三种形式——现场评估。  



现场评估是由**非员工直属主管**进行的评估。  



通常由以下人员执行：  



- 组织或部门的人力资源专业人员  
- 组织或部门外部人员  



其核心特征可以表示为：  



![](img/7460f70ccf1b3b8de8c943fc6f54b5a8_4.png)

```text
评估人 ≠ 员工直属经理
```



这种方法通常用于管理层晋升评估。  



### 示例说明  



![](img/7460f70ccf1b3b8de8c943fc6f54b5a8_6.png)

例如，在员工晋升为管理岗位之前，公司安排其与人力资源代表进行现场评估。  

![](img/7460f70ccf1b3b8de8c943fc6f54b5a8_7.png)

通过独立评估，确保晋升决策更加客观公正。  



这种方法有助于提高高层岗位选拔的公平性。  



## 五、本节总结 📚  



在本节课中，我们学习了叙述性绩效评估的三种形式：  



- **关键事件评估**：记录具体正负行为事件  
- **论文式评估**：总结整个评估周期表现  
- **现场评估**：由非直属主管进行独立评估  



我们还了解了各方法的应用场景及逻辑结构。  

![](img/7460f70ccf1b3b8de8c943fc6f54b5a8_9.png)

在未来的人力资源工作中，你很可能会接触并使用这三种叙述性评估方法。
