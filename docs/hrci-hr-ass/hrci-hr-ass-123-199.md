# HRCI《人力资源助理》课程：P199：工伤赔偿处理指南 🩹

![](img/9b3c5adbbcf75387bc075bb2b9f31f06_0.png)

在本节课中，我们将学习如何处理组织中一种特定的索赔类型——工伤赔偿。工伤赔偿是员工因工作受伤或患病时获得的一种保险保障。我们将了解其定义、覆盖范围以及具体的处理流程。

上一节我们介绍了索赔的基本概念，本节中我们来看看一种具体的索赔类型：工伤赔偿。

![](img/9b3c5adbbcf75387bc075bb2b9f31f06_2.png)

**工伤赔偿**是一种为员工提供的保险，用于覆盖因工作场所导致的伤害或疾病。其资金通常来源于州立机构或私人保险公司，旨在为符合条件的员工提供**工资替代**和**医疗福利**。

一个重要的法律要点是：如果员工获得了工伤赔偿，通常意味着他们放弃了因雇主疏忽而起诉组织的权利。

工伤赔偿涵盖一系列费用和活动，其覆盖范围包括：
*   医疗账单。
*   工资补偿。
*   医疗相关费用的报销。
*   对因工死亡员工的家属可能提供的福利。

![](img/9b3c5adbbcf75387bc075bb2b9f31f06_4.png)

处理工伤赔偿索赔时，必须做到准确、及时且尽可能公平。

以下是处理工伤赔偿索赔的具体步骤：

首先，在员工接受医疗专业人员诊治以处理紧急需求后，应记录员工对事件的陈述。事件陈述应包括：
*   伤害类型。
*   身体受伤部位。
*   伤害发生的日期、时间和地点。
*   是否有其他员工涉及。

其次，应记录目击者陈述以获取更多信息。务必收集事件发生前、发生时和发生后时间线的详细信息，以及已提供医疗救治的程度。

最后，应了解适用的相关法律，例如《美国残疾人法案》。

在记录了所有相关信息后，需要向组织的保险公司和州工伤赔偿机构提交索赔。必须遵守所在州关于报告和提交工伤赔偿索赔的截止日期，以避免出现问题。这些截止日期因州而异。

接下来，州机构将决定是否调查该事件。

![](img/9b3c5adbbcf75387bc075bb2b9f31f06_6.png)

或者，组织也可以选择不代员工提交索赔。

![](img/9b3c5adbbcf75387bc075bb2b9f31f06_8.png)

如果组织不提交索赔，员工可以直接向州机构办公室提交。

如果员工的伤害或疾病符合《家庭和医疗休假法》或《美国残疾人法案》的标准，他们可能获得更多福利。人力资源部门的责任是核实员工因其伤害或疾病获得了应得的全部福利。

在处理工伤赔偿索赔过程中，人力资源团队应咨询组织的法律顾问。在整个过程中保持持续沟通，将确保一切操作合法合规。

![](img/9b3c5adbbcf75387bc075bb2b9f31f06_10.png)

![](img/9b3c5adbbcf75387bc075bb2b9f31f06_11.png)

如果没有既定的程序来处理出现的各种情况，工伤赔偿问题可能会迅速升级。

![](img/9b3c5adbbcf75387bc075bb2b9f31f06_13.png)
