# 4：离岸外包与外包

![](img/12096be70a6ff03004790d819d9e6491_0.png)

在本节课中，我们将要学习两种常见的商业策略：外包与离岸外包。作为人力资源专业人士，理解并支持这些策略至关重要。我们将探讨它们的定义、区别以及实际应用案例。

![](img/12096be70a6ff03004790d819d9e6491_2.png)

## 外包：利用外部专业力量

上一节我们介绍了人力资源在组织重组中的作用，本节中我们来看看**外包**。外包是指企业支付费用，让外部实体来执行特定的业务职能。这些职能通常不属于组织的**核心能力**，因此聘请外部实体来维持这些职能是有益的。

**核心能力**是指一个组织视为其成功关键的特定职能。

![](img/12096be70a6ff03004790d819d9e6491_4.png)

以下是外包的一些常见例子：
*   安保服务
*   餐饮服务
*   薪资处理

例如，假设一家名为“Slicelice”的连锁餐厅销售汽水等饮料。他们将饮料机的补货和维护工作外包给第三方供应商。这种安排意味着员工只需保持机器清洁，而无需担心补充饮料或修理机器。

![](img/12096be70a6ff03004790d819d9e6491_6.png)

## 离岸外包：建立海外业务

接下来，人力资源专业人士需要熟悉的另一个商业策略是**离岸外包**。离岸外包是指组织在海外建立子公司，通常将工作从薪酬较高的美国员工转移到美国以外薪酬较低的员工。

![](img/12096be70a6ff03004790d819d9e6491_8.png)

当组织需要处理大规模、劳动密集型的职能时，离岸外包是一个流行的概念。

以下是这些职能最常见的例子：
*   客户服务
*   技术支持
*   计算机编程

例如，Connective 是一家致力于连接企业的全球公司。Connective 将其部分基于电子邮件和聊天的客户服务工作交由海外员工处理。

![](img/12096be70a6ff03004790d819d9e6491_10.png)
