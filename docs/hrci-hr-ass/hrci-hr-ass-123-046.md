# 46：背景调查与记录核查

![](img/07f28b66e0f5db9c5bfb857053c39235_0.png)

在本节课中，我们将学习组织在招聘过程中为何以及如何进行背景调查。背景调查是确保招聘质量、规避法律风险的关键环节。我们将探讨几种常见的调查类型、相关法律规定以及实施时的注意事项。

## 背景调查的必要性

组织需要核实潜在员工通过合理的背景调查，否则可能因**疏忽雇佣**而承担责任。疏忽雇佣是一项法律主张，如果雇主因未进行适当的背景调查而未能预见风险，导致其他员工受到伤害，雇主可能因此被起诉。

上一节我们介绍了背景调查的法律必要性，本节中我们来看看雇主通常会进行的几种具体调查。

## 常见的背景调查类型

根据具体情况，雇主通常会使用以下几种调查。以下是三种主要的调查类型：

1.  **信用调查**：用于发现可能影响候选人履行职责而不造成不当困难的未决问题。
    *   **法律要求**：根据《公平信用报告法》，在获取此类报告前，必须获得申请人的许可。代码示例（获取许可声明）：
        ```
        本人授权 [公司名称] 获取本人的信用报告，用于雇佣背景调查目的。
        ```

2.  **犯罪记录调查**：用于评估候选人是否适合担任保安、儿童看护等特定职位。
    *   **法律要求**：犯罪记录调查属于消费者报告的一种，同样受《公平信用报告法》约束。
    *   **最佳实践**：如果雇主基于犯罪记录决定不录用申请人，必须书面通知候选人并给予其回应时间。

3.  **机动车记录调查**：仅当驾驶是员工主要职责的一部分时，组织才应使用此项检查。

## 调查的潜在风险与注意事项

消费者报告（如信用调查）能提供有价值的信息，但也可能导致无意的歧视性招聘决定。例如，由于系统性的压迫，黑人和拉丁裔人群的信用分数往往低于白人和亚裔人群。因此，仅凭信用调查可能会无意中使某些候选人失去资格。

除了上述报告，还有**调查性消费者报告**。这类报告包含信用调查之外的信息，涉及通过访谈申请人的邻居、朋友和同事来收集其性格信息。与信用调查类似，组织在启动此类报告前也必须获得候选人的许可。

最后需要考虑的一点是，雇主也可能因**疏忽推荐**而被追责。当一个组织被问及候选人的信息时提供了不完整的信息，就会发生这种情况。

![](img/07f28b66e0f5db9c5bfb857053c39235_2.png)

## 提供推荐信息的准则

当被要求提供推荐信息时，经理和人力资源专业人员应提供关于员工在组织任职期间的完整事实信息，但应避免就该员工是否应被录用提出任何建议。

![](img/07f28b66e0f5db9c5bfb857053c39235_4.png)

传达事实和相关信息的行为受到**附条件特权原则**的保护，可免于诽谤诉讼。

## 总结

![](img/07f28b66e0f5db9c5bfb857053c39235_6.png)

本节课中我们一起学习了背景调查在招聘中的重要性。检查与报告是组织寻找合适候选人的关键工具，但实施时必须符合当地法律法规。我们介绍了信用调查、犯罪记录调查和机动车记录调查等常见类型，并强调了获取许可、避免歧视以及提供完整推荐信息等法律和道德准则。接下来，我们将探讨在选拔过程中可用于评估候选人核心任务完成能力的不同测试选项。
