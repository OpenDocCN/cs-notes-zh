# 33：候选人渠道 📊

![](img/bf2f2bd2ea2070b620d81fd9c93d1b62_0.png)

在本节课中，我们将要学习一种主动招聘策略——候选人渠道。我们将了解它的定义、构成要素、优势以及建立渠道时需考虑的关键标准。

## 概述

上一节我们探讨了多种人才获取策略。本节中，我们来看看候选人渠道这一策略。这是一种在职位空缺出现之前，就预先识别和储备合适候选人的方法，旨在加速招聘团队的选拔与聘用流程。

![](img/bf2f2bd2ea2070b620d81fd9c93d1b62_2.png)

## 什么是候选人渠道？🤔

候选人渠道，也称为人才管道，是一份经过初步筛选、且可能对您组织空缺职位感兴趣的候选人名单。招聘团队利用这份名单来接触和重新接触被动候选人，协调多元化计划，并创建更高效的招聘流程。这种方法让您能专注于组织和员工的长期成功，而非临时寻找潜在候选人。

## 候选人渠道的构成要素

候选人渠道中的人员构成因公司、其需求和人才寻源团队而异。然而，大多数名单包含一些共同的元素。以下是渠道中可能包含的几类人员：

*   **当前员工**：考虑内部晋升或转岗的现有员工。
*   **前员工**：曾为组织工作过、了解公司文化的人员。
*   **推荐人选**：通过员工或业务伙伴推荐获得的人才。
*   **行业与活动结识的人才**：通过行业出版物、网络活动等渠道结识的潜在人才。
*   **技能匹配的潜在人才**：拥有与组织特定职位通常所需技能、背景和经验相匹配的人员。

即使组织当前没有立即的空缺职位，这些人员也会被列入名单。当职位出现空缺时，便可邀请渠道中合格的候选人进行面试。

此外，主动的招聘团队还会维护一个包含各类成就人才的储备库，例如在其组织中担任中高层职位的人员。这些职位通常更难填补且耗时，提前识别潜在候选人可以节省大量时间和资源。

## 候选人渠道的优势 💪

候选人渠道是一个多功能的工具。让我们回顾一下使用这些名单的几个好处：

*   **高度可定制**：渠道的创建可以简单如一个电子表格，也可以复杂如一个数据库。公式示例：`渠道工具 = 电子表格 或 数据库`。
*   **信息记录全面**：渠道可以记录团队可能需要的信息，包括候选人姓名、联系方式、相关社交媒体资料以及与招聘团队的互动记录。
*   **提升雇主品牌**：它向潜在候选人介绍了您的雇主品牌。这种品牌认知有助于候选人判断是否愿意被考虑担任空缺职位。在入职前熟悉雇主品牌有助于提高员工保留率。
*   **提高招聘效率**：候选人渠道中可用的高质量候选人可以减少您在被动招聘上花费的时间。渠道使寻源团队能够维护人才网络，这些网络使得为意外出现的职位空缺快速招聘变得更容易。

## 建立渠道的关键标准 📝

![](img/bf2f2bd2ea2070b620d81fd9c93d1b62_4.png)

尽管每个组织的候选人渠道都因其特定需求而有所不同，但建立时仍需牢记一些通用标准。现在我们来回顾一下：

*   **获得联系许可**：您需要与渠道中的候选人保持联系。务必请他们同意接收关于工作机会、公司更新以及个性化邮件的通知。
*   **满足基本要求**：候选人必须达到或超过公司或团队的部分基本要求。例如，某些职位可能要求最低教育水平。通常还要求候选人通过初步评估，如电话或视频会议筛选。这能让寻源团队了解候选人、评估其兴趣并确定其可能的合适岗位。
*   **对组织有好感**：理想情况下，候选人渠道中的人应曾表达过对组织的积极看法。这些陈述进一步证实了他们可能对未来职位感兴趣。

## 总结

![](img/bf2f2bd2ea2070b620d81fd9c93d1b62_6.png)

本节课中，我们一起学习了候选人渠道这一主动招聘策略。我们了解到，它是一个预先建立的人才储备库，包含各类潜在候选人，能够显著提升招聘效率、强化雇主品牌并支持长期人才规划。人才获取团队应与潜在候选人培养持续的关系，为新的工作机会做好准备，而候选人渠道正是跟踪这些人员的有力工具。
