# HRCI人力资源助理（招聘、学习发展、薪酬福利）：第3课：示例：职业发展 📈

![](img/79bd8a99f4e8856dc946e4311c0cfa4e_0.png)

在本节课中，我们将通过一个真实世界的案例，学习如何为员工制定一份职业发展计划。我们将跟随Urban Attire公司的人力资源专员Mary，看她如何帮助市场部的Avery规划其职业发展路径。

![](img/79bd8a99f4e8856dc946e4311c0cfa4e_2.png)

## 案例背景介绍

上一节我们介绍了职业发展计划的基本概念，本节中我们来看看一个具体的应用实例。

Urban Attire是一家专注于现代都市休闲服饰的中型企业。公司提供一系列时尚且实用的服装选择，旗下设有工厂、实体门店和总部，拥有大量员工，人力资源部门责任重大。

公司的市场部负责紧跟潮流并与客户互动。人力资源部的Mary正在帮助市场部的Avery制定一份职业发展计划。一份有效的职业发展计划可以向Avery表明，组织对其成功有所投入，并希望保持他的工作热情与积极性。

![](img/79bd8a99f4e8856dc946e4311c0cfa4e_4.png)

## 制定计划前的准备

在讨论职业发展之前，Mary首先熟悉了Avery的个人档案。

Avery是市场部的负责人，已在Urban Attire工作了六年。他拥有市场营销学士学位和工商管理硕士学位。Avery已展现出未来成为领导者的潜力，但对管理岗位尚不确定。不过，Avery表达了对了解组织内不同机会的兴趣，希望打破常规，迎接新的挑战。

## 职业发展计划的具体步骤

在与Avery交谈后，Mary开始协助他制定具体的职业发展步骤。以下是计划的核心环节：

**第一步：参加领导力培训课程**
Mary建议的第一步是参加领导力培训课程。这能让Avery了解自己是否对在组织内担任管理角色感兴趣。Mary找到了一门专注于培养**领导技能、战略思维和决策能力**的在线课程。

**第二步：尝试岗位轮换**
接下来，Avery和Mary讨论了第二步：岗位轮换。轮岗可以帮助Avery体验市场部内的不同角色，并发现任何感兴趣的机会。这将使Avery能够形成更广阔的视野，理解公司整体的市场战略，并发现迎接新挑战的可能性。

**第三步：配备导师**
在完成培训和岗位轮换后，Avery将与市场部的一位高级经理结对，由后者担任其导师。导师将提供**指导、反馈和支持**，帮助Avery培养在未来角色中取得成功所需的技能和知识，无论他最终是否决定追求管理机会。

**第四步：参与志愿者项目**
接着，Mary与Avery讨论了他所热衷的志愿者机会。Mary介绍了公司赞助的、与Avery兴趣相符的志愿者项目。参与这些项目能让Avery在回馈社区的同时，培养新技能，接触新的思维和工作方式。志愿者活动可以与职业发展计划的前几步同时进行，以保持Avery对组织的投入和工作兴趣。经过商讨，Avery决定参与“舞会礼服学校”项目，该项目将旧礼服改造翻新，提供给无力购买新礼服的人。

**第五步：确定HR支持与反馈机制**
最后，Mary和Avery确定了Avery希望从人力资源部门获得的支持力度。Avery希望每月进行一次跟进，以调整职业发展计划。在这些反馈会议中，Avery将分享他通过该计划对自身的新认识。这确保了计划能持续贴合Avery的兴趣，同时组织也能获得可用于未来其他员工职业发展的信息。

## 总结与展望

本节课中，我们一起学习了如何为一个具体案例制定职业发展计划。

![](img/79bd8a99f4e8856dc946e4311c0cfa4e_8.png)

现在，Avery正朝着发展一个更有趣、更令人兴奋的职业道路前进。我们后续将继续关注Urban Attire公司的情况。在你未来的人力资源工作中，很可能需要帮助构建职业发展计划。帮助员工保持对工作的投入和热情，是一项任务，希望这也能让你对自己的工作保持投入和热情。😊
