# HRCI《人力资源助理》课程：P53：谈判：录用与反录用

![](img/71402d1f32261b175e549ec5d6d617aa_0.png)

在本节课中，我们将学习招聘流程中的谈判环节。这个过程发生在向候选人发出录用通知书之后。

![](img/71402d1f32261b175e549ec5d6d617aa_2.png)

成功的谈判关键在于，在进入谈判前，你需要清晰地了解自己能够向候选人提供什么，以及提供的限度。

## 谈判过程的三个核心部分

谈判过程主要包含三个核心部分：反录用、保留价格和协作式谈判。以下是这些术语的具体含义。

### 反录用

在候选人收到初始录用通知后，他们可能会回复一份反录用通知。反录用是候选人对初始录用条件提出修改的请求。候选人可能基于多种因素来确定他们的反录用要求，包括工作经验年限、教育背景以及该职位的行业平均薪酬。

为应对反录用，你需要提前确定自己能够和不能够向候选人提供什么。明确自己能提供的条件有助于确保谈判顺利进行。

### 保留价格

确保谈判顺利进行的另一个重要方法是明确你的保留价格。保留价格是你能提供给候选人的最高薪资或福利待遇。它通常被称为“离场点”，因为当候选人拒绝这个报价时，谈判双方通常会结束谈判。

明确你公司中该职位的保留价格，将帮助你避免做出无法兑现的承诺。

### 协作式谈判

顺利谈判的最后一个方面被称为协作式谈判。协作式谈判是一种双赢模式，双方在谈判中共同努力达成期望的结果。与候选人谈判的目标是，就一份既能满足组织需求与期望，也能满足未来员工需求与期望的录用条件达成一致。

进行协作式谈判时，你首先应该定义一个“可能协议区”。可能协议区标识了候选人利益与组织利益重叠的区域。

定义这些共同利益将帮助你找到可能协议的上限和下限。以这些共同利益为指导进行谈判，将确保双方都能从谈判结果中受益。

![](img/71402d1f32261b175e549ec5d6d617aa_4.png)

## 谈判后的步骤

![](img/71402d1f32261b175e549ec5d6d617aa_5.png)

一旦双方达成协议，就应该最终确定录用通知书。录用通知书应清晰概述录用条件，例如薪资、福利、津贴以及在谈判中达成的任何特殊条款。

谈判具有挑战性，但也极具回报。如果起初觉得困难，请不要担心，这是一门艺术，你一定会随着经验的积累而做得更好。在后续课程中，我们将详细探讨录用通知书的细节。

## 总结

![](img/71402d1f32261b175e549ec5d6d617aa_7.png)

本节课我们一起学习了招聘谈判的三个核心部分：反录用、保留价格和协作式谈判。掌握这些概念并提前做好准备，将帮助你更自信、更有效地进行谈判，最终为组织争取到合适的人才。
