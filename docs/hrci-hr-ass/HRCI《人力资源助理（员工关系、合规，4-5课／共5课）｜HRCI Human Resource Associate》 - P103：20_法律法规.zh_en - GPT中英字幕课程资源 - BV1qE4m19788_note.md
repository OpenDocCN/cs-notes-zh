## 🏛 《1964年民权法案》（Civil Rights Act of 1964）  



上一部分我们提到，在设计培训与发展项目时必须遵守反歧视法律。本节首先介绍最重要的一部基础性法律——《1964年民权法案》。  



**entity["legislation","Civil Rights Act of 1964","us federal law"]** 的目的是通过废除不公平的投票规则以及学校、工作场所和公共场所的种族隔离制度来打击歧视行为。  

该法律确保在工作场所中，人们不会因为种族、宗教、国籍、性取向或性别认同而受到不平等待遇。  



我们可以将其核心原则用公式表示为：  



```  
Equal Treatment = 不考虑（race + religion + national origin + sexual orientation + gender identity）  
```  



该法案中的 **Title VI** 规定：雇主不得因以下因素对员工进行不公平对待：  



以下是被明确禁止作为歧视依据的因素：  

- 种族（race）  
- 肤色（color）  
- 宗教（religion）  
- 性别（sex）  
- 国籍（national origin）  



该法律还设立了 **entity["organization","Equal Employment Opportunity Commission","us federal agency"]**（EEOC），其职责是监督雇主遵守法律规定，防止对员工的歧视行为。

## 👵 《年龄歧视就业法》（Age Discrimination in Employment Act）  



在了解了基于种族和性别的歧视之后，本节我们来看与年龄相关的法律保护。  



**entity["legislation","Age Discrimination in Employment Act","us federal law"]** 禁止雇主基于年龄对员工或求职者进行歧视。  



![](img/8f8f8d08ef519c8e364b0d874691adf0_2.png)

该法律适用于就业的各个方面。以下是涵盖的具体内容：  

- 招聘（hiring）  
- 解雇（firing）  
- 薪酬（pay）  
- 工作分配（job assignments）  
- 晋升（promotions）  
- 裁员（layoffs）  
- 培训（training）  
- 福利（fringe benefits）  
- 其他任何就业条款或条件  



法律还明确规定：  

```  
If age ≥ 40  
AND termination_reason == "age only"  
→ Illegal  
```  



也就是说，仅因员工年龄超过40岁而终止其工作属于违法行为。

## ♿ 《美国残疾人法案》（Americans with Disabilities Act）  



在探讨年龄保护之后，本节我们来看关于残疾员工权益的法律。  



**entity["legislation","Americans with Disabilities Act","us federal law"]** 保护残疾员工的合法权益。  



该法律要求雇主为残疾员工提供“合理便利”。  



其核心逻辑可以表示为：  



```  
If employee.hasDisability == True  
→ Employer must provide Reasonable Accommodation  
```  



“合理便利”是指在不过度增加组织负担的前提下，对工作环境或工作方式进行必要调整。

## 💰 《同酬法》（Equal Pay Act）  



在保障就业机会平等的同时，薪酬公平同样重要。本节介绍与薪酬相关的法律。  



**entity["legislation","Equal Pay Act","us federal law"]** 规定：男性和女性从事相似工作时，应获得相同报酬。  

该规定与职位名称无关，而是以工作内容为判断标准。  



该法律涵盖所有形式的报酬。以下是具体包括的内容：  

- 基本工资（salary）  
- 奖金（bonuses）  
- 加班费（overtime pay）  
- 福利（benefits）  



其核心原则可以表示为：  



```  
If work_A == work_B  
→ pay_A == pay_B  
```  



雇主不得因为性别差异而为相同工作支付不同薪酬。

## 🌍 法律对组织的意义  



在介绍完主要法律之后，我们可以总结这些法规对组织的重要意义。  



遵守上述法律可以帮助组织建立一个尊重员工、提供平等机会的工作环境。  

![](img/8f8f8d08ef519c8e364b0d874691adf0_6.png)

这些法律促进多元化、包容性和公平性。  

其目标是确保任何人都不会因为自身身份特征而受到歧视。
