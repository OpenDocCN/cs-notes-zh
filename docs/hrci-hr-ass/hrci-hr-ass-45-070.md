# 70：65_职能型组织

![](img/c68c5866986d1a006c1e49d14a682cb9_0.png)

![](img/c68c5866986d1a006c1e49d14a682cb9_0.png)

在本节课中，我们将探讨职能型组织结构，以及它如何影响员工与部门之间的关系。



![](img/c68c5866986d1a006c1e49d14a682cb9_2.png)

![](img/c68c5866986d1a006c1e49d14a682cb9_2.png)

职能型结构是指一个部门或分部中的人员拥有相似的专业知识或技能。

例如，组织中的会计部门或信息技术部门。我们以一家名为Connective的B2B电信公司为例来说明。该公司拥有350名员工。

Connective遵循职能型组织结构，以确保高效运营和有效协调。

Connective内部的各个部门共同促成了公司的成功，这些部门包括销售部。

市场部、运营部、会计部、人力资源部、信息技术部以及研发部。



![](img/c68c5866986d1a006c1e49d14a682cb9_4.png)

![](img/c68c5866986d1a006c1e49d14a682cb9_4.png)

上一节我们介绍了职能型结构的定义，现在让我们具体看看Connective公司的销售部门。该部门包括所有负责推动销售和管理客户关系的销售代表和主管。

每位团队成员都有与其各自职能相对应的具体目标和任务。通过将员工按其销售专长分组。

Connective确保了销售活动采取专注且专业化的方法。

这种安排带来了运营流程的简化。

生产率的提高以及客户服务的增强。Connective的所有销售员工在部门内部都有明确的汇报关系。



![](img/c68c5866986d1a006c1e49d14a682cb9_6.png)

员工向各自的团队经理汇报，而团队经理则向销售部门经理汇报。

![](img/c68c5866986d1a006c1e49d14a682cb9_6.png)

这种精简的层级结构促进了清晰的权责划分。

使得在专业化的销售领域内能够进行高效的决策。这种结构为员工提供了指导和反馈，并使他们能够有效地协作。



![](img/c68c5866986d1a006c1e49d14a682cb9_8.png)

虽然职能型组织为Connective带来了若干优势。

但认识到其潜在的缺点也至关重要。一个挑战是部门间的沟通有限。

![](img/c68c5866986d1a006c1e49d14a682cb9_8.png)

这可能会影响协作和信息共享。为了降低这种风险。

Connective提倡跨部门沟通并鼓励团队合作。



![](img/c68c5866986d1a006c1e49d14a682cb9_10.png)

![](img/c68c5866986d1a006c1e49d14a682cb9_11.png)

![](img/c68c5866986d1a006c1e49d14a682cb9_10.png)

现在你已经了解到，职能型组织允许在每个部门内部建立清晰的层级、实现专业化。

![](img/c68c5866986d1a006c1e49d14a682cb9_11.png)

并进行有效的协调。

职能型组织结构提供了一个有价值的框架，它根据员工的技能进行分组，并确保了一个精简的系统。



![](img/c68c5866986d1a006c1e49d14a682cb9_13.png)

## 总结

![](img/c68c5866986d1a006c1e49d14a682cb9_13.png)

本节课中，我们一起学习了职能型组织结构。我们了解到，这种结构将拥有相似技能或专业的员工集中在同一部门，例如销售部或IT部。通过Connective公司的例子，我们看到了这种结构如何带来清晰的汇报层级、专业化分工和高效的部门内协调。同时，我们也认识到其潜在的挑战，如部门间沟通壁垒，并了解了企业可以通过促进跨部门协作来应对。
