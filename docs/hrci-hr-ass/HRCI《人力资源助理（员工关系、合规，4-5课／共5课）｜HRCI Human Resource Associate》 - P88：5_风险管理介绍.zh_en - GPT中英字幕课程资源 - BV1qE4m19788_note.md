## 🔎风险管理的定义


风险管理是分析潜在威胁并决定如何预防的过程。

换句话说，可以用一个简单公式描述：

**风险管理 = 风险分析 + 风险预防**

风险管理的重点有两个：

![](img/1265b16c0afb79849e7b6ed04cf8434c_4.png)

```
1. 保护员工
2. 保护组织
```

本节课将对这些概念进行介绍，在后续课程中会进一步展开。



---

## 🧭风险管理的四个阶段


上一节我们了解了风险管理的总体含义，本节我们来看它的四个基本阶段。

以下是风险管理的流程：

```
阶段1：Identify —— 识别风险
阶段2：Assess —— 评估风险
阶段3：Mitigate/Eliminate —— 减轻或消除风险
阶段4：Monitor —— 监控风险管理流程
```

这些阶段将在之后的视频中进一步深入说明。

## ⚖️法律与安全要求：OSHA


在了解风险管理的流程后，本节我们来看企业在管理风险时必须遵守的重要法律要求。

组织在管理风险时应重视健康与安全法律法规。其中关键机构是**entity["organization","Occupational Safety and Health Administration","us federal workplace safety"]（OSHA）**。

以下是OSHA的功能：

```
OSHA制定并采纳安全标准
这些标准帮助预防和处理工作场所事故
大多数组织必须遵守OSHA法规
```

例如，**entity["organization","Urban Attire","example company"]**在仓库存放商品。OSHA规定新仓库员工在开始工作前必须完成三项培训，主题包括：

```
1. 动力平台（powered platforms）
2. 升降机（man lifts）
3. 车载作业平台（vehicle-mounted work platforms）
```

![](img/1265b16c0afb79849e7b6ed04cf8434c_6.png)

这些培训确保员工能够安全操作可能存在危险的设备。

## 📝风险评估：企业降低风险的另一工具


上一节介绍了法律与合规要求，本节我们来了解企业如何具体开展风险评估。

风险评估是识别潜在危险并分析其后果的过程。公式如下：

![](img/1265b16c0afb79849e7b6ed04cf8434c_8.png)

**风险评估 = 危害识别 + 后果分析**

为了更有效地评估风险，组织应邀请最可能接触风险的员工参与，因为他们通常对潜在事件与后果有最直接的认识。



---

## 🛠️风险评估示例


继续以上内容，本节我们来看一个更具体的实际案例。

例如，entity["organization","Urban Attire","example company"]每年安排两次风险评估。人力资源部门会与精心挑选的员工讨论可能的威胁，例如：

```
使用刀具或开箱器时造成的意外割伤
```

![](img/1265b16c0afb79849e7b6ed04cf8434c_11.png)

由于割伤无法完全消除，风险团队会合作寻找降低风险的方式，如：

```
改进工具
加强培训
调整操作流程
```
