**HRCI人力资源助理课程：第4课：评分绩效评估法 📊**

![](img/9c9c36c03317fafe10c45737971d38c9_0.png)

在本节课中，我们将要学习绩效评估中的评分法。之前我们已经介绍了行为法和比较法，本节我们将深入探讨如何使用评分法来评估员工绩效。

![](img/9c9c36c03317fafe10c45737971d38c9_2.png)

评分法使用数值评分或等级来进行绩效评估。评分量表旨在区分不同的绩效水平。这些量表既可以是数字型的，也可以是描述型的。

![](img/9c9c36c03317fafe10c45737971d38c9_3.png)

以下是两种主要的评分量表类型：

![](img/9c9c36c03317fafe10c45737971d38c9_5.png)

*   **数字评分量表**：这是一种使用数字的评分工具。例如，一个从1到5分的量表。
    *   **公式/示例**：`绩效评分 = {1, 2, 3, 4, 5}`，其中1代表“待改进”，5代表“卓越”。
*   **描述性评分量表**：这种量表使用短语来描述绩效水平。例如：“始终符合期望”、“有时符合期望”或“从未符合期望”。

![](img/9c9c36c03317fafe10c45737971d38c9_7.png)

除了量表，管理者还可以使用清单法。清单法提供一系列用于描述员工绩效各方面的陈述。

以下是清单法的应用方式：
管理者勾选他们认为适用于每位员工的描述。

让我们通过一个例子来具体了解。艾弗里是Urban Attire公司的经理，负责进行季度绩效评估。她为每位员工准备了一份清单，该清单使用描述性量表来评估员工绩效。

![](img/9c9c36c03317fafe10c45737971d38c9_9.png)

这份清单按类别划分，包括客户服务技能、工作技能、组织性和可靠性。每个类别下都列出了关于员工期望的简要陈述，并附有评分等级。

![](img/9c9c36c03317fafe10c45737971d38c9_10.png)

艾弗里为每位员工完成评分后，会与他们进行一次简短会议，讨论其整体绩效。随后，她会将员工文档归档，以备日后用于晋升或解雇等雇佣决策。

![](img/9c9c36c03317fafe10c45737971d38c9_12.png)
