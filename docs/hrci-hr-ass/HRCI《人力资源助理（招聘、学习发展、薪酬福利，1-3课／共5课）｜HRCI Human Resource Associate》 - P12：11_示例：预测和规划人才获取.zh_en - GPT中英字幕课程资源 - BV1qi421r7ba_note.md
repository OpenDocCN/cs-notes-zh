# 人力资源助理：第12课：预测和规划人才获取 📊

![](img/985223f5ffd191374240edb98f31be75_0.png)

在本节课中，我们将学习如何在实际场景中进行人才获取的预测和规划。通过这个示例，我们可以看到如何评估公司当前和未来的用人需求，并根据实际情况做出相应的招聘决策。

![](img/985223f5ffd191374240edb98f31be75_2.png)

## 1. 背景介绍

在这一部分，我们将以Connective公司为例，探讨如何在快速发展的环境中进行人才获取的规划。

Connective是一家现代通信公司，致力于帮助企业保持联系。随着远程工作的兴起，Connective专注于帮助分布式团队通过视频会议和基于云的电话系统等工具进行协作。该公司正在快速发展，尤其是在过去几年远程工作爆发后，需求激增。

Connective的团队完全远程工作，面临着巨大的业务需求，HR团队正在制定扩展团队的计划。Alex是负责该项目的HR负责人。

## 2. 人才需求评估

为了开始这一过程，Alex收集了来自公司各个部门的需求信息，包括各部门经理提供的业务量和他们所需的人力资源。

### 销售团队的挑战

销售团队正在努力应对大量来自其他公司的咨询请求。虽然这是一个好问题，但他们无法满足当前的需求，因此需要增加更多的销售人员。为了确定招聘人数，Alex和HR团队进行了工作量分析。

他们与销售主管合作，确定了实现销售目标所需的工作量，并将其转化为工作时长。最终，他们决定至少需要两名全职初级销售人员。

![](img/985223f5ffd191374240edb98f31be75_4.png)

![](img/985223f5ffd191374240edb98f31be75_5.png)

### 立即需求

由于销售团队即将进入一个繁忙的季度，且市场营销团队最近启动了一个新的广告活动，新客户已开始联系销售团队，因此Alex决定尽快引入一名临时工作人员来解决当前的需求。

临时工作人员预计能够帮助团队度过下个季度，如果表现良好，他们可能会成为正式员工。

## 3. 招聘流程与外部合作

Alex曾与一家专注于销售岗位的临时员工招聘机构合作过，并且合作经历非常成功。这种方式是帮助销售团队尽快获得支持的最快途径。招聘机构将负责招聘、背景调查、薪酬福利等工作，而Alex则继续寻找合适的全职销售人员。

### 临时人员的角色

在临时员工适应工作时，Alex将继续寻找长期的全职员工来填补该岗位。最终，招聘到的全职员工将可能会受到FLSA（公平劳动标准法案）的豁免影响，但这需要根据具体职位职责进一步确认。

### 灵活的工作安排

![](img/985223f5ffd191374240edb98f31be75_7.png)

与Connective其他管理人员一样，Alex也有灵活的工作安排。除了远程工作外，Alex每隔一个星期五休息，采用9天80小时的压缩工作制，即每周工作4天，每天10小时。

## 4. 总结

![](img/985223f5ffd191374240edb98f31be75_9.png)

本节课我们通过Connective公司的示例，了解了如何进行人才需求的预测与规划。在实际操作中，HR团队需要根据部门需求进行细致的工作量分析，以确定所需人员的数量。同时，通过与临时员工招聘机构的合作，快速填补岗位需求也是一种有效的策略。接下来，我们将学习工作设计与工作重设计的相关内容。
