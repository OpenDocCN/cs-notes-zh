# 4：制定愿景声明 📝

![](img/00c69009e6d31b83ac9c9e92cc371290_0.png)

在本节课中，我们将学习愿景声明的概念，了解其与使命声明的区别，并通过知名公司的实例来掌握如何制定一个有效的愿景声明。

![](img/00c69009e6d31b83ac9c9e92cc371290_2.png)

上一节我们介绍了使命声明，它定义了组织当前的核心业务与存在方式。本节中，我们来看看与之紧密相关的愿景声明。

## 愿景声明的定义与作用

愿景声明是一句鼓舞人心的陈述，它揭示了组织未来的目标和理想。使命声明定义组织当前在做什么，而愿景声明则着眼于其未来。

![](img/00c69009e6d31b83ac9c9e92cc371290_4.png)

## 愿景声明的四大核心要素

根据项目管理软件Asana的定义，一个有效的愿景声明应包含以下四个关键组成部分：

以下是构成愿景声明的四个要素：

1.  **雄心勃勃**：愿景声明应能激励员工、投资者和客户，让他们珍视组织的使命。它应对组织的未来理想和方向充满热情。
2.  **切实可行**：愿景声明在保持雄心的同时，必须是可实现的。它应是一个组织可以努力达成的、切实可行的目标。
3.  **宽泛概括**：愿景声明不应过于具体，但它需要将组织的使命与其目标联系起来。
4.  **具有战略性**：愿景声明应识别出最相关和最重要的理念。它应能体现组织使命和宗旨的抱负。

此外，愿景声明应使用清晰、日常的语言撰写，避免使用流行语或行业黑话。

## 知名公司愿景声明实例分析

接下来，我们通过对比宜家、孩之宝和西南航空的使命与愿景声明，来加深理解。

### 宜家 (IKEA)

![](img/00c69009e6d31b83ac9c9e92cc371290_6.png)

![](img/00c69009e6d31b83ac9c9e92cc371290_7.png)

*   **使命声明**：`宜家以低廉的价格提供种类繁多、设计精良、功能实用的家居用品。`
*   **愿景声明**：`为大众创造更美好的日常生活。`

宜家的使命声明解释了其作为一家**低价家具公司**的当前身份。而其愿景声明则表达了他们**为人们创造更美好生活**的未来目标。

### 孩之宝 (Hasbro)

*   **使命声明**：`孩之宝创造世界上最好的游戏和娱乐体验。`
*   **愿景声明**：`让世界对所有儿童、粉丝和家庭变得更美好。`

从使命声明中，我们可以理解孩之宝是创造流行玩具（他们称之为“游戏和娱乐体验”）的公司。其愿景声明则详细阐述了他们通过游戏**让世界对每个人变得更美好**这一雄心勃勃、切实可行、宽泛且具有战略性的目标。

### 西南航空 (Southwest Airlines)

*   **使命声明**：`西南航空通过友好、可靠且低成本的航空旅行，将人们与他们生活中重要的事物连接起来。`
*   **愿景声明**：`成为世界上最受喜爱、最高效、最盈利的航空公司。`

西南航空的使命声明表达了其对客户的专注。而其愿景声明则指明了其主要目标：**既受大众喜爱，又能实现盈利**。

## 总结

![](img/00c69009e6d31b83ac9c9e92cc371290_12.png)

本节课中，我们一起学习了愿景声明的制定。总结来说，**使命声明解释组织当前正在做什么，而愿景声明则清晰地表达了组织的目标，并揭示了公司的长期发展方向**。理解并制定好这两份声明，对于明确组织定位和引领未来至关重要。
