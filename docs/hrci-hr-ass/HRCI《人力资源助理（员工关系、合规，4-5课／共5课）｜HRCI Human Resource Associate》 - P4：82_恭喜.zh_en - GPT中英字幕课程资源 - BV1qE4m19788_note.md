## 💰 薪酬体系基础：结构与类型  

首先，你学习了薪酬体系的基本构成。  

以下是本部分的核心内容：  

- 薪酬体系（Compensation Systems）  
- 工资结构（Wages）  
- 差异化薪酬结构（Differential Pay Structures）  



薪酬体系的核心可以用公式表示为：  

```
Total Compensation = Base Pay + Differential Pay
```

其中：  

- `Base Pay` = 基本工资  
- `Differential Pay` = 因岗位、时间、技能或条件差异而产生的额外支付  



在理解结构之后，我们进一步学习薪酬与福利的基础理论。  



## ⚖️ 薪酬与福利核心原则  

接下来，你学习了薪酬与福利的基本概念与原则。  

以下是核心内容：  

- 工资公平（Wage Equity）  
- 绩效付薪（Pay for Performance）  
- 激励机制（Incentives）  



工资公平的基本逻辑可以表示为：  

```
Wage Equity = Internal Equity + External Equity
```

- `Internal Equity` = 组织内部岗位之间的公平性  
- `External Equity` = 与外部市场相比的竞争力  



绩效付薪可以表示为：  

```
Total Pay = Base Pay + Performance Bonus
```

这说明员工的总收入与绩效结果直接相关。  



在掌握这些原则之后，你开始深入研究福利体系的具体构成。  



## 🏥 福利类型与功能  

随后，你深入学习了不同类型的福利及其功能。  

以下是本部分的重点：  

- 法定福利（如社会保险）  
- 自愿福利（如补充医疗、退休计划）  
- 员工支持类福利（如健康项目、员工援助计划）  



福利的功能可以概括为：  

```
Benefits Function = Attraction + Retention + Motivation
```

即：  

- 吸引人才  
- 留住人才  
- 提升员工动力  



理解福利的类型和功能后，你还学习了技术层面的设计内容。  



## 💻 薪酬与福利技术设计  

最后，你学习了如何设计薪酬与福利管理系统。  

目标是满足组织与员工的双重需求。  

这一设计逻辑可以表示为：  

```
HR Technology Design = Organizational Needs + Employee Needs
```

也就是说，薪酬与福利系统必须：  

- 支持组织战略目标  
- 提高员工使用体验与满意度  
- 确保数据准确与合规性
