## 什么是工作评估？

![](img/60ab5d5769c02f0753c4579208175585_4.png)

工作评估是一个系统性的过程，它根据岗位所需的职责和技能水平对所有岗位进行分类。

工作评估发生在为组织创建薪酬体系的早期阶段。该过程旨在通过分析岗位职责和技能，为所有岗位建立公平、公正的薪酬水平。这能确保员工根据其岗位的相对价值获得适当的报酬。

组织使用不同的技术来完成工作评估。四种主要技术是点数法、排序法、分类法和因素比较法。这些技术用于根据岗位在组织内部的价值来定价。它们关注的是内部因素，如技能、职责和复杂性。这些技术与基于市场的工作评估不同，后者使用外部市场数据（如劳动力市场）来定义岗位价值。

接下来，让我们逐一探索这些技术。

## 四种工作评估技术

以下是四种核心的工作评估技术。

![](img/60ab5d5769c02f0753c4579208175585_6.png)

### 1. 点数法

点数法通过审查岗位的不同部分来确定合适的薪酬。

在评估过程中，岗位被分解为更小的组成部分，每个部分根据其重要性被分配点数。点数最高的岗位获得最高的薪酬。例如，一个需要大量经验的岗位组成部分可能会获得更多点数。一旦岗位的所有部分都被赋予点数，组织就可以确定该岗位的价值，并设定公平的薪酬。

**核心公式**：
```
岗位总价值 = Σ (各因素得分 × 权重)
```

### 2. 排序法

排序法根据岗位对组织的重要性来排列岗位顺序。

这种技术相当直接，最适合岗位数量较少、易于管理的小型组织。岗位被置于一个等级体系中，顶层代表价值高的专业岗位。然后，所有岗位都使用这个等级体系进行比较。

### 3. 分类法

在这种评估方法中，通过将具有共同特征的职位分组来创建岗位类别。

岗位类别随后被分级，并分配一个薪酬范围。例如，一个大型组织在各个部门（如市场、销售和人力资源部）都有团队领导。在分类法中，所有团队领导被归入同一等级，而这些部门的负责人则被归入更高的等级或类别。

### 4. 因素比较法

因素比较法是排序法和点数法的结合。

![](img/60ab5d5769c02f0753c4579208175585_8.png)

因素比较法先对岗位因素进行排序，然后为每个因素分配一个货币价值。在这种方法中，岗位薪酬是基于一系列因素的集合来确定的，而不是针对个人。例如，一个组织为其销售部门建立薪酬体系。首先，它对部门内的岗位要求或因素进行排序，这些因素可能包括教育背景、经验或技术技能。接着，它为每个因素分配一个货币金额。这些因素的总和决定了薪酬。
