# HRCI人力资源助理课程：P47：不同的测试选项 📊

![](img/0ce67201587caa93b41f158ae4ed826c_0.png)

在本节课中，我们将深入探讨评估潜在员工的不同方法，具体介绍组织在招聘过程中可能使用的各类测试。我们将了解纸笔测试、医学测试、实操测试以及测谎测试等，并讨论雇主在实施这些测试时需要考虑的关键因素。

上一节我们介绍了员工评估的基础知识，本节中我们来看看几种具体的评估工具。

![](img/0ce67201587caa93b41f158ae4ed826c_2.png)

## 纸笔测试 📝

纸笔测试是传统的书面预聘测试，不过现在许多已实现计算机化。这类测试通常用于评估员工的综合能力。

以下是常见的纸笔测试类型：
*   **一般能力或智力测试**：评估认知能力。
*   **诚信测试**：评估候选人的诚实度和可靠性。
*   **性格类型测试**：了解候选人的行为倾向和个性特征。
*   **语言流利度测试**：评估语言掌握水平。
*   **岗位特定知识测试**：考察与职位直接相关的专业知识。

**核心要点**：所有测试都必须与**工作内容相关**。同时，雇主需要考虑**测试效度**，即测试是否能准确预测工作表现。例如，一个非常聪明的申请人可能会揣测雇主的意图，并给出迎合期望的答案。

## 医学测试与实操测试 🏥🔧

在招聘过程中，雇主可能要求申请人进行特定的医学测试，以确保其身体条件能够胜任工作职责。

以下是医学测试的常见形式：
*   **体格检查**
*   **药物测试**

**重要提示**：雇主必须谨慎，不能利用体格检查不公平地筛选掉残障人士申请人。

雇主也可能使用实操测试来评估申请人执行工作的能力。

以下是实操测试的常见形式：
*   **技能测试**：例如打字、起草信件、操作机器等。
*   **驾驶测试**：如果工作需要操作车辆。
*   **公文筐练习**：要求申请人整理备忘录和其他文件，以确定哪些事项需要优先处理。
*   **试用期**：支付报酬让申请人工作一段时间，以观察其是否适合长期聘用。
*   **短期咨询安排**：用以评估申请人是否适合该职位并满足绩效要求。

## 工作样本评估与评估中心 🎨🏢

对于创意类职位，如写作或平面设计，工作样本评估非常普遍。

这些评估可以在**评估中心**进行，以最好地衡量求职者满足不同工作需求的能力。

评估中心可能采用以下方法的组合：
*   面试
*   公文筐测试
*   角色扮演练习
*   小组讨论

**成本考量**：由于成本高昂，评估中心最常被州和地方政府以及大型企业使用，小型组织使用较少。

## 测谎测试 🕵️

尽管1988年的《雇员测谎保护法》限制了雇主对员工使用测谎测试，但仍存在一些例外情况。

![](img/0ce67201587caa93b41f158ae4ed826c_4.png)

以下人员可能被要求接受测谎测试：
*   国家安全或国防部门的雇员。
*   能够接触装甲车服务的雇员。
*   从事制药产品相关工作的雇员。

![](img/0ce67201587caa93b41f158ae4ed826c_6.png)

此外，如果准雇员因前一家公司的财务损失或损害而接受调查，也可能被要求进行测谎测试。
