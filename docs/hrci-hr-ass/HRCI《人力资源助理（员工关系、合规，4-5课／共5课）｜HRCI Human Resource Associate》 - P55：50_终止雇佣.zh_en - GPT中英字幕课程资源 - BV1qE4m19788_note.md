# HRCI《人力资源助理（员工关系、合规，4-5课／共5课）》：第55章：员工终止雇佣

![](img/6011557cc0722f3e4b71e15eff31d6d1_0.png)

在本节课中，我们将学习员工终止雇佣的相关内容。终止雇佣对员工和雇主来说都是一项艰难的任务，尤其是在处理不当的情况下。以下是一些终止雇佣的重要概念和最佳实践。

![](img/6011557cc0722f3e4b71e15eff31d6d1_2.png)

## 1. 终止雇佣的挑战 💼

终止雇佣对任何一方都可能产生负面影响，尤其是处理不当时。大多数雇主倾向于避免直接解除员工的职务，通常采取渐进的纪律处分措施，如警告和其他补救措施，试图改善员工的表现。然而，有时这些方法并未能奏效。

## 2. 美国“雇佣自由原则”（Employment at Will）🇺🇸

在美国，雇佣自由原则允许雇主无需给出理由即可解雇员工，前提是解雇理由不违法。违法的解雇理由包括基于年龄、性别或种族的歧视。大多数公司会为特定原因解雇员工，因此，大多数解雇被认为是“因故终止雇佣”（termination for cause）。

### **因故终止雇佣的原因**：
- 表现不佳
- 故意违反公司政策
- 偷窃或挪用公款
- 未通过毒品或酒精测试
- 不服从
- 将公司财产用于个人事务
- 在简历上撒谎

有时，解雇是因为招聘过程出现问题，员工与职位不匹配。通过确保候选人与职位描述匹配，并在招聘时进行选择性筛选，可以避免这种情况。

## 3. 不当解雇的法律风险 ⚖️

不当解雇的代价可能很高。不当解雇诉讼可能导致巨额法律费用，如果公司被判定为错误，可能需要支付高额赔偿金。因此，雇主在解雇员工时必须非常谨慎。

### **不当解雇的三大例外**：
1. 公司不能解雇受到联邦或州平等就业和工作场所法律保护的员工。例如，不能因为举报不安全工作环境的员工而解雇。
2. 通过普通法例外，员工受隐性合同保护，例如员工手册中的规定。
3. 公共政策豁免保护员工拒绝违法行为时不会被解雇。

## 4. 建设性解雇 🛠️

![](img/6011557cc0722f3e4b71e15eff31d6d1_4.png)

建设性解雇是指雇主通过恶化工作条件迫使员工辞职。例如，降职以羞辱员工、骚扰或威胁解雇等行为可能构成建设性解雇。

![](img/6011557cc0722f3e4b71e15eff31d6d1_6.png)

## 5. 终止雇佣的最佳实践 🌟

![](img/6011557cc0722f3e4b71e15eff31d6d1_7.png)

处理解雇时，应该以公正、一致、人道且尊重的方式进行。人力资源部门通常协助管理层在解雇员工时提供支持。在许多情况下，人力资源代表会记录或协助解雇过程。

### **良好实践的关键**：
- 确保员工有机会收拾个人物品，并简短告别同事，这样有助于减少不满情绪。
- 即使被解雇的员工通常会感到不愉快，但如果他们受到尊重，许多尴尬的情况可以避免。
- 对于裁员情况，人力资源部门通常会提供职业安置服务，帮助被解雇员工寻找新工作。

![](img/6011557cc0722f3e4b71e15eff31d6d1_9.png)
