# 4周：合规策略与实施

![](img/b7b4f4859679faa86aa8d976dda78e79_0.png)

欢迎来到人力资源风险管理与合规学习的第四周。本周，你将学习合规策略与实施。首先，你将了解人力资源在组织重组中的角色，这包括增长、集中、并购与裁员。你还将学习更多关于离岸外包与业务外包的知识。接下来，你将探索人力资源在业务连续性中的作用，即组织为确保在紧急事件期间能够运行业务职能而进行的规划与行动。你将具体审视资产剥离与无薪假。

作为一名人力资源专业人士，你通常需要负责许多事务，包括组织重组与业务连续性，因此理解它们至关重要。让我们开始吧。
