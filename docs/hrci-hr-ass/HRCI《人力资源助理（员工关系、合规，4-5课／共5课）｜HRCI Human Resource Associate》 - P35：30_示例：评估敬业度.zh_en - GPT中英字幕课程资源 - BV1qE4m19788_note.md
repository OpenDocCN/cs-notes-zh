## 📊 二、使用问卷调查衡量包容性  



首先，Neri使用了在推行新包容性举措之前就采用过的同一份调查问卷。



该调查采用五点量表，用于衡量员工对某个陈述的认同程度。



核心评分方式如下：



![](img/0e5cb74604812cbef1f6c3b7a606fd0c_4.png)

```text
1 = 强烈不同意  
2 = 不同意  
3 = 中立  
4 = 同意  
5 = 强烈同意
```



其中一个关键问题是：  

> 你是否认为Urban Attire是一个包容性的工作场所？  



为了理解变化趋势，可以通过以下公式计算平均得分：



```text
平均得分 = 所有员工评分总和 ÷ 参与人数
```



调查结果显示了一个虽小但令人鼓舞的提升：



- 之前的平均得分为 4.0（同意）  
- 几个月积极推进包容性后，平均得分提升至 4.5（介于“同意”与“强烈同意”之间）  



这一增长可以表示为：



```text
增长值 = 4.5 - 4.0 = +0.5
```



Neri对这一进步感到满意，并计划在下一次全员大会上分享该数据。

![](img/0e5cb74604812cbef1f6c3b7a606fd0c_6.png)

虽然这些数据难以像量化调查那样直接对比，但整体反馈同样积极。



Neri对Urban Attire的发展方向感到满意。

## 📈 四、分析人力资源数据：晋升多元化趋势  



接下来，Neri审查了人力资源数据，以验证晋升是否更加多元化。



她发现，公司晋升结构现在与员工整体构成更加一致。



可以用以下公式表示晋升结构匹配度：



![](img/0e5cb74604812cbef1f6c3b7a606fd0c_8.png)

```text
匹配度 = 晋升群体比例 ≈ 公司员工群体比例
```



这意味着晋升更加公平，反映了整体员工构成。



Neri认为，即使没有正式修改政策，仅仅公开表达对晋升多样性不足的担忧，也推动了进步。



她计划在未来数月和数年内持续监测这一问题及其他相关指标。



为了确保持续改进，公司需要：  



- 保留基准数据（Benchmark Data）  
- 分析重复出现的趋势（Recurring Trends）  
- 基于数据做出决策  



可以总结为：



```text
持续改进 = 基准数据 + 趋势分析 + 长期监测
```
