# 185：457退休计划 💼

![](img/48459541ed787b31511d7088b6c64b51_0.png)

在本节课中，我们将学习另一种退休计划——457计划。这是一种由州或地方政府以及某些非营利组织提供的退休储蓄工具。

![](img/48459541ed787b31511d7088b6c64b51_2.png)

上一节我们介绍了不同类型的退休计划，本节中我们来看看457计划的具体内容。

## 什么是457计划？

457退休计划由州、地方政府以及部分非营利组织提供。该计划中的供款和收益享受**税收递延**待遇，这意味着资金在提取之前无需缴税。

最常见的类型是**457(b)**计划，主要面向州和地方政府雇员。另一种是**457(f)**计划，面向高薪酬的政府及部分非政府雇员。

## 457计划的核心特点

以下是457计划的一些关键运作方式：

*   **供款额度**：参与者可以将**100%** 的工资存入其457计划，只要不超过计划设定的上限。
*   **税收优势**：由于供款是税前的，参与者在工作期间可以更有效地储蓄。公式表示为：`应纳税收入 = 总收入 - 457计划供款`。
*   **退休预算**：参与者规划退休收入时，必须将未来的税款考虑在内。

## 深入解析457(b)计划

现在，让我们进一步了解457(b)计划的具体细节。

*   **税前供款**：向457(b)计划的供款从个人收入中**税前扣除**，这降低了其应税收入。
*   **投资增值**：与401(k)计划类似，雇员可以选择将其457(b)供款用于投资以实现增值。
*   **提前取款**：与401(k)不同，参与者从457(b)计划中**提前取款不会产生10%的罚款**。

以下是截至2021年的供款限额：

*   **年度基本限额**：雇员每年最多可供款 **$19,500**。
*   **追加供款**：如果计划允许追加供款，雇员每年可额外供款 **$6,500**。
*   **双倍限额追加**：部分457(b)计划允许在特定情况下使用双倍限额。例如，如果雇员距离退休不足三年，其供款上限可达 **$39,000**，而非$19,500。

请注意，美国国税局(IRS)通常会每年更新这些限额，因此需要密切关注其官网的最新公告。
