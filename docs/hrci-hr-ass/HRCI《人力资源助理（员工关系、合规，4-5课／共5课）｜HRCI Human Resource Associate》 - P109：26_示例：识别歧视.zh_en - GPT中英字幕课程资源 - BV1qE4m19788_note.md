# HRCI《人力资源助理（员工关系、合规，4-5课／共5课）》：P109：26_示例：识别歧视

![](img/43b600f9a4206dc5ba6bd93b914cfd78_0.png)

在本节课中，我们将学习如何识别组织中的歧视现象，特别是在薪酬和招聘过程中。通过这个案例分析，我们将探讨如何通过数据分析来识别潜在的歧视问题。

## 1. 了解背景

![](img/43b600f9a4206dc5ba6bd93b914cfd78_2.png)

Connective 是一家现代化的通信公司，致力于帮助分布式工作团队通过视频会议和基于云的电话系统等软件工具保持高效协作。其员工团队完全远程工作，分布在不同地区。Alex 是 Connective 的人力资源员工，负责分析公司的员工数据，确保公司的人力资源政策符合多元化和包容性的要求。

## 2. 使用群体分析法进行薪酬分析

首先，Alex 使用**群体分析法**来检查不同群体员工的薪酬是否合理。特别是，他关注那些来自受保护群体（如女性、少数族裔等）的员工是否与其他员工在薪酬上有显著差距。

### 分析过程：
- Alex 将来自受保护群体的员工与非受保护群体的员工的薪酬数据进行对比。
- 结果表明，尽管有一两个潜在问题需要进一步研究，但整体薪酬看起来是合理的，员工的薪酬与他们的职责和技能相符。
- Alex 认为，Connective 采取的薪酬范围控制和职位评估措施对薪酬公平性起到了积极作用。

**公式：**
- **薪酬公平性分析公式**：
  - 若（受保护群体薪酬 - 非受保护群体薪酬）>0，且差距超出合理范围，则存在潜在歧视问题。

## 3. 检查晋升数据

接下来，Alex 对晋升数据进行了群体分析，目的是检测晋升过程中是否存在性别或其他群体的偏见。结果揭示了一个潜在的令人担忧的趋势——女性员工晋升的概率略低于男性员工。

### 分析过程：
- 数据显示，尽管 Alex 亲自推动了确保晋升公平的举措，但女性员工在晋升方面确实面临不利情况。
- Alex 意识到，尽管薪酬上似乎没有明显问题，但晋升过程中仍可能存在不平等现象。

**公式：**
- **晋升公平性分析公式**：
  - 若（女性员工晋升率 - 男性员工晋升率）< 0，且差距显著，则可能存在晋升歧视。

## 4. 进行可用性分析

最后，Alex 进行了一项**可用性分析**，目的是评估受保护群体中有多少人符合公司招聘职位的要求。由于 Connective 的员工完全远程工作，进行此类分析变得更加复杂。

![](img/43b600f9a4206dc5ba6bd93b914cfd78_4.png)

### 分析过程：
- Alex 从 Denver（科罗拉多州）的劳动市场数据入手，分析了符合工作要求的人数。
- 结果令人惊讶，Connective 的员工群体中有相对较高比例的人来自受保护群体，这表明公司在招聘过程中成功地吸引了更多的多元化人才。

**公式：**
- **可用性分析公式**：
  - 若（Connective 受保护群体员工比例 - Denver 地区受保护群体员工比例）> 0，则公司在招聘上取得了成功的多元化进展。

## 5. 总结

![](img/43b600f9a4206dc5ba6bd93b914cfd78_6.png)

通过分析薪酬、晋升和招聘数据，Alex 发现 Connective 在多元化和包容性方面做得很好，尤其是在吸引受保护群体员工方面取得了显著进展。然而，晋升过程中的性别差异仍需改进。在本节课中，我们一起学习了如何通过数据分析识别组织中潜在的歧视问题，并了解了如何采取措施促进组织的多元化与公平。
