# HRCI《人力资源助理（员工关系、合规）》：第4-5课：积极风险与消极风险 ⚖️  



![](img/6f843890c7e1e0eba10041fd667d3977_0.png)

在本节课中，我们将学习**什么是风险**，以及如何区分**积极风险**和**消极风险**。我们还会通过实际案例理解风险对组织的影响，并掌握在人力资源工作中如何有效管理风险。  



## 一、风险的基本概念 🔍  



当人们想到“风险”时，通常会想到负面的事情，例如自然灾害或员工受伤。  

然而，并非所有风险都是坏事，有些风险甚至可能带来益处。  



因此，在学习风险管理时，我们需要改变对风险的单一认知。  



![](img/6f843890c7e1e0eba10041fd667d3977_2.png)

### 风险的基本定义  



风险可以用一个简单的公式表示：  



```
风险 = 事件发生的可能性 × 事件产生的影响
```  



如果影响是负面的，则属于消极风险；如果影响是正面的，则属于积极风险。  



在了解了风险的基本概念之后，接下来我们分别看看什么是积极风险。  



## 二、积极风险（Positive Risk）🌱  



积极风险是一种**可能为组织带来成功的机会**。  



它可能帮助组织创造价值、获得优势或实现新的发展机会。  



如果这种风险发生，可能会对以下目标产生积极影响：  



以下是积极风险可能带来的收益：  



![](img/6f843890c7e1e0eba10041fd667d3977_4.png)

- 创新能力提升  
- 节约成本  
- 节省时间  
- 提高绩效  
- 获得竞争优势  



### 案例说明  



例如，Connective 公司近期对稳定且用户友好的视频通话系统需求大幅增加。  



因此，公司决定投入大量资金开发新的软件安全系统，以防止数据泄露。  



这项投资属于积极风险，因为它有可能保护公司免受黑客及其他恶意攻击者的侵害。  



引入新软件本身也存在风险，例如：  



以下是可能出现的风险情况：  



- 技术故障  
- 员工培训影响生产效率  



然而，新软件也可能带来多种好处。  



以下是该软件的潜在优势：  



- 降低法律费用  
- 减少人工成本  
- 保护企业声誉  



公司认为这些优势超过了可能存在的负面风险，因此对这一决策感到满意。  



在了解了积极风险之后，我们再来看看什么是消极风险。  



## 三、消极风险（Negative Risk）⚠️  



消极风险是一种**可能造成损害的威胁**。  



消极风险通常存在于组织的日常运营中，而且往往比较容易被察觉。  



消极风险可能影响项目、组织或个人，并导致以下不良结果：  



以下是消极风险可能带来的后果：  



- 损失  
- 损害  
- 失败  
- 其他不理想的结果  



### 案例延续  



![](img/6f843890c7e1e0eba10041fd667d3977_6.png)

继续前面的例子，由于公司已经投资了新的安全软件系统，他们决定不再为视频通话增加密码功能。  



![](img/6f843890c7e1e0eba10041fd667d3977_7.png)

公司认为，现有的安全系统已经足够强大，因此无需额外的密码保护。  



不设置密码属于一种消极风险。  



虽然当前的软件系统很强大，但技术不断变化。  



未来，黑客可能会突破该系统。  



如果发生这种情况，可能会产生以下影响：  



- 损害公司声誉  
- 因法律诉讼产生高额费用  
- 降低员工生产效率  
- 造成财务损失  



因此，忽视潜在威胁可能会带来严重后果。  



## 四、风险管理的关键原则 🎯  



![](img/6f843890c7e1e0eba10041fd667d3977_9.png)

在管理风险时，必须同时考虑积极风险和消极风险。  



这样做有助于实现更加平衡的决策，并最大化组织成功的可能性。  



风险管理的核心思路可以表示为：  



```
全面识别风险 → 评估影响 → 制定应对策略 → 持续监控
```  



![](img/6f843890c7e1e0eba10041fd667d3977_11.png)

在人力资源管理工作中，理解如何区分并应对这两类风险极其重要。  



![](img/6f843890c7e1e0eba10041fd667d3977_12.png)

这样可以帮助你更有效地评估并降低组织风险。

## 五、本节课总结 🧾  



本节课中，我们一起学习了风险的基本概念，并明确了风险不仅包括威胁，也包括机会。  



我们理解了积极风险可以带来创新、节约成本和竞争优势，而消极风险可能导致损失和失败。  



通过案例分析，我们掌握了在实际工作中如何平衡两类风险。  



![](img/6f843890c7e1e0eba10041fd667d3977_14.png)

在人力资源管理中，正确识别和管理风险，是支持组织持续成功的重要能力。
