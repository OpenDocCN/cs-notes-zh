## 一、培训回报率（ROI）💰  



上一节我们明确了目标的重要性，接下来需要通过具体指标来衡量成果。首先介绍最核心的指标：培训投资回报率（Training Return on Investment，ROI）。



培训投资回报率是指通过培训所获得的盈利或效率提升，与培训投入成本之间的关系。



其核心计算公式为：



```text
ROI = (培训收益 - 培训成本) / 培训成本 × 100%
```



ROI 用于衡量培训是否带来了实际的经济价值或效率提升。



![](img/2bb6b574b11b08e1808f77d3c099642f_2.png)

## 二、人均培训成本 👥  



在了解 ROI 之后，我们还需要关注成本结构。以下是另一个常见指标：人均培训成本。



人均培训成本是指培训总成本除以参与培训的员工人数。



其计算公式为：



```text
人均培训成本 = 培训总成本 / 参与培训员工人数
```



该指标可以帮助人力资源部门控制预算，并评估资源分配是否合理。



## 三、学习者参与度 📈  



在分析成本之后，我们还需要关注学习过程本身。接下来介绍学习者参与度。



学习者参与度是指学习者在培训过程中投入的时间和努力程度。



参与度之所以重要，是因为它会影响其他指标，例如投资回报率。



如果员工在培训中投入更多精力，通常更有可能将所学知识应用到工作中，从而提升整体培训效果。



## 四、培训满意度 😊  



在关注学习过程后，我们进一步评估员工的主观体验。本节介绍培训体验满意度。



培训满意度是指员工在培训结束时所感受到的满意程度。



该指标通常通过问卷调查进行评估，例如：



- 课程内容是否有帮助  
- 培训讲师是否清晰易懂  
- 培训是否满足预期  



满意度可以反映培训体验质量，但不能单独代表培训效果。



## 五、培训课程数据与成绩指标 📝  



除了主观评价之外，我们还可以通过数据和成绩来评估培训效果。以下是常见的数据型指标：



- 课程报名数据（Course Enrollment Data）  
- 课程完成率（Course Completion Data）  
- 测评成绩，尤其是通过率（Pass Rates）  



![](img/2bb6b574b11b08e1808f77d3c099642f_4.png)

这些数据可以用于评估培训项目的有效性。



例如，通过率可以用以下公式表示：



```text
通过率 = 通过人数 / 参加考试人数 × 100%
```



如果通过率较高，说明学习效果较好；如果通过率较低，可能需要优化课程内容。



## 六、比较性衡量方法 🔍  



在了解具体指标后，本节介绍如何使用这些指标进行效果评估。



指标用于评估政策、流程或项目的有效性。



比较性衡量是指将当前指标与基准值进行比较，以判断干预措施是否有效，或是否需要进行调整。



![](img/2bb6b574b11b08e1808f77d3c099642f_6.png)

例如，一家公司记录员工使用病假的数量作为基准数据。



公司发现员工用完了所有病假，并且没有结转剩余天数。



## 七、干预措施与效果评估 🏃  



在发现问题之后，人力资源部门决定采取干预措施以改善员工健康状况。



公司开展了一项健康促进活动，通过记录员工每日步数并给予奖励来鼓励运动。



实施后，人力资源部门可以比较项目实施前后的病假天数。



评估逻辑如下：



```text
若 实施后病假天数 < 实施前病假天数 → 项目有效
若 实施后病假天数 ≥ 实施前病假天数 → 需要调整项目
```



如果项目实施后病假减少，说明项目有效。



如果病假数量没有变化或增加，则可能需要对项目进行改进。
