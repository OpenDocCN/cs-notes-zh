# 151：29_示例：确定基本工资

![](img/adcd20e2fbf38318d6f355046ace84ef_0.png)

## 📋 概述
在本节课中，我们将通过一个现实世界的示例，学习如何为一个新设立的职位确定基本工资。我们将跟随人力资源专家Jay的步骤，了解他在为一家披萨连锁店“Slice You”设计一个新的物流管理职位薪酬时，需要考虑的各个因素。

![](img/adcd20e2fbf38318d6f355046ace84ef_2.png)

在前两节课中，我们学习了薪酬、激励和工资公平性。本节中，我们将通过一个具体案例来看看这些概念如何应用。

## 🍕 案例背景：Slice You公司
Slice You是一家制作美味且价格合理的披萨连锁餐厅。该组织专注于大学市场，并希望在尽可能多的大学附近开设分店。

Jay是Slice You总部的一名人力资源专员。最近，物流部经理找到Jay，请求设立一个新职位来协助管理配送工作。目前，这项工作包含在物流经理的职责范围内，但随着公司发展，责任也随之增加。Jay很乐意提供帮助。设计薪酬是他不常有机会做的事情，但这总是一个既具挑战性又有回报的过程。

## 🔍 第一步：进行薪酬基准调查
首先，Jay进行了一项可靠的薪酬基准调查。他查阅了类似职位的招聘信息、行业团体的文章以及从调查中收集的大型数据集。

经过初步研究并与物流团队沟通后，大家一致认为将此职位设为**FLSA豁免的带薪职位**是合理的。现在，Jay需要决定这个职位具有竞争力的薪资是多少。

## ⚖️ 第二步：确定薪酬时需考虑的因素
以下是Jay在确定薪酬时需要考虑的几个关键因素：

*   **外部竞争力**：确保薪酬与其他公司的类似职位相比具有竞争力。
*   **内部公平性**：确保薪酬与Slice You内部的其他职位相比是公平的。
*   **法律合规性**：确保该职位符合所有适用的劳动法规定。

为了考虑申请者不同的经验水平和教育背景，Jay设定了一个**薪酬范围**，而不是一个固定的薪资数字。这样，一位非常适合但经验较少的人可以从薪酬范围的下限开始，而经验更丰富的人则可以从上限开始。

![](img/adcd20e2fbf38318d6f355046ace84ef_4.png)

由于这不是一个按小时计薪的职位，Jay无需担心不同类型的差别工资。但他确实花时间确定了该职位的**总薪酬**。

## 💰 第三步：计算总薪酬
考虑到Slice You提供的额外福利和激励措施，如医疗保险和股票期权，Jay清楚地了解了公司除了基本工资外，将为这个新职位支付的总薪酬。

Jay认为设定的基本工资是公平且具有竞争力的。希望申请该职位的人才将充满热情地为Slice You工作。

![](img/adcd20e2fbf38318d6f355046ace84ef_6.png)

## 📝 总结
本节课中，我们一起学习了确定基本工资的实际步骤。确定基本工资需要进行大量研究，这是组织创建新职位时的重要一步。确定的基本工资是一个起点，但在面试申请者时，通常还会涉及一些谈判。在接下来的课程中，你将了解更多关于激励和福利的内容。
