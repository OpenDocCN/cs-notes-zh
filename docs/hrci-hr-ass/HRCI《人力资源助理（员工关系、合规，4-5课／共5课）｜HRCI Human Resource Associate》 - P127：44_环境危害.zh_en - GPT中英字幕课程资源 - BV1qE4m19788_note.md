## 📘 课程概述  

在本节课中，我们将学习工作场所中的**环境健康危害**。  

在上一阶段的学习中，我们已经了解了人力资源在工作场所健康与安全中的角色，并认识了根据 entity["organization","Occupational Safety and Health Administration","us federal agency"]（OSHA）定义的多种安全隐患，例如物体打击事故以及火灾和爆炸等风险。  

本节课将在此基础上，进一步探讨另一类重要风险——环境健康危害，帮助人力资源专业人员更好地识别与管理这些问题。  



## 🌱 一、什么是环境健康危害  

环境健康危害是指存在于员工工作环境中，**可能对员工健康产生负面影响的任何因素**。  

可以用一个简单公式表示：  

```
环境健康危害 = 工作环境中的因素 + 对员工健康的负面影响
```  

环境危害主要分为三大类：  

![](img/816f8868d26d22cf105e95e020844ae4_2.png)

- 生物危害  
- 化学危害  
- 物理危害  

接下来，我们将逐一进行说明。  



## 🦠 二、生物危害（Biological Hazards）  

在了解总体分类后，本节我们先来看第一类——生物危害。  

生物危害是指可能使员工暴露于**传染性疾病**的特定活动、条件或受污染物体。  

可以表示为：  

```
生物危害 = 受污染物体 / 不良卫生条件 / 可能传播疾病的行为
```  

以下是常见的生物危害类型：  

- 受污染的食物  
- 卫生条件差  
- 被污染的针头刺伤  

员工最常接触到的传染性疾病包括：  

- **乙型肝炎（Hepatitis B）**：一种肝脏疾病，常通过被污染的针头或错误处理体液传播。  
- **HIV（人类免疫缺陷病毒）**：会导致AIDS，可通过被污染的针头或错误处理体液传播。  
- **结核病（Tuberculosis，TB）**：一种肺部疾病，通常通过咳嗽或打喷嚏传播。  



## 🧪 三、化学危害（Chemical Hazards）  

在了解生物危害之后，我们继续讨论第二类——化学危害。  

化学危害是指员工经常接触或存在于工作环境中的物质，这些物质可能导致疾病甚至死亡。  

可以表示为：  

```
化学危害 = 有害化学物质 × 暴露时间
```  

以下是常见的化学危害：  

![](img/816f8868d26d22cf105e95e020844ae4_4.png)

- 有毒化学品  
- 放射性物质  
- 爆炸物  
- 致病污染物（例如煤尘）  

![](img/816f8868d26d22cf105e95e020844ae4_5.png)

这些物质如果管理不当，将对员工健康造成严重威胁。  



![](img/816f8868d26d22cf105e95e020844ae4_7.png)

## ⚙️ 四、物理危害（Physical Hazards）  

在介绍完化学危害后，我们来看最后一类——物理危害。  

物理危害是指可能危及员工安全与健康的**环境条件、设计缺陷或不安全物体**。  

可以表示为：  

```
物理危害 = 不安全环境 / 设计缺陷 / 危险物体
```  

物理危害主要包括三种类型：  



### 1️⃣ 人体工学危害（Ergonomic Hazards）  

人体工学危害通常由重复性动作或不自然姿势引起。  

![](img/816f8868d26d22cf105e95e020844ae4_9.png)

可能导致：  

- 肌肉骨骼损伤  
- 腕管综合征等职业病  



### 2️⃣ 有形危害（Tangible Hazards）  

有形危害是指可能引发事故的不安全物体、条件或程序。  

以下是常见示例：  

- 未安装防护装置的机械设备  
- 湿滑地面  
- 不完善的安全流程  



### 3️⃣ 压力危害（Stress Hazards）  

压力危害来源于极端的情绪或身体压力。  

可能导致：  

- 焦虑  
- 恐慌发作  
- 过度疲劳  
- 心脏问题  

以下情况可能引发压力危害：  

- 长时间或不可预测的工作时间  
- 紧迫的截止期限  
- 员工之间的紧张关系  
- 欺凌行为或人际冲突  



## 💰 五、危险津贴（Hazard Pay）  

在了解各种环境危害后，我们还需要理解与其相关的补偿机制。  

危险津贴是指为补偿员工在工作中面临的**身体和心理风险**而支付的额外报酬。  

![](img/816f8868d26d22cf105e95e020844ae4_11.png)

可以用公式表示为：  

```
总工资 = 基本工资 + 危险津贴
```  

危险津贴通常有两种支付方式：  

- 额外的小时危险补贴  
- 在完成危险任务后一次性发放奖金  



## 🌍 六、环境健康危害的重要性  

![](img/816f8868d26d22cf105e95e020844ae4_13.png)

环境健康危害包括工作场所中的生物、化学和物理风险。  

![](img/816f8868d26d22cf105e95e020844ae4_14.png)

识别并妥善应对这些危害，有助于：  

- 建立安全的工作环境  
- 保护员工健康  
- 提升员工福祉  
- 降低企业法律与合规风险

我们了解了三大类危害：生物危害、化学危害和物理危害，并掌握了它们的典型表现形式与可能后果。同时，我们还学习了危险津贴的概念及其计算方式。
