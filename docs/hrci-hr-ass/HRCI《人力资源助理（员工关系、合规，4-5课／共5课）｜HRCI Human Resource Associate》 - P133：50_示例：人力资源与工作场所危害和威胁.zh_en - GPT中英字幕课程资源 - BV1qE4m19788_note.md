## 🏬 案例背景：披萨连锁企业的安全问题  

  

上一节我们学习了工作场所中的危险与威胁类型。本节中，我们通过一个具体组织的案例来理解这些概念如何在现实中应用。  

  

本案例的组织是 SliceU，一家在大学校园附近拥有多家门店的披萨连锁餐厅。  

  

该公司以制作美味且价格合理的披萨而闻名。  

  

Jay 是在 SliceU 总部工作的人力资源专员。  

  

最近，两家不同门店发生了意外事件，引起了 Jay 的注意。  

  

在这两家门店中，各有一名员工因地面湿滑而摔倒。  

  

幸运的是，两次事件中都没有人员严重受伤。  

  

但 Jay 希望主动解决这一问题，防止类似事件再次发生。  



---  



## ⚠️ 第一步：识别工作场所危险  

  

在明确问题后，本节我们来看 Jay 如何识别危险。  

  

首先，Jay 将湿滑地面识别为一种**有形危险（Tangible Hazard）**。  

  

这种物理危险通常涉及不安全的物体状态或操作流程。  

  

核心概念可以表示为：  

```
Hazard = 不安全的物体条件 + 不安全的操作流程
```  

  

SliceU 一直以清洁为企业自豪。  

  

每晚打烊时，员工都会拖地，这是例行的收尾工作之一。  

  

然而，在两起滑倒事件中，拖地工作是在其他员工仍在完成收尾工作的情况下进行的。  

  

这说明流程安排存在风险。  

  

作为人力资源团队成员，Jay 对这个问题进行了深入思考，并与各门店经理进行了沟通。  

  

Jay 的角色使其有权制定公司范围内的事故预防计划。  



---  



## 🛠️ 第二步：制定预防方案  

  

识别危险之后，本节我们来看解决方案的设计。  

  

Jay 制定了一项新的操作规范，要求：  

```
拖地时间 = 所有其他收尾工作完成之后
```  

  

![](img/dd7b20c888f4285d804817a518cffa6b_4.png)

此外，Jay 为每个门店订购了新的地面干燥风扇。  

  

湿地面无法完全避免，但可以减少其持续时间和风险。  

  

新流程的目标是：  

```
安全性提升 = 优化流程 + 改善设备
```  

  

Jay 还制作了一系列信息通知、传单和海报，向团队成员说明即将实施的变化。  



---  



## 🎨 第三步：提升安全意识  

  

除了制度改进，本节我们来看如何提升员工参与度。  

  

为了提高员工对新安全计划的关注度，Jay 邀请设计团队创建了一个名为 Sppy 的动画角色。  

  

Sppy 被用于所有与该安全计划相关的宣传材料。  

  

![](img/dd7b20c888f4285d804817a518cffa6b_6.png)

Jay 认为有趣的形象可以吸引员工注意。  

  

同时，Sppy 的活力形象也符合 SliceU 的品牌身份。  

  

这一做法体现了：  

```
员工参与度 = 安全信息 + 趣味表达 + 品牌一致性
```  



---  



## 📢 第四步：实施与反馈  

  

在完成方案设计后，本节我们来看执行过程。  

  

Jay 将新材料分发给所有员工，并确保每家门店都张贴了新的拖地操作规范。  

  

员工对新的指导说明和地面风扇表示认可。  

  

有员工在邮件中回复，表达对 Sppy 形象的喜爱。  

  

![](img/dd7b20c888f4285d804817a518cffa6b_8.png)

Jay 相信，这一新计划将有助于预防未来的滑倒事故。  



---  



## 🧩 人力资源的关键职责  

  

通过这个案例，我们可以总结出人力资源在安全管理中的关键作用。  

  

以下是人力资源在安全管理中的核心职责：  

- 识别工作场所危险  
- 评估风险等级  
- 制定预防措施  
- 推动公司范围内的执行  
- 提升员工安全意识  

  

核心逻辑可以概括为：  

```
识别危险 → 制定方案 → 宣传培训 → 执行监督 → 持续改进
```  

  

当员工在工作场所感到安全和被关怀时，他们更有可能保持高效和忠诚。
