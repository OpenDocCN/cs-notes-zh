# HRCI人力资源助理课程：第1课：薪资介绍 💰

![](img/cf1565de52367b0e80312776d2996bff_0.png)

在本节课中，我们将要学习薪资管理的基本概念。薪资是人力资源管理中的一个重要环节，它涉及员工薪酬的计算与发放。

## 薪资职责的组织归属

薪资管理的职责在不同组织中可能有所不同。

以下是几种常见的情况：
*   在一些组织中，薪资任务由人力资源部门完成。
*   在另一些组织中，薪资任务由财务或会计部门履行。
*   有时，薪酬和薪资相关的任务会被外包。

如果任务被外包，人力资源部门的责任是选择并监督服务供应商。

## 薪资记录的法律要求

此外，《公平劳动标准法案》要求将非豁免员工的薪资信息记录在组织的档案中。这项记录保存工作很可能是一项人力资源任务。

上一节我们介绍了薪资职责的归属，本节中我们来看看薪资管理的具体内容。

## 薪资的构成：计算与扣款

薪资管理不仅涉及计算应发工资，还包括管理各项扣款。

扣款主要分为两类：强制性扣款和自愿性扣款。

### 强制性扣款

一些扣款是强制性的，例如社会保障金、医疗保险以及州和联邦所得税。这些费用从员工工资中预扣，并定期提交给美国国税局或其他相关政府机构。代表员工预扣和提交的金额会在每年年底通过W2表格进行报告。

还有其他一些发生频率较低的扣款，例如法院命令或税收扣押。对于这些扣款，会从员工工资中预扣或扣押一部分，以清偿其债务。

工资被扣押的员工受到《消费者信用保护法案》（CCPA）的保护。CCPA禁止雇主解雇因单一债务而被扣押工资的员工，并限制了一个工资周期内可被扣款的金额。

### 自愿性扣款

![](img/cf1565de52367b0e80312776d2996bff_2.png)

除了强制性扣款，雇主可能还会预扣额外的自愿性扣款。自愿性扣款对应的是可选福利，例如401K养老金计划供款、健康保险相关费用和工会会费。

## 课程总结

本节课中我们一起学习了薪资管理的基础知识。总结来说，薪资管理包括计算工资和管理员工工资单上的各项扣款。这些任务的职责可能因组织而异，但人力资源部门很可能是其中的关键参与者。
