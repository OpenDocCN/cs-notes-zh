## 一、OSHA简介 🏛️  



在开始之前，我们先回顾一个重要机构。  

![](img/0a82e9d0658da6165c95dfa8f3f2e6a3_2.png)

你已经了解了**职业安全与健康管理局（OSHA）**，即  

entity["organization","Occupational Safety and Health Administration","us labor agency"]。  

它是美国劳工部下属机构，负责执行健康与安全法规。  

其核心职责可以用公式表示为：  

```text
OSHA职责 = 制定安全标准 + 执行检查 + 违规处罚
```

## 二、现场调查流程 🔍  



上一节我们介绍了OSHA的基本职能，本节我们来看现场调查的具体步骤。  

当OSHA认为有必要调查某个工作场所的安全状况时，会开展**现场调查（Onsite Investigation）**。  

调查遵循固定流程，具体如下：  



### 以下是现场调查的步骤：  



1. **调查员到达现场**  

   合规安全与健康官员（Compliance Safety and Health Officer）到达工作场所，并通知相关负责人将开始调查。  



2. **核实身份与调查范围**  

   雇主可以核实调查员的证件，并了解调查的内容和原因。  



3. **现场检查**  

   调查员通常会与一名管理层成员和一名员工一起检查工作场所，并指出发现的违规情况。  



![](img/0a82e9d0658da6165c95dfa8f3f2e6a3_4.png)

4. **员工陪同限制**  

   由于合规安全与健康官员数量有限，员工并没有陪同调查员的权利。  



### 调查优先原则  

由于人力有限，OSHA会优先调查风险最高的工作场所。  

那么，OSHA如何判断哪些场所风险最高？  

下面我们将介绍调查优先级的划分。



---

## 三、调查优先级划分 📊  



上一节我们提到OSHA会优先处理高风险场所。本节我们来看五个优先等级。  

OSHA按照重要性将调查分为五个等级，从低到高排列如下：  



### 以下是五个调查优先级：  



1. **后续检查（最低优先级）**  

   在已确认违规后进行，用于核实雇主是否已整改并符合相关标准。  



2. **计划性或高危行业检查**  

   针对高危行业或危险活动进行的例行检查。  



3. **员工投诉引发的检查（中等优先级）**  

   因员工投诉而触发的调查。  



4. **灾难事故触发的检查（最高优先级）**  

   当工作场所存在紧迫危险时，优先级最高。  



### 灾难（Catastrophe）的定义  

![](img/0a82e9d0658da6165c95dfa8f3f2e6a3_6.png)

```text
灾难 = 3人或以上住院  或  ≥1人死亡
```  

当满足上述条件时，OSHA将立即展开调查。



---

## 四、非现场调查 🖥️  



在某些情况下，OSHA可能不会进行现场调查，而是改为**非现场调查（Offsite Investigation）**。  

这类调查通常通过电话进行。  



### 非现场调查流程如下：  



1. 调查员通过电话通知雇主涉嫌违规类型。  
2. 雇主必须在5天内书面说明整改方式。  
3. 书面回复可通过邮寄或传真提交。  



### 以下情况可能适用非现场调查：  



- 没有迹象表明员工因违规而处于危险中  
- 工作场所不属于高风险行业  
- 雇主没有违规历史  
- 雇主曾积极配合OSHA要求  



可以表示为：  

```text
非现场调查条件 =
无直接危险
+ 非高风险行业
+ 无违规历史
+ 良好配合记录
```

## 五、六种违规类型与处罚 ⚖️  



了解调查流程后，我们来看违规类型及处罚标准。  

目前，OSHA认定六种违规类型，严重程度逐级增加。  



### 以下是六种违规类型：  



### 1️⃣ 轻微违规（De Minimis Violation）  

- 违反标准  
- 不影响员工健康或安全  
- 仅通知，不开罚单  



### 2️⃣ 非严重违规（Other-than-Serious Violation）  

- 违反标准  
- 影响员工健康或安全  
- 无紧迫危险  

处罚：  

```text
最高罚款 = $7,000 / 每项违规
```  



### 3️⃣ 严重违规（Serious Violation）  

- 存在员工受伤或死亡的紧迫风险  

处罚：  

```text
最高罚款 = $7,000 / 每项违规
```  



![](img/0a82e9d0658da6165c95dfa8f3f2e6a3_8.png)

### 4️⃣ 未整改违规（Failure to Abate Violation）  

- 在整改期限后仍未纠正违规  

处罚：  

```text
最高罚款 = $7,000 × 每超期一天
```  



### 5️⃣ 重复违规（Repeat Violation）  

- 再次违反相同或类似标准  

处罚：  

```text
最高罚款 = $70,000 / 每项违规
```  



### 6️⃣ 故意违规（Willful Violation）  

- 故意忽视或违反OSHA标准  

处罚：  

```text
罚款范围 = $5,000 ～ $70,000
```  

如果违规导致员工死亡，雇主可能面临额外罚款甚至监禁。
