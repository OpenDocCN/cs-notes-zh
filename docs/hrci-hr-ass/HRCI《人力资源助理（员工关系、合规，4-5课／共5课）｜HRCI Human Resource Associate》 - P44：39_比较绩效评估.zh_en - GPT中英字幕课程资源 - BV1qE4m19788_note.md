## 👥 比较绩效评估方法概述



在上一节课中我们介绍了行为式绩效评估方法。本节课中我们来看看另一类方式——**比较绩效评估方法**。  
比较评估的核心是**将每位员工与其他员工进行表现上的比较**，常见方式包括：

- 排名法  
- 两两比较法  
- 强制分布法  
- 要素比较法

以下我们将依次介绍这些方法。



---

## 🥇 排名法（Ranking Method）



排名法会把所有员工按绩效从高到低排列。  

![](img/3dc56db545fa9818a431ec63bbba254f_2.png)

其核心逻辑可表示为：

![](img/3dc56db545fa9818a431ec63bbba254f_3.png)

```
员工绩效顺序：E1 > E2 > E3 > … > En
```

在进入方法细节前，我们先看它的典型特点。以下是排名法的主要要点：

- 适用于员工数量较少的团队  
- 通过排序快速识别最佳与最差表现  
- 简单直接，操作难度低  

例如，Connective 的市场团队有六名平面设计师，HR 团队与部门主管使用排名法将六名设计师按绩效排序，并作出管理决策。

上一节我们学习了排名法的基础逻辑，接下来我们看看如何从两人角度进行比较。



---

## 🔄 两两比较法（Paired Comparison Method）



两两比较法指一次只比较两位员工的绩效，其逻辑如下：

```
比较(Ei, Ej) → 记录表现较好的员工
```

以下是两两比较法的核心特点：

- 每次只对两位员工做出比较  
- 通过多轮比较得到综合评价  
- 适合小团队且员工表现差异明显时使用  

例如，Slicelice U 有三名披萨厨师：Sam、Alex、Devon。比较方式如下：

- Sam 对比 Alex → 记录优胜者  
- Sam 对比 Devon → 记录优胜者  
- Alex 对比 Devon → 记录优胜者  

通过比较结果即可得出整体排序。  
上一节我们讨论了逐对比对方式，本节我们转向一个强调整体分布的方法。



---

## 📉 强制分布法（Forced Distribution Method）



强制分布法（也称强制排序法）要求将员工依照绩效**按正态分布曲线**进行分类：

![](img/3dc56db545fa9818a431ec63bbba254f_5.png)

**绩效人数分布公式示意：**

![](img/3dc56db545fa9818a431ec63bbba254f_6.png)

```
高绩效：10％  
中间绩效：70％  
低绩效：20％
```

以下是强制分布法的应用特点：

- 能减少主管因过度宽松或过度严格产生的偏差  
- 使评价更客观一致  
- 适合人数较多、绩效数据可量化的团队  

例如，Connive 在为规模较大的销售团队做评估时使用强制分布法，因为销售代表人数多并且绩效数据可直接比较。

上一节我们看到了按整体统计规律分类的方法，本节我们来学习更细化、更侧重能力项的方式。



---

## 🧩 要素比较法（Factor Comparison Method）



要素比较法会将员工的表现拆分为多个要素并分别评分。  

公式化表达为：

```
总绩效 = Σ（各绩效要素得分）
```

以下是该方法的主要特点：

- 强调员工在具体能力或工作要素上的表现  
- 更适合具备固定技能要求的岗位  
- 评价结构清晰，可针对性提升  

![](img/3dc56db545fa9818a431ec63bbba254f_8.png)

例如，Urban Attire 使用此方法评估其销售人员。每位销售人员会依照：

![](img/3dc56db545fa9818a431ec63bbba254f_9.png)

- 客户服务技能  
- 店铺职责  
- 工作努力程度  

等要素分别评分，再合成为综合评估结果。

## 🧭 小结：选择适合的比较评估方法



在本节课中我们一起学习了四种比较绩效评估方法：

- 排名法  
- 两两比较法  
- 强制分布法  
- 要素比较法  

![](img/3dc56db545fa9818a431ec63bbba254f_11.png)

每种方法都有其适用场景，而如何选择最适合团队的方式，是 HR 需要不断磨练的专业技能。
