## 一、平等就业机会（EEO）的基本概念 🏛️



![](img/2cc8461dbd989ca7a167260da43b66b4_2.png)

![](img/2cc8461dbd989ca7a167260da43b66b4_3.png)

平等就业机会（EEO）是指一系列立法和政策，要求所有员工在就业过程中受到平等对待，不得因以下因素而受到区别对待：

- 种族（Race）  
- 国籍（National Origin）  
- 年龄（Age）  
- 宗教（Religion）  
- 性别（Sex）  



这一原则来源于《1964年民权法案》第六章（Title VI of the Civil Rights Act of 1964）。该法案设立了平等就业机会委员会（EEOC），负责执行和管理平等就业相关法律。



在了解了EEO的基本定义之后，接下来我们将系统梳理与平等就业机会相关的核心法律概念。



## 二、非法歧视（Illegal Discrimination） 🚫



非法歧视是指基于受法律保护的特征，对员工进行不公平或不公正的对待。这种行为是法律明令禁止的。



常见的受保护特征包括：

- 种族  
- 性别  
- 残疾  



例如，一名身体残疾的员工在获得合理便利后能够胜任工作，但其主管仍对其进行不当对待。



具体行为包括：

- 员工提出在特定工作区域设置轮椅通道的合理请求，主管拒绝该请求。  
- 主管发表贬低残疾的言论，营造敌意工作环境。  



在此情形下，主管的行为构成基于残疾的非法歧视，员工有权提出申诉。



![](img/2cc8461dbd989ca7a167260da43b66b4_5.png)

## 三、受保护群体（Protected Classes） 🛡️



受保护群体是指因共享某一特征而受到法律保护，不得因该特征而遭受歧视或骚扰的群体。



这些特征包括：

- 种族  
- 性别  
- 年龄  
- 残疾  
- 性取向  



美国联邦与州法律均对这些群体提供保护。



例如，若某雇主在招聘过程中持续拒绝具备资格的黑人申请者，仅因其种族原因，这种做法违反了联邦和州关于反种族歧视的法律。



理解受保护群体后，我们进一步区分不同形式的歧视类型。



## 四、差别对待（Disparate Treatment） ⚖️



差别对待是指雇主基于员工的受保护特征，对其进行不同于其他员工的不公平对待。



其核心逻辑可以表示为：  

```
若 A组 ≠ B组 的对待标准  
且 差异原因 = 受保护特征  
则 可能构成差别对待
```



例如，某雇主长期拒绝晋升表现优秀且符合资格的LGBTQI+员工，但却晋升业绩与资历相当的异性恋员工。



此种基于性取向或性别认同的区别对待，构成差别对待。



在理解了“有意区别对待”后，下一节我们将探讨“表面中立但实际不公平”的情况。



## 五、差别影响（Disparate Impact） 📊



差别影响是指某项政策或做法虽然表面上中立，但实际上对某一受保护群体产生了不成比例的负面影响。



![](img/2cc8461dbd989ca7a167260da43b66b4_7.png)

其核心逻辑可表示为：  

```
政策本身 = 中立  
结果影响 ≠ 各群体均等  
且 不利影响集中于受保护群体  
→ 可能构成差别影响
```



例如，某建筑公司因体型、身高或力量因素，招聘的男性人数远多于女性。



即便公司未明确规定排除女性，但该做法实际上削弱了女性的就业机会，可能构成差别影响。



为了判断是否存在差别影响，可以使用“第4/5规则”。



## 六、真实职业资格（BFOQ） 👔



![](img/2cc8461dbd989ca7a167260da43b66b4_9.png)

真实职业资格（Bona Fide Occupational Qualification，BFOQ）是指某些岗位确实需要特定条件，即使这些条件看似具有歧视性，只要与业务运营和核心职责直接相关，就可能合法。



合法判定标准可表示为：  

```
岗位要求 ∈ 核心业务需要  
且 与基本工作职责直接相关  
→ 可认定为合法BFOQ
```



例如，航空公司规定飞行员必须在一定年龄退休。该要求旨在保障飞行安全与运营效率，因此可能构成合法的真实职业资格。



接下来，我们通过量化标准来分析潜在歧视问题。



## 七、第4/5规则（Four-Fifths Rule） 📈



第4/5规则用于分析是否存在潜在就业歧视。



其计算公式为：  

```
选择率 = 录用人数 ÷ 申请人数  

若：
某群体选择率 < 0.8 × 最高选择率群体  

→ 存在潜在差别影响
```



示例：



以下是两个申请群体的录用情况：  

- 无残疾申请者：100人中录用65人 → 选择率 = 65%  
- 有残疾申请者：100人中录用35人 → 选择率 = 35%  



计算对比：  

![](img/2cc8461dbd989ca7a167260da43b66b4_11.png)

```
35% ÷ 65% = 0.54 （54%）
```  



由于54% < 80%，根据第4/5规则，存在潜在对残疾申请者的不利差别影响。



在确认存在差异后，还需通过法律测试进一步判断是否构成违法歧视。



## 八、麦克唐纳-道格拉斯测试（McDonnell Douglas Test） 🧾



麦克唐纳-道格拉斯测试用于判断工作场所是否发生歧视行为。



其逻辑流程可表示为：  

```
步骤1：员工提出初步歧视证据  
步骤2：雇主提供合法、非歧视性理由  
步骤3：员工证明该理由为借口  
```



例如，一名员工申请晋升但未获录用，认为原因与其宗教信仰有关。



员工可提供以下证据：

- 其宗教身份  
- 未被录用的通知邮件  
- 证明另一名具有相同经验、技能和资格的员工被晋升  



随后，雇主必须提供合法理由，例如业绩不佳。



如果雇主提供理由，员工需进一步证明该理由只是掩饰歧视的借口。



![](img/2cc8461dbd989ca7a167260da43b66b4_13.png)

通过该测试，可以系统判断是否存在违法歧视行为。
