So how does the divide and conquer algorithm work？The algorithm comprises two steps with an optional third。

 namely， divide and conquer， which are the mandatory steps and combine， which is an optional step。

In the divide step， the input is split into smaller segments and processed individually。

In the con step， every task associated with a given segment is solved。

The optional last step combined is combining all the solved segments。

 This will not happen in every instance， but will， for the example provided。

Merge sort is a sophisticated approach for sorting an array。It starts by haalvelving the array。
