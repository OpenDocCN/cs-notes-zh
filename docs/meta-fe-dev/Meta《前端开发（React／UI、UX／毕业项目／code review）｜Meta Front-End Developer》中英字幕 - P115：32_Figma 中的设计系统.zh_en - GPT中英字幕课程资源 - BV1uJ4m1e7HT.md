These are auto layout and component features。The first is the auto layout feature。

This essentially converts whatever you are creating into its own frame。

 giving you more control and customization options。

On the online order page， I created some category buttons for the menu items， starters， mains。

 desserts， a la carte and so on。 I created these manually by adjusting the background button shape to fit the text。

This is how I originally created these categories。 I duplicated the starters button。

 and typed in mains。However， the button shape is too big。 Auto layout will rectify this。

 I select the means elements and go to the right hand panel and click the plus icon to create an auto layout。

 and the button shape automatically resizes to the size of the text。Now。

 it is changed into an auto layout frame in the layersers panel。The icon indicates that。

And the properties are in the right sidebar。Now， if I hover over my text element。

 it has an outer box around it。This is the frame。 Now。

 the button has padding or some space between the text and the top。

 bottom and both sides of the text， which is adjustable。

Then I create a component from this button by clicking the Cre component icon at the top frame。

 it resembles four diamonds。This has now created a master component。

You can create instances and variations based on this component。Now。

 let's make instances of this component。 I can duplicate the button by pressing C D on Windows or command D on Mac。

 or I can click and drag on the master button component and press alt as I drag。

 The layers panel shows that a second button was created。 but the icon is a single diamond。

 This tells us that this is an instance of the master button。 I rename it yellow button。😊。

I click on the yellow button instance and change the fill of the frame to yellow。

The instance of only this button changed to yellow。 Now I go to my master component。

I change the radius to 8， and all instances will retain this change。

The component master button is the master。 If I want to change all instances， I do it from here。

 If I want instances to have individual characteristics， I do it inside that particular instance。
