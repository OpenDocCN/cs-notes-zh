To interface with something means interacting or communicating with it。

 you can interface with other humans and you can interface with computers and applications。

People do it every day to communicate with colleagues and friends by email and text messages。

 or with technology， for example， a printer by changing its settings。

A graphical user interface or GUI presents interactive visual components people can interact with on screen to communicate with technology for consistency。

 we refer to it as a UI。
