 especially as we have worked out the correct answer to compare to， In fact。

 if we take this approach， we might apply the ideas of black box testing。

 coming up with test cases for what we think will be difficult at step 1。 That is。

 If we get them all right the first time， that is best。

 Test driven development and is used in industry。 This process meshes naturally with the seven step method as developing those test cases works well for step 1。
