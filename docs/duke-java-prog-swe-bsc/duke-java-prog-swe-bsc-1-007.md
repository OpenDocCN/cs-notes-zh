## 标记语言与所见即所得的区别

你可能习惯于编辑Word文档，在那里你输入文本，所见即所得。

但对于标记语言，你所写的并非你直接得到的。你编写的HTML将以一种更美观的方式显示，即你指定它要显示的方式。
