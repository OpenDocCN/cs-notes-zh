# 018：分析随机化算法的五个基本工具

在本节课中，我们将学习分析随机化算法的五个核心工具。这些工具对于理解算法性能、证明近似算法的保证以及进行概率分析至关重要。我们将从简单的期望线性性开始，逐步深入到更强大的尾不等式，如马尔可夫不等式、切比雪夫不等式和切尔诺夫界，最后介绍并集界。课程将通过哈希表和近似算法等具体例子来阐释这些工具的应用。

## 工具一：期望的线性性

上一节我们介绍了随机变量的基本概念。本节中，我们来看看第一个也是最基础的工具：期望的线性性。

期望的线性性指出，无论随机变量是否独立，其和的期望等于期望的和。这是一个极其强大且应用广泛的工具。

**公式**：
`E[Σᵢ Xᵢ] = Σᵢ E[Xᵢ]`

其强大之处在于，我们可以将一个复杂的随机变量（例如快速排序的比较次数）分解为许多简单随机变量（通常是0-1指示变量）的和。分析复杂随机变量的期望，就简化为分析这些简单随机变量的期望并求和。

### 应用：最大3-SAT问题

以下是最大3-SAT问题的一个简单随机化近似算法分析。

*   **问题描述**：给定一个包含m个子句的3-CNF公式，每个子句包含三个不同的文字。目标是找到一个真值赋值，使得满足的子句数量最大化。
*   **算法**：为每个变量独立、均匀地随机分配真值（True或False）。
*   **分析**：我们关心满足的子句数量的期望值。

**证明**：
1.  定义指示随机变量 `Xⱼ`，当子句 `j` 被满足时为1，否则为0。
2.  对于任意一个包含三个不同文字的子句，只有当其三个变量均被赋予使文字为假的值时，该子句才不被满足。随机赋值下，这种情况发生的概率是 `(1/2)³ = 1/8`。因此，一个子句被满足的概率是 `1 - 1/8 = 7/8`。
3.  根据期望的线性性，满足子句总数的期望 `E[Σⱼ Xⱼ] = Σⱼ E[Xⱼ] = Σⱼ (7/8) = (7/8)m`。
4.  由于最优解 `OPT ≤ m`，该随机算法在期望意义下是一个 `7/8` 近似算法。

这个结果有一个重要的推论：**任何**3-SAT公式都存在一个真值赋值，可以满足至少 `7/8` 的子句。

## 工具二：马尔可夫不等式

期望的线性性适用于分析平均值。但如果我们想要“高概率”的保证，即随机变量以很大概率接近其期望值，就需要尾不等式。马尔可夫不等式是其中最弱但假设最少的一个。

马尔可夫不等式适用于任何非负随机变量。它给出了随机变量超过其期望值某个倍数的概率上界。

**公式**：
对于非负随机变量 `X` 和任意常数 `c > 1`，有 `P(X ≥ c * E[X]) ≤ 1/c`。

### 应用：哈希分析（弱假设）

考虑一个哈希场景：我们有 `n` 个桶和 `n` 个待哈希的元素。我们假设哈希函数满足：**对于任意特定元素，它被映射到任意特定桶的概率是 `1/n`**。我们关心某个特定桶 `i` 的负载（即映射到该桶的元素数量）。

*   **期望**：利用期望线性性，易得 `E[负载] = 1`。
*   **应用马尔可夫不等式**：我们想知道负载远大于期望的概率。例如，负载大于 `n`（即所有元素都哈希到桶 `i`）的概率。
    `P(负载 ≥ n) = P(负载 ≥ n * E[负载]) ≤ 1/n`。

这个界限 `1/n` 在仅假设P1的情况下是紧的（例如，考虑一族总是将所有元素映射到同一个桶的常数哈希函数）。这表明我们需要更强的假设来获得更好的浓度保证。

## 工具三：切比雪夫不等式

为了获得比马尔可夫不等式更强的浓度界限，我们需要关于随机变量的更多信息，特别是方差。切比雪夫不等式利用方差来提供更好的尾部概率控制。

切比雪夫不等式给出了随机变量偏离其期望值超过 `t` 倍标准差的概率上界。

**公式**：
对于随机变量 `X`，其期望 `μ` 和方差 `σ²` 有限，对于任意 `t > 1`，有 `P(|X - μ| ≥ tσ) ≤ 1/t²`。

### 应用：哈希分析（成对独立假设）

现在我们对哈希函数家族加强假设，要求其满足**成对独立性**：对于任意两个不同的元素 `x` 和 `y`，以及任意两个桶 `i` 和 `j`，有 `P(h(x)=i ∧ h(y)=j) = 1/n²`。这比之前的单点均匀性假设强。

*   **期望**：`E[负载] = 1`（同前）。
*   **方差计算**：负载 `X` 是 `n` 个指示变量 `X_y` 的和。在成对独立假设下，方差可加：`Var(X) = Σ Var(X_y)`。每个 `X_y` 是伯努利变量，`Var(X_y) ≤ 1/n`。因此，`Var(X) ≤ n * (1/n) = 1`，标准差 `σ ≤ 1`。
*   **应用切比雪夫不等式**：计算负载大于等于 `n` 的概率。
    `P(负载 ≥ n) = P(负载 - 1 ≥ n-1) ≤ P(|负载 - 1| ≥ n-1) ≤ Var(X)/(n-1)² ≈ 1/n²`。

这个界限 `1/n²` 比马尔可夫不等式给出的 `1/n` 要好，但在实际哈希表场景中仍然显得较弱。同样，在仅假设P1和P2（成对独立）下，这个界限也可以是紧的。

## 工具四：切尔诺夫界

当我们拥有最强的独立性假设时，可以获得指数级小的尾部概率，这是通过切尔诺夫界实现的。切尔诺夫界适用于独立随机变量的和，并且要求每个变量有界。

切尔诺夫界有两种形式，分别控制随机变量超过或低于其期望值的概率。这里我们主要关注上界。

**公式（上界）**：
设 `X = Σ X_j`，其中 `X_j ∈ [0, 1]` 且相互独立。对于任意 `δ > 0`，有：
`P(X ≥ (1+δ)E[X]) ≤ [e^δ / (1+δ)^(1+δ)]^(E[X])`。

**公式（下界，简化版）**：
`P(X ≤ (1-δ)E[X]) ≤ e^(-δ²E[X]/2)`。

关键点在于，界限是**指数形式**的。只要偏差 `δ` 或期望 `E[X]` 足够大，概率就会变得非常小。

### 应用：哈希分析（完全独立假设）

现在假设哈希函数是完全随机的：每个元素独立且均匀地映射到各个桶。此时，桶 `i` 的负载 `X` 是 `n` 个独立的0-1指示变量的和。

*   **期望**：`E[X] = 1`。
*   **应用切尔诺夫界**：
    1.  计算负载超过 `log n` 的概率。取 `1+δ = log n`，则 `δ ≈ log n - 1`。代入上界公式，得到的概率上界远小于任何多项式 `1/n^d`，实际上趋于零的速度比多项式逆更快。
    2.  计算负载超过 `(3 log n)/(log log n)` 的概率。代入计算可得，该概率至多为 `1/n²`。
    3.  若计算负载超过 `n`（所有元素进一个桶）的概率，切尔诺夫界会给出一个像 `1/2^n` 这样的指数小概率，这更符合我们对良好哈希函数的直觉。

### 应用扩展：负载均衡与随机舍入

切尔诺夫界在负载均衡和随机舍入算法中非常有用。

*   **重负载哈希**：如果哈希 `n log n` 个元素到 `n` 个桶，则每个桶的期望负载为 `log n`。此时，即使取一个较小的常数倍偏差（如 `δ=3`，即负载不超过 `4 log n`），由于 `E[X]` 较大，切尔诺夫界也能给出很强的浓度保证（例如概率小于 `1/n²`）。这表明在适度负载下，随机分配能实现很好的均衡。
*   **随机舍入（多商品流）**：在近似算法中，我们常先求解问题的线性规划（LP）松弛，得到分数解。随机舍入将这些分数解释为概率，从而随机地将解“舍入”为整数解。
    *   **问题**：在图中为 `k` 个源-汇对寻找路径，使得共用同一条边的路径数尽可能少（最小化最大边拥塞）。
    *   **算法**：
        1.  求解多商品流LP松弛（每条边容量为1，为每对发送1单位流）。
        2.  若LP不可行，则原问题也无解。
        3.  若LP可行，则对每个商品 `i`，根据LP解中从 `s_i` 到 `t_i` 各路径上的流量比例，随机选择一条路径。
    *   **分析**：对于任意一条边，根据LP约束，流经它的总流量（期望负载）至多为1。各商品路径的选择是独立的。因此，可以使用切尔诺夫界证明：**以高概率，没有边会被超过 `O(log n / log log n)` 条路径使用**。这是一个经典的应用，将哈希分析的思想迁移到了算法设计领域。

## 工具五：并集界

并集界是一个简单但极其常用的工具，用于控制多个“坏事件”中至少发生一个的概率。

并集界指出，多个事件并集的概率不超过它们各自概率之和。

**公式**：
对于任意事件 `E₁, E₂, ..., E_k`，有 `P(∪ᵢ Eᵢ) ≤ Σᵢ P(Eᵢ)`。

### 应用：从单一边界到全局边界

在哈希和随机舍入的分析中，我们通常先针对**单个**桶或边得到一个高概率的优良性质（例如“桶 `i` 负载过高”的概率很小）。但我们关心的是**所有**桶或边的情况。

并集界使我们能够将针对单个对象的概率界限推广到整体。例如：
*   在哈希分析中，如果每个桶负载过高的概率 ≤ `1/n²`，那么通过并集界，**存在**某个桶负载过高的概率 ≤ `n * (1/n²) = 1/n`。因此，**所有**桶负载都良好的概率至少为 `1 - 1/n`。
*   在随机舍入中，如果每条边拥塞过高的概率 ≤ `1/m²`（`m` 为边数），那么存在边拥塞过高的概率 ≤ `m * (1/m²) = 1/m`，从而以高概率得到全局优良解。

## 总结

本节课中我们一起学习了分析随机化算法的五个基本工具：
1.  **期望的线性性**：分析和的期望，无需独立性假设。
2.  **马尔可夫不等式**：基于期望的非负随机变量尾部概率上界，假设最少。
3.  **切比雪夫不等式**：利用方差提供更好的浓度保证，适用于（成对）独立变量和。
4.  **切尔诺夫界**：为独立有界随机变量和提供指数级强的尾部概率界限，是证明“高概率”保证的利器。
5.  **并集界**：将针对单个事件的概率界限联合起来，得到全局保证。

这些工具构成了分析随机化算法性能的理论基础，在算法设计、机器学习、网络优化等领域都有广泛应用。从简单的随机赋值算法到复杂的随机舍入方案，它们帮助我们理解、量化并证明随机化带来的好处。
