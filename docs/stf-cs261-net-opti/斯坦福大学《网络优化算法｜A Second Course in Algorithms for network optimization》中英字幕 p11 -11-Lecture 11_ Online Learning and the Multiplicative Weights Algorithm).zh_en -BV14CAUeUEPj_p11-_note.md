# 网络优化算法：11：在线学习与乘性权重算法

在本节课中，我们将要学习在线学习的基本概念，并深入探讨一个著名的算法——乘性权重更新方法。我们将看到，这个算法不仅本身非常强大，而且其思想还能用于推导线性规划和极小极大定理的相关结论。

## 概述

在线问题是指算法的输入是逐步到达的，而非一次性全部给出。算法必须在每个时间步做出不可撤销的决策，而决策时只能基于过去的信息。本节课我们将介绍一个简单的在线决策模型，并学习如何通过乘性权重算法来最小化“遗憾”。

## 在线问题与遗憾最小化

在线问题的核心挑战在于，算法必须在信息不完全的情况下做出决策。例如，调度作业或处理动态到达的图节点。这使得问题比离线版本困难得多。

为了衡量在线算法的性能，我们引入“遗憾”的概念。遗憾是指算法获得的累计奖励与某个基准策略（例如，事后看来最好的单一固定动作）所获奖励之间的差距。我们的目标是设计算法，使遗憾尽可能小。

## 算法性能的局限性

在深入算法之前，我们需要了解性能的理论上限。

1.  **无法与全知基准竞争**：我们无法与一个知道所有未来奖励的“先知”算法竞争。对手可以构造奖励序列，使得任何在线算法的期望奖励远低于先知算法。
2.  **确定性算法不够好**：任何确定性算法都可能被对手针对，导致线性遗憾。因此，随机化是必要的。
3.  **随机算法的下界**：即使使用随机化算法，期望遗憾的下界是 **Ω(√(T log N))**，其中 T 是时间步数，N 是动作数量。这意味着任何算法都无法避免这个量级的遗憾。

## 乘性权重算法

乘性权重算法是一个简单而强大的随机化算法，它恰好能达到 **O(√(T log N))** 的期望遗憾上界，因此是最优的。

该算法基于两个原则：
1.  根据动作过去的表现来调整选择它的概率：表现越好，概率越高。
2.  对表现差的动作进行“指数级”惩罚：迅速降低其被选中的概率。

以下是算法的具体步骤：

**初始化**：
为每个动作 `i` 分配初始权重 `w_i(1) = 1`。

**循环（对于每个时间步 t = 1 到 T）**：
1.  **选择动作**：以与权重成比例的概率随机选择一个动作 `a_t`。即，选择动作 `i` 的概率为：
    `P_t(i) = w_i(t) / (Σ_j w_j(t))`
2.  **接收奖励**：观察到奖励向量 `r_t`，其中 `r_t(i)` 是动作 `i` 在时间步 `t` 的奖励（在 -1 到 1 之间）。
3.  **更新权重**：对于每个动作 `i`，更新其权重：
    `w_i(t+1) = w_i(t) * (1 + η * r_t(i))`
    其中 `η` 是一个介于 0 和 1/2 之间的学习率参数，稍后将确定其最优值。

## 算法分析

我们将证明乘性权重算法的期望遗憾上界为 **O(√(T log N))**。证明的关键是引入一个中间量：所有权重的和 `Φ(t) = Σ_i w_i(t)`。

**证明分为两部分**：

**第一部分：将权重和与算法奖励关联**
通过分析权重更新规则，我们可以推导出以下关系：
`Φ(t+1) = Φ(t) * (1 + η * (算法在时间步t的期望奖励))`
通过归纳和利用不等式 `1+x ≤ e^x`，我们最终得到：
`Φ(T) ≤ N * exp(η * Σ_t (算法在时间步t的期望奖励))`

**第二部分：将权重和与最佳固定动作关联**
设 `a*` 为事后看来最好的固定动作，其总奖励为 `OPT`。所有权重的和至少包含动作 `a*` 的权重：
`Φ(T) ≥ w_a*(T) = 初始权重 * Π_t (1 + η * r_t(a*))`
利用泰勒展开和对数不等式 `ln(1+x) ≥ x - x^2 (当 |x| ≤ 1/2)`，我们可以得到：
`w_a*(T) ≥ exp(η * OPT - η^2 * T)`

**结合两部分**：
将两部分的不等式结合，取对数并整理，我们得到：
`算法的总期望奖励 ≥ OPT - ηT - (ln N)/η`
选择 `η = √((ln N)/T)` 来平衡误差项，即可得到：
`算法的总期望奖励 ≥ OPT - 2√(T ln N)`
因此，期望遗憾（`OPT - 算法奖励`）的上界为 **2√(T ln N)**。

## 时间平均遗憾

将总遗憾上界除以时间步数 T，我们得到时间平均遗憾上界：
`平均遗憾 ≤ 2√((ln N)/T)`
这意味着，如果我们希望平均每步遗憾不超过 `ε`，只需要运行算法约 `T = O((ln N)/ε^2)` 步即可。

## 应用：极小极大定理的新证明

乘性权重算法不仅本身有用，还能为经典结论提供新的证明视角。例如，我们可以用它来证明博弈论中的**极小极大定理**。

**定理陈述**：
对于一个 m×n 的收益矩阵 A，行玩家和列玩家在允许随机化策略时，有：
`max_x min_y (x^T A y) = min_y max_x (x^T A y)`
其中 x 和 y 分别是行和列上的概率分布。

**证明思路（利用乘性权重）**：
1.  设想行玩家和列玩家独立地使用乘性权重算法来选择混合策略，进行 T 轮重复博弈。
2.  对于行玩家，其奖励向量定义为 `A * q_t`（q_t 是列玩家在 t 轮的策略）。
3.  对于列玩家，其奖励向量定义为 `-p_t^T * A`（p_t 是行玩家在 t 轮的策略，负号因为其收益与行玩家相反）。
4.  令 `x̂` 和 `ŷ` 分别为行、列玩家 T 轮策略的平均。
5.  根据乘性权重算法的遗憾界，行玩家的平均收益 `v` 满足：对于任意纯策略（行）i，有 `v ≥ (e_i^T A ŷ) - ε`。这意味着 `v ≥ max_x (x^T A ŷ) - ε`。
6.  同理，对于列玩家（其目标是使行玩家收益最小），有 `v ≤ min_y (x̂^T A y) + ε`。
7.  结合两个不等式，得到 `max_x min_y (x^T A y) ≥ min_y max_x (x^T A y) - 2ε`。
8.  由于 `ε` 可以任意小，且反方向不等式 `max_x min_y (x^T A y) ≤ min_y max_x (x^T A y)` 总是成立，因此两者相等。这就完成了极小极大定理的证明。

这个证明展示了在线学习理论与经典优化、博弈论之间的深刻联系。

## 总结

本节课我们一起学习了在线学习的基本框架和核心挑战。我们介绍了“遗憾最小化”的目标，并探讨了算法性能的理论极限。随后，我们详细介绍了**乘性权重算法**，它通过维护和更新动作的权重，以概率方式选择动作，实现了最优的遗憾上界 **O(√(T log N))**。最后，我们看到了这个强大算法的一个精彩应用：为博弈论中的**极小极大定理**提供了一个简洁而富有洞察力的新证明。这体现了不同算法思想之间相互启发和增强的力量。
