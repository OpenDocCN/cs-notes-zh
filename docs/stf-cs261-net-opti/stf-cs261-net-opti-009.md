# 009：线性规划对偶理论 — 第二部分

在本节课中，我们将继续深入学习线性规划对偶理论。我们将回顾互补松弛条件，并通过二分图匹配等具体案例来理解其对偶的实际应用。最后，我们将探讨强对偶定理，并了解其证明背后的核心思想。

## 课程回顾与互补松弛条件

上一节我们介绍了线性规划对偶的基本概念和弱对偶定理。本节中，我们来看看互补松弛条件及其意义。

互补松弛条件如下：假设有原始问题和对偶问题的可行解 `x` 和 `y`。如果满足以下两个条件，则 `x` 和 `y` 都是各自问题的最优解：
1.  原始变量 `xj` 只有在对应的对偶约束为紧（即等式成立）时才非零。
2.  对偶变量 `yi` 只有在对应的原始约束为紧时才非零。

互补松弛条件的证明基于弱对偶的推导过程。如果条件1和2成立，那么弱对偶证明中的两个不等式实际上都变成了等式，从而原始目标函数值等于对偶目标函数值。根据之前的推论，这证明了 `x` 和 `y` 都是最优解。

## 互补松弛条件的直观理解

我们可以通过一个物理比喻来直观理解互补松弛条件。将线性规划的目标函数视为沿方向 `c` 施加的力，将约束视为可行域的边界墙。在最优解 `x*` 处，尽管目标函数仍在施加力，但解保持静止，这意味着所有作用力之和为零。

对偶变量 `y` 可以解释为这些约束墙所施加的反作用力的大小。互补松弛条件（条件2）表明，只有那些与最优解 `x*` 接触（即紧约束）的边界墙才能施加力。不接触的约束（非紧约束）不能施加任何力。因此，只有紧约束对应的对偶变量才可能非零。

## 互补松弛条件与算法设计范式

互补松弛条件为设计求解线性规划及其特例（如网络流、匹配）的算法提供了一个通用范式。该范式基于三个共同保证最优性的条件：原始可行性、对偶可行性和互补松弛性。

通用策略是：在算法中始终维持其中两个条件作为不变式，初始时违反第三个条件，然后逐步工作以恢复第三个条件。这产生了三种主要的算法设计思路：
*   维持原始可行性和对偶可行性，逐步满足互补松弛性。
*   维持原始可行性和互补松弛性，逐步满足对偶可行性。
*   维持对偶可行性和互补松弛性，逐步满足原始可行性。

## 案例分析：最小成本完美二分图匹配

现在，我们通过一个熟悉的例子——最小成本完美二分图匹配，来具体应用对偶理论和互补松弛条件。

### 原始线性规划

首先，我们将问题表述为线性规划。对于二分图 `G=(V, W, E)`，决策变量 `xe` 表示边 `e` 是否被选入匹配（理想值为0或1）。目标是最小化总成本 `∑ ce * xe`。约束条件是每个顶点恰好与一条边关联：

```
最小化 ∑ ce * xe
约束条件：
   对于所有 v ∈ V: ∑ xe = 1 (e ∈ δ(v))
   对于所有 w ∈ W: ∑ xe = 1 (e ∈ δ(w))
   对于所有 e ∈ E: xe ≥ 0
```

### 对偶线性规划

应用对偶变换规则，我们得到对偶问题。对偶变量 `pv` 和 `pw` 对应于原始问题中每个顶点的约束（因为原始约束是等式，所以对偶变量无符号限制）。对偶问题如下：

```
最大化 ∑ pv + ∑ pw
约束条件：
   对于所有边 e=(v,w): pv + pw ≤ ce
```

这正是匈牙利算法中使用的顶点价格（`p`）。对偶可行性条件 `pv + pw ≤ ce` 等价于匈牙利算法中的第一个不变式：所有边的约化成本非负。

### 互补松弛条件与匈牙利算法

对于这个原始-对偶对，互补松弛条件意味着：如果一条边 `e` 在匹配中（即 `xe > 0`），那么对应的对偶约束必须是紧的，即 `pv + pw = ce`。这正是匈牙利算法的第二个不变式：匹配中的边必须是“紧边”（约化成本为零）。

因此，匈牙利算法完美地体现了之前提到的算法设计范式：
*   **维持的不变式**：对偶可行性（所有约化成本 ≥ 0）和互补松弛性（匹配中的边是紧边）。
*   **逐步实现的目标**：原始可行性（得到一个完美匹配）。

当算法终止时，它同时满足了原始可行性、对偶可行性和互补松弛性，从而保证了找到的解是最优的。

### 重要推论

通过对匈牙利算法正确性的分析，我们可以得出关于该线性规划的两个重要结论：
1.  **强对偶成立**：对于此类匹配问题的线性规划及其对偶，最优目标函数值总是相等，没有对偶间隙。
2.  **整数解最优性**：该线性规划总是存在一个0-1最优解（即一个完美的二分图匹配）。分数解不会提供更好的目标值。

## 强对偶定理

强对偶定理是线性规划理论的核心结果。它指出：对于任何一对互为对偶的线性规划，如果两者都是可行的，那么它们的最优目标函数值必定相等。

**定理（强对偶）**：设 `P` 和 `D` 是一对原始-对偶线性规划。如果 `P` 和 `D` 都可行，则 `P` 的最优值等于 `D` 的最优值。

强对偶定理推广了诸如最大流-最小割定理、霍尔定理等许多组合优化中的最小最大定理。从强对偶可以推导出最优性条件：原始解 `x` 和对偶解 `y` 都是最优的，**当且仅当**它们可行且目标函数值相等（或等价地，满足互补松弛条件）。

## 强对偶证明思路

强对偶的完整证明较为复杂，但我们可以借助两个关键概念来理解其核心思想：分离超平面定理和法卡斯引理。

### 分离超平面定理

分离超平面定理是一个直观的几何事实：给定一个**闭凸集** `C` 和集外一点 `z`，总存在一个超平面将 `z` 和 `C` 分离。即存在向量 `α` 和标量 `β`，使得对于所有 `x ∈ C` 有 `α^T x ≥ β`，而对 `z` 有 `α^T z < β`。

### 法卡斯引理

法卡斯引理可以看作是强对偶定理的“可行性版本”。它关注线性系统的可行性证明。

考虑以下线性系统（系统1）：
```
存在 x ≥ 0，使得 A x = b
```

法卡斯引理断言：**系统1要么可行，要么存在一个“不可行性证明” `y`（系统2），但两者不能同时成立。**
系统2为：
```
存在 y，使得 y^T A ≥ 0 且 y^T b < 0
```

**理解**：如果系统1不可行，你总可以找到一组行乘子 `y`，通过线性组合约束得到矛盾（`y^T A ≥ 0` 但 `y^T b < 0`），从而证明其不可行。这类似于对偶解为原始问题提供上界（或下界）证明。

### 从分离超平面到法卡斯引理

法卡斯引理的证明巧妙地运用了分离超平面定理：
1.  定义集合 `Q = {A x | x ≥ 0}`，即所有能使系统1可行的右端项 `b` 的集合。`Q` 是一个闭凸锥。
2.  如果系统1对给定的 `b` 不可行，则 `b ∉ Q`。
3.  应用分离超平面定理：存在超平面分离 `b` 和 `Q`。设其法向量为 `α`，可以调整使得分离条件为：对所有 `d ∈ Q` 有 `α^T d ≥ 0`，但对 `b` 有 `α^T b < 0`。
4.  验证 `α` 满足系统2的条件：
    *   `α^T b < 0` 直接满足。
    *   由于 `A` 的每一列都属于 `Q`（取 `x` 为单位向量），因此 `α^T (A的每一列) ≥ 0`，这意味着 `α^T A ≥ 0`。
5.  因此，`α` 就是我们要找的证明不可行的 `y`。

这样，我们就用法卡斯引理证明了：如果一个线性系统不可行，总存在一个特定形式的证明（行乘子 `y`）来验证这一点。

### 从法卡斯引理到强对偶

虽然我们在此不展开严格推导，但强对偶定理可以通过类似法卡斯引理的思路，或者直接用法卡斯引理来证明。基本思想是：如果原始和对偶都可行但存在对偶间隙，可以构造出一个新的线性系统，并用法卡斯引理导出矛盾。这将在习题集中进一步探讨。

## 总结

本节课中我们一起学习了：
1.  **互补松弛条件**及其作为最优性充要条件的意义。
2.  互补松弛条件如何为算法设计提供通用范式，并以**匈牙利算法**作为经典案例，说明其如何维持对偶可行性与互补松弛性，最终达成原始可行性以获得最优匹配。
3.  **强对偶定理**的陈述及其重要性：对于可行的原始-对偶对，最优值相等，无对偶间隙。
4.  强对偶定理证明的核心思想，包括**分离超平面定理**和**法卡斯引理**。法卡斯引理保证了线性系统不可行时，总存在一个由行乘子构成的“证明”，这体现了与强对偶相似的精神——对偶解总是足够好的界限证明。

理解这些概念为我们分析和设计网络优化算法提供了强大的理论基础。
