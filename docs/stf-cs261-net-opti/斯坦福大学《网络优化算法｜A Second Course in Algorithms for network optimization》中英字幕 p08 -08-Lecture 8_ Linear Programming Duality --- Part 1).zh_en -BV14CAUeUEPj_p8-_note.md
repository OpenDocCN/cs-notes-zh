# 斯坦福大学《网络优化算法》课程笔记：第8讲：线性规划对偶理论 — 第一部分

在本节课中，我们将学习线性规划对偶理论的基础知识。对偶理论是理解线性规划的核心，它为我们提供了一种证明解最优性的强大方法。我们将从一个简单的例子开始，逐步推导出对偶线性规划的一般形式，并将其应用于最大流问题，以展示其实际应用。

---

## 线性规划回顾

上一节我们介绍了线性规划的基本概念。线性规划包含决策变量、线性约束（可以是等式或不等式）以及一个需要最大化或最小化的线性目标函数。

考虑以下线性规划示例：
- 最大化：`x1 + x2`
- 约束条件：
    1. `4x1 + x2 ≤ 2`
    2. `x1 + 2x2 ≤ 1`
    3. `x1 ≥ 0`
    4. `x2 ≥ 0`

该问题的可行解区域是四个半平面的交集。最优解是使目标函数 `x1 + x2` 最大的点，即尽可能向东北方向移动。通过观察，我们可以找到最优解为 `(3/7, 2/7)`，对应的目标函数值为 `5/7`。

---

## 推导对偶线性规划

本节中，我们将探讨如何为线性规划问题推导出对偶问题。对偶问题为我们提供了原问题最优值的上界。

### 上界推导

对于上述线性规划，我们如何证明其最优值不会超过某个特定值？以下是几种方法：

1.  **使用单个约束**：
    - 由于 `x1 + x2 ≤ 4x1 + x2`（因为 `x1 ≥ 0`），且根据第一个约束 `4x1 + x2 ≤ 2`，我们可以得出 `x1 + x2 ≤ 2`。这是一个上界，但可能不够紧。

2.  **使用另一个约束**：
    - 根据第二个约束 `x1 + 2x2 ≤ 1`，我们可以得出 `x1 + x2 ≤ 1`。这个上界比第一个更紧。

3.  **组合约束**：
    - 我们可以将两个约束线性组合。例如，取 `(1/7)*(4x1 + x2) + (3/7)*(x1 + 2x2)`。通过代数运算，这等于 `x1 + x2`。
    - 由于 `4x1 + x2 ≤ 2` 且 `x1 + 2x2 ≤ 1`，我们可以得出 `x1 + x2 ≤ (1/7)*2 + (3/7)*1 = 5/7`。
    - 这个上界 `5/7` 是紧的，因为我们已知一个可行解 `(3/7, 2/7)` 的目标函数值正好是 `5/7`。

### 一般形式推导

现在，我们将上述思想推广到一般形式的线性规划。考虑以下原问题（P）：
- 最大化：`c^T x`
- 约束条件：`A x ≤ b`
- 非负约束：`x ≥ 0`

其中，`c` 和 `x` 是 n 维向量，`A` 是 m×n 矩阵，`b` 是 m 维向量。

为了推导上界，我们引入对偶变量 `y`（m 维向量），并考虑线性组合 `y^T A x`。我们希望这个组合能“控制”原目标函数 `c^T x`。具体来说，我们需要：
```
A^T y ≥ c
```
并且 `y ≥ 0`。

如果上述条件满足，那么对于任何可行解 `x`，我们有：
```
c^T x ≤ (A^T y)^T x = y^T A x ≤ y^T b
```
这里用到了 `x ≥ 0`、`A x ≤ b` 和 `y ≥ 0`。

因此，`y^T b` 是原问题目标值的一个上界。为了得到最紧的上界，我们应最小化 `y^T b`。这就引出了对偶问题（D）：
- 最小化：`b^T y`
- 约束条件：`A^T y ≥ c`
- 非负约束：`y ≥ 0`

### 弱对偶性

由上述构造可知，对于任何原问题可行解 `x` 和对偶问题可行解 `y`，都有：
```
c^T x ≤ b^T y
```
因此，原问题的最优值不超过对偶问题的最优值。这称为**弱对偶性**。

## 应用于最大流问题

本节我们将对偶理论应用于最大流问题，展示其实际意义。

### 最大流的路径形式线性规划

考虑最大流问题的另一种线性规划形式，其中决策变量 `f_P` 表示流在每条从源点 `s` 到汇点 `t` 的路径 `P` 上的流量：
- 最大化：`∑_P f_P` （总流量）
- 约束条件：对于每条边 `e`，`∑_{P: e∈P} f_P ≤ u_e` （容量约束）
- 非负约束：`f_P ≥ 0`

### 推导对偶问题

根据对偶的一般规则，我们为每个原约束（即每条边 `e`）引入对偶变量 `l_e`（可以理解为边 `e` 的长度或“价格”）。对偶问题为：
- 最小化：`∑_e u_e * l_e`
- 约束条件：对于每条 `s-t` 路径 `P`，`∑_{e∈P} l_e ≥ 1`
- 非负约束：`l_e ≥ 0`

### 对偶问题的解释

对偶问题可以这样理解：我们为每条边分配非负长度 `l_e`，要求任意 `s-t` 路径的总长度至少为 1（即 `s` 和 `t` 被足够“分开”），同时最小化加权总长度 `∑ u_e * l_e`（可以看作网络的总“体积”）。

### 与最小割的联系

有趣的是，任何 `s-t` 割 `(A, B)` 都自然地对应一个对偶可行解：
- 对于所有从 `A` 指向 `B` 的边 `e`，设 `l_e = 1`。
- 对于其他边，设 `l_e = 0`。

由于任何 `s-t` 路径都必须穿过割 `(A, B)`，因此路径长度约束 `∑_{e∈P} l_e ≥ 1` 自动满足。此时，对偶目标值 `∑ u_e * l_e` 正好等于割 `(A, B)` 的容量。

### 对偶性的意义

根据弱对偶性，我们有：
```
最大流值 ≤ 对偶最优值 ≤ 最小割容量
```
然而，根据最大流最小割定理，实际上等号成立：
```
最大流值 = 对偶最优值 = 最小割容量
```
这说明了两个重要事实：
1.  **强对偶性**：对于这个特定的线性规划及其对偶，最优值之间没有间隙。
2.  **整数最优解**：尽管线性规划允许分数解，但此对偶问题总存在一个最优解是 `0/1` 的，对应一个最小割。这意味着分数解并不比整数解更好。

## 一般线性规划的对偶规则

上一节我们针对特定形式的线性规划推导了对偶。本节我们给出适用于任何线性规划的一般对偶规则。

对于**最大化**原问题（P），其对偶问题（D）的转换规则如下：

| 原问题 (P) | 对偶问题 (D) |
| :--- | :--- |
| 最大化 `c^T x` | 最小化 `b^T y` |
| 约束矩阵 `A` | 约束矩阵 `A^T` |
| 第 i 个约束：`≤ b_i` | 对偶变量 `y_i ≥ 0` |
| 第 i 个约束：`≥ b_i` | 对偶变量 `y_i ≤ 0` |
| 第 i 个约束：`= b_i` | 对偶变量 `y_i` 无约束 |
| 变量 `x_j ≥ 0` | 第 j 个约束：`≥ c_j` |
| 变量 `x_j ≤ 0` | 第 j 个约束：`≤ c_j` |
| 变量 `x_j` 无约束 | 第 j 个约束：`= c_j` |

对于**最小化**原问题，只需将上述规则中的“最大化”与“最小化”互换，并将不等号方向全部反转即可。对偶的对偶即是原问题。

弱对偶性在所有情况下都成立：对于可行解 `x` 和 `y`，有 `c^T x ≤ b^T y`（最大化原问题）或 `c^T x ≥ b^T y`（最小化原问题）。

---

## 弱对偶性的推论

根据弱对偶性，我们可以得出一些有用的推论：

1.  如果原问题（最大化）的目标值无上界（可无限增大），则对偶问题**不可行**（无解）。因为任何对偶可行解都会给出一个有限的上界。
2.  如果对偶问题（最小化）的目标值无下界（可无限减小），则原问题**不可行**。
3.  如果找到原问题可行解 `x` 和对偶问题可行解 `y`，使得 `c^T x = b^T y`，那么 `x` 和 `y` 分别是原问题和对偶问题的**最优解**。这是证明最优性的一个有力证书。

---

## 互补松弛条件

互补松弛条件是上述推论3的进一步细化，它提供了检验最优性的具体条件。

假设 `x` 和 `y` 分别是原问题和对偶问题的可行解。互补松弛条件如下：

1.  对于每个原变量 `x_j`，如果 `x_j > 0`，则在对偶中对应的第 `j` 个约束必须取等号（即“紧”）。
2.  对于每个对偶变量 `y_i`，如果 `y_i > 0`，则在原问题中对应的第 `i` 个约束必须取等号。

如果 `x` 和 `y` 满足互补松弛条件，那么它们分别是原问题和对偶问题的最优解。

例如，在最大流问题中，互补松弛条件对应于：在最小割中，所有前向边都是饱和的（流量等于容量），所有后向边的流量为零。这正是福特-富尔克森算法终止时找到的流和割所满足的条件。

本节课中我们一起学习了线性规划对偶理论的第一部分。我们从具体例子出发，理解了如何通过组合约束来获得原问题最优值的上界，并由此形式化地定义了对偶线性规划。我们证明了弱对偶性，并看到了它在最大流最小割定理中的体现。最后，我们介绍了一般线性规划的对偶转换规则、弱对偶性的推论以及互补松弛条件。对偶理论为我们理解和证明算法的最优性提供了统一而强大的框架。在下节课中，我们将深入探讨强对偶性及其证明。
