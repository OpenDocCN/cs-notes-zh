# 计算机科学的数学基础：L2.7.1：偏序关系 📚

![](img/559c2df900d71d932b56ddb44e816727_1.png)

在本节课中，我们将学习**偏序关系**。偏序关系是描述有向图的另一种方式，它为我们提供了一个关于**数学结构公理化**和**数学思想**的有趣案例。我们将从讨论用于公理化偏序关系和有向图的一些基本属性开始。

![](img/559c2df900d71d932b56ddb44e816727_3.png)

## 传递性

上一节我们介绍了有向图中的路径概念。本节中，我们来看看路径的一个基本属性：**传递性**。

![](img/559c2df900d71d932b56ddb44e816727_5.png)

考虑有向图中的路径。如果存在一条从顶点 `u` 到顶点 `v` 的路径，以及一条从顶点 `v` 到顶点 `w` 的路径，那么将这两条路径连接起来，就得到了一条从 `u` 到 `w` 的路径。

用有向图 `G` 的正路径关系 `G+` 来表达，这意味着：如果 `u G+ v` 且 `v G+ w`，那么 `u G+ w`。

将这个属性抽象到任意关系上，就得到了**传递性**的定义。对于一个定义在集合 `A` 上的关系 `R`（即 `R` 的定义域和陪域相同），如果满足 `u R v` 且 `v R w` 能推出 `u R w`，那么这个关系就是**传递的**。

![](img/559c2df900d71d932b56ddb44e816727_7.png)

我们刚刚看到，任何有向图 `G` 的正路径关系 `G+` 都是传递的。另一种理解传递性的方式是：`u R v` 表示存在一条从 `u` 到 `v` 的边。传递性意味着，如果存在从 `u` 到 `v` 的边和从 `v` 到 `w` 的边，那么也必须存在一条从 `u` 到 `w` 的边。换句话说，如果存在一条长度为2的路径，那么就必须存在一条长度为1的路径。通过简单的归纳法可以得出，如果两个顶点之间存在任何正长度的路径，那么它们之间也存在一条长度为1的路径。

![](img/559c2df900d71d932b56ddb44e816727_9.png)

以下是传递性的核心定义：

![](img/559c2df900d71d932b56ddb44e816727_11.png)

![](img/559c2df900d71d932b56ddb44e816727_13.png)

**关系 `R` 是传递的，当且仅当 `R` 等于某个有向图 `G` 的正路径关系 `G+`。**

这个定理的证明基本上是平凡的：我们可以让关系 `R` 本身作为有向图的边集，那么 `R` 自然就是该图的正路径关系。

## 非对称性与严格偏序

![](img/559c2df900d71d932b56ddb44e816727_15.png)

![](img/559c2df900d71d932b56ddb44e816727_17.png)

现在，让我们转向**有向无环图**。在有向无环图中，如果存在一条从顶点 `u` 到顶点 `v` 的正长度路径，那么由于图中没有环，就不可能存在一条从 `v` 回到 `u` 的路径。这个属性被称为**非对称性**。

![](img/559c2df900d71d932b56ddb44e816727_19.png)

因此，DAG 中的正路径关系 `D+` 具有非对称性：如果 `u` 能通过正长度路径到达 `v`，那么 `v` 就不可能通过正长度路径回到 `u`。

![](img/559c2df900d71d932b56ddb44e816727_21.png)

将这个属性抽象出来：对于一个任意关系 `R`，如果 `u R v` 能推出 `非(v R u)`，那么这个关系就是**非对称的**。根据定义，无环有向图 `D` 的 `D+` 关系是非对称的。

**严格偏序**就是一个同时满足**传递性**和**非对称性**的关系。

![](img/559c2df900d71d932b56ddb44e816727_23.png)

以下是严格偏序的一些例子：
*   **集合间的真包含关系**：如果集合 `A` 真包含于集合 `B`，那么 `B` 不可能真包含于 `A`。
*   **MIT 课程间的间接先修关系**：如果课程 `A` 是课程 `B` 的先修课，那么 `B` 不可能是 `A` 的先修课。
*   **实数上的小于关系**：如果 `x < y`，那么 `y` 不可能小于 `x`。

![](img/559c2df900d71d932b56ddb44e816727_25.png)

![](img/559c2df900d71d932b56ddb44e816727_27.png)

综合之前的推理，我们可以得出一个重要结论：

**一个关系 `R` 是严格偏序，当且仅当 `R` 是某个有向无环图 `D` 的正路径关系 `D+`。**

![](img/559c2df900d71d932b56ddb44e816727_29.png)

![](img/559c2df900d71d932b56ddb44e816727_31.png)

因此，定义严格偏序的公理（传递性和非对称性）抽象地刻画了“关系来源于一个有向无环图”这一特性。

![](img/559c2df900d71d932b56ddb44e816727_33.png)

## 全序性

![](img/559c2df900d71d932b56ddb44e816727_35.png)

偏序的另一个重要属性是**全序性**（有些作者称之为**线性**）。

![](img/559c2df900d71d932b56ddb44e816727_37.png)

全序性的简单定义是：对于任意两个不同的元素，其中一个必定“大于”另一个（相对于该关系而言）。

![](img/559c2df900d71d932b56ddb44e816727_39.png)

![](img/559c2df900d71d932b56ddb44e816727_41.png)

最熟悉的例子是实数上的**小于关系**或**小于等于关系**。给定任意两个不同的实数 `x` 和 `y`，要么 `x < y`，要么 `y < x`。

![](img/559c2df900d71d932b56ddb44e816727_43.png)

形式化定义如下：对于一个关系 `R`，如果对于任意 `x ≠ y`，都有 `x R y` 或 `y R x` 成立，那么这个关系就是**全序的**。

另一种说法是：在关系 `R` 下，不存在**不可比较**的元素。全序关系看起来就像一条链：给定一组元素，总会有一个最大的，然后是次大的，依此类推（假设是有限集）。

全序的基本例子是数字间的大小关系（如 `<` 或 `≤`）。非全序的基本例子是集合的包含关系，因为完全可以存在两个集合，它们互不包含。

![](img/559c2df900d71d932b56ddb44e816727_45.png)

## 自反性与弱偏序

![](img/559c2df900d71d932b56ddb44e816727_47.png)

**弱偏序**是严格偏序的一个小变体，也是一个熟悉的概念。它放松了严格偏序中“元素不能与自身相关”的限制。

![](img/559c2df900d71d932b56ddb44e816727_49.png)

在弱偏序中，我们不仅放松了这个限制，反而要求**每个元素都必须与自身相关**。

![](img/559c2df900d71d932b56ddb44e816727_51.png)

弱偏序的例子包括：
*   **集合间的（非严格）包含关系**：`A ⊆ A` 总是成立。
*   **实数上的小于等于关系**：`x ≤ x` 总是成立。

![](img/559c2df900d71d932b56ddb44e816727_53.png)

区分弱偏序和严格偏序的属性是**自反性**。一个定义在集合 `A` 上的关系 `R` 是**自反的**，当且仅当对于 `A` 中的所有元素 `a`，都有 `a R a`。

![](img/559c2df900d71d932b56ddb44e816727_55.png)

我们可以立即观察到，有向图的**路径关系 `G*`**（包括长度为0的路径）是自反的，因为根据定义，从任何顶点到其自身都存在一条长度为0的路径。

![](img/559c2df900d71d932b56ddb44e816727_57.png)

## 反对称性

当我们引入自反性后，需要调整非对称性的概念，使其允许元素与自身相关。调整后的属性称为**反对称性**。

![](img/559c2df900d71d932b56ddb44e816727_59.png)

更精确地说，反对称性与非对称性的区别在于：
*   在非对称性中，`a R a` 是绝对不允许的。
*   在反对称性中，`a R a` 是允许的，不被禁止。

![](img/559c2df900d71d932b56ddb44e816727_61.png)

![](img/559c2df900d71d932b56ddb44e816727_63.png)

![](img/559c2df900d71d932b56ddb44e816727_65.png)

一个关系的**反对称性**形式化定义为：如果 `u R v` 且 `u ≠ v`，那么 `非(v R u)`。第一行是非对称性的陈述，附加条件是它只在 `u` 和 `v` 不相等时才必须成立。这就是形式化地说“反对称性等同于非对称性，但允许 `a R a`”。

![](img/559c2df900d71d932b56ddb44e816727_67.png)

包含长度为0路径的有向图路径关系 `G*` 是**反对称的**。

## 总结：偏序的公理化

![](img/559c2df900d71d932b56ddb44e816727_69.png)

现在，让我们把所有这些概念整合起来。

![](img/559c2df900d71d932b56ddb44e816727_71.png)

![](img/559c2df900d71d932b56ddb44e816727_73.png)

*   **弱偏序**是**传递的**、**反对称的**和**自反的**关系。在弱偏序中，我们要求每个元素都与自身相关。
*   **严格偏序**是**传递的**和**非对称的**关系。在严格偏序中，没有元素与自身相关。

一个简短的说明：非对称性意味着没有元素与自身相关；自反性意味着所有元素都与自身相关。可能存在某些图，其中一些元素与自身相关，另一些则不，这样的关系只是传递且反对称的，但它们不常出现，因此我们不给它们专门命名。

最后，总结整个故事的定理是：

![](img/559c2df900d71d932b56ddb44e816727_75.png)

**一个关系 `R` 是弱偏序，当且仅当 `R` 等于某个有向图 `G` 的路径关系 `G*`。**

![](img/559c2df900d71d932b56ddb44e816727_77.png)

![](img/559c2df900d71d932b56ddb44e816727_79.png)

本节课中，我们一起学习了偏序关系的核心概念：**传递性**、**非对称性/反对称性**、**自反性**、**严格偏序**、**弱偏序**以及**全序性**。我们看到了这些抽象的公理性质如何精确地对应到有向图（特别是有向无环图）的路径关系上，从而将图论与序论紧密地联系起来。
