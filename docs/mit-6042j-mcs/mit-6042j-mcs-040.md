# 计算机科学的数学基础：2.2.1：同余模n 📐

在本节课中，我们将学习数论中的一个核心概念——**同余**。这个概念由高斯在18世纪初提出，至今仍在计算机科学，特别是加密和哈希等领域有广泛应用。我们将从定义开始，逐步理解其性质和应用。

![](img/37cdff322361b6c3f2e589fe1e26fb4d_1.png)

## 同余的定义

同余是两个整数 `a` 和 `b` 之间的一种关系，它由另一个大于1的整数 `n` 决定。

![](img/37cdff322361b6c3f2e589fe1e26fb4d_3.png)

**定义**：如果 `n` 整除 `(a - b)`，即 `(a - b)` 是 `n` 的倍数，则称 `a` 与 `b` 模 `n` 同余。记作：
`a ≡ b (mod n)`

![](img/37cdff322361b6c3f2e589fe1e26fb4d_5.png)

这是一个关键定义。例如，`30 ≡ 12 (mod 9)`，因为 `30 - 12 = 18`，而 `9` 整除 `18`。

![](img/37cdff322361b6c3f2e589fe1e26fb4d_7.png)

## 同余的等价表述

理解同余的另一种方式是：两个数模 `n` 同余，当且仅当它们除以 `n` 的余数相同。

**引理**：`a ≡ b (mod n)` 当且仅当 `a` 和 `b` 除以 `n` 的余数相同。

![](img/37cdff322361b6c3f2e589fe1e26fb4d_9.png)

![](img/37cdff322361b6c3f2e589fe1e26fb4d_11.png)

我们可以用这个引理来验证之前的例子：`30` 除以 `9` 余 `3`，`12` 除以 `9` 也余 `3`，因此它们模 `9` 同余。

上一节我们介绍了同余的基本定义，本节中我们来证明这个关键的等价引理。

![](img/37cdff322361b6c3f2e589fe1e26fb4d_13.png)

### 引理的证明

![](img/37cdff322361b6c3f2e589fe1e26fb4d_15.png)

**证明**：
设 `a` 除以 `n` 的商为 `q_a`，余数为 `r_a`；`b` 除以 `n` 的商为 `q_b`，余数为 `r_b`。根据除法算法，有：
`a = q_a * n + r_a`
`b = q_b * n + r_b`
其中 `0 ≤ r_a, r_b < n`。

*   **充分性（右推左）**：如果 `r_a = r_b`，那么 `a - b = (q_a - q_b) * n`。显然 `n` 整除 `(a - b)`，所以 `a ≡ b (mod n)`。
*   **必要性（左推右）**：如果 `a ≡ b (mod n)`，即 `n` 整除 `(a - b)`。将 `a` 和 `b` 的表达式代入：
    `a - b = (q_a - q_b) * n + (r_a - r_b)`
    由于 `n` 整除 `(a - b)` 和 `(q_a - q_b) * n`，因此 `n` 也必须整除 `(r_a - r_b)`。但 `r_a` 和 `r_b` 都在 `[0, n)` 区间内，它们的差 `(r_a - r_b)` 的绝对值小于 `n`。在 `(-n, n)` 区间内，能被 `n` 整除的数只有 `0`。因此 `r_a - r_b = 0`，即 `r_a = r_b`。

证毕。这个引理非常重要，它意味着我们可以用“余数相等”来理解和操作同余。

![](img/37cdff322361b6c3f2e589fe1e26fb4d_17.png)

## 同余的基本性质

由于同余本质上意味着余数相等，它继承了等式的许多良好性质。

![](img/37cdff322361b6c3f2e589fe1e26fb4d_19.png)

以下是同余关系的三个基本性质：

1.  **自反性**：对于任意整数 `a`，有 `a ≡ a (mod n)`。
2.  **对称性**：如果 `a ≡ b (mod n)`，那么 `b ≡ a (mod n)`。
3.  **传递性**：如果 `a ≡ b (mod n)` 且 `b ≡ c (mod n)`，那么 `a ≡ c (mod n)`。

![](img/37cdff322361b6c3f2e589fe1e26fb4d_21.png)

这些性质很容易从“余数相等”的角度理解。例如，传递性：`a` 和 `b` 余数相同，`b` 和 `c` 余数相同，自然 `a` 和 `c` 余数也相同。

## 余数引理与算术运算

![](img/37cdff322361b6c3f2e589fe1e26fb4d_23.png)

一个非常有用的技术结果是**余数引理**：任何整数 `a` 都与其自身除以 `n` 的余数 `r` 模 `n` 同余。即 `a ≡ r (mod n)`，其中 `r = a % n`。

**证明**：根据定义，`a` 和 `r` 除以 `n` 的余数都是 `r`，因此由之前的引理可知它们同余。

![](img/37cdff322361b6c3f2e589fe1e26fb4d_25.png)

这个引理是**余数算术**的基础，它允许我们在同余计算中，随时用一个更小的余数来替换原数。

![](img/37cdff322361b6c3f2e589fe1e26fb4d_27.png)

更重要的是，同余与加法、乘法运算**兼容**。这意味着我们可以像处理普通等式一样处理同余式。

![](img/37cdff322361b6c3f2e589fe1e26fb4d_29.png)

以下是同余运算的规则：

![](img/37cdff322361b6c3f2e589fe1e26fb4d_31.png)

*   **加法规则**：如果 `a ≡ b (mod n)`，那么对于任意整数 `c`，有 `a + c ≡ b + c (mod n)`。
*   **乘法规则**：如果 `a ≡ b (mod n)`，那么对于任意整数 `c`，有 `a * c ≡ b * c (mod n)`。
*   **加法组合规则**：如果 `a ≡ b (mod n)` 且 `c ≡ d (mod n)`，那么 `a + c ≡ b + d (mod n)`。
*   **乘法组合规则**：如果 `a ≡ b (mod n)` 且 `c ≡ d (mod n)`，那么 `a * c ≡ b * d (mod n)`。

![](img/37cdff322361b6c3f2e589fe1e26fb4d_33.png)

**推论**：在进行模 `n` 的算术时（无论是加法、乘法还是它们的组合），我们可以在任何步骤将任意一个数替换为与其同余的数（通常替换为它的余数），最终结果的同余性保持不变。这使得计算涉及大数的模运算变得非常简单。

![](img/37cdff322361b6c3f2e589fe1e26fb4d_35.png)

## 应用示例

![](img/37cdff322361b6c3f2e589fe1e26fb4d_37.png)

让我们应用所学知识解决一个具体问题：计算 `287^9 mod 4` 的值。

1.  **简化底数**：首先计算 `287 mod 4`。`287 ÷ 4 = 71 ... 3`，所以 `287 ≡ 3 (mod 4)`。根据乘法规则，`287^9 ≡ 3^9 (mod 4)`。
2.  **简化指数**：计算 `3^2 = 9`。`9 mod 4 = 1`，所以 `3^2 ≡ 1 (mod 4)`。
3.  **分解指数**：`3^9 = 3^(2*4+1) = (3^2)^4 * 3^1`。
4.  **代入计算**：因为 `3^2 ≡ 1 (mod 4)`，所以 `(3^2)^4 ≡ 1^4 ≡ 1 (mod 4)`。因此，`3^9 ≡ 1 * 3 ≡ 3 (mod 4)`。

![](img/37cdff322361b6c3f2e589fe1e26fb4d_39.png)

所以，`287^9 mod 4 = 3`。通过使用同余性质，我们避免了直接计算 `287^9` 这个巨大数字。

## 总结

本节课中我们一起学习了**同余模n**的概念。
*   我们掌握了同余的两种等价定义：`n | (a-b)` 或 `a` 和 `b` 除以 `n` 的余数相同。
*   我们探讨了同余的自反性、对称性和传递性。
*   我们学习了关键的**余数引理**，以及同余与加法、乘法运算的兼容性规则。这些规则构成了**余数算术**的基础，允许我们在计算中用更小的余数替换大数。
*   最后，我们通过一个计算 `287^9 mod 4` 的例子，演示了如何运用这些规则高效地解决模运算问题。

![](img/37cdff322361b6c3f2e589fe1e26fb4d_41.png)

理解同余是学习后续数论知识以及在计算机科学中应用相关算法（如加密）的重要基石。
