![](img/e5e281b99ab1922feb99f35c9a0cb653_19.png)

手机锁屏上的惊人内容。所以像我之前提到的，Glance 每天有超过 1.5 亿的活跃用户，其中超过 50%来自较小的城市和城镇。为了能够服务于这样的受众，在计算机或智能设备相对较少的情况下，你需要让你的应用程序变得极其。

使用起来简单，什么会比在你手机的锁屏上直接回答你的问题更简单呢？所以这就是我们的计划。在锁屏上你确实可以访问相机。你只需向右滑动。拍摄对象的图像，然后你需要点击“询问 Glance”，这将重定向到城市应用程序。

所以这就是我们计划如何最大化我们的覆盖范围，最大化我们的用户覆盖率，希望能为每一个需要帮助的人提供额外的视角。所以是的，即使你没有理解我刚才在这里说的 20 分钟的内容，在 TLDR 版本中，我们使用了 VQA V2 数据集。我们使用了基于 VIT 的模型。我们使用 Streamlet 构建了原型，并用 React 进行了前端开发。

我们整个 API 服务是使用 fast API 构建的。你可以克隆这个仓库来试试。此外，在这个仓库中，你也可以看到如何在云平台上部署它。我们展示了如何使用它，如何在 GCP 上部署。所以，自己试试看，因为不要仅仅依赖录制的演示。你需要亲自尝试。只是有几点需要注意的是，它能够正常工作。

在 Google Chrome 和 Microsoft Edge 中效果最佳。Safari 和 Firefox 有些音频问题，我们会很快修复。所以今天就到这里，伙计们。非常感谢你们。这是帮助我们建立它的团队，而不是 Shiva 和 Pablo，因为没有他们，这个应用程序就不可能实现。所以非常感谢他们。
