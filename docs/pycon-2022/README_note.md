# Python入门：1：变量与数据类型

在本节课中，我们将要学习Python编程中最基础的两个概念：变量和数据类型。理解它们是编写任何程序的第一步。

## 概述

变量是存储信息的容器。数据类型则定义了容器中可以存放什么类型的信息，以及可以对信息进行哪些操作。

## 变量

变量就像一个贴有标签的盒子，你可以把数据放进去，并通过标签名称来使用它。

以下是定义一个变量的基本语法：
```python
variable_name = value
```
*   `variable_name` 是你为变量起的名字。
*   `=` 是赋值运算符，表示“把右边的值存到左边的变量里”。
*   `value` 是你想要存储的具体数据。

上一节我们介绍了变量的概念，本节中我们来看看Python中几种常见的数据类型。

## 基本数据类型

Python有几种内置的核心数据类型，用于表示不同种类的数据。

以下是几种最常用的数据类型：

1.  **整数**
    *   表示没有小数部分的数字，可以是正数或负数。
    *   示例：`age = 25`， `temperature = -10`

2.  **浮点数**
    *   表示带有小数点的数字。
    *   示例：`price = 19.99`， `pi = 3.14159`

3.  **字符串**
    *   表示文本信息，需要用单引号 `‘ ’` 或双引号 `“ ”` 括起来。
    *   示例：`name = “Alice”`， `greeting = ‘Hello World’`

4.  **布尔值**
    *   只有两个值：`True`（真） 和 `False`（假），常用于逻辑判断。
    *   示例：`is_student = True`， `has_finished = False`

了解了基本类型后，我们来看看如何将这些数据组合起来。

## 数据组合类型

有时我们需要将多个数据项组合在一起，这时就需要用到组合数据类型。

以下是两种基础的组合类型：

1.  **列表**
    *   一个有序的、可修改的集合，用方括号 `[ ]` 表示。
    *   示例：`fruits = [“apple”, “banana”, “orange”]`

2.  **字典**
    *   一个无序的“键-值”对集合，用花括号 `{ }` 表示。
    *   示例：`person = {“name”: “Bob”, “age”: 30}`

## 总结

本节课中我们一起学习了Python的变量与数据类型。我们知道了变量是存储数据的容器，并通过等号 `=` 进行赋值。我们还认识了整数、浮点数、字符串、布尔值这几种基本数据类型，以及列表和字典这两种组合数据类型。掌握这些是后续学习控制流、函数等更复杂概念的基础。
