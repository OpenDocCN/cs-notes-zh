# 数字系统与计算机架构：P1：2.2.7 VTC 实例分析 📈

![](img/eeb373ef6358701537da06ea6242d39b_0.png)

在本节课中，我们将通过一个具体的实例，学习如何分析一个器件的电压传输特性，并判断它是否可以作为组合逻辑反相器使用。

## 概述

我们将分析一个特定器件的电压传输特性图，并尝试为其选择合适的电压阈值，以使其满足组合逻辑器件的要求。

![](img/eeb373ef6358701537da06ea6242d39b_2.png)

![](img/eeb373ef6358701537da06ea6242d39b_3.png)

---

让我们来看一个具体的例子。下图展示了一个特定器件的电压传输特性。

![](img/eeb373ef6358701537da06ea6242d39b_0.png)

![](img/eeb373ef6358701537da06ea6242d39b_5.png)

我们想知道，是否可以将这个器件用作组合逻辑反相器。

![](img/eeb373ef6358701537da06ea6242d39b_6.png)

![](img/eeb373ef6358701537da06ea6242d39b_2.png)
![](img/eeb373ef6358701537da06ea6242d39b_3.png)

![](img/eeb373ef6358701537da06ea6242d39b_8.png)

换句话说，我们能否为电压阈值 `VOL`、`VIL`、`VIH` 和 `VOH` 选择合适的值，使得图中所示的电压传输特性满足组合逻辑器件的约束条件？

![](img/eeb373ef6358701537da06ea6242d39b_5.png)
![](img/eeb373ef6358701537da06ea6242d39b_6.png)

一个反相器在其输入为数字0时输出数字1，反之亦然。

![](img/eeb373ef6358701537da06ea6242d39b_10.png)

![](img/eeb373ef6358701537da06ea6242d39b_11.png)

![](img/eeb373ef6358701537da06ea6242d39b_8.png)

实际上，当这个器件的输入电压较低时，它确实能产生一个高输出电压，因此它有可能满足要求。

![](img/eeb373ef6358701537da06ea6242d39b_10.png)
![](img/eeb373ef6358701537da06ea6242d39b_11.png)

![](img/eeb373ef6358701537da06ea6242d39b_13.png)

该器件产生的最低输出电压是 0.5 伏特。因此，如果该器件要产生一个合法的数字输出 0，我们必须选择 `VOL` 至少为 0.5 伏特。

![](img/eeb373ef6358701537da06ea6242d39b_13.png)

![](img/eeb373ef6358701537da06ea6242d39b_15.png)

我们希望当反相器的输入是有效的数字1时，它能产生一个有效的数字0。

![](img/eeb373ef6358701537da06ea6242d39b_16.png)

![](img/eeb373ef6358701537da06ea6242d39b_15.png)
![](img/eeb373ef6358701537da06ea6242d39b_16.png)

观察电压传输特性图，我们发现如果输入电压高于 3 伏特，输出电压将小于或等于 `VOL`。因此，让我们将 `VIH` 设定为 3 伏特。

![](img/eeb373ef6358701537da06ea6242d39b_18.png)

![](img/eeb373ef6358701537da06ea6242d39b_18.png)
![](img/eeb373ef6358701537da06ea6242d39b_19.png)

![](img/eeb373ef6358701537da06ea6242d39b_19.png)

我们可以将其设定为高于 3 伏特的值，但我们会将其设定得尽可能低，以便为较大的高电平噪声容限留出空间。

![](img/eeb373ef6358701537da06ea6242d39b_21.png)
![](img/eeb373ef6358701537da06ea6242d39b_22.png)

![](img/eeb373ef6358701537da06ea6242d39b_21.png)

这样，我们就确定了四个信号阈值中的两个：`VOL` 和 `VIH`。

![](img/eeb373ef6358701537da06ea6242d39b_22.png)

![](img/eeb373ef6358701537da06ea6242d39b_24.png)

![](img/eeb373ef6358701537da06ea6242d39b_24.png)

另外两个阈值通过噪声容限 `N` 与这两个阈值相关联，如下面两个公式所示：

```
VIL = VOL + N
VOH = VIH + N
```

![](img/eeb373ef6358701537da06ea6242d39b_26.png)

我们能否找到一个 `N` 值，使得当 `Vin` 小于或等于 `VIL` 时，`Vout` 大于或等于 `VOH`？

![](img/eeb373ef6358701537da06ea6242d39b_28.png)

![](img/eeb373ef6358701537da06ea6242d39b_28.png)

如果我们选择 `N` 为 0.5 伏特，那么根据公式可以得出 `VIL` 为 1 伏特，`VOH` 为 3.5 伏特。

![](img/eeb373ef6358701537da06ea6242d39b_30.png)

![](img/eeb373ef6358701537da06ea6242d39b_30.png)

将这些阈值绘制在图上，并添加上禁止区域，我们高兴地发现，该电压传输特性实际上是合法的。

![](img/eeb373ef6358701537da06ea6242d39b_32.png)
![](img/eeb373ef6358701537da06ea6242d39b_33.png)

因此，如果我们使用以下信号规范：`VOL` 为 0.5 伏特，`VIL` 为 1 伏特，`VIH` 为 3 伏特，`VOH` 为 3.5 伏特，那么我们就可以将这个器件用作组合逻辑反相器。

![](img/eeb373ef6358701537da06ea6242d39b_32.png)

![](img/eeb373ef6358701537da06ea6242d39b_33.png)

![](img/eeb373ef6358701537da06ea6242d39b_35.png)
![](img/eeb373ef6358701537da06ea6242d39b_36.png)

一切准备就绪。

## 总结

![](img/eeb373ef6358701537da06ea6242d39b_36.png)

本节课中，我们一起学习了如何通过分析电压传输特性图，为一个器件选择合适的电压阈值。我们确定了 `VOL`、`VIL`、`VIH` 和 `VOH` 的值，并验证了它们满足组合逻辑反相器的要求，最终得出结论：该器件可以在指定的信号规范下作为反相器使用。
