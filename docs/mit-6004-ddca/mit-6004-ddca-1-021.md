# 021：处理噪声 🛡️

![](img/44b41dfc5a500ed816bc80abe99b21b8_0.png)

![](img/44b41dfc5a500ed816bc80abe99b21b8_2.png)

在本节课中，我们将要学习数字系统中一个关键概念：如何处理无处不在的电气噪声。我们将看到，通过精心设计信号规范，可以为系统提供“噪声容限”，从而确保数字信号的可靠传输。

![](img/44b41dfc5a500ed816bc80abe99b21b8_4.png)

![](img/44b41dfc5a500ed816bc80abe99b21b8_5.png)

在上一节我们介绍了数字信号的基本规范，本节中我们来看看如何增强其鲁棒性以应对噪声。

![](img/44b41dfc5a500ed816bc80abe99b21b8_7.png)

## 噪声带来的问题

![](img/44b41dfc5a500ed816bc80abe99b21b8_8.png)

考虑一个组合系统，左侧的上游组合器件试图向右侧的下游组合器件发送一个数字0。

![](img/44b41dfc5a500ed816bc80abe99b21b8_10.png)

![](img/44b41dfc5a500ed816bc80abe99b21b8_11.png)

上游器件产生的输出电压略低于 **V<sub>L</sub>**。根据我们之前提出的信号规范，这符合数字0的表示。

![](img/44b41dfc5a500ed816bc80abe99b21b8_13.png)

现在，假设一些电气噪声轻微改变了导线上的电压，导致下游器件输入端检测到的电压略高于 **V<sub>L</sub>**。换句话说，接收到的信号不再符合有效的数字输入规范。

![](img/44b41dfc5a500ed816bc80abe99b21b8_15.png)

下游器件的组合行为因此无法得到保证。系统由于微小的电气噪声而产生了错误行为。这正是我们希望通过采用数字系统架构来避免的那种不稳定行为。

![](img/44b41dfc5a500ed816bc80abe99b21b8_17.png)

## 解决方案：调整信号规范

![](img/44b41dfc5a500ed816bc80abe99b21b8_18.png)

![](img/44b41dfc5a500ed816bc80abe99b21b8_20.png)

解决此问题的一种方法是调整信号规范，使输出信号必须遵守比输入信号更严格的界限。

![](img/44b41dfc5a500ed816bc80abe99b21b8_21.png)

![](img/44b41dfc5a500ed816bc80abe99b21b8_23.png)

其核心思想是确保有效的输出信号在受到噪声影响后，不会变成无效的输入信号。

![](img/44b41dfc5a500ed816bc80abe99b21b8_25.png)

我们能否通过完全避免噪声来彻底解决这个问题？这是一个美好的想法，但如果我们计划使用电子元件，这是无法实现的目标。

![](img/44b41dfc5a500ed816bc80abe99b21b8_27.png)

## 噪声的来源

![](img/44b41dfc5a500ed816bc80abe99b21b8_29.png)

![](img/44b41dfc5a500ed816bc80abe99b21b8_30.png)

电压噪声（我们将其定义为偏离标称电压值的变化）有多种来源。

![](img/44b41dfc5a500ed816bc80abe99b21b8_32.png)

以下是噪声的主要来源：
*   **电气效应**：例如，根据欧姆定律（**V = I × R**），导体上的IR压降；导体之间的电容耦合。
*   **制造差异**：元件参数与其标称值之间的制造差异，导致器件间电气行为的微小不同。
*   **环境因素**：例如热噪声，或外部电磁场引起的电压效应。

![](img/44b41dfc5a500ed816bc80abe99b21b8_34.png)

这个列表还可以继续。请注意，在许多情况下，噪声是由电路的正常操作引起的，或者是制造电路所用材料和工艺的固有特性，因此是不可避免的。

![](img/44b41dfc5a500ed816bc80abe99b21b8_36.png)

然而，我们可以预测噪声的幅度，并相应地调整我们的信号规范。

![](img/44b41dfc5a500ed816bc80abe99b21b8_38.png)

## 实施：分离的输入/输出规范

![](img/44b41dfc5a500ed816bc80abe99b21b8_39.png)

![](img/44b41dfc5a500ed816bc80abe99b21b8_41.png)

让我们看看如何实施这个方案。

![](img/44b41dfc5a500ed816bc80abe99b21b8_43.png)

我们对噪声问题的修正方案是为数字输入和数字输出提供独立的信号规范。

![](img/44b41dfc5a500ed816bc80abe99b21b8_45.png)

*   要发送数字0，数字输出必须产生小于或等于 **V<sub>OL</sub>** 的电压。
*   要发送数字1，数字输出必须产生大于或等于 **V<sub>OH</sub>** 的电压。

![](img/44b41dfc5a500ed816bc80abe99b21b8_47.png)

![](img/44b41dfc5a500ed816bc80abe99b21b8_49.png)

到目前为止，这似乎与我们之前的信号规范没有太大不同。区别在于，数字输入必须遵守不同的信号规范。

![](img/44b41dfc5a500ed816bc80abe99b21b8_51.png)

*   小于或等于 **V<sub>IL</sub>** 的输入电压必须被解释为数字0。
*   大于或等于 **V<sub>IH</sub>** 的输入电压必须被解释为数字1。

![](img/44b41dfc5a500ed816bc80abe99b21b8_53.png)

![](img/44b41dfc5a500ed816bc80abe99b21b8_54.png)

这四个信号阈值的取值需满足以下约束条件：
```
V_OL < V_IL
V_IH < V_OH
```
请注意，**V<sub>IL</sub>** 严格大于 **V<sub>OL</sub>**，而 **V<sub>IH</sub>** 严格小于 **V<sub>OH</sub>**。

![](img/44b41dfc5a500ed816bc80abe99b21b8_56.png)

## 噪声容限

![](img/44b41dfc5a500ed816bc80abe99b21b8_58.png)

输入和输出电压阈值之间的间隙称为**噪声容限**。

![](img/44b41dfc5a500ed816bc80abe99b21b8_60.png)

噪声容限告诉我们，一个有效的0或1输出信号可以承受多大的噪声，而其连接到的输入端仍能正确解读该信号。

![](img/44b41dfc5a500ed816bc80abe99b21b8_62.png)

![](img/44b41dfc5a500ed816bc80abe99b21b8_63.png)

两个噪声容限中较小的那个，称为该信号规范的**抗噪声度**。

![](img/44b41dfc5a500ed816bc80abe99b21b8_65.png)

作为数字工程师，我们的目标是设计信号规范，以提供尽可能高的抗噪声度。

![](img/44b41dfc5a500ed816bc80abe99b21b8_67.png)

![](img/44b41dfc5a500ed816bc80abe99b21b8_68.png)

遵守此信号规范的组合器件，会在噪声有机会累积并最终导致信号错误之前，消除其输入端的噪声。

![](img/44b41dfc5a500ed816bc80abe99b21b8_70.png)

![](img/44b41dfc5a500ed816bc80abe99b21b8_71.png)

## 总结

![](img/44b41dfc5a500ed816bc80abe99b21b8_73.png)

本节课中我们一起学习了数字系统如何处理噪声。关键在于设计具有**噪声容限**的信号规范，即输出信号的电压范围（**V<sub>OL</sub>**, **V<sub>OH</sub>**）比输入信号的识别范围（**V<sub>IL</sub>**, **V<sub>IH</sub>**）更严格。这样，即使信号在传输中受到一定干扰，只要干扰幅度不超过噪声容限，接收端仍能正确识别。这确保了数字信号传输的可靠性，避免了早期模拟信号示例中出现的问题。
