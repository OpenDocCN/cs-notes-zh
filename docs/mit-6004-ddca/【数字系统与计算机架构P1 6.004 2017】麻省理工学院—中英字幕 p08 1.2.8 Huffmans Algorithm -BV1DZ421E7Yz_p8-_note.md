# 数字系统与计算机架构：1.2.8：霍夫曼算法 🧮

![](img/669bf1ba546482acb500d71c7c0f8729_0.png)

在本节课中，我们将学习霍夫曼算法。这是一种用于构造最优变长编码的方法。我们将了解其工作原理，并通过一个具体例子来演示如何构建编码树。

![](img/669bf1ba546482acb500d71c7c0f8729_2.png)

## 概述

给定一组符号及其出现的概率，霍夫曼算法能够指导我们如何构造一个最优的变长编码。

![](img/669bf1ba546482acb500d71c7c0f8729_4.png)

所谓“最优”，是指在**每次编码一个符号**的假设下，没有其他变长编码能拥有更短的**期望长度**。

## 算法原理：自底向上构建

![](img/669bf1ba546482acb500d71c7c0f8729_6.png)

该算法从底向上构建编码的二叉树。

![](img/669bf1ba546482acb500d71c7c0f8729_7.png)

首先，选择**概率最小**的两个符号。这意味着它们的信息含量最高，因此应该拥有最长的编码。

![](img/669bf1ba546482acb500d71c7c0f8729_9.png)

如果在过程中遇到两个符号概率相同的情况，可以任意选择其中一个。

![](img/669bf1ba546482acb500d71c7c0f8729_11.png)

![](img/669bf1ba546482acb500d71c7c0f8729_12.png)

## 算法步骤详解

以下是霍夫曼算法的具体构建步骤。

### 第一步：选择并合并最小概率符号

在我们的运行示例中，概率最低的两个符号是 **C** 和 **D**。

![](img/669bf1ba546482acb500d71c7c0f8729_14.png)

![](img/669bf1ba546482acb500d71c7c0f8729_15.png)

将这两个符号合并为一个二叉子树，其中一个分支标记为 `0`，另一个标记为 `1`。哪个标签对应哪个分支并不重要。

![](img/669bf1ba546482acb500d71c7c0f8729_16.png)

![](img/669bf1ba546482acb500d71c7c0f8729_17.png)

从符号列表中移除 C 和 D，并用一个新构建的子树替换它们。这个新子树的根节点具有关联概率 `1.6`，即其两个分支概率之和。

### 第二步：重复合并过程

![](img/669bf1ba546482acb500d71c7c0f8729_19.png)

现在继续，在每一步都选择两个概率最低的符号（和/或子树），将选择的对象合并成一个新的子树。

![](img/669bf1ba546482acb500d71c7c0f8729_21.png)

在我们的例子中，此时符号 **A** 的概率是 `1/3`，符号 **B** 的概率是 `1/2`，而 **CD 子树** 的概率是 `1/6`。因此，我们将 **A** 与 **CD 子树** 合并。

![](img/669bf1ba546482acb500d71c7c0f8729_23.png)

### 第三步：完成最终合并

在最后一步，我们只剩下两个选择：**B** 和 **ACD 子树**。

我们将它们合并成一个新的子树，其根节点最终成为代表最优变长编码的树的根节点。

![](img/669bf1ba546482acb500d71c7c0f8729_25.png)

![](img/669bf1ba546482acb500d71c7c0f8729_26.png)

![](img/669bf1ba546482acb500d71c7c0f8729_27.png)

令人高兴的是，这棵树生成的编码正是我们一直使用的那个。

## 编码的等价性与效率

如上所述，我们可以通过交换任何子树分支上的 `0` 和 `1` 标签，来生成多种不同的变长编码。

![](img/669bf1ba546482acb500d71c7c0f8729_29.png)

![](img/669bf1ba546482acb500d71c7c0f8729_30.png)

但是，所有这些编码都将具有相同的**期望长度**。期望长度由每个符号到树根的距离决定，而不是由从根到叶路径上的标签决定。

因此，所有这些不同的编码在效率上是等价的。

![](img/669bf1ba546482acb500d71c7c0f8729_32.png)

## 总结

![](img/669bf1ba546482acb500d71c7c0f8729_37.png)

本节课中，我们一起学习了霍夫曼算法。我们了解到，该算法通过自底向上、反复合并概率最小的两个节点来构建最优编码树。最终生成的变长编码能最小化信息的期望编码长度，并且通过交换分支标签可以得到一系列等价的、效率相同的最优编码。
