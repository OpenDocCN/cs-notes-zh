# 斯坦福 CS110 计算机系统原理课程笔记索引 📚

在本教程中，我们将一起浏览斯坦福大学 CS110 课程《计算机系统原理》的完整笔记目录。这份索引涵盖了从基础概念到高级主题的二十余个核心讲座，是学习计算机系统底层原理的宝贵资料。

## 课程概述 🗺️

CS110 课程深入探讨了计算机系统的核心原理，包括操作系统、并发编程、网络通信和系统设计。通过本课程的学习，你将理解程序如何在计算机上运行，以及如何高效、安全地管理系统资源。

以下是本课程所有讲座笔记的完整列表，按逻辑顺序排列，便于你系统性地学习。

## 课程笔记目录 📖

### 第一部分：基础与文件系统

上一节我们介绍了课程的整体框架，本节中我们来看看关于计算机系统基础，特别是文件系统的核心内容。

以下是第一部分包含的讲座笔记：

*   **P1：第一讲 介绍** - 课程概述与 `main` 函数解析。
*   **P2：第二讲 文件系统** - 文件系统的基本概念与作用。
*   **P3：第三讲 Unix v6 文件系统** - 深入剖析经典的 Unix v6 文件系统实现。
*   **P4：第四讲 文件系统数据结构、系统调用与多处理器介绍** - 讲解文件系统的内部数据结构、相关的系统调用，并初步引入多处理的概念。

### 第二部分：进程管理与信号

在理解了文件系统之后，我们将视角转向程序的执行单元——进程。本节将探讨进程的创建、通信以及系统如何管理它们。

以下是第二部分包含的讲座笔记：

*   **P5：第五讲 execvp 介绍** - 详细介绍 `execvp` 系统调用，用于执行新程序。
*   **P6：第六讲 execvp、管道、dup2、信号** - 综合讲解进程替换、进程间通信（管道、重定向）以及信号的初步概念。
*   **P7：第七讲 信号** - 深入探讨信号机制，它是进程间异步通信的一种方式。
*   **P8：第八讲 竞态条件、死锁与数据完整性** - 分析并发编程中常见的陷阱：竞态条件、死锁，以及如何保证数据完整性。
*   **P9：讲座 8-1 作业 3** - 针对作业3的补充讲解，巩固进程与信号相关知识。

### 第三部分：并发编程

掌握了进程之后，我们将进入更细粒度的并发单元——线程。本节是并发编程的核心，将学习如何编写高效、正确的多线程程序。

以下是第三部分包含的讲座笔记：

*   **P10：第九讲 线程介绍** - 线程的基本概念及其与进程的区别。
*   **P11：第十讲 从 C 线程到 C++ 线程** - 对比 C 语言（如 pthreads）和 C++ 标准库中的线程实现。
*   **P12：第十一讲 多线程、条件变量与信号量** - 讲解线程同步的核心工具：条件变量和信号量。
*   **P13：第十二讲 互斥锁、条件变量、信号量回顾** - 对关键的同步原语进行总结和回顾。
*   **P14：第十三讲 冰淇淋店模拟** - 通过一个生动的模拟案例，实践多线程与同步机制的应用。

### 第四部分：网络与高级I/O

当程序需要在不同机器间通信时，网络知识就变得至关重要。本节将带你从网络基础开始，一直深入到实际的网络编程和系统设计。

以下是第四部分包含的讲座笔记：

*   **P15：第十四讲 网络入门** - 计算机网络的基本概念与协议栈介绍。
*   **P16：第十五讲 网络与客户端** - 探讨网络通信模型，重点讲解客户端角色。
*   **P17：第十六讲 网络系统调用** - 介绍进行网络编程所必需的系统调用，如 `socket`, `bind`, `connect`, `accept` 等。
*   **P18：第十七讲 Web 代理** - 学习如何实现一个 Web 代理服务器，这是网络编程的经典实践。
*   **P19：第十八讲 MapReduce** - 介绍分布式计算框架 MapReduce 的原理与思想。
*   **P20：第十九讲 系统设计原理** - 总结构建大型、可靠计算机系统的通用设计原则。
*   **P21：第二十讲 非阻塞 I/O** - 讲解高性能服务器中常用的非阻塞 I/O 模型，提升程序处理并发连接的能力。

## 总结 🎯

本节课中我们一起学习了斯坦福 CS110 《计算机系统原理》课程的全部笔记索引。我们从最基础的文件系统和进程管理出发，逐步深入到并发编程、线程同步，最后扩展到网络通信和高级系统设计。这个知识体系为你理解从单机程序到分布式系统的运行原理打下了坚实的基础。建议你按照目录顺序进行学习，并结合实践来深化理解。
