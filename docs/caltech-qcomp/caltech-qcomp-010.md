在本节课中，我们将继续探索量子纠缠的奇妙之处，并最终开始讨论量子计算。这将是下一讲的目标。但在继续之前，我想对量子纠缠进行一些更深入的探讨，特别是关于利用量子纠缠可以完成的任务。

我们已经对此有所了解，因为我们讨论过，当两个或多个参与者共享量子纠缠时，他们可以赢得一个游戏的成功概率，比没有共享纠缠时要高。这就是贝尔不等式违反的核心，并且实验已经验证了这一点。

今天，我想讨论两个相关的任务：超密编码和量子隐形传态。量子隐形传态是一个有用的协议，它被用作许多量子系统和量子网络中需要执行的任务的子程序。同时，它本身也是一个非常酷且有趣的话题。超密编码与它密切相关，我们将会看到这一点。

## 超密编码：利用量子信道发送经典比特

首先，我们来谈谈通信。我们使用量子信道，但目标仅仅是发送经典比特。这听起来有点浪费，因为使用量子信道，我们本可以做更有趣的事情。但假设我们只有量子信道可用，我们可以用它来发送量子比特，并且假设这是一个无噪声的量子信道，这是最好的情况。这意味着我们可以通过信道以完美的保真度发送量子信息。

假设爱丽丝和鲍勃通过这样一个信道连接。这意味着爱丽丝可以为她的系统（比如一个量子比特）准备任何她选择的量子态，并通过信道发送给鲍勃。鲍勃可以随心所欲地处理它，比如测量它或将其存储在量子内存中。信道无噪声意味着鲍勃接收到的状态正是爱丽丝发送的状态，就像爱丽丝在他的实验室里为他准备的一样。

现在，我们想讨论的是，如何使用连接爱丽丝和鲍勃的量子信道来发送一个或多个比特。当然可以做到。要发送一个比特，爱丽丝必须选择一对完美可区分的量子态，即相互正交的态。爱丽丝可以准备其中一个基态并发送给鲍勃。鲍勃通过与爱丽丝预先约定，知道在正确的基上进行测量以完美区分这些态。例如，他们可以约定，爱丽丝总是使用标准基态（计算基态 |0⟩ 或 |1⟩），而鲍勃知道如何进行测量以完美区分 |0⟩ 和 |1⟩。每当爱丽丝想发送一个比特给鲍勃时，她就准备这两个基态之一，通过信道发送，然后鲍勃测量它，这样比特就确实从爱丽丝传给了鲍勃。

现在，假设爱丽丝可以多次使用信道。她总共可以发送 N 个量子比特。通过我刚才描述的方式一次发送一个比特，她当然可以这样向鲍勃传递 N 个比特。一个自然的问题是：她能做得更好吗？使用信道 N 次，爱丽丝能否向鲍勃发送超过 N 个比特？事实证明答案是否定的。我今天不会在这里证明这一点，在我们的讲义后面的章节中有证明和讨论，但这学期我们不会涉及。有一个定理叫做“霍尔沃定理”，其推论是，当我们发送 N 个量子比特时，我们无法发送超过 N 个比特的经典信息。这并非一个显而易见的事实。

因为如果你考虑发送一个量子态字母表，鲍勃试图从中区分或获取信息，爱丽丝不必做我刚才描述的事情（发送两个相互正交的态之一）。她可以尝试更巧妙的方法：她可以从一个量子比特的许多态中选择一个字母表。毕竟，她可以准备并发送布洛赫球面上的任何纯态。她可以从数十亿个不同的信号态中选择一个发送给鲍勃。问题在于，鲍勃很难准确区分所有这些态。霍尔沃定理的内容是，由于准确区分非正交态非常困难，发送那么大的字母表实际上并没有好处，并不能帮助爱丽丝发送更多的比特信息。

爱丽丝可以尝试的另一个技巧是，准备多个量子比特的纠缠态，然后一次一个地通过信道发送所有这些量子比特。现在她利用了跨越信道多次使用的纠缠，这听起来是个好策略。但只要他们一开始没有任何共享的纠缠，这个定理再次表明，当爱丽丝向鲍勃发送 N 个量子比特时，如果目标确实是发送经典信息，那么鲍勃最多只能接收 N 个比特的经典信息。

## 共享纠缠如何增强经典通信？

现在，假设我们稍微改变一下规则。我们想知道爱丽丝和鲍勃共享的纠缠对于经典通信任务有多大的赋能作用。因此，我们现在想象爱丽丝和鲍勃有先见之明，准备了一些纠缠的量子比特对，以备不时之需。现在的问题是：他们能否利用这种纠缠更有效地进行通信？

假设他们很久以前就将纠缠“存入银行”，知道有一天可能会用到它。量子信道虽然无噪声，但也可能正因为无噪声，使用起来极其昂贵——每次你想从爱丽丝向鲍勃发送一个量子比特都要花费一百万美元。而今天，爱丽丝迫切需要向鲍勃发送两个比特的经典信息，这关乎生死。但她只有一百万美元，所以她只能负担得起使用一次量子信道。幸运的是，他们很久以前确实在银行里存了一些纠缠，预见到有一天可能需要它。那么，他们将如何利用这种纠缠来更有效地通信呢？

我们注意到以下情况。假设爱丽丝和鲍勃很久以前准备了一个共享的最大纠缠态（两个量子比特），一个量子比特在爱丽丝那里，一个在鲍勃那里。假设是态 |Φ⁺⟩，即 |00⟩ 和 |11⟩ 的均匀叠加态。我们注意到，如果爱丽丝独自（无需鲍勃帮助）对她的那部分纠缠态应用可能的幺正变换，她可以改变爱丽丝和鲍勃共享的纠缠态。具体来说，她可以应用泡利算子之一：恒等算子 I、泡利 X、泡利 Y 或泡利 Z。当她做这些事情时，共享的纠缠态会发生什么变化？

泡利算子 X 只是一个比特翻转，它将爱丽丝的 |0⟩ 变为 |1⟩，|1⟩ 变为 |0⟩。因此，作用在态 |Φ⁺⟩ = |00⟩ + |11⟩ 上，会变成 |10⟩ + |01⟩。如果她应用 Y，那么除了一个相位因子，它执行比特翻转，将 |1⟩ 变为 |0⟩，|0⟩ 变为 |1⟩，但也会有一个相对符号。实际上，它会将 |0⟩ 变为 |1⟩，|1⟩ 变为 -|0⟩。因此，当爱丽丝对她的量子比特和纠缠对应用 Y 时，态 |Φ⁺⟩ 变为 |10⟩ - |01⟩。如果她改为应用泡利算子 Z，它会改变基态 |0⟩ 和 |1⟩ 的相对相位。因此，当爱丽丝对她的量子比特应用 Z 时，如果初始态是 |Φ⁺⟩，它会变成 |00⟩ - |11⟩。

无论如何，我们注意到，以这种方式可以获得的四种可能态是相互正交的态。我给它们起了名字：|Φ⁺⟩、|Ψ⁺⟩、|Ψ⁻⟩ 和 |Φ⁻⟩。它们可以被完美区分——前提是你能够访问两个量子比特。如果你在实验室里同时拥有两个量子比特，这些态是相互正交的。很明显，态 |Φ⁺⟩ 和 |Φ⁻⟩ 都与 |Ψ⁺⟩ 和 |Ψ⁻⟩ 正交，它们具有不同的奇偶性。|Φ⁺⟩ 和 |Φ⁻⟩（忽略之前的笔误）都具有这样的性质：如果你取两个比特的异或，你总是得到 0（要么是 00，要么是 11）。你当然可以将其与奇校验（10 或 01）区分开来。而且，由于这些相对相位，具有偶校验的两个态（|00⟩ + |11⟩ 和 |00⟩ - |11⟩）是完美可区分的，同样，|10⟩ + |01⟩ 和 |10⟩ - |01⟩ 也是相互正交的。因此，一个正交测量可以完美区分这四种可能性。

现在，对于两个量子比特，能够完美区分四种可能性并不奇怪，标准基也可以做到这一点。我们可以选择基态 |00⟩、|01⟩、|10⟩ 和 |11⟩，它们也是完美可区分的。但这四个态的特殊之处在于，它们都是所谓的“贝尔态”，都是我们这对量子比特的最大纠缠态。因此，如果你在实验室里同时拥有两个量子比特，你可以进行所谓的贝尔测量，投影到这个贝尔基（|Φ⁺⟩、|Φ⁻⟩、|Ψ⁺⟩、|Ψ⁻⟩）上，并完美区分它们。

现在我们可以这样想：如果我准备了 |Φ⁺⟩，爱丽丝进入房间，她向我保证她将对其中一个量子比特应用 I、X、Y 或 Z 中的一个，但她不告诉我具体是哪一个。当我回到房间时，我可以进行这个贝尔测量，我可以完美地区分爱丽丝可能应用的这四种可能操作。

好的，现在我们可以看到该做什么了。记住，爱丽丝和鲍勃共享 |Φ⁺⟩，他们在不同的城市。爱丽丝在帕萨迪纳，鲍勃在我们上次检查时在滑铁卢。但是爱丽丝可以应用 I、X、Y 或 Z 中的一个，这将他们共享的纠缠态改变为这四个相互正交的可能性之一。然后她只需将她的量子比特发送给鲍勃。当那个量子比特以完美的保真度到达鲍勃的实验室（因为量子信道是无噪声的）时，鲍勃可以执行贝尔测量。他可以通过这种方式区分四种可能性，从而判断爱丽丝做了四件事中的哪一件。这样，爱丽丝成功地向鲍勃发送了两个经典比特的信息，尽管她只发送了一个量子比特。

我试图在这里用一种时空图来展示它是如何工作的。因为我小时候学过广义相对论，所以我有时喜欢把时间想象成垂直向上运行的。我们想象有人（也许是查理，或者丹佛的那个家伙，我记得是唐纳德？）准备了 |Φ⁺⟩，并将其中一个量子比特给了爱丽丝，另一个给了鲍勃。这发生在很久以前。然后爱丽丝和鲍勃闲逛了一段时间，直到今天出现了爱丽丝需要向鲍勃发送那两个经典比特的紧急需求。她获取那两个经典比特的信息，通过对她那部分纠缠态应用四种操作（I、X、Y、Z）之一来编码它们。她将那个量子比特发送给鲍勃，鲍勃进行贝尔测量，他就能完美地破译那两个经典比特。

我喜欢把这个写成一种不等式，有时称为资源不等式。像这样的不等式想要表达的是：如果你拥有左边列出的资源，大于等于号意味着你可以使用这些资源，并通过某种合适的协议来模拟右边的资源。我这里的符号含义是：Q → 表示从一个参与者向另一个参与者（在我们的例子中是从爱丽丝到鲍勃）发送一个量子比特。方括号 [qq] 表示一个共享的纠缠量子比特对，有时称为一个 E 比特或纠缠比特，它只是一个贝尔对，即完美最大纠缠的量子比特对。有了这个，我们可以模拟 2 个比特的经典信息。C → C 表示从一个参与者向另一个参与者（从爱丽丝到鲍勃）发送一个比特。SD 是我对“超密”的缩写。

这被称为超密编码。这也许有点夸张，但称为“密编码”似乎是公平的——我们设法只使用一个量子比特就发送了两个经典比特（我在这里称之为 C 比特，只是为了更容易书写）。这就是编码“密”的地方。事实上，即使我们有共享纠缠，我们也无法做得比这更好。如果我们只允许消耗这一个 E 比特的纠缠并从爱丽丝向鲍勃发送一个量子比特，我们能发送的最多经典信息就是两个比特。

现在，关于这种通信的一个有趣之处是，通信是完美加密的。我的意思是，假设有一个对手，一个窃听者，非常想知道爱丽丝必须发送给鲍勃的这两个经典比特到底是什么重要信息。因此她忍不住要干扰那些量子比特。也许她在发送途中将它们带入她的实验室进行测量。我指的是从爱丽丝到鲍勃的量子比特，也许伊芙拦截了它并对它应用了一些操作。但如果她的目标是知道爱丽丝发送给鲍勃什么消息，这对她没有任何好处，因为从爱丽丝传输到鲍勃的那个量子比特是最大纠缠态的一半。换句话说，它是一个最大混合态。无论消息是什么，它都是一个最大混合态。因此，窃听者伊芙拦截的量子比特，无论消息内容如何，都处于完全相同的状态。这就是我所说的完美加密：它不会向窃听者透露任何关于爱丽丝消息内容的信息。

现在有一种看待这个问题的方式，让它看起来不那么性感。那就是：可能爱丽丝自己准备了最大纠缠态。也许她昨天就做了。她在实验室里制作了 |Φ⁺⟩。她保留了其中一半（一个量子比特），并将另一半（另一个量子比特）发送给了鲍勃。现在今天，她意识到必须发送这两个经典比特的信息，所以她今天决定，她并不真的希望两个量子比特的纠缠态是 |Φ⁺⟩，她希望它是其他三个相互正交的最大纠缠态之一。因此，她获取这两个 C 比特，通过执行适当的操作进行编码，然后发送第二个量子比特。

所以实际上，爱丽丝正在发送这四个态（|Φ⁺⟩、|Ψ⁺⟩、|Ψ⁻⟩、|Φ⁻⟩）中的一个，因为在这个场景中，她将两个量子比特都发送给了鲍勃。从这个角度来看，爱丽丝使用了两个量子比特的通信向鲍勃发送了两个经典比特。嗯，她本可以通过两次都使用标准基来做到这一点，她可以发送两次 0 或 1 来完成同样的任务。那么超密编码有什么了不起的呢？你看，它确实与她一次发送一个经典比特的场景不同。

首先，她昨天就发送了第一个量子比特。也许她 100 年前就发送了它。但关键是，她在很久以前就把它发送给了鲍勃，远在她知道今天想发送给鲍勃的两个经典比特是什么之前。所以这是他们能够提前很久做的事情。因此，当紧要关头来临，他们必须立即通信，并且只有一百万美元，只能使用一次信道时，爱丽丝仍然有可能发送那两个经典比特的信息。即使她很久以前就发送了一个量子比特，也为时不晚。

此外，为了共享纠缠态 |Φ⁺⟩，并不需要爱丽丝在她的实验室里制作 |Φ⁺⟩，然后发送其中一个量子比特给鲍勃。也可以是反过来：鲍勃制作了 |Φ⁺⟩，并发送其中一个量子比特给爱丽丝。那么，建立那个纠缠对首先是鲍勃到爱丽丝的量子通信，然后是爱丽丝到鲍勃的量子通信。这与经典场景非常不同，因为通信首先是鲍勃到爱丽丝，然后是爱丽丝到鲍勃，但不知何故，我们设法通过这种方式从爱丽丝向鲍勃发送了两个经典比特的信息。

或者，正如我之前所说，也不一定是爱丽丝或鲍勃，可能是丹佛的纠缠大师唐纳德制作了纠缠对，并将其中一个量子比特从爱丽丝发送给鲍勃。

再次强调，正如我已经强调过的，我所描述的通信加密。在这种情况下，当他们使用纠缠字母表发送两个经典比特时，任何一个量子比特本身都不传递任何信息，只有这对量子比特一起才传递信息，因为任何一个量子比特本身都只是处于最大混合态，不告诉我们任何信息。这与爱丽丝只是两次使用经典通信协议，或者她第一次发送 0 或 1，第二次发送 0 或 1 的场景形成对比，在那种情况下，信道的每次使用当然都包含信息。

这就是超密编码。现在我要谈谈量子隐形传态，这是一个相关的任务。在某种意义上，它有点像是超密编码的“对偶”。无论“对偶”是什么意思，我们经常用这个词来表示两件事以某种方式相互关联。但在这种情况下，我可以具体说明我的意思。

如果我们看幻灯片左边描述的超密编码协议，我已经向你们描述过了：创建一个纠缠对，分发给爱丽丝和鲍勃，他们持有这些纠缠对一段时间。然后有一天，爱丽丝收到两个她想发送的经典比特信息，她对她的那部分纠缠对执行四种可能操作之一，将量子比特发送给鲍勃，然后鲍勃在他那端进行贝尔测量，他就能破译那两个经典比特。所以在这里，爱丽丝应用一个幺正变换，从爱丽丝到鲍勃的通信是量子通信（一个量子比特的价值），鲍勃进行贝尔测量，最终协议实现的是从爱丽丝到鲍勃的两个经典比特的通信。

量子隐形传态做什么呢？任务是发送量子信息从爱丽丝到鲍勃。事实上，我们可以想象一个场景，爱丽丝和鲍勃像往常一样在不同的城市，爱丽丝在帕萨迪纳，鲍勃在滑铁卢。今天有一个量子比特到达了爱丽丝的实验室，或者可能是爱丽丝在她的实验室里做的某个实验创造的。最重要的是，这个量子比特必须在不被修改或损坏的情况下转移到鲍勃那里。也许爱丽丝做了一个实验，创造了这个量子比特的奇异态，编码方式使得她的实验室没有设备来测量这个量子比特，但鲍勃有。如果它在鲍勃的实验室里，他就能进行他们需要的测量以获得实验结果。所以爱丽丝必须设法将这个量子比特从她的实验室弄到鲍勃的实验室，但它非常遥远，而且那个该死的量子信道非常不可靠——有些日子它工作，今天它不工作，现在没有连接爱丽丝和鲍勃的量子信道。而且，你知道，他们的竞争对手也在做实验，他们明天就要报告结果了，所以她最好今天无论如何都要把那个量子比特弄给鲍勃。

事实上，她可以做到。以下是她的做法：记住他们有纠缠，就像在超密编码中一样，他们有纠缠，这是很久以前准备的，他们共享的。爱丽丝收到一个量子比特（也许是她实验中创造的），她必须发送给鲍勃。她对她想发送的量子比特和她那部分纠缠对 |Φ⁺⟩ 进行贝尔测量。当她进行贝尔测量时，她是在基 |Φ⁺⟩、|Φ⁻⟩、|Ψ⁺⟩、|Ψ⁻⟩ 上进行测量，有四种可能的结果。她告诉鲍勃测量结果是什么，那是两个经典比特的信息，足以区分四种可能性。她通过电话将那两个经典比特发送给鲍勃，他们只需要进行经典通信。

现在，当鲍勃收到那个信息时，他取出他与爱丽丝共享的纠缠态中他那部分（这是很久很久以前创造的），并对他的那部分纠缠对应用四种可能操作之一：恒等 I、泡利 X、泡利 Y、泡利 Z。通过这样做，神奇地，他在他的实验室里恢复了爱丽丝曾经拥有的量子比特的完美副本。这就是量子隐形传态的工作原理。

所以你可以看到它与超密编码多么相似，只是它有点内外翻转了。在超密编码中，我们使用量子通信将爱丽丝拥有的经典信息发送给鲍勃，爱丽丝执行幺正变换，鲍勃进行贝尔测量。在这里，我们使用经典通信（两个比特的价值）将量子比特从爱丽丝发送给鲍勃，爱丽丝进行贝尔测量，鲍勃执行一个取决于爱丽丝测量结果的幺正变换。

因此，我可以将其写成一个资源不等式，这次我用 TP 缩写表示隐形传态。再次强调，这意味着左边的资源足以实现右边的资源，即通过两个比特的经典通信（从爱丽丝到鲍勃的 C → C）并消耗一个 E 比特的共享纠缠（爱丽丝和鲍勃准备或很久以前分发给他们的纠缠对），这足以实现从爱丽丝到鲍勃的一个量子比特价值的通信（这就是 Q → Q 的含义）。这就是量子隐形传态告诉我们的。

现在，如果他们当初没有量子纠缠，他们就无法做到这一点。如果他们没有量子信道，也没有共享的量子纠缠，爱丽丝就没有办法将那个量子比特发送给鲍勃。他们当然可以进行经典通信，但爱丽丝不知道她想让鲍勃测量的量子比特的状态。好吧，也许她可以测量它，但即使她可以，关键是她不知道量子比特的状态。所以也许她确实测量了它，然后她告诉鲍勃测量结果，然后鲍勃可以利用关于测量结果的信息来尝试重建状态。但如果她事先不知道量子比特的状态，她就不知道一种不会干扰它的测量方法。因此，通常当她测量它或以某种方式处理它时，她会改变量子比特的状态，然后就没有办法将状态传达给鲍勃了。

## 验证量子隐形传态协议

所以我必须说服你这真的有效。这是隐形传态协议，我已经告诉过你它是什么。爱丽丝有一个量子比特，她想发送给鲍勃，他们有一个共享的纠缠对。爱丽丝对她那部分纠缠对和刚刚出现在她实验室的量子比特进行贝尔测量，她将测量结果（她的贝尔测量结果）发送给鲍勃，然后鲍勃应用泡利算子之一来恢复量子比特。

因此，我们只需要做一个计算来验证它确实有效。

现在，让我们将与 C 和 A 共享的每个贝尔态对应的项合并在一起，爱丽丝将对 C 和 A 进行贝尔测量。所以我必须合并 a|Φ⁺⟩|0⟩ + b|Φ⁺⟩|1⟩，这给了我这一项：C 和 A 共享 |Φ⁺⟩，现在鲍勃系统有 a|0⟩ + b|1⟩。

然后我有两个 |Φ⁻⟩ 项，这一个和这一个，这给我 |Φ⁻⟩ 对于 C 和 A，乘以鲍勃系统的 a|0⟩ 减去 b|1⟩。

让我们稍微整理一下。好的，我在网上找到了这个……（检查一下）。或者再仔细听一遍，我得做点什么。

所以，如果 C 和 A 是 |Φ⁺⟩，这只是原始态 |ψ⟩ = a|0⟩ + b|1⟩ 在鲍勃系统中。
如果 C 和 A 是 |Ψ⁺⟩，那么这是对 |ψ⟩ 应用 X 的结果，因为你看，|0⟩ 变成了 |1⟩，|1⟩ 变成了 |0⟩。所以它只是对 |ψ⟩ 应用了比特翻转。我把它写成 X|ψ⟩ 对于鲍勃系统。
如果 C 和 A 是 |Ψ⁻⟩，那么除了一个相位，这只是对 |ψ⟩ 应用 Y，因为不是 a|0⟩ + b|1⟩，我得到了 a|1⟩ - b|0⟩，这翻转了比特并改变了基态的相对相位。
当 C 和 A 是 |Φ⁻⟩ 时，我有 a|0⟩ - b|1⟩，这等同于对鲍勃系统的 |ψ⟩ 应用 Z。

所以现在，我已经用 C 和 A 的贝尔基重写了这个态，即爱丽丝想发送的量子比特 |ψ⟩ 和共享纠缠 |Φ⁺⟩ 的组合。

现在，如果我们想象对 C 和 A 执行贝尔测量，我们会注意到求和中的这四项都以系数 1/2 出现。这意味着贝尔测量的四种结果中的每一种都是等概率的，它们都以概率 1/4 发生。

所以爱丽丝进行测量，得到贝尔基的四种可能性之一：|Φ⁺⟩、|Ψ⁺⟩、|Ψ⁻⟩、|Φ⁻⟩，它们都是等可能的，然后告诉鲍勃她得到了哪个结果。当鲍勃知道结果后，如果爱丽丝的结果是 |Φ⁺⟩，他什么也不做（应用恒等算子 I）；如果她的结果是 |Ψ⁺⟩，他应用 X；如果她的结果是 |Ψ⁻⟩，他应用 Y；如果她的结果是 |Φ⁻⟩，他应用 Z。你可以看到，以爱丽丝的测量结果为条件，记住 X²、Y² 和 Z² 都等于恒等算子，鲍勃应用的操作如何恰好撤销了应用于 |ψ⟩ 的泡利算子，因此鲍勃恢复了 |ψ⟩，正是今天出现在鲍勃实验室的状态。

现在，在爱丽丝发送消息给鲍勃之前，在他收到消息之前，鲍勃不知道他有什么。事实上，如果他测量他的量子比特，它只会被描述为最大纠缠态的一半，其密度算子是 (1/2)I，这不会告诉他任何关于爱丽丝试图发送给他的量子态的信息。一旦他有了消息，状态的纯度就恢复了。有了爱丽丝贝尔测量结果的知识，鲍勃知道他必须应用这四个操作 I、X、Y、Z 中的一个才能得到爱丽丝试图发送给他的状态。爱丽丝和鲍勃都不知道 |ψ⟩ 是什么。如果鲍勃还没有测量它，他对它一无所知，只知道他拥有它。

现在，他们不知道任何关于它的事情是好事，因为如你所知，如果你获取了一个未知量子态的信息，你会损坏它，会引起一些干扰，而这里没有发生这种情况。爱丽丝不知道 |ψ⟩ 是什么，鲍勃也不知道 |ψ⟩ 是什么，但协议已经将态 |ψ⟩ 完美地从爱丽丝的实验室传送到了鲍勃的实验室。

记住我们讨论过如何不能克隆未知的量子态，我想我在第一讲中提到过。嗯，这看起来有点可疑，因为爱丽丝的实验室里有一个量子比特，突然鲍勃的实验室里也有了一个。这听起来像克隆吗？也许像，但并不是。因为爱丽丝在进行贝尔测量时不得不销毁了她的量子比特副本，她无法再在她的实验室里恢复它了。她所能做的就是将测量结果发送给鲍勃，现在一旦鲍勃知道了那个经典消息，鲍勃就拥有了这个态。从来没有一个状态的两个副本同时存在的情况。为了让态 |ψ⟩ 神奇地出现在鲍勃的实验室，它必须在爱丽丝的实验室被销毁。而且，在鲍勃能够恢复它之前，必须有时间让经典消息从爱丽丝传播到鲍勃。

所以，如果你从超光速通信的角度来看，这个态并没有超光速传播。在鲍勃能够接收到那个经典消息之前，它不会出现在鲍勃的实验室。因此，正如相对论者所说，|ψ⟩ 在鲍勃实验室的重新出现，是在爱丽丝进行贝尔测量事件的未来光锥内，并且不存在任何时空切片，使得状态的两个副本同时存在。

现在，那两个经典比特，如果那个爱管闲事的伊芙又在偷听并知道了那两个经典比特是什么，她会知道这个爱丽丝发送给鲍勃的秘密量子态的某些信息吗？嗯，不，她不会知道任何关于它的信息，因为那两个经典比特是均匀随机的——贝尔测量的所有四种可能结果都是等可能的，无论状态是什么，都会发生这种情况。所以那些均匀随机的比特不会告诉我们任何关于量子态的信息。

那么，这是如何工作的呢？我们如何能够仅通过发送两个经典比特就将量子态从帕萨迪纳传送到滑铁卢？我的意思是，在某种程度上，这有点令人惊讶，甚至有点神奇。是经典比特以某种方式从爱丽丝发送信息给鲍勃吗？那不可能真的是这样，对吧？因为它们只是两个均匀随机的比特，它们本身并不告诉我们任何关于量子态的信息。量子纠缠本质上是必需的，没有它就行不通，但仅靠它本身也不够，没有经典通信也不行。正是两者结合在一起才真正做到了这一点。

记住，爱丽丝和鲍勃共享的、在协议中使用的纠缠，与我们试图发送的量子态无关。他们可能去年就准备了那个纠缠对，知道有一天会需要它。事实上，这正是发生的情况。他们去年确实做了。现在那个量子比特今天出现了，当然那个纠缠对不知道今天会出现什么量子比特。所以，仅靠纠缠本身并没有用，如果鲍勃只有纠缠的一半，他只有一个最大混合态，那并不是非常宝贵的商品。是共享纠缠和经典通信两者共同使这种隐形传态成为可能。

我一直说没有人知道 |ψ⟩ 是什么，没有人知道 |ψ⟩ 是什么，那么我们如何说隐形传态真的成功了呢？有没有办法验证它？嗯，原则上，我们可以。可能有另一个参与者查理，他准备了 |ψ⟩，他知道他是怎么做的，他知道状态是什么。所以他把它给了爱丽丝，他没有告诉爱丽丝状态是什么或他是如何准备的任何信息。但爱丽丝和鲍勃执行这个协议，然后查理去拜访鲍勃的实验室，当他到达那里时，他可以执行一个测量，将量子比特投影到两个态上：要么是查理知道的态 |ψ⟩，要么是 |ψ⟩ 的正交补 |ψ⊥⟩。因此，如果隐形传态完美工作，那么当他进行那个测量时，每次他都会得到结果 |ψ⟩，永远不会得到结果 |ψ⊥⟩。

所以爱丽丝和鲍勃可以尝试这个协议一千次。每次查理给他们一个态，他可能每次都准备一个不同的态来为难他们。但他应该总是能成功验证态 |ψ⟩，永远不会是 |ψ⊥⟩。在他这样做了一千次之后，我们相当确信这种隐形传态是真实存在的，并且状态确实以非常高的保真度被传输。

## 高维推广与信息流视角

好的，现在我想以一种更一般的方式来看待这个问题，只需进入更高维度。我认为这有助于理解隐形传态是如何工作的，并为我们提供了一种推广它的方法，这有时很有用。所以让我们稍微谈谈这个。

现在我将假设爱丽丝和鲍勃有 n 维希尔伯特空间。他们共享一个他们的 n 维系统的最大纠缠态。所以这是我们在量子比特情况下讨论的 |Φ⁺⟩ 的类比，我称之为大写 Φ。这是标准的最大纠缠态，与我们讨论状态和信道之间的对偶性（Choi-Jamiołkowski 同构）时谈到的相同。我为爱丽丝的系统选择一个基，为鲍勃的系统选择一个基，状态只是 |i⟩⊗|i⟩ 的均匀叠加，对 i 求和。

现在有一些有用的东西需要注意。假设爱丽丝对这个最大纠缠态（我称之为大写 Φ）应用某个幺正变换。我将这个幺正写为 n×n 幺正矩阵，作用在基元素 |i⟩ 上，它会给我 ∑ⱼ Uⱼᵢ |j⟩。因此，当爱丽丝应用 U 时，态大写 Φ 变成这个态：归一化因子乘以 ∑ᵢⱼ Uⱼᵢ |j⟩⊗|i⟩ 对于鲍勃系统。嗯，这里我只是取了 U 并移动了它，取了转置，所以我将 Uⱼᵢ |j⟩|i⟩ 写成 Uᵢⱼᵀ |i⟩|j⟩。但你看，如果我现在对 i 求和，那等同于我对基态 |j⟩ 应用了 Uᵀ。所以这等同于爱丽丝的恒等算子与鲍勃的 Uᵀ 的直积作用在大写 Φ 上。换句话说，无论爱丽丝对状态应用什么幺正变换，最大纠缠态的等效变化都可以通过鲍勃对他的那部分状态应用一个幺正来实现，但如果爱丽丝应用 U，那么鲍勃必须应用 Uᵀ 才能以同样的方式改变最大纠缠态。

同样地，如果他们都应用幺正，假设爱丽丝应用 U，鲍勃应用 V。嗯，我可以这样想：它是 (U⊗I) × (I⊗V)。在 U⊗I 的情况下，我可以用 I⊗Uᵀ 替换它，然后应用 I⊗V，所以结果是这两个的乘积将是 I⊗(VUᵀ)。因此，让爱丽丝应用幺正和鲍勃应用幺正，可以通过只让鲍勃应用适当的幺正来实现。同样，我们可以通过仅作用于爱丽丝一侧的变换（幺正）来获得爱丽丝应用 U 和鲍勃应用 V 所产生的状态。那么我会取 I⊗V 并将其写为 Vᵀ⊗I，然后应用 U⊗I，我会得到 (UVᵀ)⊗I，或者如果你愿意，我可以直接使用我在这里观察到的，取爱丽丝应用的 (UVᵀ)⊗I，然后取它的转置，即鲍勃应用的 (VUᵀ)，这给了我相同的效果。

所以事实上，任何最大纠缠态都可以通过爱丽丝一侧或鲍勃一侧应用幺正从任何其他最大纠缠态获得。在最大纠缠态的情况下，施密特基是模糊的，我可以为爱丽丝或鲍勃选择任何正交归一基作为施密特基。但一旦我为爱丽丝固定了正交归一基，如果我有一个特定的态，那么它将锁定鲍勃的相应基，通过施密特形式与之相关。但对于一般的纠缠态，爱丽丝有一个正交归一基，鲍勃有一个，它们彼此完美关联，所有这些态都通过在爱丽丝一侧和鲍勃一侧应用幺正相关，但我要说的是，他们不必都应用幺正，只需要其中一个应用幺正就可以得到我想要的任何纠缠态。

因此，这两个 n 维系统的任何最大纠缠态都可以写成我们的标准态（称为大写 Φ，对于爱丽丝和鲍勃的某个特定基选择），然后爱丽丝应用幺正 U 于该态。作为一种符号，我将称之为大写 Φ(U)，即取大写 Φ 并在爱丽丝一侧应用 U 得到不同的最大纠缠态，并且所有最大纠缠态都可以写成那种形式。

现在我想考虑一个类似于我们之前讨论的量子比特协议，但现在爱丽丝的实验室中进行的是最大纠缠基上的测量。现在有一个 n 维系统进入爱丽丝的实验室，并且有一个态，比如大写 Φ，或者一般来说，U 张量积大写 Φ，一个爱丽丝和鲍勃共享的最大纠缠态。

因此，为了弄清楚那里发生了什么，我想计算这个量，这个与 C 和 A 的 Φ 相关的左矢。我可以像往常一样将其视为一个线性算子，当它作用于系统 A 和 B 的状态时，结果是一个将系统 C 映射到系统 B 的算子。

我们可以通过显式计算来验证它是如何工作的。所以我写下左矢 ⟨Φ| 的表达式，作为 ∑ⱼ (1/√n) ⟨j|⟨j|。对于右矢 |Φ⟩，是 ∑ᵢ (1/√n) |i⟩|i⟩。所以我从这两个 1/√n 因子得到 1/n。现在你看，我这里有一个系统 A 的右矢和一个相应的系统 A 的左矢。但它们是正交归一基，所以这是一个内积，它将坍缩为只有 i=j 的项。因此，现在剩下的项只有那些系统 C 和系统 B 的基态具有相同索引的项。所以我要取这个系统 C 的左矢并将其移到右边，取系统 B 的右矢并将其移到左边。所以我们这里得到的是，除了归一化因子 1/n 外，看起来像一个恒等算子。但我不称它为恒等算子，因为它实际上是将一个系统映射到另一个系统，它不是作用在系统 B 上的恒等算子，而是一个将系统 C 的状态转移到系统 B 的状态的算子。所以我称它为转移算子。但它是一个转移算子，它将系统 C 的一个状态转移到系统 B 的相同状态。
