# 002：密度算符

在本节课中，我们将要学习量子计算与量子信息理论中的一个核心工具：**密度算符**。我们将从回顾封闭量子系统的公理化表述开始，然后探讨当系统与其环境相互作用（即系统是“开放”的）时，描述方式将如何变化。密度算符正是描述这种开放系统的关键数学对象。

## 回顾：封闭量子系统的公理

上一讲我们概述了量子计算的主题。本节中，我们首先回顾描述封闭量子系统的数学框架，即量子力学的公理化表述。这些公理定义了如何用数学描述系统的状态、可观测量、测量过程、动力学以及复合系统。

**状态描述**：一个量子系统的**状态**由一个复希尔伯特空间中的**射线**描述。这意味着状态是一个归一化的向量（称为**态矢**），且其整体相位没有物理意义。我们用狄拉克符号 `|ψ⟩` 表示态矢，其对应的对偶向量（**左矢**）记为 `⟨ψ|`。两个向量的内积记为 `⟨φ|ψ⟩`。

**可观测量**：**可观测量**由作用在该希尔伯特空间上的**自伴（厄米）线性算符** `A` 描述。自伴算符满足 `A = A†`，其中 `†` 表示厄米共轭。这样的算符可以进行谱分解：
`A = Σ_n a_n E_n`
其中 `a_n` 是实数本征值，`E_n` 是投影到对应本征空间的**正交投影算符**，满足 `E_n† = E_n` 和 `E_n² = E_n`，且不同本征值对应的投影算符相互正交。

**测量规则（玻恩规则）**：当在状态 `|ψ⟩` 下测量可观测量 `A` 时，得到本征值 `a_n` 的**概率**为：
`p_n = ⟨ψ| E_n |ψ⟩`
测量后，系统状态**坍缩**到对应的投影态，并重新归一化：
`|ψ'⟩ = E_n |ψ⟩ / √p_n`

**动力学**：量子态随时间的演化由**含时薛定谔方程**描述：
`iħ d/dt |ψ(t)⟩ = H(t) |ψ(t)⟩`
其中 `H(t)` 是系统的**哈密顿算符**，也是一个自伴算符。演化是**幺正**的，即存在一个幺正算符 `U(t)` 使得 `|ψ(t)⟩ = U(t) |ψ(0)⟩`。

**复合系统**：由两个子系统 `A` 和 `B` 组成的复合系统 `AB`，其希尔伯特空间是子系统希尔伯特空间的**张量积**：
`H_AB = H_A ⊗ H_B`
如果 `dim(H_A) = d_A`，`dim(H_B) = d_B`，则 `dim(H_AB) = d_A * d_B`。

## 从封闭系统到开放系统

上一节我们回顾了理想化的封闭量子系统规则。然而，所有真实的量子系统都与其环境存在相互作用，无法完全隔离。这种系统称为**开放量子系统**。开放系统与环境的相互作用会导致**退相干**，这是量子计算中错误的主要来源。为了研究开放系统，我们通常将其视为一个更大封闭系统（“宇宙”）的一部分。我们只关注其中一部分（“系统”），而无法观测或操控其余部分（“环境”）。

## 密度算符的引入

现在，我们来看看当只关注复合系统的一部分时，如何描述该部分的状态。考虑一个由 Alice（系统 A）和 Bob（系统 B）共享的复合系统。假设复合系统处于一个纯态 `|ψ_AB⟩`，但 Alice 只能访问系统 A。

让我们从一个具体例子开始。假设复合系统处于一个特殊的纠缠态：
`|ψ_AB⟩ = a |0_A⟩⊗|0_B⟩ + b |1_A⟩⊗|1_B⟩`
其中 `|a|² + |b|² = 1`。

如果 Alice 在她的系统 A 上测量一个可观测量 `M_A`（对应总可观测量 `M_A ⊗ I_B`），其期望值计算如下：
`⟨M_A⟩ = ⟨ψ_AB| (M_A ⊗ I_B) |ψ_AB⟩`
经过推导（利用 Bob 系统上基矢的正交性），这个期望值可以写为：
`⟨M_A⟩ = Tr_A ( ρ_A M_A )`
其中 `ρ_A` 是一个作用在系统 A 希尔伯特空间上的算符，称为**密度算符**（或密度矩阵）：
`ρ_A = |a|² |0_A⟩⟨0_A| + |b|² |1_A⟩⟨1_A|`

这个表达式有清晰的物理解释：它相当于 Alice 的系统以概率 `|a|²` 处于态 `|0⟩`，以概率 `|b|²` 处于态 `|1⟩` 的**统计混合**。重要的是，对于 Alice 可能进行的任何只涉及系统 A 的测量，仅凭密度算符 `ρ_A` 就足以预测所有结果。它**完全描述**了 Alice 系统的所有可观测性质，但丢失了系统 A 与 B 之间关联（纠缠）的信息。

## 密度算符的一般定义与性质

更一般地，对于一个复合系统 `AB` 的任意纯态 `|ψ_AB⟩`，系统 A 的密度算符通过对复合系统的投影算符 `|ψ_AB⟩⟨ψ_AB|` 在系统 B 上进行**偏迹**得到：
`ρ_A = Tr_B ( |ψ_AB⟩⟨ψ_AB| )`
如果 `|ψ_AB⟩ = Σ_{i,μ} a_{iμ} |i_A⟩⊗|μ_B⟩`，那么密度算符的矩阵元为：
`(ρ_A)_{ij} = Σ_μ a_{iμ} a_{jμ}^*`

一个算符 `ρ` 要成为有效的密度算符，必须满足以下三个性质：
1.  **厄米性**：`ρ† = ρ`。这意味着它的本征值是实数。
2.  **半正定性**：对于任意态矢 `|φ⟩`，有 `⟨φ| ρ |φ⟩ ≥ 0`。这意味着它的本征值都是非负实数。
3.  **迹为1**：`Tr(ρ) = 1`。这保证了概率归一化。

由于密度算符是厄米且半正定的，它总可以在某个正交基下对角化：
`ρ = Σ_a p_a |a⟩⟨a|`
其中 `{|a⟩}` 是一组正交归一基矢，`p_a` 是非负实数，且 `Σ_a p_a = 1`。这给出了密度算符的**系综解释**：系统以概率 `p_a` 处于纯态 `|a⟩`。

*   **纯态**：如果密度算符只有一个非零本征值（为1），即 `ρ = |ψ⟩⟨ψ|`，则系统处于纯态。
*   **混合态**：如果密度算符有多个非零本征值，则系统处于混合态。

## 施密特分解与纠缠

上一节我们定义了密度算符。现在，利用密度算符的对角化，我们可以将复合系统 `AB` 的任意纯态 `|ψ_AB⟩` 写成一个标准形式，即**施密特分解**。

选择系统 A 的基矢 `{|i_A⟩}`，使其恰好是 A 的密度算符 `ρ_A` 的本征态基。那么，总态可以写成：
`|ψ_AB⟩ = Σ_i √p_i |i_A⟩⊗|i_B'⟩`
其中 `p_i` 是 `ρ_A` 的本征值（即概率），`{|i_B'⟩}` 是系统 B 中的一组正交归一态（可通过重新标定得到）。数 `√p_i` 称为**施密特系数**。

施密特分解揭示了以下关键点：
*   系统 A 和系统 B 的密度算符 `ρ_A` 和 `ρ_B` 具有**相同的非零本征值谱** `{p_i}`。
*   非零本征值的个数称为态的**施密特秩**。
*   如果施密特秩为1，则总态是一个**乘积态**（可分离态）：`|ψ_AB⟩ = |ψ_A⟩⊗|ψ_B⟩`。此时系统间没有纠缠。
*   如果施密特秩大于1，则总态是**纠缠态**（不可分离态）。

纠缠是一种纯粹的量子关联，无法通过经典手段实现。以下是一些关于纠缠的重要说明：

**纠缠的创建**：如果 Alice 和 Bob 最初共享一个乘积态（无纠缠），他们仅通过各自的**局域操作**和彼此间的**经典通信**，**无法创建**纠缠。要创建纠缠，需要执行联合的量子操作，或者通过量子信道直接传输量子系统。

**量子关联 vs 经典关联**：纠缠态中的关联是“量子”的，体现在它们存在于**多个测量基**中。例如，著名的 Bell 态 `(|00⟩ + |11⟩)/√2` 不仅在 Z 基（`|0⟩, |1⟩`）下关联，在 X 基（`|+⟩, |-⟩`）下也关联。而经典关联（如一对总是相同的硬币）只在一种“观看方式”下关联。

**量子擦除**：这个概念阐明了相干叠加与统计混合的区别。当系统 A 与系统 B 纠缠时，从 A 的视角看，关于 A 某个基（如 Z 基）的信息可能被 B “记录”。这使得 A 无法表现出该基下两个态之间的干涉效应（即相干叠加的行为）。然而，如果 Bob 对 B 进行一个特定测量（如 X 基测量），他可能会“擦除”那个记录信息，从而在 A 中恢复干涉的可能性。这演示了“信息”的存在与否如何影响量子系统的行为。

## 总结

本节课中我们一起学习了描述开放量子系统的核心工具——**密度算符**。我们从封闭系统的量子力学公理出发，探讨了当只关注更大系统的一部分时，需要用密度算符来描述该子系统的状态。密度算符是一个满足厄米性、半正定性且迹为1的算符，它提供了系统所有可观测量的完整统计信息。我们还学习了如何通过**施密特分解**来分析复合纯态的纠缠结构，并讨论了纠缠的基本性质及其与经典关联的区别。这些概念为我们后续研究开放系统的演化、广义测量以及退相干现象奠定了坚实的基础。
