# 量子计算：第7讲B：贝尔多胞形补充说明

在本节课中，我们将对第7讲的内容进行一个简短的补充说明。我们将回顾“局部构型”与“非局部构型”的核心概念，明确它们之间的区别，并修正一个关于“极集”定义的口误。

---

上一节我们介绍了贝尔多胞形的基本概念。本节中，我们将更详细地探讨构成这些多胞形的“构型”，并区分局部与非局部的不同。

## 局部构型

一个**局部构型**意味着，对于爱丽丝的每一个输入，都存在一个特定的输出以概率1发生，其他输出概率为零。鲍勃的情况亦然。其行为是完全确定性的，且一方的输出不依赖于另一方的输入。

我们可以用以下公式来描述一个局部构型：

```
P(x, y | a, b) = δ(x, x_a) * δ(y, y_b)
```

其中：
*   `δ` 是克罗内克δ函数（当两个参数相等时为1，否则为0）。
*   `x_a` 是由爱丽丝的输入 `a` 唯一确定的输出值。
*   `y_b` 是由鲍勃的输入 `b` 唯一确定的输出值。

以下是关于局部构型数量的计算：
*   假设爱丽丝和鲍勃各有 `M` 种可能的输入设置。
*   对于每一种输入，有 `V` 种可能的输出结果。
*   爱丽丝需要为她的 `M` 种输入各确定一个输出，因此有 `V^M` 种可能性。
*   同理，鲍勃也有 `V^M` 种可能性。
*   因此，总的局部构型数量为 `V^(2M)`。

## 非局部构型

一个**非局部构型**同样是确定性的，但允许一方的输出依赖于另一方的输入。这意味着爱丽丝的输出可以同时取决于她自己的输入 `a` 和鲍勃的输入 `b`。

其概率分布公式如下：

```
P(x, y | a, b) = δ(x, x_{ab}) * δ(y, y_{ab})
```

其中 `x_{ab}` 和 `y_{ab}` 现在由输入对 `(a, b)` 共同决定。

以下是关于非局部构型数量的计算：
*   此时，爱丽丝的输出取决于 `M * M = M^2` 种可能的输入组合。
*   为这 `M^2` 种组合中的每一种确定一个输出，有 `V^(M^2)` 种可能性。
*   鲍勃亦然，也有 `V^(M^2)` 种可能性。
*   因此，总的非局部构型数量为 `V^(2 * M^2)`。

## 构型数量对比与意义

通过比较局部与非局部构型的数量，我们可以直观理解贝尔多胞形的复杂性。

以一个常见实验为例：
*   爱丽丝和鲍勃各有2种输入设置 (`M=2`)。
*   每次测量有2种可能结果 (`V=2`)。

计算可得：
*   **局部构型数量**：`2^(2*2) = 2^4 = 16`
*   **非局部构型数量**：`2^(2*2^2) = 2^(2*4) = 2^8 = 256`

显然，非局部构型的数量远多于局部构型。如果我们考虑所有非局部构型的凸组合（即允许概率混合），所形成的多胞形将比仅由局部构型形成的多胞形（即**局部多胞形**或**贝尔多胞形**）复杂得多，拥有更多的面和更高的维度。

## 关于“极集”的修正

在之前的讲解中，关于多胞形的“极集”定义存在一个口误，在此进行修正：

*   **错误表述**：极集是“包含该多胞形的所有极端半空间”的集合。
*   **正确表述**：一个多胞形的**极集**，是**所有**能包含该多胞形的（线性）半空间的集合。然后，我们从极集中挑选出那些“极端的”元素，这些极端元素正好对应于原多胞形的各个面。在贝尔不等式的语境下，这些极端半空间所对应的线性不等式，就是我们要找的**贝尔不等式**。
