# Python编程入门：第1章：变量与数据类型

在本节课中，我们将要学习Python编程中最基础的两个概念：变量和数据类型。它们是构建所有程序的基石，就像盖房子需要砖块和水泥一样。理解它们，是开启编程之旅的第一步。

## 什么是变量？📦

上一节我们提到了变量是编程的基础，本节中我们来看看它的具体定义。你可以把变量想象成一个贴有标签的盒子。这个盒子（变量）可以用来存放数据（比如数字、文字等），而标签（变量名）则帮助我们记住盒子里装的是什么。

在Python中，创建一个变量并给它赋值非常简单，只需要使用等号 `=`。

```python
# 创建一个名为“message”的变量，并存入一段文字
message = "你好，世界！"
# 创建一个名为“number”的变量，并存入一个数字
number = 42
```

当你运行 `print(message)` 时，计算机会找到名为“message”的盒子，取出里面的内容“你好，世界！”并显示出来。

## 基本数据类型🔢

变量这个“盒子”里可以放不同种类的东西，这些东西的“种类”就是数据类型。Python有几种内置的基本数据类型，最常用的有三种。

以下是三种最基础的数据类型介绍：

*   **整数**：没有小数部分的数字，例如 `-10`, `0`, `100`。
*   **浮点数**：带有小数点的数字，例如 `3.14`, `2.0`, `-0.5`。
*   **字符串**：由字符组成的文本，需要用单引号 `‘ ’` 或双引号 `“ ”` 括起来，例如 `‘Python’`, `“学习编程”`。

## 如何查看和转换类型？🔄

我们知道了数据的类型，但如何确认一个变量里数据的类型呢？又如何在需要时改变它的类型呢？

Python提供了一个非常方便的函数 `type()` 来查看任何变量的数据类型。

```python
age = 18
print(type(age))  # 输出：<class 'int'>，表示这是一个整数

price = 9.99
print(type(price)) # 输出：<class 'float'>，表示这是一个浮点数

name = "小明"
print(type(name))  # 输出：<class 'str'>，表示这是一个字符串
```

有时，我们需要将一种数据类型转换为另一种，这个过程称为“类型转换”。例如，将数字转换成文字以便拼接，或者将数字字符串转换成真正的数字以便计算。

以下是常用的类型转换函数：

*   `int()`：将数据转换为整数。例如 `int(“10”)` 会得到整数 `10`；`int(3.99)` 会得到 `3`（直接去掉小数部分）。
*   `float()`：将数据转换为浮点数。例如 `float(“3.14”)` 会得到浮点数 `3.14`；`float(5)` 会得到 `5.0`。
*   `str()`：将数据转换为字符串。例如 `str(100)` 会得到字符串 `“100”`。

```python
# 类型转换示例
num_str = "50"
real_num = int(num_str) + 5  # 先将字符串"50"转为整数50，再加5
print(real_num)  # 输出：55

result = "我的年龄是：" + str(age)  # 将整数18转换为字符串"18"，再与其他字符串拼接
print(result)    # 输出：我的年龄是：18
```

## 动手试一试：一个小例子💻

现在，让我们把上面学到的知识组合起来，完成一个简单的程序。这个程序会询问用户的名字和年龄，然后打印一句问候语。

```python
# 1. 使用 input() 函数获取用户输入，输入的内容默认是字符串类型
user_name = input("请输入你的名字：")
age_str = input("请输入你的年龄：")  # 注意：这里 age_str 是字符串，如 "20"

# 2. 将年龄从字符串转换为整数，以便将来可以进行数学运算
age_int = int(age_str)

# 3. 计算明年的年龄
age_next_year = age_int + 1

# 4. 使用 str() 将数字转换回字符串，与其他文本拼接并输出
print("你好，" + user_name + "！")
print("你明年就" + str(age_next_year) + "岁了。")
```

运行这个程序，按照提示输入，你会看到类似下面的输出：
```
请输入你的名字：安娜
请输入你的年龄：25
你好，安娜！
你明年就26岁了。
```
