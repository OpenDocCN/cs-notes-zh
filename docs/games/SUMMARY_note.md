# GAMES 图形学系列课程导航 🗺️

在本教程中，我们将一起梳理 GAMES 图形学系列课程的整体结构。这个系列涵盖了从计算机图形学基础、游戏引擎原理到计算成像、物理仿真等前沿领域的丰富内容。通过本教程，你可以快速了解课程体系，找到适合自己的学习路径。

## 课程体系概览 📚

GAMES 系列课程是一个系统性的图形学学习资源，主要分为以下几个核心模块：

*   **基础入门与工具**：为初学者打下坚实的数学、编程和工具使用基础。
*   **核心图形学原理**：深入讲解计算机图形学的核心理论与算法。
*   **游戏引擎技术**：聚焦现代游戏引擎的架构、渲染、动画、物理等关键技术。
*   **计算成像与前沿应用**：探索成像原理、计算摄影、VR/AR显示等交叉领域。
*   **高级专题与科研**：涉及几何处理、物理仿真、等几何分析等专业方向。

上一节我们介绍了课程的整体框架，接下来，我们将详细浏览每个模块下的具体课程。

## 模块一：基础入门与工具 🛠️

这个模块旨在帮助初学者快速上手，掌握图形学学习所必需的数学知识和研发工具。

以下是该模块包含的课程列表：

*   **GAMES001-图形学中的数学**：从线性代数到微分几何、优化理论，系统讲解图形学所需的数学知识。
*   **GAMES002-图形学研发基础工具**：涵盖Linux/Windows命令行、Git、编译工具、三维软件等必备工具的使用。
*   **GAMES003-科研基本素养**：指导如何建立领域视野、选择科研课题，培养科研能力。

## 模块二：核心图形学原理 🎨

这个模块是计算机图形学的经典核心课程，由浅入深地讲解渲染、几何、动画等基本原理。

上一节我们学习了基础工具，本节中我们来看看图形学的核心理论课程。以下是该模块包含的课程列表：

*   **GAMES101-现代计算机图形学入门-闫令琪**：广受好评的图形学入门课程，涵盖渲染、光栅化、着色、几何、光线追踪等核心内容。
*   **GAMES102-几何建模与处理**：深入讲解曲线曲面建模、网格处理、参数化等几何处理技术。
*   **GAMES103-基于物理的计算机动画入门**：介绍刚体、柔体、流体等基于物理的动画仿真原理。
*   **GAMES105-计算机角色动画基础**：专注于角色动画，包括运动学、蒙皮、数据驱动动画等内容。

## 模块三：游戏引擎技术 🎮

这个模块深入剖析现代游戏引擎的架构与实现，是进入游戏行业或引擎开发的重要学习资源。

在掌握了图形学核心原理后，我们可以进一步探索如何将这些技术集成到复杂的游戏引擎中。以下是该模块包含的课程列表：

*   **GAMES104-现代游戏引擎：从入门到实践**：全面讲解游戏引擎的架构分层、渲染、动画、物理、网络、工具链等所有核心子系统。
*   **GAMES106-现代图形绘制流水线原理与实践**：深入GPU渲染管线，探讨其原理、优化技巧及最新进展。
*   **GAMES202-高质量实时渲染**：聚焦实时渲染的高级技术，如实时全局光照、环境光遮蔽、阴影等。

## 模块四：计算成像与前沿应用 📷

这个模块将图形学与光学、信号处理相结合，探索新一代成像与显示技术。

游戏引擎让我们能创造虚拟世界，而计算成像则致力于更好地捕捉和理解真实世界。以下是该模块包含的课程列表：

*   **GAMES204-计算成像**：系统介绍计算成像原理，涵盖视觉系统、相机模型、图像处理、高动态范围成像、VR/AR显示等。
*   **显示技术补充：计算光场与全息显示**：对显示技术进行专题补充。
*   **嘉宾座谈**：包含与Snap Research、智能光学仪器等业界专家的座谈内容。

## 模块五：高级专题与科研 🔬

这个模块包含更多专业细分领域和科研导向的课程，适合希望深入某个方向的研究者或开发者。

在学习了主流技术之后，我们可以根据兴趣选择更专精的领域进行深造。以下是该模块包含的课程列表：

*   **GAMES201-高级物理引擎实战指南2020**：深入讲解物理引擎中的拉格朗日视角、欧拉视角、有限元等高级话题。
*   **GAMES203-三维重建和理解**：介绍从图像或扫描数据中重建三维模型，以及三维深度学习等技术。
*   **GAMES301-曲面参数化**：专门研究曲面参数化的理论、方法及应用。
*   **GAMES302-等几何分析**：连接几何建模与仿真分析的前沿领域。
*   **GAMES401-泛动引擎(PeriDyno)物理仿真编程与实践**：以开源引擎为例，讲解物理仿真的并行编程与实践。

## 总结与学习建议 🎯

本节课中，我们一起梳理了GAMES图形学系列课程的全貌。该系列课程体系完整，难度递进，几乎涵盖了图形学所有重要方向。

对于初学者，建议的学习路径是：
1.  **从GAMES101开始**，建立对图形学的基本认知。
2.  **辅以GAMES001和GAMES002**，巩固数学基础和工具技能。
3.  **根据兴趣选择方向**：
    *   想进入游戏行业，深入学习**GAMES104**和**GAMES202**。
    *   对动画仿真感兴趣，学习**GAMES103**和**GAMES105**。
    *   对三维视觉和重建感兴趣，学习**GAMES203**。
    *   对成像和交叉学科感兴趣，学习**GAMES204**。
4.  **在基础扎实后**，可以挑战**GAMES201**、**GAMES301**等高级专题课程。

希望这份导航能帮助你在图形学的海洋中找到方向，开始你的学习之旅！
