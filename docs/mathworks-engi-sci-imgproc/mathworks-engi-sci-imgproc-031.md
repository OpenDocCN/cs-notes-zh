## 分析裂缝面积分布

为了解数据概况，我们使用直方图来查看裂缝面积的分布情况。

```matlab
histogram(dataTable.Area);
```

有趣的是，裂缝面积似乎呈现出两个不同的组别。`histogram`函数允许使用第二个输入参数来指定分组数量，这有助于进一步区分这两个组。

![](img/d3f4afa371000ca6a8337479b9766698_9.png)

```matlab
histogram(dataTable.Area, 20);
```

分界线大约在4000像素的裂缝面积处。

## 对图像进行分类标记

因此，为了标记图像，我们将使用`discretize`函数，将区域面积大于4000像素的裂缝定义为“严重”裂缝。

![](img/d3f4afa371000ca6a8337479b9766698_11.png)

```matlab
cutoffs = [0, 4000, Inf];
categories = {'mild', 'severe'};
dataTable.Risk = discretize(dataTable.Area, cutoffs, 'categorical', categories);
```

此命令根据提供的阈值将面积值划分到不同的区间，并用分类标签（此处为“轻微”和“严重”）进行标记。最后，将输出保存到名为`Risk`的新表格列中。

![](img/d3f4afa371000ca6a8337479b9766698_13.png)

此时，使用`montage`函数查看结果会很有帮助。让我们仅选择风险为“严重”的文件名。

```matlab
severeFiles = dataTable.FileName(dataTable.Risk == 'severe');
montage(severeFiles);
```

![](img/d3f4afa371000ca6a8337479b9766698_15.png)

## 使用图像浏览器查看大量图像

![](img/d3f4afa371000ca6a8337479b9766698_17.png)

以下是创建仅包含严重裂缝图像的数据存储的步骤：

```matlab
severeImageDS = imageDatastore(severeFiles);
```

然后，在图像浏览器应用程序中加载此数据存储变量。

## 检查分类结果与识别异常

![](img/d3f4afa371000ca6a8337479b9766698_19.png)

现在，让我们查看结果。这些严重裂缝似乎都被正确分类了，除了这里的两个裂缝，它们看起来像是被错误标记为严重的轻微裂缝。

![](img/d3f4afa371000ca6a8337479b9766698_21.png)

在自动化图像处理中，这类错误并不意外，很可能是不良分割的结果。我们通过肉眼发现了这些情况，但如果图像数量超过几百张，这种工作流程将非常耗时。

![](img/d3f4afa371000ca6a8337479b9766698_23.png)

那么，如果有数千张图像，如何识别潜在的错误呢？不良分割的一个常见结果是区域数量异常。

因此，在“区域数量”变量中寻找异常值应有助于我们找到分类错误的图像。

## 使用实时任务识别异常值

![](img/d3f4afa371000ca6a8337479b9766698_25.png)

首先，选择数据表作为输入数据。然后，选择“区域数量”作为指定变量，并将清理方法设置为“移除异常值”。提供了多种异常值检测算法，对于这种非正态分布的变量，“中位数”方法是一个不错的选择。

![](img/d3f4afa371000ca6a8337479b9766698_27.png)

## 查看异常图像

现在，要一起查看所有异常图像，首先使用`outlierIndices`变量获取每个异常图像的名称，然后将其作为输入传递给`montage`函数。

![](img/d3f4afa371000ca6a8337479b9766698_37.png)

```matlab
outlierFiles = dataTable.FileName(outlierIndices);
montage(outlierFiles);
```

![](img/d3f4afa371000ca6a8337479b9766698_39.png)

可以注意到，我们之前提到的两个被错误标记为严重的图像也被标记为异常值。但还有其他几个被识别出来以供进一步调查。

## 并排比较图像与其掩膜

为了并排比较图像与其分割掩膜，使用以下代码，该代码应用了课程前面描述的分割函数。`imageName`变量是你从可用异常值列表中选择的位置。

![](img/d3f4afa371000ca6a8337479b9766698_41.png)

![](img/d3f4afa371000ca6a8337479b9766698_42.png)

```matlab
% 假设有一个分割函数 segmentCrack
[originalImg, maskImg] = segmentCrack(imageName);
subplot(1,2,1); imshow(originalImg); title('原始图像');
subplot(1,2,2); imshow(maskImg); title('分割掩膜');
```

在此脚本中，我们还将此变量转换为下拉菜单控件。此功能允许你手动快速在异常图像之间切换。

![](img/d3f4afa371000ca6a8337479b9766698_44.png)

可以注意到，大量的区域几乎总是不良分割的结果。但幸运的是，我们通过检测异常值找到了它们，而不是用肉眼去寻找。
