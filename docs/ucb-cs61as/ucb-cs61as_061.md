# 第 5 课介绍

## 介绍

本课程建立在分层结构的概念之上——由部分组成的结构，这些部分本身又由部分组成，依此类推。我们将利用这些想法在课程结束时在 Racket 中制作一个简单的计算器。

### 警告

本课程比以前的课程更加密集和耗时。确保自己掌握进度，以免落后。

## 先决条件和预期内容

在进行本课程之前需要掌握第 4 课的内容。

在扩展使用`cons`和`list`构建更复杂结构之前，我们将简要介绍它们。
