# 047：程序计数器 🧮

![](img/2b644c04c90819d1d1d649e2901dba26_0.png)

在本节课中，我们将学习如何将计算机的各个组件连接起来，并让它们以可控的方式改变状态。我们将以程序计数器（PC）作为热身，它本身就像一台微型计算机。

## 概述

我们的目标是设计一个数字时序电路，以便精确控制反馈循环。我们已经描述了接口：输入总线、输出总线和控制信号。接下来，我们需要确定组件之间的连接，并建立控制信号（如写信号）变为1的时序。本节我们将以TOY-8计算机的程序计数器为例，它是一个小型数字时序电路。

## 设计需求

程序计数器需要满足两个主要功能。首先，它需要能够递增。其次，我们需要考虑当多个总线连接到同一电路时如何处理。

### 递增器设计

首先，我们需要一个计算 `x + 1` 的组合电路。虽然可以使用全加器，但作为电路设计练习，我们可以从头开始构建。

我们可以借鉴加法器的思想，但忽略Y输入，并将进位输入设为1。这相当于将数字 `x` 与0相加，且进位为1。

以下是其真值表：
*   **进位位**：当前进位位和 `x` 位。仅当两者都为1时，才需要输出进位1。这正好是**与**函数。
*   **和位**：当前进位位和 `x` 位。如果两者都为0，结果为0；如果一个是0另一个是1，结果为1；如果两者都为1，结果为0并产生进位1。这个真值表对应的是**异或**函数。

因此，我们的递增器电路类似于加法器，但使用**与门**和**异或门**，而不是多数门和奇偶门。电路顶部是输入总线 `X`，进位信号像在加法器中一样级联传递，`x + 1` 的结果始终在底部输出。

这是一个很好的组合电路设计练习，它将作为程序计数器的一个组件。

### 多总线连接问题

接下来，我们需要解决多总线连接的问题。如果一个组件的输出需要连接到多个其他组件，这没有问题，我们可以使用T型连接，即用一根导线连接，所有总线上的值都相同。

但是，如果一个组件的输入来自多个其他组件，这就是一个问题。例如，算术逻辑单元（ALU）的输入可能来自寄存器（算术运算结果）、内存（加载指令）或寄存器（加载地址）。这些总线上的值通常不同，我们如何连接它们呢？

我们需要一个**总线多路选择器**。这是一个关键的数字电路设计概念。

我们将使用一个**独热M路总线多路选择器**，它本质上就是多个我们之前设计的独热多路选择器的副本。

其接口如下：有多个输入总线（例如三个），每个输入总线对应一个选择线，还有一个输出总线。前提条件是，最多只有一条选择线为1。输出总线的值将与**被选中的**输入总线的值相同。

在示例中，三个不同的值进入顶部的三路总线多路选择器。由于“选择线2”为1，因此底部输入总线的值将出现在输出总线上，作为下一个电路的输入。

为了清晰区分输入和输出，我们约定总线从顶部进入，从底部输出。

总线多路选择器的实现非常简单：对于每一位，我们只需构建一个独热多路选择器，将选择线与对应的输入位进行**与**操作，然后将所有结果通过一个**或门**汇总。由于只有一条选择线为1，因此只有一路输入会通过，从而完成选择功能。

我们将立即使用这个组件来实现程序计数器，以便在两种不同的PC输入之间进行选择。

## 程序计数器实现

### 接口定义

程序计数器保存一个地址。它有一个输入总线（例如，在分支指令时从指令寄存器的地址位加载），一个始终可用的输出总线（连接到内存以获取指令）。

控制信号包括：
*   **递增**：当需要改变其值时，使其加1。
*   **加载**：当需要改变其值时，从输入总线加载值。
*   **写**：当递增或加载信号有效时，写控制线将导致值实际发生改变。

它是一个寄存器，当前值始终在输出总线上可用。

### 内部组件与连接

显然，PC内部需要一个4位寄存器和一个递增器。现在我们需要考虑如何连接这些组件以实现所需行为。

所需的连接包括：
1.  从输入总线到寄存器（用于加载）。
2.  从递增器到寄存器（用于存储 `x+1`）。
3.  从寄存器到递增器（提供要递增的数 `x`）。
4.  从寄存器到输出总线（提供当前地址）。

由于寄存器的输入可能来自两个不同的源（递增结果或外部输入），并且它们在不同时刻的值可能不同，因此我们需要一个**总线多路选择器**来选择输入。

### 完整电路图

实际的PC电路图如下：
*   底部是**PC寄存器**。
*   顶部是**递增器**。
*   中间是**总线多路选择器**。

连接方式如下：
*   输入总线连接到总线多路选择器的一个输入。
*   递增器的输出连接到总线多路选择器的另一个输入。
*   总线多路选择器的输出连接到PC寄存器的输入。
*   PC寄存器的输出一方面反馈给递增器作为输入，另一方面连接到输出总线。
*   总线多路选择器的选择线由“递增”和“加载”控制信号控制。当“递增”为高电平时，选择递增器的输出；当“加载”为高电平时，选择输入总线。

### 工作原理与控制时序

通过这些连接和电路，程序计数器将具有正确的行为。它支持两种不同的控制信号序列：

1.  **加载-写入序列**：当“加载”信号有效，随后产生一个“写”脉冲时，输入总线上的值将被存入PC寄存器。这对应于**分支指令**的执行。
2.  **递增-写入序列**：当“递增”信号有效，随后产生一个“写”脉冲时，递增器的输出（PC+1）将通过总线多路选择器传入PC寄存器，并在写脉冲发生时改变寄存器的值。这对应于指向**下一条指令**。

这里有一个非常重要的细节：从寄存器输出到递增器，再通过多路选择器回到寄存器，形成了一个**长反馈循环**。这就是我们需要使用**短写脉冲**的原因。我们只希望值在非常短的时间内（刚好足以让触发器翻转）发生改变。如果写信号长时间保持高电平，值就会不断循环递增，导致不可预测的行为。

设计这种脉冲序列的艺术与科学，正是处理器设计的核心。
