# 043：加法器电路 🧮

![](img/5ef38df409eea23ff4161eb66c68b141_0.png)

在本节课中，我们将学习如何构建一个能够执行加法运算的电路。我们将从理解二进制加法的需求开始，逐步构建一个高效的八位加法器电路，并探讨其设计原理和实现细节。

## 概述

现在，我们准备构建一个能够将两个数字相加的电路。这代表了我们希望用计算设备执行的那类计算。让我们来制作一个加法器。

我们将构建一个八位加法器。我们的目标是，给定两个二进制整数 **X** 和 **Y**（在本例中为八位），计算它们的和 **Z**。

以下是一个例子。我们之前已经多次进行过二进制加法。这是 `00010111`，即 16 + 4 + 2 + 1 = 23。另一个是 `00110001`，即 32 + 16 + 1 = 49。23 加 49 等于 72。这就是我们试图计算的结果。

在我们的加法器电路中，我们将接收输入。输入将贯穿整个电路：顶部是第一个二进制数 `00010111`（即 23），下一个输入是底部的二进制数 `00110001`（即 49）。这些是电路的输入。通常，输入来自顶部或左侧。输出将从底部产生，在本例中是 `01001000`，即 72。

我们希望构建一个具有此行为的电路。

我们将忽略溢出问题。实际上，如果数字变得太大，它会进位输出一个 1。我们可以捕获这个进位，但暂时忽略它。

## 加法器的抽象定义

一般来说，这是所有输入符号化标记的情况。我们将有 8 位数字：输入是 **x7** 到 **x0**，**y7** 到 **y0**；输出是 **z7** 到 **z0**。我们在左下角符号化地计算和，并忽略进位输出。

我们刚刚学习了如何实现可以执行任何二进制函数的电路，而加法就是一个二进制函数。因此，我们可能会考虑为每个输出位构建一个真值表。

对于八位加法器，我们有 16 个输入。如果算上进位，则有九个输出位。但行数太多了，有 2^16 种不同的输入可能性，即 65,536 行。如果这看起来还不算糟糕，设想一下在真正的计算机中，你可能需要一个 64 位或 128 位的加法器。如果使用这种方法，你将需要 2^256 行。我们已经计算过，这个数字比宇宙中的电子数量还要大。

香农和布尔告诉我们，构建这样的电路是可能的，但这不一定是最有效的方法。因此，我们将研究一种更有效的方法来构建加法器电路。

## 逐位相加的设计思想

这是一个非常简单的想法，它对应于我们进行加法运算时的做法：我们将一次处理一位。

那么，当我们将前两位相加时，我们需要做什么来实现进位位呢？

进位位：如果 **x**、**y** 和进位输入 **c_in** 都是 0，那么它将是 0；或者如果它们是 0、0、1，它也将是 0。只有当有两个 1 或三个 1 时，它才会是 1。

这听起来熟悉吗？确实如此，我们马上会看到。类似地，和位：如果其中恰好有一个是 1，或者如果三个都是 1（此时会产生进位输出），那么和位将是 1。当三个都是 1 时，和是 1，并且有进位输出。这些就是我们需要计算的函数。

但这些函数我们已经知道了。进位位是**多数函数**，和位是**奇校验函数**。因此，我们已经拥有了逐位计算进位与和所需的电路。我们只需要将这些电路连接在一起，就能完成我们的工作。

## 进位脉动加法器

这正是下图所示的结构。顶部是每一位的多数函数电路，底部是每一位的奇校验函数（即和）电路。

这两个函数都接收三个输入：两个输入位 **x** 和 **y**，以及一个进位位 **c_in**。进位位来自右侧的电路，从最右侧的 0 开始。然后，最右侧多数函数的输出被路由到中间并向上，作为从右数第二位的进位输入，依此类推。

因此，进位被路由出来并向上传递到其左侧的下一位。这些一位加法器通过进位链连接在一起，进位在其中“脉动”传递。这种设计被称为**进位脉动加法器**。

显然，相同的设计可以扩展到任意数量的位数。这就是我们的加法器电路。

## 电路内部结构

这是我们的加法器电路放大图。我们可以揭开盖子，展示我们之前讨论过的内部电路。顶部是每一位的多数电路，底部是每一位的奇校验电路。如果仔细观察，你可以看到和的计算以及进位的产生，完全符合我们的示例。这是一个用于将两个八位数相加的八位电路，我们可以看到电路中每个开关的功能。

这个例子再次说明了软件设计的经验同样适用于硬件。接口定义了电路的行为，实现则给出了如何构建它的细节。我们只是利用了对每一层行为的理解来实现加法器。我们需要实现一个布尔函数，我们知道如何实现这些布尔函数。我们从受控开关开始，用开关构建门，从门可以构建布尔函数，不久之后我们就有了一个加法器。

## 设计方法的重要性

![](img/5ef38df409eea23ff4161eb66c68b141_2.png)

这种方法不仅极大地简化了设计，而且使得复杂计算机系统的设计成为可能。不仅如此，它还允许在任何层级使用新技术。如果我们努力开发一个更好的加法器，我们可以在任何层级将其插入。事实上，今天的计算机使用更高效的加法器。这个（进位脉动）加法器需要的时间与位数成正比，以便进位脉动通过。通过使用对数因子，可以减少这个时间。

## 总结

本节课中，我们一起学习了如何构建一个加法器电路。我们从二进制加法的需求出发，认识到直接使用真值表方法对于多位加法器是不切实际的。接着，我们引入了逐位处理的思想，将加法分解为对每一位计算和与进位，并识别出这分别对应奇校验函数和多数函数。然后，我们通过将这些一位加法器单元串联，让进位在单元间“脉动”传递，构建出了高效的进位脉动加法器。最后，我们探讨了这种分层、模块化设计思想的重要性，它不仅是构建加法器的关键，也是设计更复杂计算机系统的基础。
