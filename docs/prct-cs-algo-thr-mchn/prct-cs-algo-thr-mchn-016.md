# 016：理论计算机科学概述

在本节课中，我们将开始探讨理论计算机科学。我们将了解什么是抽象机器和形式语言，并介绍确定性有限自动机和正则表达式这两个核心概念。这些基础理论不仅帮助我们理解计算的本质，也是许多现代计算技术的基石。

## 为什么学习理论？

上一节我们介绍了本课程的主题方向。本节中，我们来看看学习理论计算机科学的意义。

在许多科学学科中，从业者或许可以将理论视为与日常工作无关的内容。但对于计算机科学而言，情况并非如此。理论是我们学科不可或缺的一部分，每个人都应该对接下来几节课将要讨论的理论类型有一个基本的理解。

理论计算机科学为我们提供了对计算的深刻理解，这实际上是所有我们使用的现代计算机的基础。它确实帮助我们理解自然世界，并且也具有哲学意义。

在实践中，理论研究衍生出了各种各样的实用工具。例如，进行网络搜索时，你需要模式匹配理论，这与这些基本的计算问题密切相关。我们用来构建计算机的电路、用来将一种编程语言翻译成另一种的编译器，都基于理论。密码学、数据压缩以及许多其他领域的发展，都源于我们从理论研究中获得的基本理解。

因此，在本节课中，我们将在两个层面进行讨论：既会涉及一些理论问题，也会谈及一些实际问题，并最终提出一些关于计算的基本问题。

## 抽象机器与形式语言

上一节我们探讨了学习理论的意义。本节中，我们来看看理论研究的两个核心工具：抽象机器和形式语言。

我们将要讨论的一件事是抽象机器的概念。抽象机器实际上是计算的数学模型。在本次讲座中，我们有一种图形化模型，它模仿了你可能想象的真实机器。

存在不同类型的机器，它们具有不同的能力。每台机器都由非常具体的规则定义，用于接收输入并产生输出。本节课，我们将讨论一种最简单的抽象机器，称为确定性有限自动机。

另一个概念是形式语言。形式语言只是一组字符串的集合。右边我们有一个字符串集合的例子。这是一个英语回文串的集合，即正读和反读都相同的字符串。

这是一个特定的规则，它刻画了这个字符串集合的特征：正读和反读相同。你可以想象许多其他类型的规则。如果你能阐明规则，那么你就在定义一个字符串集合，一个作为形式语言的字符串集合。因此，今天我们将讨论一组特定的规则，称为正则表达式。这些规则在许多应用中非常有用。

## 基本问题与应用

上一节我们介绍了抽象机器和形式语言。本节中，我们将探讨与这些抽象概念相关的一些基本问题及其实际应用。

在本节课中，我们将解决与这些抽象相关的一些基本问题。例如，我们可能想知道，一个给定的字符串是否在由给定正则表达式定义的语言中？这本身就是一个值得探讨的有趣问题。

我们可能会说，如果我们有一台计算机，我们能回答这个问题吗？具体来说，像DFA这样的简单计算机能回答这个问题吗？即使使用这些非常简单的模型，我们也能遇到具有实用价值的有趣问题。这就是本节课大部分内容所要探讨的。
