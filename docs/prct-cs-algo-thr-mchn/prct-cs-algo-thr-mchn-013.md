在本节课中，我们将要学习一种名为**二叉搜索树**的数据结构。它是实现符号表高效操作的基础，也是每位计算机科学家都应掌握的经典数据结构。我们将了解其定义、核心特性，并学习如何实现搜索、插入和有序遍历等基本操作。

---

## 概述

我们将要使用的数据结构称为**二叉搜索树**。它是一种经典且基础的数据结构，用于开发符号表的高效实现。它通过在每个节点添加两个链接，构建了一个递归的、有序的双向链接结构。

---

## 二叉搜索树的定义

上一节我们提到了双向链接结构，本节中我们来看看二叉搜索树的具体定义。

二叉搜索树是一个递归的数据结构，它使用可比较的键和两个链接。其核心特性在于其**有序性**。

一个二叉搜索树要么是 `null`，要么是一个指向根节点的引用。每个节点包含一个键、一个值，以及指向**左子树**和**右子树**的引用。

其有序性定义如下：对于树中的任意节点，其**左子树**中所有节点的键都**小于**该节点的键；其**右子树**中所有节点的键都**大于**该节点的键。由于键是互异的，我们无需处理相等的情况。

以下是节点类的代码表示：

```java
private class Node {
    Key key;
    Value val;
    Node left, right;
    // 构造函数
    public Node(Key key, Value val) {
        this.key = key;
        this.val = val;
    }
}
```

---

## 二叉搜索树的操作

了解了定义后，我们来看看对二叉搜索树需要执行哪些标准操作。主要包括搜索、插入新键值对以及按顺序遍历所有节点。删除操作稍复杂，本课暂不讨论。

以下是这些操作的基本思路：

### 搜索操作

搜索操作的目的是在树中查找给定键对应的值，若不存在则返回 `null`。其逻辑与二分查找类似，但作用于树形结构。

搜索的递归算法如下：
1.  如果当前节点为 `null`，返回 `null`。
2.  将给定键与当前节点的键比较。
3.  如果给定键**小于**当前节点的键，则递归地在**左子树**中搜索。
4.  如果给定键**大于**当前节点的键，则递归地在**右子树**中搜索。
5.  如果**相等**，则返回当前节点关联的值。

以下是搜索方法的代码框架：

```java
public Value get(Key key) {
    return get(root, key);
}

private Value get(Node x, Key key) {
    if (x == null) return null;
    int cmp = key.compareTo(x.key);
    if      (cmp < 0) return get(x.left, key);
    else if (cmp > 0) return get(x.right, key);
    else              return x.val;
}
```

**示例**：假设在上图的树中搜索 “THE”。
1.  从根节点 “it” 开始，比较 “THE” > “it”，转向右子树。
2.  到达节点 “was”，比较 “THE” < “was”，转向左子树。
3.  到达节点 “the”，找到匹配，返回其关联的值。

### 插入操作

插入操作的目标是添加新的键值对。如果键已存在，则更新其关联的值。其逻辑与搜索类似，但需要在搜索路径的末端创建新节点。

插入的递归算法如下：
1.  如果当前节点为 `null`，则创建并返回一个包含新键值对的新节点。
2.  将给定键与当前节点的键比较。
3.  如果给定键**小于**当前节点的键，则递归地在**左子树**中插入，并将返回的节点链接为新的左子节点。
4.  如果给定键**大于**当前节点的键，则递归地在**右子树**中插入，并将返回的节点链接为新的右子节点。
5.  如果**相等**，则更新当前节点的值。
6.  返回当前节点（对于上层递归调用，这确保了链接的正确更新）。

以下是插入方法的代码框架：

```java
public void put(Key key, Value val) {
    root = put(root, key, val);
}

private Node put(Node x, Key key, Value val) {
    if (x == null) return new Node(key, val);
    int cmp = key.compareTo(x.key);
    if      (cmp < 0) x.left  = put(x.left,  key, val);
    else if (cmp > 0) x.right = put(x.right, key, val);
    else              x.val   = val;
    return x;
}
```

**示例**：插入新键 “worst” 并关联值 1。
1.  从根节点 “it” 开始，比较 “worst” > “it”，转向右子树。
2.  到达节点 “was”，比较 “worst” > “was”，转向右子树。
3.  到达 `null` 链接，创建新节点 “worst”。
4.  递归返回，将新节点的引用赋值给其父节点 “was” 的右链接。

### 有序遍历

有序遍历的目标是按键的顺序访问树中的每个节点。我们采用**中序遍历**算法。

遍历的递归算法如下：
1.  递归遍历**左子树**。
2.  访问当前节点（例如，将其键加入队列）。
3.  递归遍历**右子树**。

由于左子树的所有键小于根节点键，根节点键小于右子树的所有键，此算法能保证按键的升序输出。

以下是遍历方法的代码框架，它使用一个队列来收集键：

```java
public Iterable<Key> keys() {
    Queue<Key> q = new Queue<Key>();
    inorder(root, q);
    return q;
}

private void inorder(Node x, Queue<Key> q) {
    if (x == null) return;
    inorder(x.left, q);
    q.enqueue(x.key);
    inorder(x.right, q);
}
```

**示例**：遍历上图的树。
顺序为：访问左子树（“best”）-> 访问根（“it”）-> 访问右子树（按顺序访问 “of”, “the”, “times”, “was”）。最终队列中的键顺序为：`best, it, of, the, times, was`。
