## 回顾：不可靠信道上的可靠通信 🔄

上一节我们讨论了如何在不可靠的信道上实现可靠通信。其核心思想是，通过在信道前添加编码系统，在信道后添加解码系统，来“修复”这些有噪声的信道。

我们讨论了一个有噪声信道的简单例子，即**二进制对称信道**。该信道以一定概率 `F` 翻转比特。我们还讨论了两种简单的添加冗余（即编码）方法：
*   **重复码**
*   **（7,4）汉明码**，这是一种更复杂的添加冗余方式。

我们讨论了如何解码这些码。编码器负责添加冗余，而解码器则利用推理来尝试找出发生了什么并纠正错误。

我向大家介绍了一个我们将在后续课程中证明的结果，这可能是上世纪最杰出的数学成果之一：**香农噪声信道编码定理**。

该定理指出，如果你以巧妙的方式添加冗余，实际上并不需要添加太多冗余就能实现任意低的错误概率。对于翻转10%比特的二进制对称信道，在码率与最终错误概率的关系图中，可实现与不可实现区域的边界在码率轴上相交于一个点，即**信道容量**，大约为0.53。这意味着你只需要使用码率为1/2的编码（即添加100%的冗余），就可以获得任意小的错误概率。这非常神奇。

## 标准通信系统架构 🏗️

在一个标准的通信系统中，我们希望利用真实数据源中已存在的冗余，并消除它，从而压缩文件。因此，我们首先进行压缩，得到一个理想化的数据源，然后将其输入到我们上节课讨论的信道编码方法中。

所以，我们将用编码器为信道添加合适的冗余，但我们会将整个系统包装在一个压缩器和一个解压缩器之内。

因此，我们将讨论**数据压缩**，并在这个主题上花费几节课的时间。这是传统的做法。我们称之为**信源编码**或**数据压缩**。我们想要理解如何做到这一点。

## 理想化冗余源：偏置硬币示例 🪙

为了讨论压缩，思考理想化的冗余数据源可能会有所帮助。第一个例子是**偏置硬币**。

偏置硬币有一个结果，我们会得到很多次结果。结果存在于一个**字母表**中，该字母表有两个字符：`T`（反面）和 `H`（正面）。这两个结果的概率是：90%的概率是反面，10%的概率是正面。

我的记法是：称随机变量为 `X`。`X` 为反面的概率是 `P(X=T) = 0.9`。这是我在书中使用的记法。

![](img/37a7f5e20ddc8cde8476328934294853_2.png)

![](img/37a7f5e20ddc8cde8476328934294853_3.png)

关于这个冗余源（冗余是因为它有大量的0和很少的1），我们想问的问题是：如果我们从这个源得到由1000次结果组成的文件（即抛这枚硬币1000次），这个结果串有多少信息？我们应该如何压缩这个结果串？最后，我们应追求的最小可能压缩文件大小是多少？

我们希望为任何随机变量（这只是一个简单例子）回答这个问题。通常，我使用这个记法：一个**系综** `X` 包含三样东西：
1.  一个随机变量。
2.  一个可能结果的集合，我们称之为**字母表** `{a1, a2, ..., aI}`，其中 `I` 是字母表中的元素个数。
3.  一组概率 `{p1, p2, ..., pI}`，使得 `P(X=ai) = pi`，并且这些概率满足总和为1，且均为正数或零。

这是一个产生随机结果的通用源，偏置硬币是其中一个例子。

## 香农信息内容与熵 📊

在讨论这些例子时，我们将检验一个主张，即**香农的主张**。他认为，衡量信息内容以及说明我们期望能将事物压缩到何种程度的正确方法如下。

我们定义一个称为**香农信息内容**的量。对于一个来自上述通用系综的单一结果（随机变量取值为 `ai`），该结果的香农信息内容是：
```
h(x=ai) = log₂(1 / P(x=ai))
```
我们以**比特**为单位来度量它。

香农说，这是衡量信息内容的正确方式。当某事发生时，你可以说“啊，我获得了这么多信息”，你通过查看概率并取其倒数的对数来获得这个信息内容的度量。

它看起来是怎样的？如果你告诉我你的结果的概率 `p`，香农说，信息内容对于非常不可能的结果最大，对于确定的事情最小。事实上，如果某事肯定发生（概率为1），则其信息内容为0。如果概率为二分之一，则信息内容为1比特。

对于我们的偏置硬币，得到反面（概率0.9）的信息内容是 `log₂(1/0.9) ≈ 0.15` 比特。而每当发生更令人兴奋的、更罕见的事件（正面，概率0.1）时，我们得到 `log₂(1/0.1) ≈ 3.3` 比特。

因此，他声称这是衡量信息内容的正确方式。此外，它也是我们应追求的压缩文件长度。当我们发明压缩系统时，`H(X)` 是我们应追求的压缩文件长度。

在确信这一点之前，我会继续称之为香农信息内容，之后我将只称之为信息内容。

## 香农信息内容的性质与熵的定义 ⚙️

现在让我们对香农信息内容做一些观察。

**性质1：** 香农信息内容对于**独立的**随机变量具有**可加性**。例如，如果我有一个系综 `(X, Y)`，其中单次抽取的结果是一对随机变量 `(X, Y)`，且概率分布满足 `P(X,Y) = P(X)P(Y)`（对所有 `X, Y` 成立，这是独立的定义）。那么，随机变量取特定值 `(x, y)` 的香农信息内容 `h(x, y)` 为：
```
h(x, y) = log₂(1 / P(x, y)) = log₂(1 / (P(x)P(y))) = log₂(1/P(x)) + log₂(1/P(y)) = h(x) + h(y)
```
这符合直觉：同时告诉你两个独立事件的信息量，等于分别告诉你每个事件的信息量之和。

**性质2：** 香农信息内容对于不可能的结果最大（正如之前所述）。

现在，让我们定义另一个量。我们尚未证明这是衡量信息的正确方法（稍后将举例），但让我们先假设它是。然后我们可以定义另一个称为**熵**的量。

熵是**系综**的属性。一个系综 `X` 的熵是其平均香农信息内容：
```
H(X) = Σ_{x} P(x) * log₂(1 / P(x))
```
与香农信息内容一样，其单位也是比特。

## 应用示例：称球问题 ⚖️

让我们看一个例子来思考这些断言——这些量是合理的。让我们看看上节课留给大家思考的**称球问题**。

目标是：给定12个球，已知其中一个是异常的（或重或轻，但不知是哪种），使用天平尽可能少的次数，并保证找出异常球及其轻重。

（此处省略课堂讨论和策略分析部分）

在分析了各种首次称量方案（6对6、5对5、4对4等）后，我们计算了每种方案结果（左重、平衡、右重）的熵。香农会建议我们选择能使结果熵最大的第一次称量。计算表明，称量4个球对4个球（留4个在桌上）能得到最大的熵（约1.58比特），这恰好是许多同学凭直觉选择的策略。

接着，在第一次称量出现特定结果后，我们可以继续应用“选择使结果熵最大的行动”这一贪婪策略，最终可以在**三次称量**内解决此问题。

这展示了利用香农熵的概念作为指导，可以高效地解决此类谜题。但需注意两点：
1.  所获得的信息必须是**相关的**信息。在这个问题中，所有可能获得的信息都是相关的。
2.  这是一个**贪婪策略**，并非对所有问题都是全局最优的，但对于这个具体问题它是有效的。

## 回到偏置硬币与思考题 💭

现在让我们回到开始时讨论的理想化源：偏置硬币。偏置硬币出现正面的概率是10%，反面是90%。

我给大家留下一个思考题，称为**偏置硬币彩票**。

运作方式如下：类似于国家彩票，但为了获得中奖彩票，我们不用一堆球来抽奖，而是只用一枚偏置硬币，抛它1000次。结果是一个长度为1000的序列 `X1, X2, ..., Xn`。

在通过抛硬币得出这个结果之前，你可以去彩票亭购买彩票。每张彩票对应一个可能的长字符串（0和1的序列）。你可以购买全0的票，也可以购买任何 `2^N` 种可能的票中的一种。一旦每个人都决定购买哪些票，如果你拥有的票上的字符串与实际发生的结果 `X` 完全相同，那么你就赢得10亿英镑。

问题：
1.  如果黑帮老大要求你确保有99%的概率赢得彩票，同时最小化购买成本，你应该购买哪些彩票？
2.  你需要购买多少张彩票？请以 `2` 的某次方的形式表达你的答案。
