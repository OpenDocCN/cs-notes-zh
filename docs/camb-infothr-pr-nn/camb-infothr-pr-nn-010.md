在本节课中，我们将学习如何使用贝叶斯定理来解决参数推断问题，并进一步比较不同模型的优劣。我们将从一个简单的指数分布参数推断问题开始，逐步深入到更复杂的模型比较场景。

## 概述

本节课的核心是展示贝叶斯推断如何为参数估计和模型选择提供一个统一、自洽且无需人为干预的框架。我们将看到，与传统的统计方法（如直方图分箱、拟合优度度量）相比，贝叶斯方法能自动处理数据，给出唯一的答案，并且适用于任何规模的数据集。

## 从传统方法到贝叶斯方法

在介绍贝叶斯方法之前，让我们先回顾一下解决参数推断问题的几种传统思路。假设我们观测到一些在窗口 `[A, B]` 内按指数分布的事件坐标 `x`，我们希望推断出指数分布的参数 `λ`。

以下是几种可能的方法：

![](img/7cdbee35690287ad1747a702d1830282_4.png)

*   **方法1：直方图与拟合优度**
    将数据放入宽度为 `Δx` 的箱子（直方图），得到每个箱子的计数 `F_b`。然后，对于一个给定的 `λ`，计算每个箱子的期望计数 `E[F_b | λ]`。通过最小化实际计数与期望计数之间的差异（例如，平方和）来找到最佳的 `λ`。这种方法需要选择箱宽 `Δx` 和差异度量方式。

*   **方法2：取对数拟合直线**
    对指数分布取对数，可以得到 `log(期望计数) = 常数 - x/λ`。因此，可以对实际数据的直方图计数取对数，然后对 `x` 拟合一条直线，其斜率即为 `-1/λ`。但这种方法在计数为零时会遇到问题（`log(0)` 未定义）。

*   **方法3：使用两个箱子**
    为了简化，只将数据分成两个箱子，并可能根据数据调整箱子的边界，以确保每个箱子都有数据点。然后基于两个箱子的计数进行推断。

*   **方法4：累积分布函数（CDF）**
    比较经验累积分布函数与理论累积分布函数，并定义某种距离度量（如各点垂直距离之和）来评估拟合程度。这避免了选择箱宽，但引入了如何定义距离和采样点密度的问题。

*   **方法5：使用数据均值**
    计算数据的样本均值 `x̄`。对于无界窗口的指数分布，理论均值等于 `λ`。对于有界窗口，理论均值 `μ(λ, A, B)` 是 `λ` 的函数。通过求解 `μ(λ, A, B) = x̄` 来估计 `λ`。但这种方法可能只利用了数据的一部分信息，并且对于不同的模型，需要不同的“充分统计量”。

这些传统方法都需要研究者做出各种主观选择（箱宽、度量、统计量等），且答案可能因选择不同而不同。

## 贝叶斯推断方法

![](img/7cdbee35690287ad1747a702d1830282_6.png)

现在，让我们看看贝叶斯定理如何清晰、唯一地解决这个问题。对于单个数据点 `x`，给定参数 `λ`，其概率密度为：

```
P(x | λ, A, B) = (1/Z(λ)) * exp(-x/λ)， 其中 A ≤ x ≤ B
```

归一化常数 `Z(λ)` 为：

```
Z(λ) = ∫_A^B exp(-x/λ) dx = λ * [exp(-A/λ) - exp(-B/λ)]
```

![](img/7cdbee35690287ad1747a702d1830282_8.png)

![](img/7cdbee35690287ad1747a702d1830282_10.png)

![](img/7cdbee35690287ad1747a702d1830282_12.png)

对于一组独立的观测数据 `D = {x_1, x_2, ..., x_N}`，其似然函数为：

```
P(D | λ) = ∏_{n=1}^N P(x_n | λ) = (1/Z(λ))^N * exp(-∑_n x_n / λ)
```

根据贝叶斯定理，参数 `λ` 的后验分布为：

![](img/7cdbee35690287ad1747a702d1830282_14.png)

```
P(λ | D) ∝ P(D | λ) * P(λ)
```

![](img/7cdbee35690287ad1747a702d1830282_16.png)

其中 `P(λ)` 是我们对 `λ` 的先验认知。如果我们对 `λ` 几乎一无所知，可以假设一个在 `log λ` 上较为平坦的先验分布。

**关键点**：我们无需对数据做任何预处理（如分箱），也无需发明任何拟合优度度量。我们只需写出模型定义（`P(x|λ)`）和先验，然后计算后验分布。这个后验分布（似然函数）包含了给定数据下关于 `λ` 的所有信息。它可能是一个单峰曲线，其峰值位置给出了 `λ` 最可能的值，其宽度则反映了估计的不确定性。

值得注意的是，似然函数 `P(D|λ)` 仅通过数据总和 `∑_n x_n`（即数据的均值）依赖于数据。这意味着**均值是这个问题的一个充分统计量**。但这是指数分布模型的特性，贝叶斯框架会自动揭示出这一点，我们无需事先知道。

## 从参数推断到模型比较

上一节我们介绍了如何推断单一模型的参数。本节中我们来看看一个更高级的问题：**模型比较**。假设我们不确定数据是来自一个指数分布还是两个指数分布的混合。我们如何用数据来判断哪个模型更可信？

我们有两个假设：
*   **H1（单指数模型）**：`P(x | λ) = (1/Z) * exp(-x/λ)`，有一个参数 `λ`。
*   **H2（双指数混合模型）**：`P(x | λ1, λ2, π1, π2) = (1/Z) * [π1 * exp(-x/λ1) + π2 * exp(-x/λ2)]`，有多个参数（`λ1, λ2, π1, π2`，且 `π1+π2=1`）。为简化，我们可设 `π1 = π2 = 0.5`。

对于每个模型，我们都可以进行参数推断，得到后验分布 `P(参数 | D, H)` 以及一个关键的归一化常数，即**证据（Evidence）** 或 **边际似然（Marginal Likelihood）**：

```
P(D | H1) = ∫ P(D | λ, H1) P(λ | H1) dλ
P(D | H2) = ∫∫ P(D | λ1, λ2, H2) P(λ1, λ2 | H2) dλ1 dλ2
```

**模型比较的核心**：根据贝叶斯定理，模型的后验概率比为：

```
P(H1 | D) / P(H2 | D) = [P(D | H1) / P(D | H2)] * [P(H1) / P(H2)]
```

如果先验认为两个模型同样可能（`P(H1)=P(H2)`），那么后验概率比就等于两个模型的证据比。

![](img/7cdbee35690287ad1747a702d1830282_18.png)

![](img/7cdbee35690287ad1747a702d1830282_19.png)

**重要洞见**：当我们为一个模型进行完整的贝叶斯参数推断（计算了整个后验分布而不仅仅是最大似然值）时，我们实际上已经自动计算出了该模型的证据（即归一化常数）。因此，**参数推断和模型比较在贝叶斯框架中是统一的过程**。

![](img/7cdbee35690287ad1747a702d1830282_21.png)

### 模型比较实例分析

![](img/7cdbee35690287ad1747a702d1830282_23.png)

通过计算可以展示：
1.  **当数据确实来自双指数混合时**：模型 H2 的证据会显著高于 H1，贝叶斯因子会支持更复杂的模型。
2.  **当数据实际上来自单指数分布时**：尽管复杂模型 H2 通过调整参数总能得到更高的**最大似然值**，但其**证据**可能反而更低。这是因为证据在参数空间上进行了平均积分，复杂模型因其更大的参数空间体积而受到“奥卡姆剃刀”式的惩罚。因此，贝叶斯模型比较不会总是选择参数更多的模型。

## 总结

![](img/7cdbee35690287ad1747a702d1830282_30.png)

本节课中我们一起学习了：
1.  传统参数推断方法（如直方图、矩估计）需要大量主观选择，且缺乏统一标准。
2.  **贝叶斯参数推断**提供了一种机械化的方法：定义似然 `P(数据|参数)` 和先验 `P(参数)`，然后计算后验 `P(参数|数据)`。后验分布完整描述了参数的不确定性。
3.  贝叶斯方法能自动从模型中识别出关键的充分统计量（如指数分布中的数据和）。
4.  **贝叶斯模型比较**通过计算和比较不同模型的**证据（边际似然）** 来实现。证据是似然在参数先验下的平均值。
5.  一个关键的实践要点是：**完整的贝叶斯参数推断（计算归一化的后验）会直接给出模型证据**，从而自然衔接了参数推断和模型比较。
6.  贝叶斯框架通过积分对复杂模型进行自动的“奥卡姆剃刀”惩罚，防止过拟合，因此模型比较的结果是合理且可解释的。

![](img/7cdbee35690287ad1747a702d1830282_32.png)

贝叶斯定理为我们提供了一个强大、自洽且通用的框架，用于处理从简单参数估计到复杂模型选择的各种推断问题。
