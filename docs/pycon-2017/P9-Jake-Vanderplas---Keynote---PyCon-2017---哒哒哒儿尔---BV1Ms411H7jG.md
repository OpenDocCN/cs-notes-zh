哈勃太空望远镜自 1990 年以来，一直在地球轨道飞行，拍摄宇宙的惊人照片。这是我最喜欢的之一。这被称为超深场。这是哈勃观察的一小片天空，视场大约是满月直径的十分之一。

在那片小小的天空中，整合了数小时的观察，以捕捉足够的光子来创建这个图像。你在图像中看到的每一个斑点，除了可能两三个，是一个遥远的星系。它是数十亿颗恒星的集合。

那些星系就在宇宙中，距离我们可能有 130 亿光年或更远。因此，你现在在这些图片中看到的一些斑点，都是来自宇宙早期的星系。我们可以收集到这些光，并了解我们的星星来自何方，我们的星系来自何方，以及种类。

关于宇宙的整体动态。哈勃在教会我们这些事物方面表现得非常出色，关于早期宇宙以及非常细致的事物。一个互补的方法是斯隆数字天空调查所做的。我这里展示的是星系目录。

来自斯隆数字天空调查。斯隆调查与哈勃只观察天空中的一个小点不同，它扫描了整个天空，历时十年，能够对那里每个物体进行光谱测量。因此在这个图中，你所看到的是位置。

我们宇宙中星系的三维位置。我们正处于原点，正位于中心，向外看，那些弧形就是斯隆能够观察到的天空的一部分。而天文学家们能够做到的，就是观察这一点，看到星系并没有均匀分布。

这里有很多的团块、簇、丝状结构和空洞。关于星系空间分布的信息中，编码了关于宇宙整体的信息。物质有多少，暗能量有多少，暗物质又有多少。

因此，我们能够通过来自斯隆调查的这个庞大数据集来实现这一点。那些望远镜有点老旧，是在 90 年代发射的。近年来，现代科学的发展则依赖于一些其他的望远镜。我想强调的一个是开普勒项目。你可能听说过开普勒项目。

利用这些望远镜，我们已经能够发现围绕其他恒星旋转的行星，并开始思考，这些行星可能是什么样子的。这些行星之所以特别，是因为其中几个处于所谓的可居住区。天文学家将可居住区定义为液态水能够存在的太空区域。

因为我们没有其他好的定义，来说明生命可能是什么样子。因此我们认为，水是一个不错的搜索目标。但像这样的图片，这些艺术家创作的图像，有点具有误导性。你在像这样的行星或系统上获得的实际数据看起来是这样的。这就是我们通过开普勒所观察到的。

这是对特拉普斯特系统的 K2 观测。让我来描述一下你在这里看到的内容。你看到的是一颗单一的星星，它在像素之间摇摆，因为开普勒上的一个稳定器坏了，因此它随着时间的推移有些摇晃。而你在这里寻找的信号，正是这些行星。

你可以看到这些行星在掩蔽星星时的情形。因此，星星的亮度在此时，行星会在它前面经过。随着行星的经过，星星的亮度会略微下降。而这种低于百分之一的下降，正是我们在这些数据中寻找行星的关键。而现在这非常困难，因为你有仪器噪声。这是一颗微弱的星星。

你会发现星星本身是多变的。它上面有星斑，存在太阳耀斑以及其他所有这些现象。因此，你实际上可以通过这个极其复杂的统计建模，提取出这些行星，了解系统及行星的物理参数。

随着它们的运行，以及我们对其运作方式的了解。因此，寻找这些类型的系统归结为编写统计代码和非常复杂的数据处理管道。如果我告诉你这一点，希望你不会感到惊讶。

接纳 Python 社区的规范。我想告诉你另一个望远镜——这就是詹姆斯·韦伯太空望远镜。这实际上不是望远镜，而是一个比例模型。因此，这个望远镜是哈勃的替代品。哈勃已经在太空中运行了 27 年，历程漫长，但它很快就会耗尽能量，我们需要其他东西来替代它。

詹姆斯·韦伯将要发射到太空，并且它有一个巨大的镜子。哈勃的镜子大约是一个人的大小，而詹姆斯·韦伯的镜子直径大约是其三倍。詹姆斯·韦伯望远镜的另一个不同之处在于，一旦它在太空中，就不会像我们的眼睛所能看到的红色、绿色和蓝色那样对可见光敏感。

它更适合红外光。因此，光的波长比我们可以看到的红光稍长一些。这一点很重要，因为在红外线和詹姆斯·韦伯所具备的那种精确度下，我们可能能够实际成像那些系外行星。

正在其他恒星周围运行的行星。因此，与其仅仅看到它们在恒星前方的阴影，我们实际上可能能够看到一些行星本身。其中一个最令人兴奋的前景是，观察詹姆斯·韦伯能够进行的光谱测量。这是当你将光分解为其组成波长，并观察波长流量分布时的光谱。

我们将能够看到的是，当这些行星在恒星前方移动时，我们能够隔离出经过该行星大气层的光的成分。寻找光中的分子吸收。

然后推断大气的分子成分。这是非常令人兴奋的，这在系外行星科学中几乎是圣杯。我们实际上将能够知道这些行星的组成。而且有些事情虽然很困难，但如果你观察宇宙中存在的行星和大气中的气体类型。

有些东西是永远不会出现的，比如氧气和臭氧。这些氧气和臭氧是没有已知地球物理来源的物质。它们只来自生物，只来自基本上是细菌、植物和生物体的东西。因此，在接下来的五到十年里，我们有小概率能够嗅出。

一个行星的化学成分围绕另一颗恒星。如果我们看到像臭氧层这样的东西，这将是一个非常好的迹象，表明那颗行星上至少有某种细菌生命。我觉得这非常令人兴奋，对吧？我认为以这种方式探索宇宙是非常重要的。再一次，如果你想知道我们在天文学的每个部分是如何做的。

这是一个大型综合巡天望远镜。所以我不知道你是否能看到，但那里有一个小人的图像，对吧？所以这大约就是望远镜的大小。这将是一个大型地面望远镜。

它并不在太空中飞行。这是几年前镜面空白的照片，拍摄于他们开始与 LSST 团队打磨之前。对于这么大的镜子，你需要一个非常大的相机。因此，LSST 正在建造有史以来最大的数字相机。这是一台三千兆像素的相机，CCD 大约有一个人那么大。

为了给你一个视场的概念，我们有一张月球的图像，叠加在 CCD 芯片上。这实在是巨大得令人难以置信，对吧？

三千兆像素。如果你想象一下高清电视，那差不多是 2000 乘 1000。

向用户展示不同的接口和文件格式。因此，科学家们可能会花费相当多的时间试图使这些组件以某种方式协同工作。因此，他在 2000 年提倡 Python 作为解决方案，将所有这些工具粘合在一起。
