# Python入门：01：理解变量与赋值

在本节课中，我们将要学习Python编程中最基础也最重要的概念之一：变量与赋值。理解这个概念是编写任何程序的第一步。

## 什么是变量？

你可以把变量想象成一个贴有标签的盒子。这个盒子（变量）可以用来存放数据，而标签（变量名）则帮助我们记住盒子里装的是什么。在Python中，我们使用等号 `=` 来给变量赋值，这就像把一样东西放进盒子里并贴上标签。

以下是创建一个变量的基本语法：
```python
变量名 = 值
```
例如，`name = “小明”` 这行代码创建了一个名为 `name` 的变量，并将字符串“小明”存储在其中。

## 变量命名规则

上一节我们介绍了变量的基本概念，本节中我们来看看如何给变量起一个合适的名字。Python对变量名有一些简单的规则。

以下是命名变量时需要遵守的核心规则：
*   变量名只能包含字母、数字和下划线（`_`）。
*   变量名不能以数字开头。
*   变量名不能是Python的保留关键字（如 `if`、`for`、`while`等）。
*   变量名是区分大小写的，这意味着 `age`、`Age` 和 `AGE` 是三个不同的变量。

## 变量的赋值与重新赋值

变量之所以称为“变”量，是因为其中存储的值是可以改变的。我们通过赋值操作来更新变量盒子里的内容。

以下是赋值操作的示例：
```python
# 首次赋值
score = 95
print(score)  # 输出：95

# 重新赋值，改变变量的值
score = 100
print(score)  # 输出：100
```
可以看到，变量 `score` 的值从95被更新为了100。

## 同时给多个变量赋值

Python提供了一种简洁的语法，可以同时为多个变量赋值。这在一行代码中初始化多个变量时非常有用。

以下是多变量赋值的两种方式：
```python
# 方式一：为多个变量赋予相同的值
a = b = c = 10

# 方式二：为多个变量分别赋予不同的值
name, age, city = “张三”, 20, “北京”
```
第二种方式在等号两侧的变量和值的数量必须严格相等。

本节课中我们一起学习了Python中变量与赋值的基础知识。我们了解了变量就像一个存储数据的带标签的盒子，掌握了命名变量的规则，并学会了如何为变量赋值、重新赋值以及进行多变量赋值。这是你Python编程之旅的坚实第一步。
