在互联网中，网络层是特殊的，当我们将包发送到互联网时，我们必须使用互联网协议，这是互联网协议或IP，它把互联网连接在一起，我们将在后续的视频中学习IP的更多细节，但现在了解一些IP的基本事实是很好的。

首先，IP尽最大努力将我们的数据包送达另一端，但它不作任何承诺，其次，IP数据包可能会丢失，它们可能会按顺序送达，并且可能会被损坏，没有保证，这可能会让你感到惊讶，你可能在问互联网如何工作。
