---

## 循环语句：while语句

上一节我们介绍了条件判断语句，本节中我们来看看如何让计算机重复执行某些操作。`while`循环语句用于反复执行一系列语句，直到某个条件不再满足为止。

与`if`语句类似，`while`循环首先评估一个布尔表达式。如果表达式为真，则执行循环体内的语句序列。执行完毕后，程序会再次检查布尔表达式，如果仍为真，则重复执行。这个过程会一直持续，直到布尔表达式变为假，此时程序会跳出循环，继续执行循环之后的代码。

以下是一个简单的`while`语句示例，它使用花括号包裹了三条语句。只要布尔表达式为真，这些语句就会反复执行。

```java
while (i <= n) {
    System.out.println(v);
    i = i + 1;
    v = 2 * v;
}
```

其控制流程如下：程序顺序执行，但在循环体语句执行完毕后，会“回环”到开头再次检查布尔表达式。如果表达式最终变为假，则控制流会继续执行`while`语句之后的代码。这是一种表达需要反复执行的计算的自然方式。

---

## 深入理解while循环

让我们更仔细地看看具体发生了什么。以下是将上述代码包装成一个完整Java程序的例子，它包含`main`方法等结构。

```java
public class PowersOfTwo {
    public static void main(String[] args) {
        int n = Integer.parseInt(args[0]);
        int i = 0;
        int v = 1;
        while (i <= n) {
            System.out.println(v);
            i = i + 1;
            v = 2 * v;
        }
    }
}
```

这个程序的功能是打印2的幂。例如，如果你在谷歌搜索“2的幂”，可能会看到类似的结果。

程序执行过程如下：我们将`i`初始化为0，`v`初始化为1。以下是一个程序运行的跟踪示例（假设输入`n`为6）：
1.  检查布尔表达式 `i <= n` (0 <= 6)。为真，进入循环。
2.  打印`v`的值（1）。
3.  `i`自增为1。
4.  `v`乘以2变为2。
5.  再次检查 `i <= n` (1 <= 6)。为真，继续循环。
6.  打印`v`（2）。
7.  `i`自增为2。
8.  `v`乘以2变为4。
9.  ... 此过程持续，直到`i`变为7。
10. 检查 `i <= n` (7 <= 6)。为假，跳出循环，程序结束。

这个程序会打印出2的幂，你可以输入任何整数`N`来获得对应的2的幂列表。这是许多需要打印函数值的程序的典型模式。

## 常见错误：遗漏花括号

为了熟悉`while`循环的概念，请看看这段代码，找出其中有什么问题。

```java
while (i <= n)
    System.out.println(v);
    i = i + 1;
    v = 2 * v;
```

这段代码缺少了花括号。对于初学者来说，忘记花括号并不少见。在一些像Python这样的编程语言中，缩进就足够了，但在Java中，花括号是必须的。

思考一下如果犯了这个错误，程序实际会做什么是很有价值的。那么，当我们运行这个程序时，它实际上会做什么呢？

如果你不小心键入了那个忘记花括号的程序，你将不得不面对以下情况：程序将`i`设为0，`n`为6。只要`0 <= n`为真，它就会打印`v`。这将导致一个**无限循环**，因为没有任何代码能改变循环条件。如果你编译并运行这个程序，它将不停地打印“1”。

因此，在使用循环编写程序之前，首先要考虑的一件事是：如果程序陷入无限循环，你该如何停止你的机器？在老式机器上，可能需要直接拔掉电源。如今，根据你使用的计算机和系统，通常按`Ctrl+C`可以停止它，但你需要为你自己的特定机器查阅相关资料。

## 实际应用：计算平方根

让我们看一个实际应用，其中使用`while`循环将使我们能够执行计算。这是一个经典问题：**实现平方根函数**。

我们如何在Java中实现类似`Math.sqrt`的函数呢？有一个著名的**牛顿法**可以用来计算数`c`的平方根，非常适合解决这个问题。

方法如下：
1.  初始化变量`t`（记作`t₀`）为`c`。
2.  然后，我们将计算`t₁`， `t₂`， `t₃`，等等。
3.  每次迭代，我们将下一个`t`值设置为前一个`t`与`c/t`的平均值。
4.  我们持续这个过程，直到`tᵢ`与`c/tᵢ`在期望的精度范围内几乎相等。

注意，如果`t`恰好等于`c/t`，那么`t² = c`，所以`t`就是`c`的平方根。因此，我们正在逼近`c`的平方根。

以下是一个计算√2到小数点后7位的例子：
*   `t₀ = 2`
*   `t₁ = (2 + 2/2) / 2 = 1.5`
*   `t₂ = (1.5 + 2/1.5) / 2 ≈ 1.4166667`
*   `t₃ = (1.4166667 + 2/1.4166667) / 2 ≈ 1.4142157`
*   `t₄ = (1.4142157 + 2/1.4142157) / 2 ≈ 1.4142136`

经过几次迭代，我们就得到了√2的精确近似值。

我们将要使用这个方法，这是一个经典方法。重要的是要记住，计算科学在计算机出现之前很久就被研究了。古典时期的科学家们几个世纪以来都需要进行计算来理解他们的数学和科学模型。在没有计算机的情况下，他们对高效方法非常感兴趣，因为所有计算都是手工完成的。因此，我们肯定希望受益于前几个世纪发展起来的关于计算的知识，这是一个很好的例子。

## 代码实现牛顿法

让我们看看这个方法在代码中是什么样子，它正是使用`while`循环来实现牛顿法的基本迭代。

```java
public class Sqrt {
    public static void main(String[] args) {
        double epsilon = 1e-15; // 期望的精度（误差容限）
        double c = Double.parseDouble(args[0]); // 从命令行读取要计算平方根的数
        double t = c; // 初始化估计值t为c

        while (Math.abs(t - c/t) > epsilon * t) {
            // 当相对误差大于epsilon时继续迭代
            t = (c/t + t) / 2.0; // 牛顿法迭代公式：新的t是旧t与c/t的平均值
        }

        System.out.println(t); // 打印最终结果
    }
}
```

这个程序的工作流程如下：
1.  我们定义一个变量`epsilon`（大写EPS）来表示精度，例如`1e-15`表示15位小数精度。
2.  从命令行获取我们想要计算平方根的数`c`。
3.  将变量`t`初始化为`c`。
4.  进入`while`循环。循环的终止条件是：当 `|t - c/t| <= epsilon * t` 时，即相对误差在`epsilon`范围内。
5.  在循环体内，执行牛顿迭代：`t = (c/t + t) / 2.0`。
6.  循环结束后，打印答案`t`。

这就是牛顿法，它实现了计算平方根的目标，在本例中精度达到15位小数。一个非常简单的计算方法，仅用单个`while`循环就解决了一个有趣的计算问题。

## 方法原理简述（可选）

对于对数学原理感兴趣的同学，这里简单说明一下为什么这个方法有效。牛顿法实际上可以用于寻找任何函数`f(x)`的根（即使得`f(x)=0`的`x`值）。对于平方根问题，我们可以将其表述为求函数 `f(x) = x² - c` 的根。

该方法的一般形式（可简化为我们刚才展示的求平方根形式）是：在曲线上当前猜测点处画一条切线，这条切线与x轴的交点会给出一个更接近真实根的下一猜测点。重复这个过程，实际上相当快速地，我们就能得到根的一个估计值。你可以通过数学推导来理解这个方法的基础，并验证我之前展示的√2的计算过程确实完成了任务。

虽然涉及一些数学，但本节的主要观点是：**`while`循环对于执行需要迭代计算的任务极其有用。**
