在本节课中，我们将学习一个有趣的抽象数据类型——海龟绘图法。这是一种通过简单的指令来创建复杂图形的编程模型，最初由西摩·佩珀特提出，用于向儿童介绍计算概念。我们将了解其核心思想，并学习如何用Java实现一个海龟类。

---

## 概述

海龟绘图法提供了一个简单的图形绘制模型。我们将其视为一个抽象的绘图设备，它有一个位置和一个朝向，并可以执行“前进”和“左转”等基本操作。通过组合这些操作，我们可以绘制出从简单三角形到复杂螺旋的各种图形。本节我们将实现这个抽象数据类型，并编写测试客户端来验证其功能。

---

## 海龟模型的核心概念

海龟是一个理想化的绘图设备模型。每个海龟在图形界面中通常被表示为一个“小虫子”图标。

海龟的状态由两个值决定：
1.  **位置**：在屏幕上的坐标 `(x, y)`。
2.  **朝向**：海龟当前面对的方向，用一个角度 `angle` 表示。

例如，一个海龟可能位于屏幕中央 `(0.5, 0.5)`，方向朝上（角度为90度）；也可能位于 `(0.25, 0.75)`，方向朝左上。

---

## 海龟的操作

海龟可以执行以下基本操作：
*   **创建**：通过指定初始位置 `(x, y)` 和初始角度 `angle` 来创建一个海龟。
*   **左转**：让海龟逆时针旋转指定的角度。
*   **前进**：让海龟沿当前方向前进指定的距离，并在其身后画出一条线。

这些就是海龟全部的操作。令人惊讶的是，仅凭这些简单的操作，就能构成一个功能强大的绘图工具。

---

## 实现海龟抽象数据类型

我们将通过定义实例变量、构造函数、方法以及一个测试客户端来实现海龟类。

上一节我们介绍了海龟的概念和操作，本节中我们来看看如何用代码实现它。

### 实例变量与构造函数

首先，我们需要定义海龟的状态。与电荷类类似，海龟的实例变量包括其位置 `(x, y)` 和角度 `angle`。注意，这些变量不是 `final` 的，因为海龟需要在移动和转向时改变状态。

构造函数接收初始的 `x0`, `y0`, `angle0` 参数，并将它们赋值给实例变量。

```java
public class Turtle {
    private double x, y;     // 当前位置
    private double angle;    // 当前朝向（角度）

    // 构造函数
    public Turtle(double x0, double y0, double a0) {
        x = x0;
        y = y0;
        angle = a0;
    }
    // ... 其他方法
}
```

### 实现方法

接下来，我们实现两个核心方法：`turnLeft` 和 `goForward`。

**左转方法** `turnLeft` 的实现非常简单，只需将传入的角度增量 `delta` 加到当前角度上。

```java
public void turnLeft(double delta) {
    angle += delta;
}
```

**前进方法** `goForward` 的实现稍复杂一些，涉及一些数学计算。
1.  首先记录移动前的旧坐标 `(x0, y0)`。
2.  根据前进距离 `d` 和当前角度 `angle`（需转换为弧度），计算坐标的变化量。
    *   `dx = d * Math.cos(Math.toRadians(angle))`
    *   `dy = d * Math.sin(Math.toRadians(angle))`
3.  使用 `StdDraw.line(x0, y0, x0+dx, y0+dy)` 从旧位置到新位置画一条线。
4.  最后，更新海龟的实例变量到新位置：`x += dx; y += dy;`。

```java
public void goForward(double d) {
    double x0 = x;
    double y0 = y;
    // 计算新位置
    x += d * Math.cos(Math.toRadians(angle));
    y += d * Math.sin(Math.toRadians(angle));
    // 绘制线段
    StdDraw.line(x0, y0, x, y);
}
```

在方法内部，我们可以直接使用 `x`, `y`, `angle` 这些实例变量，而无需再次声明。

## 测试客户端：绘制三角形

在实现所有方法后，我们首先编写一个测试客户端来验证海龟的功能。以下是绘制一个等边三角形的代码：

```java
public static void main(String[] args) {
    // 在左下角创建一个朝右的海龟
    Turtle turtle = new Turtle(0.0, 0.0, 0.0);
    // 绘制三角形
    turtle.goForward(1.0);
    turtle.turnLeft(120.0);
    turtle.goForward(1.0);
    turtle.turnLeft(120.0);
    turtle.goForward(1.0);
    // 最后再左转一次，使海龟回到初始朝向
    turtle.turnLeft(120.0);
}
```

通过一系列“前进”和“左转”指令，我们无需手动计算 `√3` 等复杂数值，就轻松绘制出了一个三角形。这正是佩珀特希望通过海龟绘图引导儿童思考计算的方式。

## 应用示例：绘制正N边形

有了海龟类，我们可以轻松编写程序绘制更复杂的图形。例如，以下代码可以绘制一个正N边形：

```java
int n = Integer.parseInt(args[0]); // 从命令行读取边数
double angle = 360.0 / n; // 计算每次转弯的角度
double step = Math.sin(Math.toRadians(angle/2)) * 2; // 计算步长以适配画布

Turtle turtle = new Turtle(0.5, 0.0, angle/2);
for (int i = 0; i < n; i++) {
    turtle.goForward(step);
    turtle.turnLeft(angle);
}
```

通过改变 `n` 的值，我们可以得到三角形 (`n=3`)、正方形 (`n=4`)，当 `n` 很大时（如1440），图形近似为一个圆。

## 进阶应用：绘制螺旋（等角螺线）

只需对正N边形代码稍作修改，我们就能绘制出美丽的螺旋图案。我们引入一个“衰减”参数，让海龟每次前进的距离略微减少。

```java
int n = Integer.parseInt(args[0]);
double decay = Double.parseDouble(args[1]); // 衰减因子
double angle = 360.0 / n;
double step = Math.sin(Math.toRadians(angle/2)) * 2;

Turtle turtle = new Turtle(0.5, 0.0, angle/2);
for (int i = 0; i < 10*n; i++) { // 绕行多圈
    turtle.goForward(step);
    turtle.turnLeft(angle);
    step *= decay; // 每一步距离都衰减
}
```

当 `decay` 略小于1时，我们会得到一个向内旋转的螺旋。这种图形被称为**等角螺线**。
*   当 `n=3` 时，会得到三角形螺旋。
*   当 `n=7` 时，图案已经非常有趣。
*   当 `n` 很大且衰减很慢时，图形接近一个平滑的螺旋。

等角螺线在自然界中广泛存在，例如鹦鹉螺的贝壳、飓风的云图、星系的形状，甚至在建筑设计中也有应用。使用海龟绘图法，我们可以轻松地对这些自然现象进行建模。

## 常见错误：实例变量遮蔽

在结束之前，我们来看一个面向对象编程中常见的严重错误——**实例变量遮蔽**。

观察以下有缺陷的构造函数代码：
```java
public Turtle(double x0, double y0, double a0) {
    double x = x0; // 错误！这里声明了一个新的局部变量x
    double y = y0; // 错误！
    double angle = a0; // 错误！
}
```

问题在于，在构造函数内部使用 `double x = x0;` 这样的语句，并没有修改实例变量 `x`，而是**创建了一个同名的局部变量**。当构造函数执行完毕后，这个局部变量就消失了，实例变量的值从未被改变。

这种现象称为“遮蔽”。每个初学者在编写前几个面向对象程序时都可能犯这个错误，并且它很难调试，因为从代码上看逻辑似乎正确，但对象的状态就是不对。

**请牢记：不要在构造函数或方法内部重新声明实例变量。**
