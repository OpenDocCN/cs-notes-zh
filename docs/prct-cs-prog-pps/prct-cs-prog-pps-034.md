## 字符串数据类型回顾

上一节我们介绍了抽象数据类型的概念。本节中，我们来看看Java中一个最常用的内置数据类型：`String`（字符串）。

字符串是一个Unicode字符序列。Java的字符串抽象数据类型定义了许多操作，使我们能够编写程序来操作它们。

我们已经使用过一些简单的操作，例如：
*   **构造字符串**：在Java中，将字符序列放在引号内即可创建一个字符串。
*   **获取长度**：`s.length()` 返回字符串的字符数。
*   **获取字符**：`s.charAt(i)` 返回索引 `i` 处的字符。

但字符串数据类型还提供了许多其他强大的方法。

以下是Java字符串API中一些关键方法的概述：

*   **提取子字符串**：`s.substring(i, j)` 返回从索引 `i` 到 `j-1` 的子字符串。
*   **检查包含关系**：`s.contains(t)` 检查字符串 `s` 是否包含子字符串 `t`。
*   **检查开头/结尾**：`s.startsWith(prefix)` 和 `s.endsWith(suffix)` 分别检查字符串是否以特定前缀开头或以特定后缀结尾。
*   **查找索引**：`s.indexOf(pattern)` 返回给定模式在字符串中首次出现的索引。
*   **连接字符串**：我们通常使用内置的 `+` 运算符进行连接，其底层对应 `concat` 方法，用于将字符串 `t` 附加到 `s` 的末尾。
*   **字符串比较**：`s.compareTo(t)` 按字典顺序比较两个字符串。
*   **替换**：`s.replaceAll(a, b)` 将字符串 `s` 中所有出现的子字符串 `a` 替换为 `b`。
*   **分割**：`s.split(delimiter)` 根据给定的分隔符将字符串分割成一个字符串数组。
*   **相等性检查**：`s.equals(t)` 检查两个字符串是否具有相同的字符序列。

这些方法为字符串处理提供了极其灵活的基础。随着课程的深入，我们将看到大量使用这些方法的例子。

## 字符串处理示例

现在，让我们通过几个简单的程序来看看这些方法如何应用。

### 示例1：检查回文串

判断一个字符串是否正读反读都一样。思路是获取字符串长度，然后遍历前半部分字符，逐一与对应的后半部分字符比较。

```java
public static boolean isPalindrome(String s) {
    int n = s.length();
    for (int i = 0; i < n/2; i++)
        if (s.charAt(i) != s.charAt(n-1-i))
            return false;
    return true;
}
```

### 示例2：查找包含指定字符串的行

从标准输入中读取多行文本，并打印出包含命令行参数指定查询词的所有行。

```java
public static void main(String[] args) {
    String query = args[0];
    while (!StdIn.isEmpty()) {
        String line = StdIn.readLine();
        if (line.contains(query)) {
            StdOut.println(line);
        }
    }
}
```

这是一个在文件中查找包含特定词语（例如，使用某个变量的代码行）的简单方法。

### 示例3：识别超链接

假设我们想在一个文本文件中查找所有以 `"http://"` 开头并以 `".edu"` 结尾的字符串（即教育机构的超链接）。我们可以读取字符串并检查其开头和结尾。

```java
while (!StdIn.isEmpty()) {
    String link = StdIn.readString();
    if (link.startsWith("http://") && link.endsWith(".edu")) {
        StdOut.println(link);
    }
}
```

## 生物学应用：基因查找 🧬

字符串处理在现代编程中是一个非常重要的范式，一个突出的例子来自分子生物学。

多年来，科学家面临的挑战是找出人类基因组中的DNA序列。大约十年前，人类基因组测序已经完成。现在，我们拥有了海量的基因组数据，科学家的任务是编写程序来理解其结构，这本质上就是字符串处理。

### 基因组与基因

我们可以将基因组表示为一个仅由四个字母组成的字符串：**A**、**C**、**T**、**G**（代表四种核苷酸）。这个字符串可能极其漫长，对于某些生物可达数十亿个字符。

基因是基因组中代表功能单元的子字符串。科学家已知：
1.  基因由称为**密码子**的三核苷酸单元组成。
2.  基因起始于一个特定的**起始密码子**：`"ATG"`。
3.  基因终止于一个特定的**终止密码子**：`"TAA"`、`"TAG"` 或 `"TGA"` 之一。
4.  起始密码子和终止密码子之间的序列长度必须是3的倍数（即由完整的密码子构成）。

因此，在基因组字符串中，我们的挑战是找到所有以 `"ATG"` 开头，以 `"TAA"`、`"TAG"` 或 `"TGA"` 结尾，且中间部分长度为3的倍数的子字符串。

### 热身：验证潜在基因

首先，我们编写一个程序来判断一个给定的字符串是否符合上述规则，是一个潜在的基因。

```java
public static boolean isPotentialGene(String dna) {
    // 条件1: 长度是3的倍数
    if (dna.length() % 3 != 0) return false;
    // 条件2: 以起始密码子开始
    if (!dna.startsWith("ATG")) return false;
    // 条件3: 以终止密码子结束
    if (!(dna.endsWith("TAA") || dna.endsWith("TAG") || dna.endsWith("TGA")))
        return false;
    // 条件4: 中间不包含任何终止密码子
    for (int i = 3; i < dna.length() - 3; i += 3) {
        String codon = dna.substring(i, i+3);
        if (codon.equals("TAA") || codon.equals("TAG") || codon.equals("TGA"))
            return false;
    }
    return true;
}
```

这个程序充分利用了字符串的 `length()`、`startsWith()`、`endsWith()`、`substring()` 和 `equals()` 方法。

### 完整基因查找程序

真正的基因查找程序需要扫描可能长达数百万甚至数十亿字符的基因组字符串。思路如下：
1.  从左到右扫描基因组。
2.  当找到一个起始密码子（`"ATG"`）时，记录其位置。
3.  继续扫描，寻找终止密码子。
4.  如果找到一个终止密码子，并且从起始位置到该位置的**距离是3的倍数**，那么我们就找到了一个潜在的基因，可以将其提取并输出。
5.  然后重置状态，继续寻找下一个基因。

这个程序的代码比上面的验证函数稍复杂，但核心逻辑清晰，主要依赖于字符串的 `indexOf()`、`substring()` 和长度检查等操作。我们将其留作一个有趣的编程练习。

## 字符串的内部表示与性能 ⚙️

了解Java可能如何表示字符串有助于我们理解其性能，特别是在处理大型数据（如基因组）时。

一种可能的表示方式是：一个字符串对象包含两个字段：
1.  一个整数值，表示字符串的**长度**（便于快速获取）。
2.  一个内存地址（引用），指向连续存储所有字符的数组。

在这种表示下：
*   获取第 `i` 个字符（`charAt(i)`）非常高效。
*   提取子字符串（`substring(i, j)`）也可以很高效：新创建的字符串对象可以**指向原字符数组的某个偏移位置**，并记录新的起始索引和长度，而无需复制字符。这称为“共享底层数组”。

**重要区别：引用相等 vs. 值相等**
由于这种表示，两个不同的字符串对象（即使内容相同）在内存中是分开的。因此：
*   `s == t` 比较的是两个引用（内存地址）是否相同，通常**不**用于判断字符串内容是否相等。
*   `s.equals(t)` 才是比较两个字符串的字符序列是否完全相同的方法。

## 面向对象编程总结 🎯

本节课中我们一起学习了如何利用Java内置的抽象数据类型（如`String`）来解决问题。

面向对象编程的核心是**操纵对象**。对象是持有某种数据类型值的实体。变量名引用对象，我们通过操作这些对象来操作它们所持有的值。

在Java中：
*   **原始类型**（如 `int`, `double`）：变量直接存储值。
*   **引用类型**（如 `String`, `Color`, `Picture`）：变量存储的是指向对象（值）的**引用**。

虽然有些纯粹的面向对象语言没有原始类型，但Java为了效率保留了它们。作为一名务实的程序员，我们需要理解并利用这两种类型。

本讲的重点是，你可以编写程序来操纵像颜色、声音、图片和字符串这样的抽象概念。在下一讲中，你将学习如何**定义自己的抽象数据类型**，并编写程序来操纵它们。这将把你的编程能力提升到一个全新的水平，我们将会看到许多相关的例子。
