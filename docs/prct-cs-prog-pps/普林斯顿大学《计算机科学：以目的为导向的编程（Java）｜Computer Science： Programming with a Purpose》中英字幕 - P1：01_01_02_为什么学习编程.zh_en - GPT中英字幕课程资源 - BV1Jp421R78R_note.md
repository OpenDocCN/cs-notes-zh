# 计算机科学：以目的为导向的编程（Java）：1：为什么学习编程

在本节课中，我们将探讨开始编程需要了解的基础知识。我们会介绍一些历史背景，让内容更有趣，并解释编程的核心概念及其重要性。

## 概述

编程是让计算机执行我们期望任务的过程。随着计算机无处不在，掌握编程能力已成为一项基本技能。本节将解释为何要学习编程，介绍编程语言的概念，并以Java语言为例，展示编写第一个程序的基础知识。

## 为何要编程？

计算机已渗透到生活的方方面面。学习编程的核心目的是**控制计算机**，让它执行我们想要的任务。一个天真的想法是直接告诉计算机做什么，例如：“请模拟遵循牛顿定律的天体运动。”虽然理想，但计算机无法直接理解这样的自然语言指令。

另一种方式是使用现成的应用程序。虽然应用程序很方便，但无法满足所有个性化或特殊需求。**编程**使我们能够创造计算机程序，实现几乎任何我们设想的任务。

编程的历史悠久。例如，巴贝奇的分析机是早期计算机之一，而艾达·洛夫莱斯则被认为是第一位程序员。这说明了人类很早就认识到让机器执行指令的价值。

## 如何告诉计算机做什么？

编程就是告诉计算机做什么的过程。它并非专家的专属，而是一种自然、令人满足且富有创造性的体验，类似于写作。编程的独特之处在于，它能实现其他方式无法完成的成就。

开始编程面临一些挑战，这也是本课程初期要讨论的内容。你需要了解计算机能做什么，并学习一种**编程语言**作为与计算机沟通的方式。

著名计算机科学家唐纳德·克努特曾提出一个积极的观点：我们不应只想着如何指示计算机，而应专注于**向人类解释我们希望计算机做什么**。这描述了一种理想编程语言的状态。

## 与计算机沟通的方式

有多种方式可以指示计算机：
*   **使用应用程序**：这是最表层的方式。
*   **机器语言**：计算机的母语。它对计算机来说容易，但对人类来说非常繁琐且容易出错。例如，用机器语言实现两个数相加的代码可能非常复杂。
*   **自然语言**：人类的语言。虽然对人类容易，但计算机理解起来非常困难且容易产生歧义。例如，“Kids make nutritious snacks”这类新闻标题就展示了自然语言的歧义性。

因此，我们使用一种折中的方案：**高级编程语言**。它对计算机和人类的难度相当，是过去几十年中被证明有效的沟通方式。

## 为什么选择Java？

世界上有数百种高级编程语言。本课程选择**Java**，原因如下：
*   **广泛使用与可用**：由詹姆斯·高斯林在几十年前开发，持续发展至今。
*   **现代编程抽象**：包含一整套现代、有用的编程抽象概念。
*   **辅助与检查**：提供大量自动检查，帮助确保程序正确性。
*   **庞大的生态系统**：应用极其广泛，从火星探测器、手机、网络服务器到医疗设备和超级计算机，都有Java的身影。拥有数百万开发者和数十亿运行设备。

没有一种语言是完美的。正如C++语言的设计者比雅尼·斯特劳斯特鲁普所说：“世界上只有两类编程语言：一类是人们总在抱怨的，另一类是没人用的。”Java也有其缺点，但我们必须从一种具体的语言开始。

## 本课程的方法

我们的方法是使用Java的一个**最小子集**。我们使用的许多结构在其他语言中也很常见。本课程的目标是培养适用于多种语言的**通用编程技能**。一旦掌握了一种通用编程语言，转向另一种将不会太困难。

## 编程语言的词汇

学习编程语言有时被比作学习自然语言，但实际上要简单得多。Java的词汇量非常有限，主要包括以下几类（我们将在后续课程中详细讲解）：
*   关键字（如 `public`, `class`, `static`, `void`）
*   字面量（如数字 `1.0e-15`， 字符串 `"Hello, World"`）
*   运算符（如 `+`, `-`, `*`, `/`）
*   分隔符（如 `{`, `}`, `;`）

你的程序将由这些符号加上你自己创造的**变量名**组成。掌握这些有限的工具，你就能编写非常广泛的程序类别。

## 你的第一个程序：Hello World

自20世纪70年代以来，几乎每个人的第一个程序都是“Hello World”。在Java中，这个程序如下：

```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World");
    }
}
```

你需要将这段文本输入到一个文本文件中（例如使用编辑器），并将其保存为 `HelloWorld.java`。这个程序包含以下关键元素：
1.  **程序/类名**：`HelloWorld`，必须与文件名（`HelloWorld.java`）匹配。
2.  **main方法**：所有可执行Java程序都必须有一个 `main` 方法。这是程序的入口点。
3.  **语句**：`main` 方法体内包含要执行的指令。这里只有一条语句：`System.out.println("Hello, World");`，用于在屏幕上输出一行文本。

在初学阶段，你的任务就是确保准确输入这些字符。编译并运行这个程序，你将在屏幕上看到“Hello, World”。

## 关于程序风格

观察以下三个版本的Hello World程序：

```java
// 版本 A
public class HelloWorld {
public static void main(String[] args) {
System.out.println("Hello, World"); }}

// 版本 B (书籍网站风格)
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World");
    }
}

// 版本 C
public class HelloWorld
{
    public static void main(String[] args)
    {
        System.out.println("Hello, World");
    }
}
```

对于Java编译器而言，这三个程序是**完全相同**的。颜色、字体、注释和额外的空格（如缩进、换行）都属于**编程风格**，而非语言本身的一部分。

风格的选择取决于上下文（如在集成开发环境中、书籍中、幻灯片上或你自己的代码中）。一致的风格有助于发现错误，并使他人更容易阅读和理解你的代码。通常，在团队或班级中，大家会约定一种风格。

对于学生，建议是：如果评分者或你的上司有明确的风格要求，请遵循它。否则，你可以在保持清晰和一致性的前提下，融入一些自己的创意，形成个人风格。重要的是区分**语言规则**（必须遵守）和**风格约定**（可以灵活调整）。
