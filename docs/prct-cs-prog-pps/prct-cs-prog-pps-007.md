在本节课中，我们将要学习`while`语句的一种替代方案——`for`循环。这是一种有时能让程序更清晰、更易读的循环表达方式，其代码通常更紧凑、更易于理解。

## for循环的结构与原理

上一节我们介绍了`while`循环，本节中我们来看看`for`循环。`for`循环包含几个组成部分。

首先，程序会执行一个**初始化语句**。接着，它会评估一个**布尔表达式**。如果该表达式为`true`，则执行循环体内的语句序列。之后，执行一个**递增语句**，并重复此过程。

这是一种语言结构，其工作方式实际上等同于一个`while`循环。

以下是`for`循环的通用结构：
```java
for (初始化语句; 布尔表达式; 递增语句) {
    // 循环体语句序列
}
```

## for循环示例：打印2的幂

让我们通过一个例子来理解`for`循环：打印从2的0次方到2的n次方的所有幂。

初始化语句中，我们设置变量`v = 1`。`for`循环括号内包含初始化语句、布尔表达式和递增语句。大括号内是循环体语句序列。程序会严格按照这个顺序执行。

实际上，每个`for`循环都有一个等价的`while`循环。在`while`循环中，初始化语句位于循环开始之前，布尔表达式在每次迭代前被测试，递增语句则位于循环体语句序列之后。`for`循环只是让代码更紧凑、更易读。

## 更多for循环应用实例

以下是`for`循环的几个典型应用。

**计算前n个整数的和**
这是一个非常简单的计算，使用`for`循环可以自然地表达。
```java
int sum = 0;
for (int i = 1; i <= n; i++) {
    sum += i;
}
```

**计算n的阶乘**
同样，使用`for`循环进行乘法运算非常直观。
```java
int product = 1;
for (int i = 1; i <= n; i++) {
    product *= i;
}
```

**打印函数值表**
在`for`循环中，我们经常通过计数来跟踪某些内容，因此`int i = 0; i < n; i++`这种模式非常常见。下面的例子是打印函数值表，其中函数是`2 * PI * k / n`，`k`从0到`n`。这样，我们可以用一个`for`循环语句快速打印出感兴趣的值。

## 综合案例：打印尺子刻度

这是`for`循环的另一个应用示例，即我们上次看过的打印尺子细分刻度的问题。

给定`n`英寸，我们希望为从1到`n`的每个`i`值，将`i`夹在两个副本之间。让我们看看它的实现。

这是我们上次看到的计算，当时每次扩展刻度时都需要一条新语句。现在，我们将其放入一个`for`循环中。我们从一个空字符串开始，对于从1到`N`的每个`i`，我们将`i`放在之前计算出的尺子字符串的两个副本之间。

例如，当`i`为4时，假设此时`ruler`已经是`"1 2 1 3 1 2 1"`。我们会先放一个`ruler`的副本，然后放上`i`的值（即4），再放另一个`ruler`的副本。完成后，打印整个字符串。

运行`java Ruler 4`，我们就能得到正确的结果。

关于这个计算，有一个有趣的点值得思考：当你使用循环时，可以用很少的代码表达大量的计算。在这个案例中，这个很小的程序可以产生巨大的输出。如果你运行`java Ruler 100`，程序会尝试打印出2的100次方减1个整数，这远远超出了宇宙中可能打印出的数量。这正体现了循环的威力——它能让我们触及“无限”。
