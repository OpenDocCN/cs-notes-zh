# Python正则表达式教程：第12章：模式匹配

## 概述
在本节课中，我们将学习如何使用正则表达式来创建和匹配特定模式，以验证数据格式。我们将以十六进制颜色代码为例，详细介绍如何构建正则表达式模式来验证用户输入。

---

## 十六进制颜色代码简介

在计算机中，颜色通常使用十六进制颜色代码表示。这种代码以井号（#）开头，后跟六个字符。这些字符可以是数字0到9，或字母A到F（大小写均可）。

例如，代码 **#0076BA** 代表一种特定的蓝色。

十六进制颜色代码的结构如下：
- 前两个字符表示红色分量（00到FF）
- 中间两个字符表示绿色分量（00到FF）
- 最后两个字符表示蓝色分量（00到FF）

其中，**00** 表示该颜色分量最低（无该颜色），**FF** 表示该颜色分量最高（全该颜色）。

以下是一些示例：
- **#FF0000** 表示纯红色
- **#00FF00** 表示纯绿色
- **#0000FF** 表示纯蓝色
- **#FFFFFF** 表示白色（所有颜色组合）
- **#000000** 表示黑色（无颜色）

---

## 构建验证程序

上一节我们介绍了十六进制颜色代码的基本结构，本节中我们来看看如何用Python程序验证这种格式。

我们将编写一个程序，使用正则表达式模块 `re` 来检查用户输入的字符串是否符合十六进制颜色代码的格式。

首先，导入必要的模块并定义主函数：

```python
import re

def main():
    code = input("请输入一个十六进制颜色代码: ")
```

接下来，我们需要定义一个正则表达式模式来匹配有效的颜色代码。

---

## 定义基础模式

我们的模式需要匹配以下规则：
1.  以井号（#）开头
2.  后跟六个字符
3.  每个字符是 0-9、a-f 或 A-F

我们可以从最简单的部分开始：匹配开头的井号。

```python
pattern = r"#"
```

`r` 前缀表示这是一个“原始字符串”，确保反斜杠等字符被按字面意义解释，这在正则表达式中很重要。

然后，我们使用 `re.search()` 函数在用户输入的字符串中搜索这个模式：

```python
match = re.search(pattern, code)
if match:
    print("有效")
    print(f"匹配到: {match.group()}")
else:
    print("无效")
```

目前，这个程序只检查字符串中是否包含井号，这显然不够。

---

## 使用字符集改进模式

我们需要确保井号后面的字符在允许的范围内。这可以通过**字符集**来实现，字符集用方括号 `[]` 表示。

字符集 `[a-fA-F0-9]` 表示可以匹配一个小写a-f、大写A-F或数字0-9中的任意一个字符。

因此，我们的模式可以改进为：

```python
pattern = r"#[a-fA-F0-9]"
```

但这只匹配井号加一个有效字符。我们需要匹配六个这样的字符。

---

## 使用量词指定字符数量

为了匹配恰好六个有效字符，我们可以使用**量词**。量词用花括号 `{}` 表示，其中的数字指定了前面元素需要重复的次数。

将模式修改为：

```python
pattern = r"#[a-fA-F0-9]{6}"
```

现在，模式 `#[a-fA-F0-9]{6}` 表示：一个井号，后跟恰好六个在指定范围内的字符。

然而，这个模式仍然存在问题。它会在更长的字符串中寻找匹配。例如，对于输入 `"颜色是#AAAAAA吗"`，程序会找到 `#AAAAAA` 并认为有效，但这并非我们想要的严格匹配。

---

## 使用锚点确保精确匹配

为了解决上述问题，我们需要使用**锚点**来确保模式匹配整个字符串的特定部分。
- `^` 锚点表示匹配必须从字符串的开头开始。
- `$` 锚点表示匹配必须到字符串的结尾结束。

因此，最终的模式应该是：

```python
pattern = r"^#[a-fA-F0-9]{6}$"
```

模式 `^#[a-fA-F0-9]{6}$` 的含义是：
- `^#`：字符串必须以井号开头。
- `[a-fA-F0-9]{6}`：紧接着必须是恰好六个有效字符。
- `$`：字符串必须在此结束。

这样，只有完全符合 `#` + 六个有效字符 格式的输入才会被判定为有效。

---

## 简化字符集写法

我们可以使用连字符 `-` 在字符集中表示一个范围，使模式更简洁：

```python
pattern = r"^#[a-fA-F0-9]{6}$"
```
可以等价地写为：
```python
pattern = r"^#[a-fA-F0-9]{6}$"
```
（注意：`[a-fA-F0-9]` 已经是最简洁的写法，它等价于 `[abcdefABCDEF0123456789]`）。

---

## 完整程序代码

以下是整合了所有概念的完整程序：

```python
import re

def main():
    code = input("请输入一个十六进制颜色代码: ")

    # 定义正则表达式模式：以#开头，后跟恰好6个十六进制字符（0-9, a-f, A-F），然后结束
    pattern = r"^#[a-fA-F0-9]{6}$"

    # 在输入中搜索模式
    match = re.search(pattern, code)

    # 输出结果
    if match:
        print("有效")
        print(f"匹配到: {match.group()}")
    else:
        print("无效")

if __name__ == "__main__":
    main()
```
