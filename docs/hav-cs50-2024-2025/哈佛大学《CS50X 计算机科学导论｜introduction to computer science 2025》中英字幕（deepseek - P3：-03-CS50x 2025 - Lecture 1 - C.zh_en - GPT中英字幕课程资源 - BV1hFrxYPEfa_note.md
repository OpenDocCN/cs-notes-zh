## 从Scratch到C语言

上一节我们介绍了Scratch作为编程的起点。本节中我们来看看如何将Scratch中的概念转化为C语言代码。

在Scratch中，我们使用“当绿旗被点击”积木来启动程序。在C语言中，这对应于一个名为 `main` 的特殊函数。Scratch中的“说”积木是一个函数，它接受输入并产生一个视觉上的“副作用”。在C语言中，我们使用 `printf` 函数来实现类似的功能。

![](img/40648e07405302a90e71ffbc2e9792df_13.png)

例如，Scratch中的“说 Hello, world!”积木在C语言中变为：
```c
printf("hello, world\n");
```
这里的 `\n` 是一个转义序列，代表换行符。

## 编写第一个C程序：Hello, World

![](img/40648e07405302a90e71ffbc2e9792df_31.png)

现在，让我们动手编写第一个C程序。我们将创建一个名为 `hello.c` 的文件，并在其中写入代码。

![](img/40648e07405302a90e71ffbc2e9792df_33.png)

![](img/40648e07405302a90e71ffbc2e9792df_35.png)

以下是创建和运行程序的三个步骤：
1.  **创建源文件**：使用 `code hello.c` 命令创建一个新文件并打开编辑器。
2.  **编译程序**：使用 `make hello` 命令将源代码（`hello.c`）编译成机器代码（`hello`）。`make` 命令会调用编译器。
3.  **运行程序**：使用 `./hello` 命令执行编译好的程序。`./` 表示在当前文件夹中查找程序。

在终端中，命令提示符通常是 `$`。你可以使用 `ls` 命令列出当前目录下的文件，使用 `cd` 命令切换目录。

## C语言的基本构件

C语言程序由一些基本构件组成。让我们逐一分解。

### 函数与头文件

在C语言中，`printf` 这样的函数并不是语言本身的一部分，它们来自“库”。要使用库中的函数，你需要在程序开头“包含”对应的头文件。

![](img/40648e07405302a90e71ffbc2e9792df_41.png)

例如，要使用 `printf`，你需要添加：
```c
#include <stdio.h>
```
`stdio.h` 是“标准输入输出”头文件，包含了 `printf` 等函数的声明。如果你忘记包含它，编译器会报错。

![](img/40648e07405302a90e71ffbc2e9792df_43.png)

CS50课程还提供了一个自定义库 `cs50.h`，它包含了一些简化输入操作的函数，例如 `get_string` 和 `get_int`。我们将在后续使用它们。

### 变量与类型

在Scratch中，你可以创建变量来存储值。在C语言中，你也需要这样做，但必须明确指定变量的“类型”，告诉计算机你打算存储什么类型的数据。

C语言支持多种数据类型，例如：
*   `int`：整数。
*   `char`：单个字符。
*   `float` / `double`：浮点数（带小数点的数字）。
*   `string`：字符串（CS50库提供的类型，本质上是文本）。

当你从函数（如 `get_string`）获取一个返回值时，你需要将其存储在一个声明了合适类型的变量中。

例如，获取一个字符串：
```c
string answer = get_string("What's your name? ");
```
这里，`string` 是变量 `answer` 的类型，`=` 是赋值运算符，将 `get_string` 的返回值赋给 `answer`。

### 格式化输出

在Scratch中，你可以使用“连接”积木组合文本。在C语言的 `printf` 中，我们使用“占位符”和额外的参数来实现。

例如，要打印“Hello, David”：
```c
printf("hello, %s\n", answer);
```
`%s` 是一个占位符，表示这里要插入一个字符串。字符串的值由 `printf` 的第二个参数 `answer` 提供。多个参数用逗号分隔。

其他常用的格式码包括：
*   `%i`：用于整数。
*   `%c`：用于字符。
*   `%f`：用于浮点数。

## 控制流：条件语句

程序经常需要根据条件做出决策。在Scratch中，我们使用“如果-那么”积木。在C语言中，我们使用 `if`、`else if` 和 `else` 语句。

一个比较两个数字的C语言条件语句示例如下：
```c
if (x < y)
{
    printf("x is less than y\n");
}
else if (x > y)
{
    printf("x is greater than y\n");
}
else
{
    printf("x is equal to y\n");
}
```
注意，在C语言中，判断相等使用双等号 `==`，而不是单等号 `=`（单等号用于赋值）。同时，良好的代码设计应避免不必要的条件检查。

逻辑运算符 `||` 表示“或”，`&&` 表示“与”，可用于组合条件。

## 控制流：循环

循环用于重复执行一段代码。在Scratch中，有“重复”积木。在C语言中，我们有几种循环结构。

**`while` 循环**：在条件为真时重复执行。
```c
int i = 0;
while (i < 3)
{
    printf("meow\n");
    i++;
}
```

**`for` 循环**：一种更紧凑的循环，将初始化、条件和更新写在一行。
```c
for (int i = 0; i < 3; i++)
{
    printf("meow\n");
}
```
这是最常用、最规范的循环写法。程序员通常从0开始计数。

**`do while` 循环**：先执行一次循环体，然后再检查条件。适用于至少需要执行一次的情况，例如验证用户输入。
```c
int n;
do
{
    n = get_int("Positive integer: ");
}
while (n < 1);
```

要创建一个无限循环，可以使用 `while (true)`。你可以按 `Ctrl+C` 来终止正在运行的程序。

## 自定义函数

在Scratch中，你可以定义自己的积木块。在C语言中，你可以定义自己的函数。

![](img/40648e07405302a90e71ffbc2e9792df_56.png)

![](img/40648e07405302a90e71ffbc2e9792df_57.png)

一个没有输入和输出的函数（只有副作用）定义如下：
```c
void meow(void)
{
    printf("meow\n");
}
```
`void` 表示“无”。第一个 `void` 表示函数不返回值，第二个 `void` 表示函数不接受参数。

一个有输入的函数定义如下：
```c
void meow(int n)
{
    for (int i = 0; i < n; i++)
    {
        printf("meow\n");
    }
}
```
然后你可以这样调用它：`meow(3);`。

如果一个函数有返回值，你需要指定返回类型并使用 `return` 关键字：
```c
int get_positive_int(void)
{
    int n;
    do
    {
        n = get_int("Positive integer: ");
    }
    while (n < 1);
    return n;
}
```

![](img/40648e07405302a90e71ffbc2e9792df_59.png)

### 函数原型

![](img/40648e07405302a90e71ffbc2e9792df_61.png)

在C语言中，函数必须在使用前被声明。通常，我们将 `main` 函数放在文件顶部，而将其他自定义函数放在后面。为了让编译器提前知道这些函数的存在，我们需要在文件顶部（`#include` 之后）添加函数“原型”。

函数原型就是函数定义的第一行加上分号。例如：
```c
void meow(int n);
int get_positive_int(void);
```

## 综合示例：马里奥方块

![](img/40648e07405302a90e71ffbc2e9792df_70.png)

![](img/40648e07405302a90e71ffbc2e9792df_72.png)

让我们应用所学知识，解决一个类似绘制马里奥游戏中砖块的问题。

**打印一行砖块**：我们可以用一个循环打印一行指定数量的砖块（用 `#` 表示）。
```c
for (int i = 0; i < 4; i++)
{
    printf("#");
}
printf("\n");
```

**打印一个方阵砖块**：我们需要使用嵌套循环。外层循环控制行数，内层循环控制每行的砖块数。
```c
for (int i = 0; i < 3; i++) // 对于每一行
{
    for (int j = 0; j < 3; j++) // 对于行中的每一列
    {
        printf("#");
    }
    printf("\n"); // 一行结束后换行
}
```
更好的设计是定义一个 `print_row` 函数来打印单行，然后在主函数中循环调用它。

为了代码清晰，可以使用“常量”和“注释”：
```c
const int N = 3; // 常量，砖块方阵的大小

// 打印一行N个砖块
void print_row(void)
{
    for (int i = 0; i < N; i++)
    {
        printf("#");
    }
    printf("\n");
}
```

### 整数溢出

![](img/40648e07405302a90e71ffbc2e9792df_87.png)

计算机使用固定数量的比特位（如32位）来存储整数。这意味着整数有一个最大值。如果你尝试存储一个超过这个最大值的数，就会发生“整数溢出”，数值会“绕回”到一个很小的数（甚至是负数或零）。

![](img/40648e07405302a90e71ffbc2e9792df_88.png)

![](img/40648e07405302a90e71ffbc2e9792df_90.png)

![](img/40648e07405302a90e71ffbc2e9792df_91.png)

例如，一个32位有符号整数最大约为21亿。在之前的“加倍游戏”示例中，金钱加倍最终会溢出，导致数值异常。

![](img/40648e07405302a90e71ffbc2e9792df_93.png)

![](img/40648e07405302a90e71ffbc2e9792df_94.png)

![](img/40648e07405302a90e71ffbc2e9792df_95.png)

![](img/40648e07405302a90e71ffbc2e9792df_97.png)

![](img/40648e07405302a90e71ffbc2e9792df_98.png)

![](img/40648e07405302a90e71ffbc2e9792df_100.png)

在现实世界中，整数溢出曾导致过严重问题，例如旧版波音787飞机在连续运行248天后可能因计数器溢出而断电。

![](img/40648e07405302a90e71ffbc2e9792df_102.png)

使用 `long`（64位）类型可以存储更大的数，但它最终也会溢出。

### 浮点数精度

计算机使用固定数量的比特位表示浮点数（实数）。由于比特位有限，计算机无法精确表示所有实数（实数有无限多个）。这会导致精度误差。

例如，在C语言中，`1 / 3` 如果用整数运算结果是 `0`。即使用浮点数 `1.0 / 3.0`，打印出来也可能是 `0.3333333432674408` 而不是完美的 `0.333333...`。这种不精确性是数字计算中固有的挑战。

### 著名的日期问题

“千年虫”（Y2K）问题是因为许多旧系统用两位数字存储年份，导致2000年被误认为是1900年。类似地，“2038年问题”源于许多系统使用32位整数存储自1970年1月1日以来的秒数，该数值将在2038年1月19日溢出。解决方案是迁移到使用更多比特位的系统。

## 代码质量

在编程中，我们不仅关注代码是否能运行（正确性），还关注它是否写得好。

*   **正确性**：代码是否按照预期工作？这是最基本的要求。
*   **设计**：代码的结构是否高效、清晰？是否避免了不必要的重复或复杂的逻辑？这更为主观，但至关重要。
*   **风格**：代码的格式是否整洁、一致、易读？例如，适当的缩进、空格和命名。

在CS50中，我们提供了工具来帮助你评估和改进代码：
*   `check50`：自动检查代码的正确性。
*   `style50`：在VS Code中提供代码风格建议。
*   `design50`：在VS Code中提供代码设计建议。
