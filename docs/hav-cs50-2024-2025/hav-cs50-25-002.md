## 源代码与机器代码

我们人类编写的、像上面那样的代码被称为**源代码**。它是我们能够理解和编写的文本形式。然而，计算机的CPU只能理解由0和1组成的**机器代码**。

因此，我们需要一个工具将源代码转换为机器代码。这个工具就是**编译器**。编译器是一种特殊的软件，它读取源代码（例如用C语言编写的代码）并生成相应的机器代码（0和1），以便计算机能够执行。

在我们的课程环境中，我们将使用一个名为 `make` 的命令来调用编译器。例如，要编译一个名为 `hello.c` 的程序，我们可以在终端中输入 `make hello`。如果编译成功且没有错误信息输出，就意味着生成了一个名为 `hello` 的可执行文件（机器代码）。要运行这个程序，我们需要输入 `./hello`。

以下是编译和运行C程序的基本步骤：
1.  使用 `code hello.c` 命令创建并打开源代码文件。
2.  在编辑器中编写C代码。
3.  使用 `make hello` 命令编译代码。
4.  使用 `./hello` 命令运行编译好的程序。

## 深入理解“Hello, World”

![](img/668cb852feddba7089c907fa23070ae2_7.png)

让我们仔细分析一下第一个C程序。

![](img/668cb852feddba7089c907fa23070ae2_9.png)

```c
#include <stdio.h>

![](img/668cb852feddba7089c907fa23070ae2_11.png)

int main(void)
{
    printf("hello, world\n");
}
```

*   `#include <stdio.h>`：这行代码告诉编译器，我们想要使用标准输入输出库中的函数，特别是 `printf` 函数。`stdio.h` 是一个**头文件**，它包含了这些函数的声明。
*   `int main(void)`：这是每个C程序的入口点。程序从这里开始执行。`int` 表示这个函数会返回一个整数值，`void` 表示它不接受任何参数。目前我们只需记住这个固定的写法。
*   `{` 和 `}`：这些花括号定义了 `main` 函数的代码块，所有要执行的指令都写在这里面。
*   `printf("hello, world\n");`：这是调用 `printf` 函数来打印文本。
    *   `"hello, world\n"` 是要打印的字符串。双引号 `"` 用来标记字符串的开始和结束。
    *   `\n` 是一个**转义序列**，代表换行符，它会把光标移动到下一行。
    *   分号 `;` 在C语言中用于结束一条语句，就像英文句子以句号结束一样。

如果省略了分号或拼错了头文件名，编译器会给出错误信息，帮助我们定位问题。

## 获取用户输入

在Scratch中，我们可以使用“询问并等待”积木来获取用户输入。在C语言中，我们可以使用CS50库提供的 `get_string` 函数来实现类似功能。

首先，我们需要在代码中包含CS50库的头文件：`#include <cs50.h>`。

![](img/668cb852feddba7089c907fa23070ae2_19.png)

以下是如何获取用户姓名并打招呼的代码：

![](img/668cb852feddba7089c907fa23070ae2_21.png)

```c
#include <cs50.h>
#include <stdio.h>

int main(void)
{
    string answer = get_string("What's your name? ");
    printf("hello, %s\n", answer);
}
```

*   `string answer`：这行代码声明了一个名为 `answer` 的变量，其类型为 `string`（字符串）。`string` 类型来自CS50库，用于存储文本。
*   `get_string("What's your name? ")`：这个函数会在屏幕上显示提示信息“What‘s your name? ”，然后等待用户输入。用户输入的内容会作为函数的**返回值**。
*   `=`：这是赋值运算符。它将 `get_string` 的返回值（用户输入的姓名）存储到变量 `answer` 中。
*   `printf("hello, %s\n", answer)`：这里我们再次使用 `printf`。`%s` 是一个**占位符**，它告诉 `printf`：“在这里插入一个字符串”。逗号后面的 `answer` 指定了要插入哪个变量的值。

## 条件语句

在Scratch中，我们可以使用“如果...那么”积木来根据条件执行不同的操作。在C语言中，我们使用 `if`、`else if` 和 `else` 语句来实现条件逻辑。

以下是一个比较两个数字 `x` 和 `y` 的例子：

```c
if (x < y)
{
    printf("x is less than y\n");
}
else if (x > y)
{
    printf("x is greater than y\n");
}
else
{
    printf("x is equal to y\n");
}
```

*   `(x < y)`：这是一个**布尔表达式**，其结果为真（`true`）或假（`false`）。`<` 是“小于”比较运算符。
*   注意，在C语言中，判断相等需要使用两个等号 `==`，因为单个等号 `=` 已经被用作赋值运算符。
*   良好的程序设计意味着避免不必要的检查。例如，如果已经确定 `x` 既不小于 `y` 也不大于 `y`，那么它必然等于 `y`，因此最后的 `else` 块就足够了，不需要再写 `else if (x == y)`。

C语言中常用的比较运算符包括：
*   `<` （小于）
*   `>` （大于）
*   `<=` （小于或等于）
*   `>=` （大于或等于）
*   `==` （等于）
*   `!=` （不等于）

## 循环

循环允许我们重复执行一段代码。在Scratch中，有“重复”积木。在C语言中，我们主要有两种循环：`while` 循环和 `for` 循环。

**使用 `while` 循环：**

```c
int i = 0;
while (i < 3)
{
    printf("meow\n");
    i++;
}
```
这段代码会打印“meow”三次。`i++` 是 `i = i + 1` 的简写，表示将 `i` 的值增加1。

**使用 `for` 循环（更常用）：**

![](img/668cb852feddba7089c907fa23070ae2_23.png)

```c
for (int i = 0; i < 3; i++)
{
    printf("meow\n");
}
```
`for` 循环将计数器初始化（`int i = 0`）、循环条件（`i < 3`）和计数器更新（`i++`）都集中在一行，结构更紧凑。

**无限循环：**
有时我们需要一个永不停止的循环（例如游戏主循环），可以使用 `while (true)`。

```c
while (true)
{
    printf("meow\n");
}
```
要中断这样的循环，可以在终端中按下 `Ctrl + C`。

## 数据类型与运算符

C语言要求我们明确指定变量中存储的数据类型。这有助于编译器高效地管理内存。

以下是基本数据类型：
*   `bool`：布尔值，`true` 或 `false`。
*   `char`：单个字符，如 `'a'` 或 `'2'`。
*   `string`：字符串（来自CS50库），由多个字符组成。
*   `int`：整数，范围约为 -20亿到 +20亿（使用32位）。
*   `long`：更大的整数，范围更广（使用64位）。
*   `float`：单精度浮点数，可表示带小数点的数字。
*   `double`：双精度浮点数，比 `float` 精度更高。

**算术运算符**用于数学计算：
*   `+` 加法
*   `-` 减法
*   `*` 乘法
*   `/` 除法
*   `%` 取模（求余数）

![](img/668cb852feddba7089c907fa23070ae2_25.png)

**注意整数除法：** 在C语言中，两个整数相除结果仍是整数，小数部分会被**截断**（直接丢弃）。例如，`1 / 3` 的结果是 `0`。要进行浮点数除法，至少需要一个操作数是浮点类型，如 `1.0 / 3`。

## 创建自定义函数

在Scratch中，我们可以创建自己的积木块。在C语言中，我们可以创建**函数**来封装可重用的代码片段。

一个不接受输入也不返回值的函数（只有副作用）：

```c
void meow(void)
{
    printf("meow\n");
}
```
*   `void` 表示函数不返回值。
*   函数名后的 `(void)` 表示函数不接受任何参数。
*   函数体中的 `printf` 是它的副作用（在屏幕上打印）。

一个接受输入的函数：

```c
void meow(int n)
{
    for (int i = 0; i < n; i++)
    {
        printf("meow\n");
    }
}
```
现在，我们可以在 `main` 函数中调用 `meow(3)` 来让猫叫三声。

**关于函数声明（原型）：** 如果函数定义写在 `main` 函数之后，编译器在遇到调用时可能还不认识它。为了解决这个问题，我们可以在文件顶部（`main` 之前）提供函数的**原型**，即函数签名后加一个分号。

```c
#include <stdio.h>

![](img/668cb852feddba7089c907fa23070ae2_35.png)

![](img/668cb852feddba7089c907fa23070ae2_37.png)

void meow(int n); // 函数原型

int main(void)
{
    meow(3);
}

void meow(int n) // 函数定义
{
    for (int i = 0; i < n; i++)
    {
        printf("meow\n");
    }
}
```

## 命令行环境与Linux基础命令

我们编程的环境称为**集成开发环境**，它结合了图形界面和命令行界面。我们主要在**终端窗口**中使用**命令行界面**来输入命令。

以下是一些基本的Linux命令，用于在文件系统中导航和管理文件：
*   `ls`：列出当前目录下的文件和文件夹。
*   `cd <目录名>`：进入指定目录。`cd ..` 返回上级目录。
*   `mkdir <目录名>`：创建新目录。
*   `rm <文件名>`：删除文件。
*   `mv <源> <目标>`：移动文件（也可用于重命名）。
*   `cp <源> <目标>`：复制文件。

## 计算机的局限性

![](img/668cb852feddba7089c907fa23070ae2_58.png)

计算机的能力受其硬件限制。两个重要的概念是**整数溢出**和**浮点精度误差**。

![](img/668cb852feddba7089c907fa23070ae2_60.png)

**整数溢出：** 每种数据类型能表示的数字范围是有限的。例如，一个32位的 `int` 类型最大值约为20亿。如果尝试存储超过这个范围的数字，就会发生溢出，导致数值“绕回”到一个很小的数（甚至是负数）。历史上，类似的溢出错误曾导致波音787飞机在连续运行248天后需要重启，以及“2038年问题”（32位时间戳溢出）。

**浮点精度误差：** 计算机使用有限数量的位来表示浮点数，因此无法精确表示所有实数（如 `1/3`）。计算中会产生微小的误差。使用 `double` 可以提高精度，但不能根本解决这个问题。
