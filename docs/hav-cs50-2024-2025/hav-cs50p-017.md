## 概述

在本节课程中，我们将学习函数输出的两种形式：**返回值**与**副作用**。我们将重点探讨**返回值**的概念，理解它如何使我们的程序更加灵活，并学习如何在代码中捕获和使用函数的返回值。

---

![](img/821f662a89a060c0f46e27006d1fb7b6_2.png)

## 函数的两种输出

上一节我们介绍了函数作为程序的基本构建块，可以接收输入并产生输出。现在，让我们更深入地观察，函数实际上可以产生两种类型的输出：一种是**返回值**，另一种是**副作用**。

以下是一个简单的函数示例，它演示了**副作用**（在终端打印文本）：

```python
def greet(input):
    if "hello" in input:
        print("hello there")
    else:
        print("I'm not sure what you mean")
```

这个函数根据输入中是否包含“hello”这个词，在终端打印不同的问候语。这种直接在屏幕上输出文本的行为就是一种**副作用**。

---

## 理解返回值

**返回值**与副作用不同。它是一个函数可以传递给程序的值，以便在后续的代码中使用。目前，我们的 `greet` 函数并没有显式地返回任何值。

为了定义一个返回值，我们需要在Python中使用 `return` 关键字。

让我们修改上面的函数，使其返回文本字符串，而不是直接打印：

```python
def greet(input):
    if "hello" in input:
        return "hello there"
    else:
        return "I'm not sure what you mean"
```

经过这个修改，`greet` 函数现在有了明确定义的返回值：
*   如果输入中包含“hello”，则返回字符串 `"hello there"`。
*   如果不包含，则返回字符串 `"I'm not sure what you mean"`。

如果我们现在运行这个程序，终端上不会显示任何内容。这并不意味着程序出错了，而是因为返回值被计算出来后，我们还没有去捕获或使用它。

---

## 捕获和使用返回值

为了在程序中使用函数的返回值，我们需要将其存储在一个变量中。

以下是捕获和使用返回值的步骤：

1.  调用函数并传入参数。
2.  使用赋值运算符（`=`）将函数的返回值存储到一个变量中。

```python
# 调用greet函数，并将返回值存储在变量‘greeting’中
greeting = greet("hello computer")

# 现在可以打印这个变量
print(greeting)
```

运行这段代码，我们将在终端看到 `hello there`。如果我们传入 `"how's the weather"`，则会看到 `I'm not sure what you mean`。

---

## 返回值的优势：灵活性与复用性

返回值的核心优势在于它提供了极大的灵活性。当函数只产生副作用（如直接打印）时，输出形式是固定的。

然而，有了返回值，我们可以在调用函数后，以多种方式处理和修改这个结果。

以下是一些展示灵活性的例子：

*   **添加前缀**：我们可以在打印返回值前添加一些内容。
    ```python
    print("hm... " + greeting)
    ```
    输出：`hm... hello there`

*   **拼接字符串**：我们可以将返回值与其他字符串连接起来。
    ```python
    print(greeting + ", Carter")
    ```
    输出：`hello there, Carter`

*   **进行逻辑判断**：我们可以根据返回值来决定后续的程序流程。
    ```python
    if greeting == "hello there":
        print("The computer recognized your greeting!")
    ```

如你所见，**返回值**允许我们在代码的各个部分修改和使用函数的输出结果。而**副作用**（如 `print`）则是立即产生一个变化（在屏幕上显示文本）。

---

## 总结

本节课中，我们一起学习了：
1.  函数输出的两种类型：**返回值**与**副作用**。
2.  **返回值**是函数传递给调用者供后续使用的数据，使用 `return` 关键字定义。
3.  **副作用**是函数执行时对外部环境产生的直接影响（如打印输出、修改文件）。
4.  通过将返回值**赋值给变量**，我们可以在程序中捕获并灵活地使用它。
5.  返回值的核心优势在于其**灵活性**和**可复用性**，使得程序逻辑更清晰，代码更易于维护。
