## 第一个程序：聊天机器人 💬

以下是创建一个简单聊天机器人的步骤，它使用OpenAI的API来回答问题。

1.  打开一个代码编辑器（如VS Code）。
2.  在终端窗口中，创建一个Python文件（例如 `chat.py`）。
3.  在文件中输入以下代码：

    ```python
    from openai import OpenAI

    client = OpenAI()

    response = client.responses.create(
        model="gpt-4",
        input="In one sentence, what is CS50?"
    )

    print(response.output_text)
    ```

4.  运行这个程序，你将得到关于CS50的一句话描述。

这个程序是“硬编码”的，只回答一个问题。我们可以让它变得更动态，接受用户输入。

## 计算机如何表示信息？ 💡

上一节我们体验了编程的威力。现在，本节中我们来看看这一切的基础：计算机如何表示和处理信息。

计算机科学从根本上是关于信息的研究：如何表示信息、如何处理信息。这引出了“计算思维”的概念，即将计算机科学的思路应用于解决现实世界的问题。

### 二进制：计算机的语言

计算机的“字母表”只有两个字符：0和1。这被称为二进制系统（base-2）。一个二进制数字称为一个“比特”（bit）。

我们可以用简单的开关状态来理解比特：
*   **0** 可以表示为“关”（如灯泡熄灭）。
*   **1** 可以表示为“开”（如灯泡点亮）。

单个比特能表示的信息有限。为了表示更大的数字，我们将多个比特组合在一起。8个比特构成一个“字节”（byte）。

### 从二进制到数字

我们熟悉的十进制系统（base-10）使用0-9这十个数字。二进制系统使用同样的“位值”原理，但每一位代表的是2的幂次方。

例如，二进制数 `111` 的计算方式是：
`(4 * 1) + (2 * 1) + (1 * 1) = 7`

通过组合更多的比特，我们可以表示更大的数字。一个字节（8比特）可以表示从0到255（共256个）不同的值。

## 表示字母和文本 🔤

计算机不仅需要表示数字，还需要表示字母、符号等文本信息。这是通过给每个字符分配一个唯一的数字代码来实现的。

早期广泛使用的标准是**ASCII**（美国信息交换标准代码）。例如：
*   大写字母 **A** 对应数字 **65**。
*   大写字母 **B** 对应数字 **66**。

当计算机在文本环境中遇到比特模式 `01000001`（即十进制65）时，它会将其显示为字母“A”。

![](img/ab549d91158e114e58bfae1ce3bd6c67_5.png)

然而，ASCII只能表示256个字符，不足以涵盖全球所有语言。因此，现代系统使用 **Unicode**，它使用更多比特（如16或32位），能够表示数万个字符，包括各种文字符号和**表情符号**。

## 表示颜色和图像 🎨

计算机如何表示颜色呢？答案仍然是数字。

一种常见的方法是**RGB模型**，通过混合不同强度的红色（Red）、绿色（Green）和蓝色（Blue）来产生各种颜色。每种颜色的强度通常用一个字节（0-255）来表示。

例如：
*   `rgb(255, 0, 0)` 表示纯红色。
*   `rgb(0, 0, 0)` 表示黑色（无光）。
*   `rgb(255, 255, 255)` 表示白色（全光）。

一张图片由成千上万个微小的彩色点组成，这些点被称为**像素**（pixel）。每个像素的颜色由RGB值定义。因此，一张图片本质上就是一个巨大的数字矩阵。

视频则可以看作是一系列快速连续显示的图片（例如每秒30帧），从而产生运动的效果。

## 算法：解决问题的步骤 🧩

![](img/ab549d91158e114e58bfae1ce3bd6c67_7.png)

![](img/ab549d91158e114e58bfae1ce3bd6c67_8.png)

上一节我们了解了计算机如何表示各种信息。现在，本节中我们来看看计算机如何处理这些信息以解决问题，这就要用到**算法**。

算法是解决特定问题的一系列明确步骤。它是计算机程序背后的“秘方”。

### 一个例子：在电话簿中找人

假设你需要在一本厚厚的电话簿中找到“John Harvard”的电话号码。

![](img/ab549d91158e114e58bfae1ce3bd6c67_10.png)

![](img/ab549d91158e114e58bfae1ce3bd6c67_12.png)

1.  **算法A（低效）**：从第一页开始，一页一页地翻，直到找到为止。如果电话簿有1000页，在最坏情况下你需要翻1000次。
2.  **算法B（高效）**：打开电话簿的中间。根据字母顺序，判断目标名字是在左半部分还是右半部分。扔掉错误的那一半，在剩下的一半中重复这个过程。这种方法快得多。

第二种算法体现了一种“分而治之”的思想。在计算机科学中，我们不仅追求写出能工作的代码，更追求写出**高效、设计优良**的代码。

## 从算法到代码：伪代码 📝

在编写实际代码之前，我们通常先用人类语言描述算法步骤，这被称为**伪代码**。

以下是“在电话簿中找人”高效算法的伪代码：

1.  拿起电话簿。
2.  打开电话簿到中间页。
3.  查看当前页。
4.  如果“John Harvard”在当前页上：
    *   打电话。
    *   结束。
5.  否则，如果“John Harvard”在当前页之前（按字母顺序）：
    *   在左半部分取中间页。
    *   回到第3步。
6.  否则，如果“John Harvard”在当前页之后：
    *   在右半部分取中间页。
    *   回到第3步。
7.  否则（如果只剩下空白页或确认不存在）：
    *   退出。

这段伪代码包含了几个核心编程概念：
*   **函数**：如“拿起”、“打开”、“查看”等动作。
*   **条件**（if/else）：根据不同情况做出决定。
*   **布尔表达式**：答案为“是/否”的问题（如“是否在当前页？”）。
*   **循环**：重复执行某些步骤（“回到第3步”）。

## 初识编程：Scratch 🐾

![](img/ab549d91158e114e58bfae1ce3bd6c67_16.png)

上一节我们通过伪代码描述了算法。现在，本节中我们来看看如何用一种直观的方式将这些概念转化为实际可运行的程序——使用**Scratch**。

Scratch是麻省理工学院开发的一款图形化编程语言。它通过拖拽彩色代码块来创建程序，非常适合初学者理解编程逻辑。

### Scratch 环境简介

![](img/ab549d91158e114e58bfae1ce3bd6c67_18.png)

![](img/ab549d91158e114e58bfae1ce3bd6c67_20.png)

Scratch界面主要分为三个区域：
1.  **代码块区域**：按类别（如运动、外观、声音、控制）排列的各种功能块。
2.  **编程区**：将代码块拖拽到这里并拼接起来，形成脚本。
3.  **舞台区**：角色（精灵）活动的地方，有一个坐标系（x, y）。

![](img/ab549d91158e114e58bfae1ce3bd6c67_22.png)

![](img/ab549d91158e114e58bfae1ce3bd6c67_23.png)

### 第一个Scratch程序：“Hello, World”

1.  从 **事件** 类别中，拖出 `当绿旗被点击` 块。
2.  从 **外观** 类别中，拖出 `说 Hello!` 块，并将其拼接在第一个块下面。
3.  点击绿色旗帜，角色就会说“Hello!”。

这个简单程序包含了：
*   **输入**：`Hello!` 这个文本。
*   **算法/函数**：`说` 这个代码块封装了显示文字的步骤。
*   **输出/副作用**：屏幕上出现的对话气泡。

### 让程序互动：询问并回答

1.  从 **侦测** 类别中，拖出 `询问 你叫什么名字？ 并等待` 块，放在绿旗块下。
2.  从 **外观** 类别中，拖出 `说 Hello!` 块。
3.  我们需要将回答和问候语连接起来。从 **运算** 类别中，拖出 `连接 Apple 和 banana` 块。
4.  将 `连接` 块放入 `说` 块的输入框中。在第一个空格输入 `Hello, `（注意空格），从 **侦测** 类别中将 `回答` 变量拖入第二个空格。
5.  现在运行程序，输入你的名字，角色就会个性化地问候你。

这个程序引入了 **变量**（`回答`）和 **返回值** 的概念。`询问` 块不仅产生副作用（弹出对话框），还会返回用户输入的值，供后续代码使用。

### 使用循环和条件

*   **循环**：避免重复代码。例如，要让猫叫三声，可以使用 **控制** 类别中的 `重复执行 10 次` 块，将 `播放声音 Meow 直到播放完毕` 和 `等待 1 秒` 放入其中，并将次数改为3。
*   **条件**：实现交互逻辑。例如，使用 ` forever` 循环和 `如果...那么` 块，配合 `侦测` 中的 `碰到 鼠标指针？` 条件，可以实现“当鼠标碰到猫时，猫发出叫声”的效果。

### 创建自定义积木（函数）

如果你发现某些代码块组合经常被使用（比如“喵叫并等待”），可以将其定义为一个新积木，实现代码的**抽象**和**复用**。
1.  点击 **我的积木** 类别中的 `制作新的积木`。
2.  给积木命名（如“喵叫”）。
3.  在定义的模块下，拼接好喵叫的具体步骤。
4.  之后，你就可以像使用内置积木一样，使用这个自定义的“喵叫”积木了。
