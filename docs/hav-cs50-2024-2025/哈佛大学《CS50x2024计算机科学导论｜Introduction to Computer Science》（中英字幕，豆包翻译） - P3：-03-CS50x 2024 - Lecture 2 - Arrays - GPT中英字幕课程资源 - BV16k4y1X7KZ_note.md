## 编译过程详解

上一节我们介绍了如何使用 `make` 命令编译程序，本节中我们来看看编译过程背后的四个具体步骤。

![](img/9bd24c440e3404fd993981bcfdb792d7_10.png)

编译过程并非一步到位，它实际上包含四个步骤：预处理、编译、汇编和链接。

![](img/9bd24c440e3404fd993981bcfdb792d7_12.png)

![](img/9bd24c440e3404fd993981bcfdb792d7_13.png)

### 预处理
预处理是编译的第一步，主要处理源代码中以 `#` 开头的指令，例如 `#include`。预处理器会找到这些文件，并将其内容复制粘贴到源代码的相应位置。

**代码示例：**
```c
#include <stdio.h>
#include <cs50.h>
```

### 编译
编译步骤将预处理后的 C 代码转换为另一种称为汇编语言的低级语言。汇编语言更接近机器指令，但仍需进一步转换。

![](img/9bd24c440e3404fd993981bcfdb792d7_15.png)

![](img/9bd24c440e3404fd993981bcfdb792d7_16.png)

### 汇编
汇编步骤将汇编代码转换为机器可以理解的二进制代码，即 0 和 1 的序列。

### 链接
链接是最后一步，它将多个编译后的文件（例如，你编写的代码和使用的库代码）合并成一个可执行文件。

## 调试技巧

在编程过程中，调试是发现和修复错误的关键环节。以下是几种有效的调试方法。

### 使用 `printf` 调试
`printf` 是一种快速但基础的调试工具，通过在代码中插入打印语句来观察变量的值。

![](img/9bd24c440e3404fd993981bcfdb792d7_21.png)

![](img/9bd24c440e3404fd993981bcfdb792d7_23.png)

**代码示例：**
```c
printf("i is %i\n", i);
```

### 使用调试器
调试器（如 VS Code 中的 `debug50`）允许你逐行执行代码，观察变量的变化，从而更精确地定位问题。

以下是使用调试器的基本步骤：
1.  在代码中设置断点。
2.  使用 `debug50` 命令启动调试器。
3.  使用“步过”、“步入”等按钮控制执行流程。

![](img/9bd24c440e3404fd993981bcfdb792d7_25.png)

### 橡皮鸭调试法
橡皮鸭调试法是一种心理技巧，通过向一个无生命的物体（如橡皮鸭）解释你的代码逻辑，往往能帮助你发现思维中的漏洞。

## 数组

数组是一种数据结构，用于在内存中连续存储多个相同类型的值。

### 数组的声明与初始化
以下是声明和初始化数组的方法：

**代码示例：**
```c
int scores[3];
scores[0] = 72;
scores[1] = 73;
scores[2] = 33;
```

### 数组在内存中的表示
数组在内存中是连续存储的。例如，一个包含三个整数的数组会占用连续的 12 个字节（假设每个整数占 4 个字节）。

![](img/9bd24c440e3404fd993981bcfdb792d7_33.png)

![](img/9bd24c440e3404fd993981bcfdb792d7_35.png)

![](img/9bd24c440e3404fd993981bcfdb792d7_37.png)

### 使用循环处理数组
循环是处理数组元素的强大工具，可以避免重复代码。

![](img/9bd24c440e3404fd993981bcfdb792d7_39.png)

![](img/9bd24c440e3404fd993981bcfdb792d7_41.png)

**代码示例：**
```c
for (int i = 0; i < 3; i++)
{
    scores[i] = get_int("Score: ");
}
```

## 字符串与内存

![](img/9bd24c440e3404fd993981bcfdb792d7_50.png)

字符串在 C 语言中实际上是字符数组，并以空字符 `\0` 结尾。

### 字符串的本质
字符串是由字符组成的数组，最后一个字符是空字符 `\0`，用于标识字符串的结束。

![](img/9bd24c440e3404fd993981bcfdb792d7_52.png)

![](img/9bd24c440e3404fd993981bcfdb792d7_54.png)

**代码示例：**
```c
string s = "hi!";
// 等价于
char s[] = {'h', 'i', '!', '\0'};
```

![](img/9bd24c440e3404fd993981bcfdb792d7_56.png)

### 字符串长度
可以使用 `strlen` 函数获取字符串的长度，该函数位于 `string.h` 库中。

![](img/9bd24c440e3404fd993981bcfdb792d7_58.png)

**代码示例：**
```c
#include <string.h>
int length = strlen(s);
```

![](img/9bd24c440e3404fd993981bcfdb792d7_60.png)

![](img/9bd24c440e3404fd993981bcfdb792d7_61.png)

### 字符串操作
C 语言提供了许多库函数来操作字符串，例如 `toupper` 函数可以将字符转换为大写。

![](img/9bd24c440e3404fd993981bcfdb792d7_63.png)

![](img/9bd24c440e3404fd993981bcfdb792d7_65.png)

**代码示例：**
```c
#include <ctype.h>
char upper = toupper('a'); // 返回 'A'
```

## 命令行参数

命令行参数允许用户在运行程序时直接提供输入，而不是在程序运行时交互式输入。

### 使用命令行参数
`main` 函数可以接受两个参数：`argc`（参数计数）和 `argv`（参数向量，即字符串数组）。

![](img/9bd24c440e3404fd993981bcfdb792d7_71.png)

**代码示例：**
```c
int main(int argc, string argv[])
{
    if (argc == 2)
    {
        printf("Hello, %s\n", argv[1]);
    }
    else
    {
        printf("Hello, world\n");
    }
}
```

![](img/9bd24c440e3404fd993981bcfdb792d7_73.png)

### 退出状态
程序可以通过 `return` 语句返回一个退出状态码。通常，`0` 表示成功，非零值表示错误。

**代码示例：**
```c
if (argc != 2)
{
    printf("Missing command line argument\n");
    return 1;
}
return 0;
```

## 密码学简介

![](img/9bd24c440e3404fd993981bcfdb792d7_77.png)

密码学是加密和解密信息的科学与艺术，旨在确保信息传输的安全。

### 基本概念
*   **明文**：原始的可读信息。
*   **密文**：加密后的不可读信息。
*   **密钥**：用于加密和解密的秘密值。
*   **密码**：用于加密和解密的算法。

### 凯撒密码
凯撒密码是一种简单的替换密码，通过将字母在字母表中移动固定的位置（由密钥决定）来进行加密。

**加密公式：**
`密文字母 = (明文字母 + 密钥) mod 26`

**解密公式：**
`明文字母 = (密文字母 - 密钥) mod 26`

![](img/9bd24c440e3404fd993981bcfdb792d7_79.png)

**代码示例（概念性）：**
```c
// 假设 key = 1
plaintext:  H I !
ciphertext: I J !
```
