## 什么是人工智能？

![](img/f188cf513e3ffed77d8724f663f84c34_11.png)

## CS50的AI应用：橡皮鸭调试器

![](img/f188cf513e3ffed77d8724f663f84c34_19.png)

![](img/f188cf513e3ffed77d8724f663f84c34_20.png)

在CS50课程中，我们开发了一款基于AI的“橡皮鸭调试器”，旨在引导学生解决问题，而非直接提供答案。

该系统的工作原理基于提示工程。我们为AI设定了一个系统提示，指导其扮演友好助教的角色。

以下是一个简化的代码示例，展示了如何与OpenAI的API进行交互以创建聊天机器人。

```python
import os
from openai import OpenAI
from dotenv import load_dotenv

load_dotenv()

![](img/f188cf513e3ffed77d8724f663f84c34_22.png)

client = OpenAI(api_key=os.getenv("API_KEY"))

![](img/f188cf513e3ffed77d8724f663f84c34_23.png)

system_prompt = "You are a friendly and supportive teaching assistant for CS50. You are also a rubber duck."
user_prompt = input("What's your question? ")

![](img/f188cf513e3ffed77d8724f663f84c34_25.png)

chat_completion = client.chat.completions.create(
    messages=[
        {"role": "system", "content": system_prompt},
        {"role": "user", "content": user_prompt}
    ],
    model="gpt-4"
)

response_text = chat_completion.choices[0].message.content
print(response_text)
```

![](img/f188cf513e3ffed77d8724f663f84c34_27.png)

这段代码接收用户输入，结合系统提示，调用AI模型生成回应。

## 从规则到学习：AI的演进

早期AI依赖于硬编码的规则，例如在游戏中。

### 决策树与井字棋

井字棋可以通过决策树和极小化极大算法来求解，确保最优策略。

然而，对于更复杂的游戏（如国际象棋或围棋），可能的走法组合数量巨大，无法通过穷举计算找到最优解。

### 机器学习与强化学习

这引出了机器学习的概念，即让计算机从数据中自行学习。强化学习是其中一种方法，通过奖励和惩罚来塑造行为。

强化学习的核心思想是：尝试行动，获得反馈（奖励或惩罚），然后调整策略以最大化长期奖励。

![](img/f188cf513e3ffed77d8724f663f84c34_33.png)

![](img/f188cf513e3ffed77d8724f663f84c34_34.png)

为了找到更优解，有时需要在策略中引入随机性进行探索。

## 神经网络与深度学习

深度学习基于神经网络，其灵感来源于生物神经元。

![](img/f188cf513e3ffed77d8724f663f84c34_39.png)

### 基本概念

神经网络由输入层、隐藏层和输出层组成，通过调整神经元之间的连接权重来学习数据中的模式。

例如，在一个二维坐标系中，我们可以训练一个神经网络来根据点的坐标（X, Y）预测其颜色（红或蓝）。这本质上是在寻找最佳的分界线。

### 大语言模型

ChatGPT等大语言模型是特殊的神经网络，它们在大量文本数据上进行训练，以预测序列中下一个词的概率。

其核心技术包括“注意力机制”，它使模型能够衡量输入序列中不同部分之间的关系。

![](img/f188cf513e3ffed77d8724f663f84c34_41.png)

![](img/f188cf513e3ffed77d8724f663f84c34_42.png)

然而，这些模型有时会产生“幻觉”，即自信地生成不正确或虚构的信息。
