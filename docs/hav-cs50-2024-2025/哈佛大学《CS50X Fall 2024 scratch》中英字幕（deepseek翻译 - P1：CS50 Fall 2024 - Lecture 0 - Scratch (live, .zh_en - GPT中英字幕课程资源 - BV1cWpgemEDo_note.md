## 什么是计算机科学？ 🤔
计算机科学是研究信息的学科，具体来说，是研究如何表示信息以及如何处理信息。更根本地说，这门课程将教授**计算思维**，即应用计算机科学的思想来解决我们感兴趣的问题。

计算机科学的核心是**解决问题**。我们可以用一个简单的模型来理解：你有一个需要解决的问题，即**输入**；你的目标是产生一个解决方案，即**输出**。解决问题的关键在于如何将输入转化为输出。

## 信息的表示：二进制
在计算机内部，所有信息都使用**二进制**系统表示，即仅由0和1组成。这是因为计算机的物理基础（如晶体管）只有两种状态：开或关。

一个二进制位称为一个**比特**。通过组合多个比特，我们可以表示更大的数字。例如，使用3个比特，我们可以表示从0到7的8个不同的数字。

![](img/82260bc4476d8e3df68de9ec8950444e_9.png)

**公式**：`二进制数 = (位值1 * 比特1) + (位值2 * 比特2) + ...`，其中位值是2的幂次方（1, 2, 4, 8, 16...）。

![](img/82260bc4476d8e3df68de9ec8950444e_11.png)

## 从数字到字母：ASCII与Unicode
为了表示字母和符号，人们制定了标准映射。**ASCII**码使用7或8个比特，为英文字母、数字和标点符号分配了特定的数字代码。例如，大写字母A的ASCII码是65。

![](img/82260bc4476d8e3df68de9ec8950444e_13.png)

![](img/82260bc4476d8e3df68de9ec8950444e_15.png)

然而，ASCII只能表示有限的字符。为了支持全球所有语言，现代计算机使用**Unicode**标准，它可以使用16、24甚至32比特来表示字符，包括各种文字和表情符号。

## 从字母到多媒体：颜色、图像、视频与声音
计算机使用**RGB**系统表示颜色，通过红、绿、蓝三种颜色的强度值（各占一个字节）来混合出各种色彩。

图像由无数个称为**像素**的点组成，每个像素的颜色由RGB值定义。视频则是快速连续播放的一系列图像（例如每秒30帧），从而产生运动错觉。

声音可以通过数字表示，例如用数字表示音高、音量和持续时间，从而合成音乐或语音。

## 算法：解决问题的步骤
**算法**是解决问题的精确的、一步一步的指令集。关键在于“精确”。例如，在电话簿中找人，有几种不同的算法：
1.  **逐页查找**：从第一页开始，一页一页找。虽然正确，但速度慢。
2.  **隔页查找（并修正）**：一次翻两页，如果翻过了就退回一页。速度稍快，但仍不够高效。
3.  **二分查找**：从中间打开电话簿，根据人名所在位置（前一半或后一半）扔掉无关的一半，在剩下的一半中重复此过程。这种方法效率最高。

我们可以用**伪代码**来描述算法，它使用类似英语的结构，但逻辑严谨。

**伪代码示例（电话簿查找）**：
```
1. 拿起电话簿
2. 打开到电话簿中间
3. 查看页面
4. 如果人在此页
5.     打电话给人
6. 否则，如果人在书的前半部分
7.     打开到左半部分的中间
8.     回到第3行
9. 否则，如果人在书的后半部分
10.    打开到右半部分的中间
11.    回到第3行
12.否则
13.    退出
```

在伪代码中，我们看到了编程的核心构建块：
*   **函数**：执行特定动作（如“打开到中间”）。
*   **条件语句**：根据布尔表达式（是/否问题）决定执行哪条路径。
*   **循环**：重复执行某些步骤。
*   **变量**：存储数据（如“答案”）。

Scratch允许我们通过拖放代码块来编程，无需担心复杂的语法。让我们创建一个简单的程序。

**步骤1：说“Hello, World”**
1.  从“事件”类别中，拖出 `当绿旗被点击` 积木。
2.  从“外观”类别中，拖出 `说 Hello!` 积木，将其拼接在绿旗积木下方。
3.  将“Hello!”改为“Hello, World”。
4.  点击绿旗，角色就会说“Hello, World”。

![](img/82260bc4476d8e3df68de9ec8950444e_32.png)

![](img/82260bc4476d8e3df68de9ec8950444e_33.png)

**步骤2：实现交互——询问并回答**
1.  删除“说”积木。
2.  从“侦测”类别中，拖出 `询问 你叫什么名字？ 并等待` 积木，拼接在绿旗下。
3.  从“外观”类别中，拖出 `说 Hello!` 积木，拼接在询问积木下。
4.  从“运算”类别中，拖出 `连接 Apple 和 Banana` 积木。
5.  将“连接”积木拖入“说”积木的输入框中。它会自动适应。
6.  将“Apple”改为“Hello, ”。
7.  从“侦测”类别中，拖出 `回答` 积木（它是一个圆形的值），放入“连接”积木的“Banana”位置。
8.  点击绿旗，输入你的名字，角色会个性化地问候你。

在这个例子中：
*   `询问` 是一个**函数**，它有一个**输入**（问题文本）和一个**返回值**（`回答`）。
*   `连接` 是另一个函数，它接受两个输入，并输出拼接后的字符串。
*   最终，`连接` 的输出成为了 `说` 函数的输入。

**步骤3：使用循环和自定义函数**
如果我们想让猫叫三声，可以重复“播放声音”和“等待”积木。但更好的方法是使用**循环**和**自定义函数**。

1.  从“控制”类别中，拖出 `重复执行 10 次` 积木。
2.  从“声音”类别中，拖出 `播放声音 Meow 直到播放完毕` 和 `等待 1 秒` 积木，放入循环内。
3.  将重复次数改为3。
4.  点击绿旗，猫会叫三声，每次间隔一秒。

为了更好的代码组织，我们可以创建自定义函数（积木）：
1.  点击“我的积木”类别中的“制作新的积木”。
2.  将积木命名为“喵叫”，点击“确定”。
3.  在出现的 `定义 喵叫` 积木下，放入播放声音和等待的积木。
4.  现在，在“我的积木”类别中会出现一个粉色的 `喵叫` 积木。你可以在主程序中使用它来代替那两块积木，使代码更清晰、更易复用。

**步骤4：添加交互与条件判断**
我们可以让程序对用户的动作做出反应。

1.  从“控制”类别中，拖出 `重复执行` 和 `如果...那么` 积木。
2.  从“侦测”类别中，拖出 `碰到 鼠标指针？` 积木（这是一个布尔表达式），放入“如果”后面的六边形缺口处。
3.  在“如果”积木内部，放入 `播放声音 Meow 直到播放完毕`。
4.  点击绿旗，当你用鼠标指针触碰角色时，它就会喵喵叫。
