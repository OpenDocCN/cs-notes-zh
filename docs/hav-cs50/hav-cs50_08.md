## 变量

+   变量声明也得到了简化。在 C 语言中，你可能会有 `int counter = 0;` 这样的代码。在 Python 中，同样的行会写成 `counter = 0`。你不需要声明变量的类型。

+   Python 更倾向于使用 `counter += 1` 来实现加一，失去了 C 语言中 `counter++` 的能力。
