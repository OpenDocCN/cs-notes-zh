# 063：基于变分法的图像修复

## 概述

![](img/2007d66689610644159b017f2458b563_1.png)

在本节课中，我们将学习一种基于**变分法**的图像修复技术。与之前直接从偏微分方程入手的方法不同，我们将从一个**能量泛函**的公式出发，通过欧拉-拉格朗日方程将其转化为微分方程进行求解。这种方法能同时处理图像边缘的平滑延续和内部颜色的填充。

## 核心思想与模型

![](img/2007d66689610644159b017f2458b563_3.png)

上一节我们介绍了基于偏微分方程的修复方法。本节中，我们来看看基于能量最小化的变分方法。

其核心思想是：对于图像中需要修复的缺失区域，我们希望**边缘能够平滑地延续**，同时让颜色信息像水流一样自然地“流入”填充区域。这个类比并非随意，因为其背后的数学方程与描述自然界中流体运动的**传输方程**非常相似。

## 已知梯度场的修复

首先，我们从一个简化问题开始：假设已经有人告诉了我们缺失区域内每个像素点的**归一化梯度场**。

设给定的归一化梯度向量场为 **θ**，对于图像 **I**，其梯度为 **∇I**。归一化梯度定义为梯度除以其幅值，但当梯度幅值为0时需特殊处理。

![](img/2007d66689610644159b017f2458b563_5.png)

我们的目标是找到一个图像 **I**，使其在修复区域Ω内的梯度与给定的 **θ** 尽可能一致。这可以通过最小化以下能量泛函来实现：

```
E(I) = ∫_Ω |∇I / |∇I| - θ|² dΩ
```

当图像 **I** 的归一化梯度完全等于 **θ** 时，上述积分为零，达到最小。对上述能量泛函应用**欧拉-拉格朗日方程**，我们可以推导出对应的偏微分方程。在达到稳态时，方程简化为：

```
div(∇I / |∇I|) = div(θ)
```

其中，**div** 代表散度算子。求解此方程，即可得到与给定梯度场 **θ** 一致的图像 **I**。

下图展示了在已知梯度场下修复的效果：

![](img/2007d66689610644159b017f2458b563_5.png)

可以看到，虽然灰度值可能不完全准确，但基本的几何结构得到了恢复。然而在现实中，我们并没有现成的梯度场 **θ**，因此需要更完整的模型。

## 完整的变分修复模型

在实际情况中，我们既不知道要修复的图像 **I**，也不知道其梯度场 **θ**。因此，我们需要一个能同时求解两者的完整模型。

以下是完整的能量泛函：

```
E(I, θ) = ∫_Ω [ A * |∇I - θ|² + B * |∇θ|² + C * |θ|² ] dΩ
```

我们需要同时最小化这个能量泛函以求得 **I** 和 **θ**。其中包含三个部分：

1.  **数据一致性项 (`A * |∇I - θ|²`)**: 确保求解出的图像 **I** 的梯度与估计的边缘方向场 **θ** 一致。
2.  **边缘平滑项 (`B * |∇θ|²`)**: 促使边缘方向场 **θ** 在区域内平滑变化。实际上，`div(θ)` 代表图像等照度线的**曲率**。这项惩罚剧烈的曲率变化，从而实现边缘的平滑延续。
3.  **正则化项 (`C * |θ|²`)**: 这是一个技术性项，使整个问题在数学上更“适定”，保证解的良好性。参数 A, B, C 用于控制各项的权重。

求解这个模型需要同时处理两个未知量。方法是固定其中一个（如 **I**），对另一个（**θ**）应用欧拉-拉格朗日方程求其演化方程；然后交换，固定 **θ** 求 **I** 的演化方程。如此迭代求解一组耦合的偏微分方程。

## 结合结构与纹理的修复

在实际图像中，除了大的结构和均匀颜色，还包含丰富的纹理和颗粒细节。单一的修复方法难以兼顾。

更先进的策略是先将图像分解为两部分：
*   **结构层**: 包含图像的宏观轮廓和平滑颜色变化。
*   **纹理层**: 包含所有的细颗粒、噪声和重复模式。

分解技术（如 Meyer 提出的模型）可以实现这种分离。

然后，对这两部分分别采用最适合的修复方法：
1.  **结构层**: 使用本节介绍的**变分法**或之前的偏微分方程方法进行修复，以保持边缘和颜色的平滑延续。
2.  **纹理层**: 使用一种基于**复制-粘贴**的非局部方法进行修复（这将在下一个视频中详细介绍）。

最后，将修复好的结构层和纹理层重新相加，就得到了既保持结构连贯性，又恢复纹理细节的完整图像。

## 总结

本节课中我们一起学习了基于变分法的图像修复技术。我们从构建一个包含**数据一致性**、**边缘平滑**和**正则化**项的能量泛函出发，通过**欧拉-拉格朗日方程**将其转化为可求解的偏微分方程。这种方法能够优雅地同时恢复图像和其边缘方向场。最后，我们了解到将图像分解为**结构**与**纹理**两部分，并分别用针对性方法修复，是处理复杂真实图像的有效策略。在下一个视频中，我们将深入探讨用于纹理修复的复制-粘贴技术。
