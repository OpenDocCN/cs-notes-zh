# 022：22_03_07_7-非局部均值

## 📖 概述

在本节课中，我们将要学习一种名为“非局部均值”的图像去噪方法。我们将探讨局部平均的局限性，理解非局部均值的基本思想，并学习其工作原理。

## 🔍 局部平均的局限性

上一节我们介绍了局部平均操作。局部平均具有一些非常有趣的数学性质。它是一个非常简单的操作，并且正如我们之前看到的几个例子，它非常有用。但是我们也观察到，它可能会模糊图像中的信息，可能会模糊边缘。

那么，我们如何解决这个问题？我们有几种不同的方向可以探索。一个方向是基本上用不同的局部操作来替换这个局部平均，我们很快就会看到这一点。另一个方向是，我们可能会问自己，是否仅限于在局部邻域内进行平均？也许我们可以进行一种更智能的平均。

## 💡 非局部均值的基本思想

记住，平均就像热流。那么，我们是否应该在非常热的像素和非常冷的像素之间进行平均？或者我们应该尝试只在相似的像素之间进行平均，并尝试停止在非常不同的像素之间进行平均？这是一个非常有趣的概念，实际上可以用多种不同的方式、沿着不同的方向来实现。

但让我们只解释一种极其优雅的方法，部分原因在于它非常简单，它实际上被称为“非局部均值”。我们不再局限于查看局部邻域内的平均值，而是将查看整个图像中的平均值。让我们看看为什么以及如何做到这一点。

这个想法非常简单。让我们观察这张图像。我们实际上看到图像具有重复性，有时这被称为自相似性。例如，如果我们看这个局部邻域，它与这个局部邻域非常相似，也与这个局部邻域非常相似。我们还观察到，这里的这个局部邻域与这个局部邻域非常相似，甚至与这个也很相似，它们相距很远，但非常相似，这里也一样。

以前我们只考虑平均这里的像素，也许我们实际上应该平均相似的邻域。也许我们应该平均这三个，这样我们就可以用与其他像素的某种关系来替换这个像素。但我们不应该将这些与那些混合，因为局部邻域是不同的。

因此，非局部均值或非局部平均的基本思想是尝试进行平均，尝试利用平均的特性，但不将其限制在局部邻域内。我们为什么要这样做以及如何做到这一点？

## 🧠 核心原理与动机

基本思想是：如果我们假设这些是相同的事物、相同的结构，但可能带有不同的噪声，让我们看看其效果。

假设我们有一个特定的像素，称之为 **P**。这是我们观察到的。它是干净像素加上一些噪声。但噪声基本上是一个随机变量，所以它总是略有不同。因此，我们多次观察这个像素。我们观察到一次带有某种噪声。我们再次观察到它。相同的像素，但带有不同的噪声。我们可以多次观察它。假设我们观察了 **n** 次。总是同一个像素，但带有噪声的不同实现。

如果我们平均所有这些，我们可以很容易地证明（这是信号分析的基本特性），噪声的减少与平均次数的平方根成反比。所以我们总是在平均同一个像素，噪声是随机变量，基本上噪声的减少与 **√n** 成反比。我们平均得越多，减少的噪声就越多。当然，非常重要的是，我们总是平均完全相同的像素，只有噪声在变化。

这是非局部均值的动机之一，即：也许我可以找到完全相同或至少非常相似的重复结构，周围的噪声都不同，我可以通过平均来减少噪声。正因为如此，基本思想就是首先寻找相似的邻域，然后对它们进行平均。

## ⚙️ 非局部均值的工作流程

因此，如果我想替换这里的中心像素，我会查看其邻域，寻找相似的邻域，然后用这些相似邻域中心的平均值来替换该像素。

那么，我如何找到相似的邻域？关于非局部均值有大量的文献，基本上包括两部分：第一部分是如何找到相似的邻域。一旦找到它们，第二部分是如何对它们进行平均？是进行加权平均，还是计算它们之间的不同关系？但再次强调，对于这两个选项，最简单的方法是：通过比较邻域来寻找相似的邻域，比如说，计算它们之间的均方误差，并确保这个误差非常小。这是一个非常非常简单的操作。

以下是具体步骤：

1.  遍历图像，查看邻域（比如 3x3、5x5 或 7x7）。
2.  确定哪些邻域与当前邻域相似。
3.  可以包含所有相似的邻域，也可以只包含最相似的那些。
4.  一旦找到它们，进入每个邻域的中心，比如说，对中心像素值进行平均，并用这个平均值替换当前像素。

如果你想清理这里的某个像素，你会寻找与该像素周围区域相似的邻域。你遍历图像，找到一些相似的邻域，然后用所有这些相似邻域中心的平均值来替换这个像素。

再次强调，这只是一个例子。你可以用不同的方式寻找相似的邻域；一旦找到它们，你可以进行加权平均或其他函数操作，而不是简单的平均。但概念始终相同：我们不是仅仅平均邻域周围的像素来替换这个像素，而是基本上在整个图像中寻找具有某种相似性度量的相似邻域，一旦找到它们，我们就将它们组合起来，以这种方式基于我刚才解释的概念对像素进行去噪。
