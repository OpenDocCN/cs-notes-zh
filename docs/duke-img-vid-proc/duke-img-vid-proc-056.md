# 056：变分法

## 📚 概述

在本节课中，我们将学习一个在数学和图像处理领域都至关重要的概念——**变分法**。我们将从寻找函数极值的基本思想出发，扩展到寻找**泛函**的极值，并推导出求解此类问题的关键方程——**欧拉-拉格朗日方程**。最后，我们会看到如何通过迭代求解这个方程来找到最优解，为下一课中将其应用于图像去噪和增强打下基础。

## 🔍 什么是变分法？

变分法的核心思想非常简单，它是寻找函数极值方法的扩展。在常规微积分中，我们寻找使某个函数取得极值的**点**。而在变分法中，我们寻找的是使某个**泛函**取得极值的**函数**。

那么，什么是泛函呢？在图像处理的语境下，我们的未知量 `u` 可以是一幅图像。泛函 `F` 则是依赖于图像 `u` 及其导数的函数。例如，它可能是图像本身的平方、图像的梯度，或者是图像经过某种变换后的结果。

我们的目标是找到使泛函 `F(u)` 最小化的那个图像 `u`，正如我们寻找使普通函数 `f(x)` 最小化的那个点 `x` 一样。

一个函数 `u` 成为泛函最小化子的一个必要条件是满足以下方程，它被称为**欧拉-拉格朗日方程**：

```
∂F/∂u - d/dx (∂F/∂u_x) = 0
```

这正是我们在图像处理中需要求解的那类偏微分方程。

## 📏 一个直观的例子：最短路径问题

为了更好地理解，让我们看一个经典的例子。假设给定两个点，我们需要找到连接这两点的、长度最短的曲线。这里，最小化子将是一个**函数**（即曲线的方程），而不是一个点。

任何连接这两点的曲线长度可以表示为以下积分：

```
L(u) = ∫ √(1 + (u_x)^2) dx
```

其中，曲线由参数方程 `(x, u(x))` 描述，`u_x` 是 `u` 对 `x` 的导数。

我们知道，两点之间线段最短，即解应该是一条直线。让我们用变分法来验证。对于这个泛函，`F = √(1 + (u_x)^2)`。将其代入欧拉-拉格朗日方程并计算，我们得到：

```
u_xx / [√(1 + (u_x)^2)]^3 = 0
```

![](img/e743d4e067104f59f0fa2881174cd48e_1.png)

![](img/e743d4e067104f59f0fa2881174cd48e_2.png)

要使该式为零，分子必须为零，即 `u_xx = 0`。这意味着 `u_x` 是常数，因此函数 `u(x)` 是形如 `Ax + B` 的直线。系数 `A` 和 `B` 由曲线必须经过两个给定点的**边界条件**确定。这就从数学上证明了最短路径是直线。

## 🔄 从函数极值到泛函极值

为了理解欧拉-拉格朗日方程从何而来，以及它为何重要，让我们先回顾一下如何寻找普通函数的极值。

### 寻找函数极值

对于一个函数 `f(x)`，在点 `x` 处取极值的必要条件是导数 `f'(x) = 0`。我们可以通过**梯度下降法**来找到这个点：

1.  从任意点 `x_0` 开始。
2.  沿负梯度方向（即 `-f'(x)` 方向）移动一小步。
3.  重复步骤2，直到 `x` 不再变化（达到稳态）。

用数学公式描述这个动态过程就是：

```
dx/dt = -f'(x)
```

![](img/e743d4e067104f59f0fa2881174cd48e_4.png)

当系统达到稳态时，`dx/dt = 0`，这意味着 `f'(x) = 0`，即我们找到了极值点。

### 扩展到泛函极值

寻找泛函极值的思路完全平行于寻找函数极值，只是我们将“点”的扰动换成了“函数”的扰动。

1.  我们从一个候选函数 `u`（黄色曲线）开始。
2.  我们对其进行一个微小的函数扰动 `η(x)`（红色曲线），得到新函数 `ũ = u + η`。
3.  我们要求，当用 `ũ` 代替 `u` 时，泛函 `F` 关于扰动 `η` 的导数为零。这是 `u` 成为极值点的必要条件。
4.  经过一系列推导（对扰动求导并令其为零），我们最终得到了**欧拉-拉格朗日方程**。

得到这个方程后，我们可以采用与梯度下降法类似的思路：

![](img/e743d4e067104f59f0fa2881174cd48e_6.png)

![](img/e743d4e067104f59f0fa2881174cd48e_7.png)

1.  从一个初始图像（函数）`u_0` 开始。
2.  沿负的“欧拉-拉格朗日方向”移动该函数。这相当于求解一个偏微分方程：

    ```
    ∂u/∂t = -[∂F/∂u - d/dx (∂F/∂u_x)]
    ```

![](img/e743d4e067104f59f0fa2881174cd48e_9.png)

3.  不断迭代，直到函数 `u` 不再变化（达到稳态）。
4.  在稳态时，`∂u/∂t = 0`，这意味着欧拉-拉格朗日方程成立，我们找到了使泛函极值的函数。

![](img/e743d4e067104f59f0fa2881174cd48e_11.png)

## 🎯 总结

本节课我们一起学习了**变分法**的核心思想：

*   **目标不同**：变分法寻找的是使**泛函**（函数的函数）取极值的**函数**，而常规微积分寻找的是使**函数**取极值的**点**。
*   **关键方程**：函数成为泛函极值点的必要条件是满足**欧拉-拉格朗日方程**。
*   **求解方法**：可以通过类似梯度下降的迭代过程，让函数沿特定方向演化，直至达到满足欧拉-拉格朗日方程的稳态，从而找到解。
*   **广泛适用**：像水平集方法一样，变分法是一门强大的数学工具，其应用远超出图像处理范畴。图像处理领域，特别是偏微分方程方向，成功地将这些来自连续数学的技术引入并加以应用。

在下一课中，我们将看到一个具体的、对图像处理非常有用的泛函例子，并学习如何利用变分法进行图像去噪和增强。
