# 048：主动轮廓导论与IPOL、Photoshop演示

## 概述
在本节课中，我们将要学习图像与视频处理中一个非常重要的概念——主动轮廓。主动轮廓，也被称为“蛇模型”，广泛应用于商业产品中，是图像与视频分割的有力工具。我们将通过直观的演示来理解其基本思想，并观察它在实际工具中的应用。

## 主动轮廓的基本概念
主动轮廓的核心思想是，从一个或多个初始曲线开始。例如，我们可以围绕感兴趣的目标（如一朵花）绘制一条初始曲线。

主动轮廓的基本原理是设计一种速度场，驱动这条曲线向目标边界移动。随着曲线的移动，它最终会贴合到目标的轮廓上。

因此，我们需要设计一种速度，能够将任何初始轮廓推向我们感兴趣的物体边界。
