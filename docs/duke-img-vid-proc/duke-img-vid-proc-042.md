## 线段检测器简介 🧠

线段检测器是作者描述的一种技术，其核心目标是生成图像中连贯的线段或曲线，而不是仅仅依赖像梯度这样的局部边缘检测器。其基本思想在于整合局部信息，以形成对场景中重要结构的全局理解。
