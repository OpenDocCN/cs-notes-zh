# 004：04_01_04_2-课程安排与说明

## 概述

在本节课中，我们将了解本课程的整体安排与学习说明。我们将介绍课程的时间结构、每周内容、评估方式以及学习资源，帮助你为接下来的学习做好准备。

## 课程结构与安排

上一节我们通过众多实例了解了图像与视频处理的应用范围。本节中，我们来看看本课程的具体安排。

课程将持续九周，包括本周。每周内容独立成章，我将每周开启一个新主题，并在当周内完成讲解。

以下是每周课程的视频形式说明：
*   视频由多个短片段组成，而非长时间连续录制。
*   这种设计便于你随时暂停或回看，你也可以一次性观看所有视频。

## 评估与练习

了解课程结构后，我们来看看学习过程中的评估与练习环节。

每周都会设有测验。希望获得证书的同学需要完成这些测验。如果你仅对观看视频内容感兴趣，不打算参与测验，也完全欢迎。

此外，课程还提供一些推荐的计算机练习。这些练习并非强制要求，但如果你有时间，我强烈建议你尽可能多地完成，以便亲自实践我们在九周内学到的各种技术。

## 学习资源与课程内容

完成了对评估方式的介绍，接下来我们看看有哪些辅助学习资源以及课程的内容划分。

课程网站提供了大量信息，包括推荐书籍、相关网站链接以及我们将在课程中补充的额外材料。如果你想拓展视频内容之外的知识，这些资源会很有帮助。

最后需要说明的是，课程内容大致分为两部分：
*   前四到五周将涵盖基础与核心材料，这些知识至今仍然非常重要。
*   后四周左右将涉及更高级的材料，主要是当前图像处理领域的前沿课题，即你在大学和企业中能看到的前沿研究方向。

## 总结

本节课中，我们一起学习了本课程的时间安排、视频组织形式、测验与练习要求，以及丰富的辅助学习资源。我们还了解了课程将从基础内容过渡到前沿课题的整体设计。接下来，我们将在新视频中正式开始图像与视频处理相关知识的学习。
