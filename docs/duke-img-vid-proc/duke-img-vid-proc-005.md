# 005：05_01_05_3-无处不在的图像 📸

在本节课中，我们将要学习图像在现实世界中的广泛应用。我们将看到，图像不仅限于我们肉眼可见的照片，而是涵盖了从X射线到卫星图像，从显微镜成像到计算机生成图形的广阔光谱范围。理解图像的多样性是深入学习图像处理的基础。

在前两节课中，我们看到了许多图像的例子。在我身后，我写了“图像无处不在”。我想展示一些额外的例子，只是为了说明无论你感兴趣或从事哪个领域，图像都是其中的一部分。

为了帮助理解接下来我将使用的一些幻灯片，我先说明一下。我将使用这张美丽的图像，它基本上是美国航天器在月球外拍摄的第一张照片。在接下来的幻灯片中，你会看到一些带有书籍封面的内容，那本书是我们为这门课程在网站上推荐的书籍之一，通常还包括该图的说明文字。因此，课程最初几周的许多材料、图像或幻灯片都取自这本书。如果你有这本书或能通过图书馆获取，这可以帮助你查阅书中的图像和周边材料，而不仅仅依赖于我的视频讲座。

现在，让我们继续看不同的图像示例。

我想说明的一个重要观点是，我们最熟悉的是肉眼可见的图像，但这实际上只是整个光谱中非常非常小的一部分。图像实际上可以在更广阔的光谱范围内被捕捉。正如你在这里看到的，我们用相机拍摄的图像只是整个光谱中非常小的一部分。

*   **自然图像**：当然，还有我们习惯看到的自然图像，基本上就是我们用相机拍摄，可以用肉眼观看的图像。这些图像同样用于无数应用，再次包括印刷电路板应用、各种制造过程的检测等。当然，还有我们自己的图像，比如一张美元钞票的图片，或者一张汽车牌照的图片。所有这些都使用相同类型的技术拍摄，但可能用于非常不同的应用。

再次强调，理解我们能够获取的图像类型不仅存在巨大差异，这一点非常重要。有时，我们使用多种模态来获取同一场景的图像，从而获得不同的信息。不同的模态会为我们提供关于同一场景的不同类型信息。换句话说，我们观察的是同一个场景，但使用不同的模态或不同的相机。
