图像与视频处理：P64：智能剪切与粘贴

在本节课中，我们将要学习一种新的图像修复技术——智能剪切与粘贴。我们将首先介绍其基本概念，然后探讨如何将其与之前视频中讨论的变分公式相结合，以获得出色的修复效果。

上一节我们介绍了基于偏微分方程和变分公式的图像修复方法。本节中我们来看看另一种强大的技术。

为什么需要另一种图像修复技术？原因有很多。其中之一是，偏微分方程和变分公式是修复平滑区域的优秀工具，但在修复纹理区域时效果不佳。我们需要将纹理自然地扩展到待修复区域内部，这正是智能剪切与粘贴技术大显身手的地方。

让我们放大这个区域来解释其原理。

我们需要修复这个内部区域，可以看到这里存在我们希望延续的纹理模式。虽然可以编写偏微分方程来将纹理流向这个区域，但要编写一个能处理多方向复杂纹理的方程则困难得多。

但我们有另一种技术。还记得我们讨论过的用于图像去噪的非局部均值方法吗？我们将利用类似的思路，但这次是用于图像修复。

以下是基本思路。假设我们在这里取一个图像块。这个图像块很特殊，它包含一些已知信息和一些未知信息。与非局部均值方法类似，我可以在图像中寻找相似的图像块，但这次我仅在已知像素区域进行比较。例如，我取一个与目标块大小相同的块，但只比较它们的已知区域部分。

我会不断在图像中移动搜索。比如，我可能在这里找到一个图像块。同样，我只比较它们的已知区域。我不断寻找，直到找到一个或多个在已知区域上基本相等或非常相似的图像块。假设我在这里找到了一个。

那么，如何填充未知区域呢？我基本上复制这个匹配图像块的未知部分到目标块的未知区域。我可以复制整个区域，也可以只复制几个边界像素；可以只从一个匹配块复制，也可以找到多个相似块并复制它们的平均值或其他函数组合。但核心概念与非局部均值相同：寻找相似图像块（基于已知区域的相似性），然后将匹配块的未知部分复制过来。

完成这一步后，这个像素点就变成了已知点。我们可以继续向内传播：现在这个新填充的区域变成了已知区域，我们可以继续处理其相邻的未知像素，如此反复，直到填充完所有区域。基本思路是，我们从那些被大量已知像素包围的未知像素开始填充，然后逐步处理其他像素，直到完成整个修复。

现在，我们可以将刚才描述的过程与变分公式结合起来。我们可以用变分公式来表述它，并且可以添加额外的约束条件。例如，我们可以要求：如果一个图像块在这里找到了一个相似的“伙伴”，那么我希望它旁边的图像块也能在“伙伴”的旁边找到相似的图像块。这有助于加速搜索过程，并能确保纹理被平滑地复制过来。

这可以通过变分公式来实现。例如，我们可以引入平移向量的概念：这个图像块是从这里平移过来的，那个图像块是从那里平移过来的。我们可以强制要求这些平移向量平滑变化。具体做法是，对平移向量的坐标求导，并惩罚导数过大的情况，这意味着你在图像中跳跃式地寻找匹配块。所有这些概念都可以整合到一个统一的、非常丰富的变分公式中。

我们可以将之前视频中看到的平滑延续的偏微分方程/变分公式，与智能剪切与粘贴技术，全部整合到一个大型变分公式中。这样就能获得非常出色的结果。
