## 概述

在继续深入学习稀疏建模之前，我们先对压缩感知做一个简要说明。压缩感知运用了我们已学习的一些相同概念，但其目标并非对信号进行建模或表示，而是旨在提供一种全新的采样或感知范式。

## 压缩感知的基本原理

![](img/d27f79eb423fdfb5ed8c390762fe4d58_5.png)

![](img/d27f79eb423fdfb5ed8c390762fe4d58_6.png)

上一节我们介绍了稀疏建模，本节中我们来看看压缩感知如何利用类似的数学框架实现不同的目标。

![](img/d27f79eb423fdfb5ed8c390762fe4d58_7.png)

我们有一个信号 **X**，并假设该信号可以表示为字典与一个稀疏向量的乘积，即 **X = Dα**。这是压缩感知中我们希望感知的信号。

在压缩感知中，我们不直接感知全部的 **n** 维信号点。相反，我们用一个行数少于 **n** 的矩阵 **Q**（称为“感知矩阵”或“胖矩阵”）与信号 **X** 相乘，从而感知更少的数据点。这正是“压缩”感知名称的由来。

以下是其数学表达过程：

1.  原始信号模型：**X = Dα**
2.  感知过程：两边同时乘以感知矩阵 **Q**，得到 **QX = QDα**
3.  定义新字典：令 **D̃ = QD**，感知信号为 **X̃ = QX**
4.  最终方程：**X̃ = D̃α**

我们最终感知到的是维度更低的信号 **X̃**。压缩感知的目标是，在已知感知矩阵 **Q** 和字典 **D** 的情况下，从 **X̃** 中恢复出稀疏向量 **α**。

## 压缩感知与稀疏建模的关系与区别

从数学形式上看，压缩感知与稀疏建模非常相似：我们都有一个信号、一个字典，并且需要重构稀疏系数 **α**。

然而，压缩感知的核心是解决一个根本性问题：为这种恢复的可能性提供理论条件。这些条件涉及感知矩阵 **Q** 的性质、稀疏表示所用字典 **D** 的结构，以及稀疏向量 **α** 的稀疏程度。压缩感知领域提供了大量优美的数学理论和恢复条件。

最重要的是，我们必须明确区分两者的目标：
*   **稀疏建模** 是一种**信号模型**，旨在用稀疏的方式表示信号。
*   **压缩感知** 是关于设计新的**感知协议**。它主要研究如何为稀疏信号（即能用字典稀疏表示的信号）设计全新的采样或感知理论。

简而言之，如果说经典的奈奎斯特采样理论是针对带限信号的，那么压缩感知试图建立的就是针对稀疏信号的采样理论。

## 总结

本节课中我们一起学习了压缩感知的基本概念。我们了解到，压缩感知利用稀疏表示的思想，但通过引入感知矩阵来减少实际采集的数据量，从而构建了一种新的信号采集范式。它与稀疏建模共享数学工具，但追求不同的应用目标。理解这一区别对于掌握这两个领域至关重要。
