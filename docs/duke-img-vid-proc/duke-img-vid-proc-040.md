# 040：40_05_02_2-边缘与区域

## 概述
在本节课中，我们将要学习图像与视频分割中两种关键的信息类型：边缘和区域。我们将通过具体的图像示例，探讨如何利用这两种信息来分离图像中的目标物体。

## 边缘信息
上一节我们介绍了图像分割的基本概念，本节中我们来看看如何利用边缘信息进行分割。

![](img/80f18dd33766636d269a46ab19f5a5e5_1.png)

观察一张花朵的图像。假设我们的目标是将这朵花从背景中分割出来。在某些区域，存在非常清晰的边界。我们之前讨论过边缘和梯度。

梯度可以用以下公式表示：

`∇I = [∂I/∂x, ∂I/∂y]`

它本质上就是图像在x和y方向上的导数向量。几周前我们讨论过如何实现它。例如，在x方向的导数，我们可以使用一个简单的滤波器 `[+1, -1]` 来计算。

当我们计算这个导数向量的幅值时，在花朵与背景对比明显的区域，梯度值会很高。这能帮助我们分离花朵与背景。在图像分割算法中，梯度信息将经常被使用。

然而，在花朵内部也可能存在梯度，但我们不希望这些内部梯度成为分割的一部分。我们只希望利用花朵与背景之间的大差异，而不是花朵内部的差异。这增加了分割的难度，但也说明，如果我们把花朵内部视为均匀区域作为分割约束，将难以成功，因为其内部变化很大。因此，边缘对于分割此类目标将非常重要。

## 区域信息
让我们看一张略有不同的图像。

现在观察这张图像。我们清楚地看到一个球体。然而，这里基本上没有边缘。我们能看到球，但看不到任何明显的边缘，却能理解其形状。因此，在这种情况下，我们将无法使用边缘信息。

如果我们在这个区域的任何像素点计算梯度，梯度值将为0或非常低，这无助于我们找到边界或感兴趣的目标。

另一方面，我们可以利用形状和区域信息。我们可以尝试理解这是一个相对均匀的区域。实际上，我们可以利用这一事实来帮助我们获得分割结果。

为了进一步说明边缘信息在此案例中无效，让我们看看这张图像的边缘检测结果。

这里我取了之前彩色图像的绿色通道，并使用Matlab计算了边缘。我们看到一些杂散的边缘，这些我们可以通过预处理（如去噪）来消除。但在球体所在的区域，几乎没有边缘。这里只有非常微弱的边界。

因此，为了能将这个球体从背景中分割出来，我们需要其他类型的信息。这个简单的例子表明，仅有边缘信息是不够的。

## 总结
本节课中我们一起学习了图像分割中边缘与区域信息的作用。我们看到，对于边界清晰的物体（如花朵），梯度信息非常有效；而对于缺乏明显边界的均匀区域（如球体），则需要依赖形状和区域一致性等信息。
