图像与视频处理：P37：37_04_08_8-演示：维纳滤波与盒式滤波

在本节课中，我们将通过Matlab演示维纳滤波的实际应用，并将其与简单的盒式滤波（局部平均）进行对比，以直观展示维纳滤波在图像去噪中的优越性能。

---

### 概述

我们将加载一张彩色图像，将其转换为黑白图像，并添加高斯噪声以模拟受损情况。随后，我们将分别应用维纳滤波和盒式滤波（局部平均）来处理这张噪声图像，并比较两者的恢复效果。

---

### 加载与预处理图像

首先，我们加载一张之前使用过的土星图像。这是一张彩色图像，因此需要将其转换为黑白图像。转换操作与之前演示中的步骤相同。

```matlab
% 示例代码：加载并转换图像为灰度图
originalImage = imread('saturn.jpg');
grayImage = rgb2gray(originalImage);
```

### 添加噪声

接下来，我们向这张黑白图像添加高斯噪声。我们之前在学习不同类型的噪声时已经见过此操作。此处，我们需要指定噪声的方差。

```matlab
% 示例代码：添加高斯噪声
noiseVariance = 0.01; % 噪声方差
noisyImage = imnoise(grayImage, 'gaussian', 0, noiseVariance);
```

### 应用维纳滤波

现在，我们应用维纳滤波来处理这张刚刚生成的噪声图像。Matlab中有内置函数可以直接执行此操作。

```matlab
% 示例代码：应用维纳滤波
wienerFilteredImage = wiener2(noisyImage, [5 5]); % 使用5x5窗口
```

在展示维纳滤波的结果之前，我们想将其与另一种滤波方法进行对比。

![](img/cff56f3296b0f76d335fb752108b239f_1.png)

### 应用盒式滤波（局部平均）

![](img/cff56f3296b0f76d335fb752108b239f_3.png)

为了进行对比，我们创建一个半径为10的圆形结构元素，并用它来滤波噪声图像。此操作本质上是计算局部平均值，窗口半径为10。关于边界条件的处理细节（如代码中的相关参数）在此不过多赘述，它主要指示Matlab如何处理图像边缘。

```matlab
% 示例代码：创建圆形结构元素并进行局部平均滤波
diskRadius = 10;
h = fspecial('disk', diskRadius);
localAveragedImage = imfilter(noisyImage, h, 'replicate');
```

至此，我们得到了四张图像：原始图像、噪声图像、局部平均滤波后的图像以及维纳滤波后的图像。

### 结果对比与分析

现在，让我们来查看并对比这些结果。图像将并排显示以便比较。

![](img/cff56f3296b0f76d335fb752108b239f_1.png)

通过对比，我们可以清晰地看到维纳滤波的工作效果有多么出色。

![](img/cff56f3296b0f76d335fb752108b239f_3.png)

**第一张是原始图像**，我们之前已经见过。
**第二张是噪声图像**，图像质量因噪声而严重下降。
**第三张是维纳滤波的结果**，它成功地从噪声图像中恢复出了大量原始图像的细节。
**第四张是局部平均（盒式滤波）的结果**。

观察可知，维纳滤波能够从噪声图像中有效地恢复图像，其效果远优于局部平均法。将图像放大后仔细观察，差异更为明显：维纳滤波得到的图像明显比局部平均法得到的图像**更清晰、更锐利**。

维纳滤波之所以能取得如此出色的效果，是因为它利用了图像的统计信息。在本例中，维纳滤波算法自动估计了所需的参数。这对应了我们上一节课末尾所介绍的最简单的维纳滤波实现方式。即便如此，它依然在图像恢复任务中表现卓越。

让我们再次回顾：这是原始图像，这是噪声图像。维纳滤波算法从这张噪声图像出发，并没有产生模糊的结果，而是生成了一张非常清晰的图像，因此它完成了非常出色的工作。其效果明显优于会导致图像模糊的局部平均法。
