# 054：曲线演化 🌀

在本节课中，我们将学习**曲线演化**的核心概念。这是理解**活动轮廓**技术的关键数学基础。我们将探讨曲线如何随时间变化，以及不同的演化规则如何影响曲线的形状，最终将其应用于图像分割。

## 曲线演化简介

上一节我们回顾了微分几何的基础知识，本节中我们来看看**曲线演化**。其基本思想非常简单：我们拥有一条曲线，曲线上的每一点都定义了一个速度，这个速度决定了该点如何移动。

![](img/7a5291ba0a49b734eb32fd59a2c362d2_1.png)

这可以用一个偏微分方程来描述。如果我们用显式形式书写，方程表示曲线的**时间导数**等于某个速度函数。曲线由参数 `P` 参数化，其随时间的变化率由该点的速度决定，且速度本身也可能随时间变化。

![](img/7a5291ba0a49b734eb32fd59a2c362d2_3.png)

用更简洁的形式表示，即：
```
∂C/∂t = V
```
其中，`C` 是曲线，`t` 是时间，`V` 是速度场。

在某个初始时刻，我们有一条曲线。稍后，根据每个点的速度，曲线可能会变形为不同的形状。这种曲线的变形，即形状边界的运动，在图像和视频处理中能带来非常有趣的应用。

## 速度分解：法向与切向

![](img/7a5291ba0a49b734eb32fd59a2c362d2_5.png)

我们首先探讨曲线演化速度的一个重要性质。曲线上每一点的速度 `V` 都可以分解为两个分量：**法向分量**（垂直于曲线）和**切向分量**（沿着曲线切线方向）。

![](img/7a5291ba0a49b734eb32fd59a2c362d2_7.png)

以下是关于速度分解的关键点：
*   切向分量不改变曲线的形状，只影响曲线上点的参数化速度。
*   因此，从几何形状变化的角度看，只有法向分量是重要的。
*   通常，我们只考虑法向的演化，方程简化为：`∂C/∂t = V_N * N`，其中 `V_N` 是法向速度，`N` 是单位法向量。
*   如果需要，可以任意添加切向速度，但这不会改变曲线的几何形态。

例如，如果速度纯粹是切向的（`V = 函数 * T`），那么曲线形状完全不变，只是点沿着曲线“滑动”。

## 曲线演化实例

现在，让我们看几个具体的曲线演化例子，它们具有不同的法向速度 `V_N`。

### 1. 曲率流（欧几里得）

第一种重要的演化是**曲率流**。其法向变形速度与曲线在该点的**曲率** `κ` 成正比。

其方程可以写为：
```
∂C/∂t = κ * N
```
或者，等价于关于弧长参数 `s` 的二阶导数：`∂C/∂t = ∂²C/∂s²`。熟悉微分方程的同学会发现，这类似于**热流方程**的几何版本。

曲率流具有以下迷人特性：
*   **欧几里得不变性**：对曲线的旋转和平移变换保持不变。
*   **正则化（平滑）作用**：任何简单（不自交）的闭合曲线，在曲率流作用下，都会逐渐变得平滑，最终变成一个凸形。
*   **收敛为圆**：如果继续演化，凸形曲线会不断收缩，曲率趋于均匀，最终变成一个圆点（半径为0的圆）。若保持面积不变进行缩放，则会收敛为一个完美的圆。

### 2. 仿射曲率流

第二种演化是**仿射曲率流**。它在仿射变换（如倾斜、拉伸）下具有不变性，适用于相机视角变化的场景。

其方程涉及仿射曲率 `μ`（我们之前讨论过）：
```
∂C/∂t = μ^(1/3) * N
```
从几何角度看，它与基于仿射弧长的二阶导数投影到法向的结果等价。

仿射曲率流的行为与欧几里得曲率流类似：
*   **仿射不变性**：对仿射变换保持不变。
*   **正则化作用**：任何曲线都会演化成一个凸形。
*   **收敛为椭圆**：继续演化，曲线会收缩并趋于一个具有恒定仿射曲率的形状——椭圆。在仿射几何中，所有椭圆都是等价的（因为圆经仿射变换可得椭圆）。

### 3. 常速流（波传播）

第三种是**常速流**，这是最简单的演化形式。曲线上每一点都以恒定速度沿法向移动。

其方程为：
```
∂C/∂t = ±1 * N
```
正号表示向外扩张，负号表示向内收缩。

常速流具有独特的物理意义和性质：
*   **创建平行曲线（偏移）**：演化出的新曲线与原曲线处处等距。
*   **解释物理现象**：它对应于物理学中的**惠更斯原理**，描述了波前在没有障碍物时的传播方式。
*   **改变拓扑结构**：与曲率流不同，常速流可以改变曲线的拓扑结构。例如，一条曲线可以分裂成多条。
*   **产生奇点（激波）**：当曲线向内收缩时，尖锐的凸起部分会相遇、碰撞，形成角点或“激波”。这是因为没有曲率项来平滑这些高曲率区域。

## 曲线演化中的几何量变化

了解曲线在演化过程中，其长度、面积和曲率等几何量如何变化非常有用。以下是这些变化率的一般公式（对于闭合曲线）：

*   **长度变化**：`dL/dt = -∫ V * κ ds`
*   **面积变化**：`dA/dt = -∫ V ds`
*   **曲率变化**（反应-扩散方程）：`∂κ/∂t = V * ∂²κ/∂s² + V * κ²`

我们可以将这些公式应用于具体的演化：

![](img/7a5291ba0a49b734eb32fd59a2c362d2_9.png)

**对于常速流 (V=1)**：
*   长度以恒定速率减少：`dL/dt = -2π`，因此 `L(t) = L0 - 2πt`。
*   面积变化与长度成正比：`dA/dt = -L(t)`。
*   曲率会爆发（产生激波）：`κ(t) = κ0 / (1 - κ0*t)`，当 `t → 1/κ0` 时，曲率趋于无穷。

**对于欧几里得曲率流 (V=κ)**：
*   面积以恒定速率减少：`dA/dt = -2π`，因此 `A(t) = A0 - 2πt`。
*   曲线完全收缩为一点的时间是：`t_collapse = A0 / (2π)`。
*   整个演化过程保持平滑，不会产生激波。

## 与活动轮廓的联系

最后，我们将曲线演化与**活动轮廓**（Active Contours）联系起来。活动轮廓本质上就是一种受图像信息引导的曲线演化过程。

一个典型的活动轮廓演化方程如下：
```
∂C/∂t = g(I) * κ * N - (∇g(I) · N) * N
```
其中：
*   `g(I)` 是依赖于图像 `I` 的函数，例如 `g(I) = 1 / (1 + |∇I|)`。它在图像梯度强（边缘）的地方值小，目的是让曲线在物体边界处停止演化。
*   `κ * N` 是曲率流项，促使曲线平滑并收缩。
*   `-(∇g · N) * N` 是图像势能项，将曲线拉向图像边缘。

![](img/7a5291ba0a49b734eb32fd59a2c362d2_11.png)

因此，活动轮廓可以理解为：**一个由曲率流平滑驱动，但被图像边缘函数“刹停”的曲线演化过程**。我们也可以加入常速流项来加速初始演化，但需注意可能引入奇点。

![](img/7a5291ba0a49b734eb32fd59a2c362d2_13.png)

这个概念可以推广到三维曲面演化，用于分割体积数据中的物体。

## 总结

![](img/7a5291ba0a49b734eb32fd59a2c362d2_15.png)

本节课中我们一起学习了**曲线演化**的核心框架。我们首先定义了曲线演化的基本方程，并理解了只有法向速度改变形状。接着，我们探讨了三种重要的演化实例：**曲率流**（平滑并收敛于圆/椭圆）、**仿射曲率流**（具有仿射不变性）和**常速流**（产生偏移和激波）。通过分析长度、面积和曲率的变化公式，我们量化了这些演化的行为。最后，我们揭示了**活动轮廓技术**的本质——它是一种由图像信息调制的曲线演化，是连接微分几何理论与实际图像分割应用的桥梁。
