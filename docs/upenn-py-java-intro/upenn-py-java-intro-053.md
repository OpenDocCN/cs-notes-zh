# 053：使用range函数的for循环 🚀

在本节课中，我们将要学习如何使用Python中的`range`函数来创建`for`循环。`range`函数可以生成一个数字序列，它常被用来替代列表，以便重复执行某段代码特定次数。

## 概述

上一节我们介绍了使用列表进行`for`循环迭代。本节中我们来看看如何使用`range`函数来实现类似的功能。`range`函数的基本格式如下：

**`range(start, stop, step)`**

![](img/9c0584e28c1b62533fdc6e88916d3101_1.png)

其中：
*   `start` 是序列的起始值（可选，默认为0）。
*   `stop` 是序列的结束值（**不包含**该值本身）。
*   `step` 是步长，即每次递增的值（可选，默认为1）。

需要注意的是，`range`函数的参数必须是整数，不能是浮点数。

## 基础用法示例

以下是`range`函数在不同参数下的几种基础用法。

### 示例1：默认起始值

当只提供一个参数时，它被视为`stop`值。循环将从0开始，到`stop-1`结束。

```python
for x in range(10):
    print(x)
```
这段代码会打印数字0到9。

### 示例2：指定起始值

![](img/9c0584e28c1b62533fdc6e88916d3101_3.png)

我们可以同时指定`start`和`stop`值。

![](img/9c0584e28c1b62533fdc6e88916d3101_5.png)

```python
for x in range(0, 10):
    print(x)
```
这段代码同样会打印数字0到9。

### 示例3：生成特定范围的序列

通过调整`start`和`stop`值，我们可以生成任意整数区间。

![](img/9c0584e28c1b62533fdc6e88916d3101_7.png)

```python
for x in range(1, 7):
    print(x)
```
这段代码会打印数字1到6。

![](img/9c0584e28c1b62533fdc6e88916d3101_9.png)

### 示例4：使用步长（step）

`step`参数允许我们控制序列中数字的间隔。

```python
for x in range(0, 30, 7):
    print(x)
```
这段代码会打印：0, 7, 14, 21, 28。它以7为步长，从0递增到30（不包含30）。

![](img/9c0584e28c1b62533fdc6e88916d3101_11.png)

![](img/9c0584e28c1b62533fdc6e88916d3101_13.png)

### 示例5：反向迭代

通过设置负的`step`值，我们可以实现反向迭代。

```python
for x in range(5, -1, -1):
    print(x)
```
这段代码会倒序打印数字5到0：5, 4, 3, 2, 1, 0。

## 实际应用：筛选奇数

现在，让我们结合条件判断，利用`range`函数解决一个实际问题：找出1到1200之间的所有奇数。

![](img/9c0584e28c1b62533fdc6e88916d3101_15.png)

![](img/9c0584e28c1b62533fdc6e88916d3101_16.png)

以下是实现此功能的步骤：
1.  创建一个空列表`odd_numbers`用于存储结果。
2.  使用`range(1, 1201)`生成1到1200的数字序列。
3.  在循环中，使用取模运算符`%`判断每个数字是否为奇数（即`number % 2 != 0`）。
4.  如果是奇数，则使用`append()`方法将其添加到列表中。
5.  循环结束后，打印整个列表。

```python
odd_numbers = []
for number in range(1, 1201):
    if number % 2 != 0:
        odd_numbers.append(number)
print(odd_numbers)
```
运行这段代码，你将得到一个包含1到1200之间所有奇数的列表。
