Python is an interpreted programming language， which means it does not need to be compiled or converted from one language to another。



![](img/1cf653be010ce470329292208bced848_2.png)

In contrast， Java is a compiled programming language。 Python is interpreted by a Python interpreter。

 This means that sometimes it's difficult to debug your Python programs。



![](img/1cf653be010ce470329292208bced848_4.png)

![](img/1cf653be010ce470329292208bced848_5.png)
