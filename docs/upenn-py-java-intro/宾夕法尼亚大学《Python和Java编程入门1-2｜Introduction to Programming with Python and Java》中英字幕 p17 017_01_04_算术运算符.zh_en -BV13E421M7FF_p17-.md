Arithmetic operators are used to perform mathematical operations。 The plus sign indicates addition。

 and the minus sign indicates subtraction。The asterisk or star is used for multiplication while the forward slash is used for division。

The double forward slash is used for integer division。

 which divides and returns the largest whole number， discarding the fractional result。For example。

3 integer divided by2 equals 1。The double asterisk or star is used for exponentiation or calculating one number to the power of another。

For example，2 to the power of three equals 8。The percentage sign indicates the modulus operation。

 which divides and returns the remainder。 For example，7 modulus or mod5 equals 2。
