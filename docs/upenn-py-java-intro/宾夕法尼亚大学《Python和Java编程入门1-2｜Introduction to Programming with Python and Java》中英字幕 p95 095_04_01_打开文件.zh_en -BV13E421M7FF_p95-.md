When opening a file in Python， you essentially do three things， open the file， use the file。

 meaning read， write or append to the file， and close the file。😡。



![](img/896a1c6571fa2be1debe0216f6d9b8a8_2.png)

![](img/896a1c6571fa2be1debe0216f6d9b8a8_3.png)

To open a file， use Python's built in open function。

 where the path to file is a string specifying the file， and the mode is the type of file access。



![](img/896a1c6571fa2be1debe0216f6d9b8a8_5.png)

![](img/896a1c6571fa2be1debe0216f6d9b8a8_6.png)

Path to file can be just the name of a file if it's in the same directory as the program。

 or an absolute or relative path to the file。

![](img/896a1c6571fa2be1debe0216f6d9b8a8_8.png)

![](img/896a1c6571fa2be1debe0216f6d9b8a8_9.png)

To read or write to a file， the mode can be one of the following R to indicate you just want to read the file。



![](img/896a1c6571fa2be1debe0216f6d9b8a8_11.png)

W to indicate you want to write to the file。

![](img/896a1c6571fa2be1debe0216f6d9b8a8_13.png)

A to indicate that you want to append to the end of an already existing file and R plus to indicate you want to read and write to the file at the same time。



![](img/896a1c6571fa2be1debe0216f6d9b8a8_15.png)
