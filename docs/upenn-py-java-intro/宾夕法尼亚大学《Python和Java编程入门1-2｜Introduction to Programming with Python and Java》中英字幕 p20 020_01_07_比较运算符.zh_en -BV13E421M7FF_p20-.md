Comparison operators compare values and determine their relationship。

 Use the double equal sign to test if two values are equal to each other and use the exclamation equal sign to test if two values are not equal。



![](img/49b605e40cf3d567f911951e268e4c51_2.png)

![](img/49b605e40cf3d567f911951e268e4c51_3.png)

Use the less than sign to test if one value is less than another。



![](img/49b605e40cf3d567f911951e268e4c51_5.png)

And use the greater than sign to test if one value is greater than another。



![](img/49b605e40cf3d567f911951e268e4c51_7.png)

The less than equal sign tests if one value is less than or equal to another， and likewise。

 the greater than equal sign tests if one value is greater than or equal to another。
