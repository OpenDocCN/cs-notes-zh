# Python和Java编程入门：1-2：变量赋值 📝

在本节课中，我们将要学习编程中的一个核心概念——变量。我们将了解什么是变量，如何为变量赋值，以及Python语言中变量的一些重要特性。

![](img/85419ed2fa4cd74f3049b967827b92ec_0.png)

## 什么是变量？ 🔍

变量是值的符号名称或引用。当你为变量赋值时，变量就获得了该值的类型。你可以使用变量来存储各种数据。

## 变量的命名规则 📛

以下是关于变量命名的一些关键规则：

![](img/85419ed2fa4cd74f3049b967827b92ec_2.png)

![](img/85419ed2fa4cd74f3049b967827b92ec_3.png)

*   变量名可以包含字母、数字和下划线。
*   变量名不能以数字开头。
*   变量名区分大小写。

## Python的动态类型特性 🔄

Python是一种动态类型语言。这意味着如果你在代码的后续部分为同一个变量赋予一个不同的值，该变量会更新为新的值及其类型。

例如，你可以先写 `x = 5`，然后稍后写 `x = “hello”`。变量 `x` 的类型会从整数变为字符串。

## 变量的运算与更新 ➕➖

![](img/85419ed2fa4cd74f3049b967827b92ec_5.png)

![](img/85419ed2fa4cd74f3049b967827b92ec_6.png)

变量可以参与运算，并且其值可以被更新。

![](img/85419ed2fa4cd74f3049b967827b92ec_7.png)

上一节我们介绍了变量的基本概念，本节中我们来看看如何对变量进行运算。

这里我们将 `x` 增加1，其写法等同于 `x = x + 1`。

![](img/85419ed2fa4cd74f3049b967827b92ec_9.png)

```python
x += 1
```

![](img/85419ed2fa4cd74f3049b967827b92ec_10.png)

同样地，我们将 `y` 减少1，其写法等同于 `y = y - 1`。

```python
y -= 1
```

![](img/85419ed2fa4cd74f3049b967827b92ec_12.png)

## 变量类型的改变 🧩

正如之前提到的，在Python中，变量的类型可以改变。

例如，将 `x` 设置为字符串 `”5″`，会将 `x` 的类型从整数更改为字符串。

![](img/85419ed2fa4cd74f3049b967827b92ec_14.png)

```python
x = “5” # 此时x的类型是字符串（str），而不是整数（int）
```
