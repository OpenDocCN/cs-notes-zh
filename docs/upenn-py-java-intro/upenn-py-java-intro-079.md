# 079：更多列表操作 📝

在本节课中，我们将学习Python中关于列表的两种重要操作：列表的加法与乘法。这些操作能帮助我们高效地组合或重复列表中的元素。

![](img/c59a77cf29ed68ca38fa40d378647c3b_0.png)

上一节我们介绍了列表的基本索引和切片操作，本节中我们来看看如何对列表进行拼接和重复。

## 列表的加法 ➕

列表的加法操作可以将两个列表连接起来，生成一个包含两个列表所有元素的新列表。新列表的元素顺序是第一个列表的元素在前，第二个列表的元素在后。

![](img/c59a77cf29ed68ca38fa40d378647c3b_2.png)

以下是列表加法的示例：

![](img/c59a77cf29ed68ca38fa40d378647c3b_3.png)

```python
Ls1 = [2, 3, 4]
Ls2 = [7, 8, 9]
Ls3 = Ls1 + Ls2
print(Ls3)  # 输出: [2, 3, 4, 7, 8, 9]
```

在这个例子中，`Ls3` 是一个新列表，它包含了 `Ls1` 的所有元素（2, 3, 4），后面紧跟着 `Ls2` 的所有元素（7, 8, 9）。

## 列表的乘法 ✖️

列表的乘法操作可以将一个列表重复指定的次数，从而生成一个包含多个重复序列的新列表。

![](img/c59a77cf29ed68ca38fa40d378647c3b_5.png)

![](img/c59a77cf29ed68ca38fa40d378647c3b_6.png)

以下是列表乘法的示例：

![](img/c59a77cf29ed68ca38fa40d378647c3b_7.png)

```python
Ls3 = [2, 3, 4, 7, 8, 9]
Ls4 = Ls3 * 3
print(Ls4)  # 输出: [2, 3, 4, 7, 8, 9, 2, 3, 4, 7, 8, 9, 2, 3, 4, 7, 8, 9]
```

![](img/c59a77cf29ed68ca38fa40d378647c3b_8.png)

在这个例子中，`Ls4` 是一个新列表，它由 `Ls3` 的内容重复三次组成。

![](img/c59a77cf29ed68ca38fa40d378647c3b_9.png)

本节课中我们一起学习了Python列表的两种便捷操作：使用 `+` 运算符进行列表拼接，以及使用 `*` 运算符进行列表重复。这些操作是构建和操作数据序列的基础工具。
