在本节课中，我们将要学习如何创建Python列表，以及如何访问、修改和检查列表中的元素。列表是Python中一种非常基础且强大的数据结构，用于存储一系列有序的元素。

![](img/f82b151e05bdd30f1c78ed5a3c4845ac_2.png)

## 创建与查看列表

首先，我们可以创建一个包含若干随机元素的列表，并将其存储在一个变量中。

![](img/f82b151e05bdd30f1c78ed5a3c4845ac_4.png)

```python
list1 = [1, 2, 3, 4]
```

![](img/f82b151e05bdd30f1c78ed5a3c4845ac_5.png)

我们可以打印这个列表来查看其内容。同时，我们可以使用Python内置的 `len()` 函数来获取并打印列表的长度。

![](img/f82b151e05bdd30f1c78ed5a3c4845ac_7.png)

```python
print(list1)
print(len(list1))
```

![](img/f82b151e05bdd30f1c78ed5a3c4845ac_9.png)

`len(list1)` 返回的是列表的长度，即列表中包含的元素个数。

## 访问列表元素

![](img/f82b151e05bdd30f1c78ed5a3c4845ac_11.png)

我们可以获取列表中的特定元素。例如，通过使用方括号 `[]` 和索引 `1`，我们可以获取并打印列表中的第二个元素。

![](img/f82b151e05bdd30f1c78ed5a3c4845ac_13.png)

```python
print(list1[1])
```

需要记住的是，列表元素的索引从 `0` 开始。因此，索引 `1` 对应的是列表中的第二个元素。

![](img/f82b151e05bdd30f1c78ed5a3c4845ac_15.png)

如果我们尝试获取列表中不存在的元素，例如第五个元素，Python会报错。

![](img/f82b151e05bdd30f1c78ed5a3c4845ac_16.png)

```python
print(list1[4])  # 这将导致错误
```

![](img/f82b151e05bdd30f1c78ed5a3c4845ac_18.png)

由于我们的列表 `list1` 总共只有四个元素，索引 `4` 指向的第五个元素并不存在。在这种情况下，Python会抛出一个“列表索引超出范围”的错误。

## 修改列表内容

![](img/f82b151e05bdd30f1c78ed5a3c4845ac_20.png)

上一节我们介绍了如何访问列表元素，本节中我们来看看如何修改列表。你可以使用 `append()` 方法向列表末尾添加新元素，或者使用 `pop()` 方法从列表中移除元素。

![](img/f82b151e05bdd30f1c78ed5a3c4845ac_21.png)

以下是向列表添加元素的方法：
```python
list1.append(5)
```

![](img/f82b151e05bdd30f1c78ed5a3c4845ac_23.png)

如果使用 `pop()` 方法时不提供索引参数，它将移除列表中的最后一个元素。
```python
list1.pop()
```

如果使用 `pop()` 方法时提供了索引参数，它将移除该特定索引位置上的元素。例如，以下代码会移除列表中的第二个元素（索引为1）。
```python
list1.pop(1)
```

![](img/f82b151e05bdd30f1c78ed5a3c4845ac_25.png)

## 检查元素是否存在

![](img/f82b151e05bdd30f1c78ed5a3c4845ac_27.png)

最后，我们可以检查某个特定元素是否存在于列表中。以下是检查元素是否存在的方法：
```python
print(3 in list1)
```

![](img/f82b151e05bdd30f1c78ed5a3c4845ac_28.png)

本节课中我们一起学习了Python列表的基本操作：包括创建列表、获取列表长度、通过索引访问元素、向列表添加元素、从列表中移除元素，以及检查某个元素是否在列表中。掌握这些操作是使用Python进行数据处理的基础。
