# 105：CSS盒模型 📦

## 概述
在本节课中，我们将要学习CSS盒模型。盒模型是CSS布局的核心概念，它描述了网页上每个元素如何被渲染为一个矩形盒子，并控制其尺寸和间距。

上一节我们介绍了CSS背景颜色，本节中我们来看看CSS盒模型。对于CSS初学者，盒模型可能看起来有些复杂，但我们会将其分解，用简单的方式解释清楚。

## 什么是CSS盒模型？
简单来说，盒模型是一种描述网页元素布局方式的概念。每个HTML元素都被表示为一个矩形盒子，这个盒子由四个部分组成：**内容区**、**内边距**、**边框**和**外边距**。

以下是盒模型四个部分的图示说明：

![](img/baa914e28b35ed40219e4250da10b980_1.png)

让我们逐一了解每个部分：

1.  **内容区**
    内容区是HTML元素的实际内容，例如文本或图像。

2.  **内边距**
    内边距是内容区与边框之间的空间，用于在元素内部增加间距。

3.  **边框**
    边框是围绕元素和内边距的边界线，可以自定义其颜色、宽度和样式。

4.  **外边距**
    外边距是边框与相邻元素之间的空间，用于在元素外部增加间距。

![](img/baa914e28b35ed40219e4250da10b980_1.png)

理解盒模型对于创建CSS布局至关重要，因为它允许你控制页面上元素的尺寸和间距。通过使用`width`、`height`、`padding`、`border`和`margin`等CSS属性，你可以调整盒模型以实现期望的设计。

## 盒模型实战演示
让我们通过一个HTML和CSS文档的例子来具体看看。

这是我们的HTML文档，其中创建了两个`<div>`元素，并为它们编写了一些CSS样式。

![](img/baa914e28b35ed40219e4250da10b980_3.png)

```css
* {
    margin: 0;
    padding: 0;
}

div {
    height: 50px;
    background-color: lightblue;
}
```

默认情况下，我们已将整个HTML文档的外边距和内边距设置为0。同时，为两个`<div>`设置了相同的高度和一些背景色。

在浏览器中，它目前看起来是这样的：

![](img/baa914e28b35ed40219e4250da10b980_3.png)

![](img/baa914e28b35ed40219e4250da10b980_5.png)

如果我们检查元素，可以看到当前这个`<div>`元素的盒模型。目前，该`<div>`没有任何内边距、边框或外边距，因此我们看不到这些部分。内容区的尺寸是宽度（视口宽度）乘以高度50px。

![](img/baa914e28b35ed40219e4250da10b980_7.png)

现在，让我们看看如何设置这些值，以及它们如何影响盒模型。

![](img/baa914e28b35ed40219e4250da10b980_9.png)

### 设置内边距
首先，为第一个`<div>`添加内边距。

```css
.div1 {
    padding: 20px;
}
```

现在你可以看到，内容被20px的内边距从四周包围。

![](img/baa914e28b35ed40219e4250da10b980_11.png)

![](img/baa914e28b35ed40219e4250da10b980_7.png)

![](img/baa914e28b35ed40219e4250da10b980_13.png)

有时，我们可能需要为盒子的每个边设置不同的内边距。以下是实现方法：

我们可以分别为每个边设置属性：
```css
.div1 {
    padding-left: 10px;
    padding-top: 5px;
    padding-right: 10px;
    padding-bottom: 5px;
}
```
这样，我们为顶部和底部设置了5px内边距，为左侧和右侧设置了10px内边距。

![](img/baa914e28b35ed40219e4250da10b980_15.png)

![](img/baa914e28b35ed40219e4250da10b980_17.png)

![](img/baa914e28b35ed40219e4250da10b980_11.png)

但是，有一种更简洁的写法。`padding`属性可以接受多个值，按**上、右、下、左**的顺序（顺时针）分别设置各边的内边距。

```css
.div1 {
    /* 顺序：上 右 下 左 */
    padding: 10px 50px 5px 100px;
}
```
*   第一个值（10px）应用于**顶部**。
*   第二个值（50px）应用于**右侧**。
*   第三个值（5px）应用于**底部**。
*   第四个值（100px）应用于**左侧**。

![](img/baa914e28b35ed40219e4250da10b980_19.png)

### 设置边框
接下来，添加边框。我们将为所有边设置统一的边框。

![](img/baa914e28b35ed40219e4250da10b980_19.png)

![](img/baa914e28b35ed40219e4250da10b980_20.png)

```css
.div1 {
    padding: 50px;
    border: 10px solid black;
}
```
现在，你可以看到一个10px宽的黑色实线边框出现在内边距的外围。

![](img/baa914e28b35ed40219e4250da10b980_22.png)

![](img/baa914e28b35ed40219e4250da10b980_22.png)

我们可以比较两个`<div>`的盒模型。第二个`<div>`的盒模型值仍然是空的，而第一个`<div>`的盒模型已经填充了内边距和边框的值。

### 设置外边距
最后，添加外边距。

![](img/baa914e28b35ed40219e4250da10b980_24.png)

```css
.div1 {
    padding: 50px;
    border: 10px solid black;
    margin: 20px;
}
```
现在，元素周围出现了20px的外边距，它将第一个`<div>`与第二个`<div>`分隔开来。

![](img/baa914e28b35ed40219e4250da10b980_28.png)

![](img/baa914e28b35ed40219e4250da10b980_26.png)

与`padding`类似，`margin`属性也可以为每个边设置不同的值。

![](img/baa914e28b35ed40219e4250da10b980_28.png)

```css
.div1 {
    /* 顺序：上 右 下 左 */
    margin: 10px 20px 30px 50px;
}
```

![](img/baa914e28b35ed40219e4250da10b980_30.png)

![](img/baa914e28b35ed40219e4250da10b980_32.png)

![](img/baa914e28b35ed40219e4250da10b980_32.png)

我们也可以仅为第二个`<div>`添加外边距。

![](img/baa914e28b35ed40219e4250da10b980_34.png)

![](img/baa914e28b35ed40219e4250da10b980_36.png)

```css
.div2 {
    margin: 50px;
}
```
现在可以看到，外边距也添加到了第二个元素上。你不必同时设置内边距、边框和外边距，可以根据需要直接添加任何一个属性。

![](img/baa914e28b35ed40219e4250da10b980_38.png)

![](img/baa914e28b35ed40219e4250da10b980_38.png)

检查元素时，我们会发现第一个元素的外边距延伸到这里，第二个元素的外边距延伸到这里。实际上，相邻元素的外边距有时会发生重叠（外边距合并），但每个元素始终会根据其周围的元素计算自己的外边距。

## 总结
本节课中我们一起学习了CSS盒模型。我们了解到，每个HTML元素都可以看作一个由**内容区**、**内边距**、**边框**和**外边距**组成的盒子。通过`padding`、`border`和`margin`属性，我们可以精确控制元素的内部空间、边界和外部间距，这是进行网页布局的基础。希望你能够在接下来的项目中运用这些知识。
