🎼Hi there in this lesson we will be diving into angular pipes and forms。

 Angular pipes allows you to transform data before displaying it in the view。😊。

🎼We will learn how to use built in pipes like currency， date and uppercase。

 as well as how to create our own custom pipes。🎼We will also explore how to use pipes in combination with directives to format and display data in different ways Additionally we will be delving into angular forms where we will be learning about two types of forms that is temporary driven forms and reactive forms。

🎼Tempate driven forms use two way data binding to keep the model and the view in sync while reactive forms gives you more control over how data is validated and processed。

🎼By the end of this lesson， you will have a solid understanding of angular pipes and forms and you will be able to use them effectively in your projects see you in the next video。
