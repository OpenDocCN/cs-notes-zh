# Java全栈开发 专项课程（上）：5.06：HTML格式化标签 🎨

在本节课中，我们将要学习HTML中的格式化标签。这些标签用于改变网页上文本的外观，例如加粗、倾斜或添加下划线。通过掌握这些标签，你可以更好地控制网页内容的视觉呈现。

![](img/2c36776887bfa39776684575e86d8860_1.png)

上一节我们介绍了一些基础的HTML标签及其用例。本节中，我们来看看专门用于文本格式化的标签。

## 加粗标签

![](img/2c36776887bfa39776684575e86d8860_3.png)

![](img/2c36776887bfa39776684575e86d8860_4.png)

首先，我们介绍加粗标签。加粗标签用于使文本以粗体显示。使用此标签时，只需将需要加粗的文本包裹在开始和结束的加粗标签中。

![](img/2c36776887bfa39776684575e86d8860_6.png)

其语法结构如下：
```html
<b>需要加粗的文本</b>
```

![](img/2c36776887bfa39776684575e86d8860_8.png)

在编辑器中，如果将段落标签`<p>`替换为加粗标签`<b>`，可以观察到文本的字体粗细发生了变化。加粗标签也可以嵌套在段落标签内部使用，以实现段落内部分文本的加粗效果。

![](img/2c36776887bfa39776684575e86d8860_10.png)

![](img/2c36776887bfa39776684575e86d8860_12.png)

## 倾斜标签

![](img/2c36776887bfa39776684575e86d8860_14.png)

接下来是倾斜标签。倾斜标签用于使文本以斜体显示。使用此标签时，只需将需要倾斜的文本包裹在开始和结束的倾斜标签中。

![](img/2c36776887bfa39776684575e86d8860_16.png)

其语法结构如下：
```html
<i>需要倾斜的文本</i>
```

![](img/2c36776887bfa39776684575e86d8860_18.png)

![](img/2c36776887bfa39776684575e86d8860_20.png)

在浏览器中查看，被包裹的文本会呈现为斜体。与加粗标签类似，倾斜标签也可以嵌套在段落标签内部使用。

![](img/2c36776887bfa39776684575e86d8860_22.png)

## 下划线标签

现在，我们来看下划线标签。下划线标签用于为文本添加下划线。使用此标签时，只需将需要添加下划线的文本包裹在开始和结束的下划线标签中。

![](img/2c36776887bfa39776684575e86d8860_24.png)

![](img/2c36776887bfa39776684575e86d8860_26.png)

其语法结构如下：
```html
<u>需要添加下划线的文本</u>
```

![](img/2c36776887bfa39776684575e86d8860_28.png)

在HTML中，下划线标签使用`<u>`表示。保存并刷新页面后，可以看到指定文本已被添加下划线。

![](img/2c36776887bfa39776684575e86d8860_30.png)

## 删除线标签

另一个重要的标签是删除线标签。删除线标签用于在文本上添加一条横线，表示文本被划掉或删除。使用此标签时，只需将需要划掉的文本包裹在开始和结束的删除线标签中。

![](img/2c36776887bfa39776684575e86d8860_32.png)

![](img/2c36776887bfa39776684575e86d8860_34.png)

其语法结构如下：
```html
<s>需要划掉的文本</s>
```

![](img/2c36776887bfa39776684575e86d8860_36.png)

在HTML中，删除线标签通常使用`<s>`表示。保存后，可以看到指定文本被成功划掉。

![](img/2c36776887bfa39776684575e86d8860_38.png)

## 插入文本标签

插入文本标签用于指示被插入的文本，例如更正或补充内容。它常与删除线标签配合使用，以显示内容的更新。

![](img/2c36776887bfa39776684575e86d8860_40.png)

其语法结构如下：
```html
<ins>插入的文本</ins>
```

例如，可以先使用删除线标签划掉旧文本，然后紧接着使用插入标签添加新文本，从而清晰地展示内容的修改。

![](img/2c36776887bfa39776684575e86d8860_42.png)

## 上标与下标标签

以下是用于处理数学和化学公式的标签。

![](img/2c36776887bfa39776684575e86d8860_44.png)

**上标标签**使文本显示在基线之上，常用于数学公式中的指数。

其语法结构如下：
```html
<sup>上标内容</sup>
```
例如，要书写“2的平方”，可以写为 `2<sup>2</sup>`。

![](img/2c36776887bfa39776684575e86d8860_46.png)

![](img/2c36776887bfa39776684575e86d8860_48.png)

![](img/2c36776887bfa39776684575e86d8860_50.png)

**下标标签**使文本显示在基线之下，常用于化学分子式。

其语法结构如下：
```html
<sub>下标内容</sub>
```
例如，要书写水的化学式“H₂O”，可以写为 `H<sub>2</sub>O`。

![](img/2c36776887bfa39776684575e86d8860_52.png)

## 小型文本标签

小型文本标签用于使文本显示得比周围文本更小，通常用于脚注或免责声明。

![](img/2c36776887bfa39776684575e86d8860_54.png)

其语法结构如下：
```html
<small>小型文本</small>
```
例如，可以在页面底部使用此标签添加一行小字说明。

![](img/2c36776887bfa39776684575e86d8860_56.png)

## 标记文本标签

![](img/2c36776887bfa39776684575e86d8860_58.png)

标记文本标签用于高亮显示文本，类似于荧光笔的效果。

![](img/2c36776887bfa39776684575e86d8860_60.png)

其语法结构如下：
```html
<mark>需要高亮的文本</mark>
```
使用此标签包裹的文本会在页面上以高亮背景色显示。

![](img/2c36776887bfa39776684575e86d8860_62.png)

## 强调与着重标签

最后，我们介绍两个具有语义含义的标签。

**着重标签**在视觉上与加粗标签相似，但它告知浏览器此文本在文档中具有重要性。

其语法结构如下：
```html
<strong>重要的文本</strong>
```

![](img/2c36776887bfa39776684575e86d8860_64.png)

**强调标签**在视觉上与倾斜标签相似，但它告知浏览器应以强调的语气（如加重音）来朗读此文本。

其语法结构如下：
```html
<em>需要强调的文本</em>
```

![](img/2c36776887bfa39776684575e86d8860_66.png)

虽然`<strong>`和`<b>`、`<em>`和`<i>`的视觉效果可能相同，但它们在语义上传达了不同的含义，对辅助阅读工具（如屏幕阅读器）更为友好。

![](img/2c36776887bfa39776684575e86d8860_68.png)
