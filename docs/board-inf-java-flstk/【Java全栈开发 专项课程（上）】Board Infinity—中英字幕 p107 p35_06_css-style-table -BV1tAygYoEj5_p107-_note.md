# Java全栈开发 专项课程（上）：107：CSS样式化表格 🎨

在本节课中，我们将要学习如何使用CSS来美化HTML表格。表格是展示结构化数据的绝佳方式，但为了使其更具视觉吸引力，我们需要为其添加样式。

上一节我们介绍了CSS图标，本节中我们来看看如何为表格添加边框、颜色和响应式设计，使其看起来更专业、更易读。

## 概述

我们将通过一个简单的HTML表格示例，逐步应用各种CSS属性来改善其外观。核心步骤包括设置边框、合并边框、对齐文本、设置表头样式以及为表格行添加交替的背景色。

## 创建基础表格

![](img/58204e30325de9e99017497c5c73acf2_1.png)

首先，我们有一个基础的HTML表格结构，它目前没有任何样式，因此在浏览器中看起来只是简单的文本排列。

![](img/58204e30325de9e99017497c5c73acf2_3.png)

![](img/58204e30325de9e99017497c5c73acf2_5.png)

```html
<table>
  <tr>
    <th>姓名</th>
    <th>年龄</th>
    <th>城市</th>
  </tr>
  <tr>
    <td>张三</td>
    <td>25</td>
    <td>北京</td>
  </tr>
  <tr>
    <td>李四</td>
    <td>30</td>
    <td>上海</td>
  </tr>
</table>
```

## 添加基础样式

为了让表格看起来像表格，我们需要添加边框。以下是第一步CSS代码，为整个表格和每个单元格添加边框。

![](img/58204e30325de9e99017497c5c73acf2_7.png)

```css
table {
  border: 1px solid black;
  width: 500px;
  margin: 20px; /* 为表格添加一些外边距 */
}

![](img/58204e30325de9e99017497c5c73acf2_9.png)

td, th {
  border: 1px solid black;
}
```

![](img/58204e30325de9e99017497c5c73acf2_11.png)

应用此样式后，表格有了边框，但每个单元格的边框是分开的，看起来不够整洁。

## 合并表格边框

![](img/58204e30325de9e99017497c5c73acf2_13.png)

![](img/58204e30325de9e99017497c5c73acf2_15.png)

为了解决边框分离的问题，我们可以使用 `border-collapse` 属性。这个属性可以将相邻单元格的边框合并为单一边框。

![](img/58204e30325de9e99017497c5c73acf2_17.png)

```css
table {
  border-collapse: collapse;
}
```

应用此属性后，表格的边框变得清晰、统一。

![](img/58204e30325de9e99017497c5c73acf2_19.png)

## 对齐文本与设置表头样式

![](img/58204e30325de9e99017497c5c73acf2_21.png)

目前，表格内的文本（特别是表头）可能没有对齐。我们可以使用 `text-align` 属性使其居中。同时，为了让表头更突出，我们可以为其设置背景色和文字样式。

以下是相关CSS代码：

![](img/58204e30325de9e99017497c5c73acf2_23.png)

![](img/58204e30325de9e99017497c5c73acf2_25.png)

```css
td, th {
  text-align: center; /* 使所有单元格内容居中 */
}

th {
  background-color: black; /* 设置表头背景色 */
  color: white; /* 设置表头文字颜色 */
  font-weight: bold; /* 加粗表头文字 */
  text-align: left; /* 表头文字左对齐 */
  padding: 10px; /* 为表头添加内边距 */
}
```

![](img/58204e30325de9e99017497c5c73acf2_27.png)

## 为表格行添加交替颜色

![](img/58204e30325de9e99017497c5c73acf2_29.png)

![](img/58204e30325de9e99017497c5c73acf2_31.png)

为了提高长表格的可读性，一个常见的做法是为奇数行和偶数行设置不同的背景色。我们可以使用 `:nth-child()` 伪类选择器来实现斑马纹效果。

以下是实现代码：

```css
tr:nth-child(even) {
  background-color: #f2f2f2; /* 为偶数行设置浅灰色背景 */
}
```

![](img/58204e30325de9e99017497c5c73acf2_33.png)

## 添加悬停效果

为了提升交互体验，我们可以为表格行添加悬停效果，当鼠标悬停在某一行时，改变其背景色。

![](img/58204e30325de9e99017497c5c73acf2_35.png)

```css
tr:hover {
  background-color: yellow; /* 悬停时背景变为黄色 */
  color: black; /* 悬停时文字变为黑色 */
}
```

![](img/58204e30325de9e99017497c5c73acf2_37.png)

## 总结

![](img/58204e30325de9e99017497c5c73acf2_39.png)

本节课中我们一起学习了如何使用CSS来样式化HTML表格。我们掌握了以下核心技能：

1.  使用 `border` 和 `border-collapse` 属性来创建和优化表格边框。
2.  使用 `text-align` 和 `padding` 来调整文本对齐和单元格间距。
3.  使用背景色、文字颜色和字体粗细来突出显示表头。
4.  利用 `:nth-child()` 伪类为表格行创建交替的背景色，增强可读性。
5.  使用 `:hover` 伪类为表格添加简单的交互效果。

![](img/58204e30325de9e99017497c5c73acf2_41.png)

通过这些步骤，你可以将任何朴素的HTML表格转变为结构清晰、视觉美观的数据展示组件。尝试将这些技巧应用到你的项目中，以提升用户体验。
