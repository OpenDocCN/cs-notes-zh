🎼Hi there。 In this lesson， will learn about Angular data binding。

 one of the most important and powerful features of Angular framework。😊。

🎼You will start by understanding what data binding is and why it is important in modern web development。

🎼You will then learn about the different types of data binding available in Angular。

 including one way and two way data binding。🎼You will explore how to use angulars interpolation syntaxs to bind data from your components class to your templates ST as well as how to use property binding to pass data from a parent component to a child component。

🎼You will also learn about even binding， which allows you to respond to user input such as a mouse click or keyboard input and update your components data accordingly。

🎼Finally， you will dive into the Angulars two way data mining syntax。

 which combines property and even mining to enable bidirectional data flow between your components class and your templates ST。

🎼By the end of this lesson， you will have a solid understanding of how data binding works in Angular and how to use it effectively in your web applications see you in the next video。
