🎼Hi， there。In Java programming， the need of persistent storage arises when you need to store data beyond the life of your application。

Java Peristence API is a widely used standard for object relation mapping in Java applications。

🎼In this lesson， you will learn about JA and how it is used to manage the persistence of Java objects to relational databases。

😊。

🎼JPAA is a specification for ORM frameworks that allow you to map Java objects to relational databases。

🎼You will learn about the different layers in JP architecture and how they work together。

🎼Next you will learn about hibernet GPA annotations and JA configuration files。😊。

🎼Hybernet is one of the most popular JA provider and provides a rich set of annotations for mapping Java objects to relational databases。
