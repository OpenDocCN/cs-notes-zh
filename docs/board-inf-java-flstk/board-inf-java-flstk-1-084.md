# 084：表格标签详解

在本节课中，我们将要学习HTML中的表格标签。表格是网页上以结构化、有组织的方式呈现数据和信息的重要工具。

上一节我们介绍了HTML列表，本节中我们来看看如何使用HTML创建和定制表格。

## 表格的基本结构

要创建HTML表格，需要使用`<table>`标签。在`<table>`标签内部，可以定义多个其他标签来构建表格的结构。

![](img/10a7c380f4765d2161d197852c5821bb_1.png)

以下是构成表格的核心标签：
*   `<thead>`：定义表格的头部，通常包含列标题。
*   `<tbody>`：定义表格的主体，包含实际的数据行。
*   `<tfoot>`：定义表格的页脚，通常包含汇总信息。
*   `<tr>`：在`<thead>`、`<tbody>`或`<tfoot>`内部使用，用于定义一行。
*   `<td>`：在`<tr>`内部使用，用于定义标准的数据单元格。
*   `<th>`：在`<tr>`内部使用，用于定义表头单元格，通常替代`<td>`用于标题列，浏览器会默认将其内容加粗并居中显示。

## 创建一个简单的表格

现在，让我们使用这些标签来创建一个HTML页面上的表格。

首先，我们使用`<table>`标签。在其内部，我们创建表格的三个主要部分：头部(`<thead>`)、主体(`<tbody>`)和页脚(`<tfoot>`)。

```html
<table>
  <thead>
  </thead>
  <tbody>
  </tbody>
  <tfoot>
  </tfoot>
</table>
```

此时保存并查看浏览器，不会看到任何内容，因为我们还没有在表格中添加行和列。

![](img/10a7c380f4765d2161d197852c5821bb_3.png)

![](img/10a7c380f4765d2161d197852c5821bb_5.png)

让我们从头部开始。在`<thead>`内部，我们使用`<tr>`创建一行。假设我们的表格要展示“国家”和“识字率”，因此需要两个列标题。我们使用`<th>`标签来创建这两个标题单元格。

```html
<table>
  <thead>
    <tr>
      <th>国家</th>
      <th>识字率</th>
    </tr>
  </thead>
  <tbody>
  </tbody>
  <tfoot>
  </tfoot>
</table>
```

现在，我们为表格主体添加数据。在`<tbody>`内部，我们创建多个`<tr>`行，每行包含两个`<td>`数据单元格。

![](img/10a7c380f4765d2161d197852c5821bb_7.png)

![](img/10a7c380f4765d2161d197852c5821bb_9.png)

```html
<tbody>
  <tr>
    <td>印度</td>
    <td>80%</td>
  </tr>
  <tr>
    <td>美国</td>
    <td>75%</td>
  </tr>
  <tr>
    <td>埃及</td>
    <td>60%</td>
  </tr>
</tbody>
```

最后，我们在页脚(`<tfoot>`)添加一行汇总信息。

```html
<tfoot>
  <tr>
    <td>世界</td>
    <td>76%</td>
  </tr>
</tfoot>
```

## 使用属性定制表格

![](img/10a7c380f4765d2161d197852c5821bb_15.png)

除了基本标签，还可以使用属性来定制表格的外观和行为。

![](img/10a7c380f4765d2161d197852c5821bb_17.png)

例如，`border`属性可以为表格添加边框。

![](img/10a7c380f4765d2161d197852c5821bb_19.png)

```html
<table border="1">
```

![](img/10a7c380f4765d2161d197852c5821bb_21.png)

![](img/10a7c380f4765d2161d197852c5821bb_22.png)

`cellspacing`属性可以控制单元格之间的间距。将其设置为`0`可以合并边框间的空隙，设置更大的值则会增加间距。

![](img/10a7c380f4765d2161d197852c5821bb_24.png)

```html
<table border="1" cellspacing="10">
```

![](img/10a7c380f4765d2161d197852c5821bb_26.png)

![](img/10a7c380f4765d2161d197852c5821bb_28.png)

`align`属性可以设置单元格内内容的水平对齐方式（如左对齐、居中、右对齐）。注意，`<th>`单元格的内容默认是居中的。

![](img/10a7c380f4765d2161d197852c5821bb_30.png)

![](img/10a7c380f4765d2161d197852c5821bb_31.png)

```html
<td align="center">印度</td>
```

![](img/10a7c380f4765d2161d197852c5821bb_33.png)

我们可以为所有`<td>`标签添加此属性，使其内容也居中显示。

![](img/10a7c380f4765d2161d197852c5821bb_35.png)

```html
<tr>
  <td align="center">印度</td>
  <td align="center">80%</td>
</tr>
```

![](img/10a7c380f4765d2161d197852c5821bb_37.png)

## 总结

本节课中我们一起学习了HTML表格。表格是在网站上结构化展示数据和信息的强大工具。

我们掌握了表格的基本结构，包括`<table>`、`<thead>`、`<tbody>`、`<tfoot>`、`<tr>`、`<td>`和`<th>`标签的用法。我们还学习了如何使用`border`、`cellspacing`和`align`等属性来初步调整表格的样式。

![](img/10a7c380f4765d2161d197852c5821bb_39.png)

通过组合使用表格标签及其属性，你可以创建出符合特定需求的表格。后续我们还可以使用CSS来进一步美化和定制表格的外观。希望你能理解表格的构成，并在自己的网页中实践应用。
