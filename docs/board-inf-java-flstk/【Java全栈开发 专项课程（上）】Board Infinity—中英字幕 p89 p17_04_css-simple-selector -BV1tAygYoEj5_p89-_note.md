# Java全栈开发 专项课程（上）：P17-04：CSS 简单选择器 🎯

在本节课中，我们将要学习 CSS 中的简单选择器。选择器是 CSS 的核心，它决定了样式规则将应用到哪些 HTML 元素上。通过掌握简单选择器，你将能够精确地控制网页的外观。

上一节我们介绍了 CSS 的基本语法，本节中我们来看看如何通过不同的选择器来定位元素。

## 概述

![](img/25dc6dee72e91c19b79b41d5624fe021_1.png)

简单选择器是一种根据元素的标签名、类名或 ID 来匹配单个元素的选择器。它们是最基础也是最常用的选择器类型。我们将逐一探讨四种主要的简单选择器：标签选择器、类选择器、ID 选择器和通用选择器。

![](img/25dc6dee72e91c19b79b41d5624fe021_3.png)

## 1. 标签选择器

![](img/25dc6dee72e91c19b79b41d5624fe021_5.png)

标签选择器通过 HTML 元素的标签名称来匹配所有该类型的元素。

其语法非常简单，直接使用标签名作为选择器即可。例如，要为所有段落 `<p>` 设置样式，选择器就是 `p`。

![](img/25dc6dee72e91c19b79b41d5624fe021_7.png)

![](img/25dc6dee72e91c19b79b41d5624fe021_9.png)

以下是一个使用标签选择器的代码示例：

```css
a {
  background-color: blue;
}
```

在上一个视频中，我们创建了一个 HTML 页面，其中包含 `<h1>` 和 `<h2>` 标签。当我们使用 `a` 作为选择器并应用上述 CSS 时，所有 `<a>`（锚点）标签的背景色都会变为蓝色。

![](img/25dc6dee72e91c19b79b41d5624fe021_11.png)

我们也可以将此规则应用于其他标签。例如，将选择器改为 `h2`，那么所有 `<h2>` 标签的背景色会改变。如果改为 `body`，则整个页面的背景色会发生变化。

![](img/25dc6dee72e91c19b79b41d5624fe021_13.png)

## 2. 类选择器

类选择器用于匹配所有具有特定 `class` 属性的 HTML 元素。类（Class）代表一组元素，因此一个类可以包含多个元素。对该类应用的样式会作用于所有使用该类的元素。

![](img/25dc6dee72e91c19b79b41d5624fe021_15.png)

类选择器的语法是在类名前加上一个点 `.`。

以下是如何使用类选择器的示例。假设我们有一些 `<h2>` 标签，我们希望其中特定的几个拥有红色背景。

首先，为这些 `<h2>` 标签添加一个类名，例如 `red-background`：

![](img/25dc6dee72e91c19b79b41d5624fe021_17.png)

```html
<h2 class="red-background">标题1</h2>
<h2 class="red-background">标题2</h2>
```

![](img/25dc6dee72e91c19b79b41d5624fe021_19.png)

然后，在 CSS 中这样定义样式：

```css
.red-background {
  background-color: red;
}
```

这样，所有具有 `class="red-background"` 的元素的背景色都会变为红色。类选择器的优势在于它可以轻松地为一组元素统一样式。

![](img/25dc6dee72e91c19b79b41d5624fe021_21.png)

## 3. ID 选择器

ID 选择器用于匹配具有特定 `id` 属性的**单个** HTML 元素。`id` 是元素的唯一标识符，每个元素的 `id` 在文档中应该是独一无二的。因此，ID 选择器用于定位并样式化某个特定的元素。

ID 选择器的语法是在 ID 名前加上一个井号 `#`。

让我们看一个例子。假设我们有一个额外的 `<h2>` 标签，并且只想改变它的文字颜色。

首先，为该标签赋予一个唯一的 ID，例如 `change-color`：

![](img/25dc6dee72e91c19b79b41d5624fe021_23.png)

```html
<h2 id="change-color">这个标题颜色会变</h2>
```

![](img/25dc6dee72e91c19b79b41d5624fe021_25.png)

然后，在 CSS 中针对这个 ID 设置样式：

![](img/25dc6dee72e91c19b79b41d5624fe021_27.png)

```css
#change-color {
  color: white;
}
```

![](img/25dc6dee72e91c19b79b41d5624fe021_29.png)

这样，只有这个特定的 `<h2>` 标签的文字颜色会变为白色。ID 选择器提供了最精确的定位方式。

## 4. 通用选择器

![](img/25dc6dee72e91c19b79b41d5624fe021_31.png)

通用选择器可以匹配页面上的**所有** HTML 元素。它使用星号 `*` 表示。

![](img/25dc6dee72e91c19b79b41d5624fe021_33.png)

通用选择器通常用于设置一些全局的、基础的样式。例如，我们可以设置整个文档的默认文字颜色：

![](img/25dc6dee72e91c19b79b41d5624fe021_35.png)

```css
* {
  color: brown;
}
```

![](img/25dc6dee72e91c19b79b41d5624fe021_37.png)

这会将页面上所有元素的文字颜色初始设置为棕色。需要注意的是，其他更具体的选择器（如 ID 选择器或类选择器）的样式规则会覆盖通用选择器的规则，因为 CSS 遵循**层叠**和**优先级**规则。

## 总结

本节课中我们一起学习了 CSS 的四种简单选择器：
*   **标签选择器**：通过标签名匹配元素，例如 `p {}`。
*   **类选择器**：通过类名匹配一组元素，例如 `.className {}`。
*   **ID 选择器**：通过唯一 ID 匹配单个元素，例如 `#idName {}`。
*   **通用选择器**：匹配所有元素，例如 `* {}`。

![](img/25dc6dee72e91c19b79b41d5624fe021_39.png)

![](img/25dc6dee72e91c19b79b41d5624fe021_41.png)

简单选择器是精确控制网页样式的基础工具。通过组合使用它们，开发者可以轻松创建美观且功能完善的网页。建议你亲自动手实践这些选择器，以加深理解。我们下节课再见！
