在本节课中，我们将要学习 CSS（层叠样式表）的基础语法。CSS 是一种用于描述网页外观和格式的语言，它允许我们为 HTML 元素添加颜色、间距、布局等样式，从而让网页更加美观和用户友好。

---

## 概述：为什么需要 CSS？ 🤔

目前为止，我们创建的网页都相对简单。我们只是描述了页面的结构，并说明了希望在哪里列出内容。然而，我们通常希望以特定的方式对网页进行样式设置，例如添加颜色、间距和其他布局元素。为此，我们将使用 CSS 语言，特别是其最新版本 CSS3。

CSS3 使我们能够告诉网页浏览器，我们希望 HTML 页面以何种方式进行样式处理，而不仅仅是在白色背景上显示黑色文本。我们可以指定特定的 CSS 属性，以确保页面的外观符合我们的期望。

## 添加 CSS 的基本方法 🛠️

![](img/5b25f9189e795f9881b6a35947d0e23f_13.png)

上一节我们介绍了 CSS 的作用，本节中我们来看看如何将 CSS 代码添加到我们的页面中。

![](img/5b25f9189e795f9881b6a35947d0e23f_15.png)

我将创建一个新文件，命名为 `style.html`，以演示一些为页面添加样式的基本想法，并将之前的 “hello” 代码复制到其中。

![](img/5b25f9189e795f9881b6a35947d0e23f_17.png)

除了 “hello world” 之外，我还希望在 `h1` 中显示一个大标题，例如 “欢迎来到我的网页”。所以现在如果我打开 `style.html`，这就是我看到的：顶部有一个大标题，上面写着 “欢迎来到我的网页”，下面是 “hello world” 文本。

![](img/5b25f9189e795f9881b6a35947d0e23f_19.png)

现在想象一下，我想为页面顶部的标题添加一些样式。也许我希望它不再是左对齐，而是居中，并且颜色从黑色变为蓝色。

### 内联样式

![](img/5b25f9189e795f9881b6a35947d0e23f_23.png)

我们可以使用 `style` 属性为单个 HTML 元素添加样式，这被称为内联样式。

以下是具体的操作步骤：

1.  在 `h1` 元素中添加 `style` 属性。
2.  在引号内，设置 CSS 属性。例如，将颜色改为蓝色：`color: blue;`。
3.  可以添加多个属性，用分号分隔。例如，同时设置居中对齐：`text-align: center;`。

**代码示例：**
```html
<h1 style="color: blue; text-align: center;">欢迎来到我的网页</h1>
```

现在，当我刷新页面时，会看到 “欢迎来到我的网页” 这个标题变为蓝色并且居中。

## 样式的继承与分离 🧩

上一节我们学习了如何为单个元素添加样式，本节中我们来看看样式如何从父元素继承，以及如何将样式代码与 HTML 结构分离。

### 样式的继承

HTML 元素不仅可以直接样式化，它们还可以从父元素继承样式信息。例如，在 DOM 结构中，`body` 元素是 `h1` 元素的父元素。

![](img/5b25f9189e795f9881b6a35947d0e23f_35.png)

如果我将 `style` 属性从 `h1` 移动到 `body` 标签，那么 `body` 内部的所有内容（包括标题和段落）都会应用这些 CSS 样式。

**代码示例：**
```html
<body style="color: blue; text-align: center;">
    <h1>欢迎来到我的网页</h1>
    hello world
</body>
```

刷新页面后，正文的两个部分（标题和文本）都变为蓝色并居中。

![](img/5b25f9189e795f9881b6a35947d0e23f_37.png)

### 将样式移至 `<head>` 区域

虽然内联样式有效，但当页面有多个元素需要相同样式时，代码会变得冗余且难以维护。更好的方法是将样式代码移动到网页的 `<head>` 部分。

![](img/5b25f9189e795f9881b6a35947d0e23f_39.png)

以下是具体的操作步骤：

1.  在 `<head>` 标签内，添加一个 `<style>` 标签。
2.  在 `<style>` 标签内，编写 CSS 规则。例如，选择所有 `h1` 元素并为其设置样式。

**代码示例：**
```html
<head>
    <style>
        h1 {
            color: blue;
            text-align: center;
        }
    </style>
</head>
<body>
    <h1>欢迎来到我的网页</h1>
    <h1>第二个标题</h1>
    hello world
</body>
```

这样，所有 `h1` 元素都会应用相同的样式，代码只需编写一次，更易于管理和修改。

## 使用外部样式表 🌐

![](img/5b25f9189e795f9881b6a35947d0e23f_41.png)

上一节我们将样式移到了 `<head>` 中，但如果多个网页需要共享相同的样式，重复复制代码仍然不够理想。本节中我们来看看如何将 CSS 代码放入一个完全独立的文件中。

我们可以创建一个单独的 `.css` 文件来存放所有样式规则，然后在 HTML 文件中链接它。

![](img/5b25f9189e795f9881b6a35947d0e23f_45.png)

以下是具体的操作步骤：

![](img/5b25f9189e795f9881b6a35947d0e23f_49.png)

1.  创建一个新文件，例如 `styles.css`。
2.  将 CSS 代码（如 `h1 { color: blue; text-align: center; }`）写入这个文件。
3.  在 HTML 文件的 `<head>` 部分，使用 `<link>` 标签链接到这个 CSS 文件。

**代码示例 (`styles.css`):**
```css
h1 {
    color: blue;
    text-align: center;
}
```

![](img/5b25f9189e795f9881b6a35947d0e23f_57.png)

**代码示例 (`index.html` 的 `<head>` 部分):**
```html
<head>
    <link rel="stylesheet" href="styles.css">
</head>
```

![](img/5b25f9189e795f9881b6a35947d0e23f_59.png)

现在，所有链接到 `styles.css` 的 HTML 页面都会使用相同的样式。如果需要更改样式（例如将蓝色改为红色），只需在 `styles.css` 文件中修改一次即可。

![](img/5b25f9189e795f9881b6a35947d0e23f_61.png)

## 常用的 CSS 属性 📐

我们已经学会了如何应用 CSS，现在让我们看看几种最常用、最流行的 CSS 属性，它们可以让我们的网页看起来更符合期望。

### 控制元素尺寸

默认情况下，HTML 对页面上的所有内容使用默认大小。但我们可以使用 CSS 更精确地控制任何特定元素的大小。

![](img/5b25f9189e795f9881b6a35947d0e23f_69.png)

以下是控制元素尺寸的属性：

![](img/5b25f9189e795f9881b6a35947d0e23f_71.png)

*   **`width`**: 设置元素的宽度，例如 `width: 100px;`。
*   **`height`**: 设置元素的高度，例如 `height: 400px;`。
*   **`background-color`**: 设置元素的背景颜色，例如 `background-color: orange;`。

### 控制元素间距

![](img/5b25f9189e795f9881b6a35947d0e23f_75.png)

为了让页面布局更友好，我们需要控制元素内部和元素之间的空间。

![](img/5b25f9189e795f9881b6a35947d0e23f_77.png)

![](img/5b25f9189e795f9881b6a35947d0e23f_79.png)

以下是控制元素间距的属性：

![](img/5b25f9189e795f9881b6a35947d0e23f_81.png)

*   **`padding`**: 在元素**边框内部**添加空间，使内容与边框产生距离，例如 `padding: 20px;`。
*   **`margin`**: 在元素**边框外部**添加空间，使元素与其他元素或屏幕边缘产生距离，例如 `margin: 20px;`。

![](img/5b25f9189e795f9881b6a35947d0e23f_83.png)

**综合示例：**
```html
<div style="width: 200px; height: 200px; background-color: orange; padding: 20px; margin: 20px;">
    hello world
</div>
```

## 字体与边框样式 ✏️

![](img/5b25f9189e795f9881b6a35947d0e23f_89.png)

除了尺寸和间距，我们还可以使用 CSS 来更改元素的实际外观，例如字体和边框。

![](img/5b25f9189e795f9881b6a35947d0e23f_91.png)

### 字体样式

![](img/5b25f9189e795f9881b6a35947d0e23f_93.png)

![](img/5b25f9189e795f9881b6a35947d0e23f_95.png)

我们可以控制文本的字体、大小和粗细。

![](img/5b25f9189e795f9881b6a35947d0e23f_97.png)

以下是控制字体样式的属性：

*   **`font-family`**: 指定字体，例如 `font-family: Arial, sans-serif;`（`sans-serif` 是备用字体）。
*   **`font-size`**: 指定字体大小，例如 `font-size: 28px;`。
*   **`font-weight`**: 指定字体粗细，例如 `font-weight: bold;`（粗体）。

![](img/5b25f9189e795f9881b6a35947d0e23f_99.png)

### 边框样式

![](img/5b25f9189e795f9881b6a35947d0e23f_101.png)

![](img/5b25f9189e795f9881b6a35947d0e23f_103.png)

我们可以为元素添加边框，以分隔页面中的不同部分。

以下是控制边框样式的属性：

![](img/5b25f9189e795f9881b6a35947d0e23f_105.png)

*   **`border`**: 这是一个简写属性，可以同时设置边框的宽度、样式和颜色，例如 `border: 3px solid black;`。

![](img/5b25f9189e795f9881b6a35947d0e23f_107.png)

![](img/5b25f9189e795f9881b6a35947d0e23f_109.png)

**综合示例：**
```html
<div style="font-family: Arial; font-size: 28px; font-weight: bold; border: 3px solid black; padding: 10px;">
    hello world
</div>
```

## 使用 CSS 美化表格 📊

![](img/5b25f9189e795f9881b6a35947d0e23f_111.png)

让我们将学到的 CSS 知识应用到一个实际例子中：美化一个 HTML 表格。

![](img/5b25f9189e795f9881b6a35947d0e23f_113.png)

假设我们有一个简单的表格，显示海洋信息。没有样式时，它看起来并不美观。

以下是美化表格的步骤：

1.  **为整个表格添加边框**：`table { border: 1px solid black; }`
2.  **为每个单元格添加边框**：`td, th { border: 1px solid black; }`
3.  **合并单元格边框**：`table { border-collapse: collapse; }`（这会让相邻边框合并为一条线）。
4.  **为单元格内容添加内边距**：`td, th { padding: 5px; }`

**最终 CSS 代码示例：**
```css
table {
    border: 1px solid black;
    border-collapse: collapse;
}
td, th {
    border: 1px solid black;
    padding: 5px;
}
```

通过添加这些简单的 CSS 规则，表格的外观得到了显著改善。

## CSS 选择器：ID 与类 🎯

![](img/5b25f9189e795f9881b6a35947d0e23f_119.png)

![](img/5b25f9189e795f9881b6a35947d0e23f_121.png)

上一节我们样式化了所有同类元素，但有时我们只想样式化特定的一个或一组元素。本节中我们来看看如何使用 ID 和类选择器来实现更精确的样式控制。

### ID 选择器

ID 是为 HTML 元素指定的**唯一**名称。在 CSS 中，使用 `#` 符号后跟 ID 名来选择该元素。

![](img/5b25f9189e795f9881b6a35947d0e23f_123.png)

**操作步骤：**
1.  在 HTML 元素中添加 `id` 属性，例如 `<h1 id="foo">标题1</h1>`。
2.  在 CSS 中使用 `#foo` 来选择并样式化这个特定元素。

![](img/5b25f9189e795f9881b6a35947d0e23f_125.png)

**代码示例：**
```css
#foo {
    color: blue;
}
```

### 类选择器

类是为 HTML 元素指定的**非唯一**名称，多个元素可以共享同一个类名。在 CSS 中，使用 `.` 符号后跟类名来选择所有属于该类的元素。

![](img/5b25f9189e795f9881b6a35947d0e23f_127.png)

**操作步骤：**
1.  在 HTML 元素中添加 `class` 属性，例如 `<h1 class="baz">标题1</h1>`， `<h1 class="baz">标题2</h1>`。
2.  在 CSS 中使用 `.baz` 来选择并样式化所有具有该类名的元素。

**代码示例：**
```css
.baz {
    color: blue;
}
```

使用类和 ID 可以帮助我们更清晰、更有组织地管理样式，特别是在设计复杂的网页时。

## CSS 选择器的特异性 ⚖️

当多个 CSS 规则可以应用于同一个 HTML 元素时，浏览器需要决定最终应用哪个样式。这个决策过程基于“特异性”（Specificity）。

![](img/5b25f9189e795f9881b6a35947d0e23f_131.png)

特异性规则按照以下优先级顺序（从高到低）决定样式的应用：

![](img/5b25f9189e795f9881b6a35947d0e23f_133.png)

![](img/5b25f9189e795f9881b6a35947d0e23f_135.png)

1.  **内联样式**：直接写在 HTML 元素 `style` 属性中的样式，优先级最高。
2.  **ID 选择器**：例如 `#foo`。
3.  **类选择器、属性选择器、伪类**：例如 `.baz`, `[type="text"]`, `:hover`。
4.  **元素类型选择器**：例如 `div`, `h1`。
5.  **通用选择器**：例如 `*`。

![](img/5b25f9189e795f9881b6a35947d0e23f_137.png)

![](img/5b25f9189e795f9881b6a35947d0e23f_139.png)

**示例分析：**
```css
div { color: blue; }       /* 特异性较低 */
#foo { color: red; }       /* 特异性较高 */
```
如果一个 `div` 元素的 ID 是 `foo`，那么它的文字颜色将是红色，因为 ID 选择器的特异性高于元素类型选择器。

![](img/5b25f9189e795f9881b6a35947d0e23f_141.png)

了解特异性规则对于调试复杂的样式冲突非常重要。

## 更多 CSS 选择器 🔍

除了基本的选择器，CSS 还提供了许多其他强大的选择器，用于更精确地定位元素。

![](img/5b25f9189e795f9881b6a35947d0e23f_143.png)

![](img/5b25f9189e795f9881b6a35947d0e23f_145.png)

以下是几种常见的高级选择器：

![](img/5b25f9189e795f9881b6a35947d0e23f_147.png)

*   **后代选择器**：选择某个元素内部的所有特定后代元素，用空格表示。
    *   **示例**：`ul li { color: blue; }` 会选择 `ul` 内的所有 `li` 元素（包括嵌套的）。
*   **子元素选择器**：选择某个元素的直接子元素，用 `>` 表示。
    *   **示例**：`ul > li { color: blue; }` 只会选择作为 `ul` **直接子元素**的 `li`。
*   **属性选择器**：根据元素的属性及属性值来选择元素。
    *   **示例**：`a[href="https://facebook.com"] { color: red; }` 会选择 `href` 属性等于指定值的所有链接。
*   **伪类**：用于定义元素的特殊状态。
    *   **示例**：`button:hover { background-color: orange; }` 会在用户将鼠标悬停在按钮上时，将按钮背景色改为橙色。

![](img/5b25f9189e795f9881b6a35947d0e23f_149.png)

**悬停效果示例代码：**
```html
<style>
    button {
        width: 200px;
        height: 50px;
        font-size: 24px;
        background-color: green;
    }
    button:hover {
        background-color: orange;
    }
</style>
<body>
    <button>点击我</button>
</body>
```
