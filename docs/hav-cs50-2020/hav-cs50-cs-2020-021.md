## 🤖 什么是人工智能？

人工智能是计算机科学的一个分支，致力于创建能够执行通常需要人类智能的任务的系统。这包括玩游戏、识别手写、过滤垃圾邮件以及生成逼真的图像。与使用大量`if`和`else`语句的简单程序不同，真正的人工智能系统能够更灵活地推理和学习。

## 🌳 决策制定与决策树

上一节我们介绍了人工智能的目标，本节中我们来看看计算机如何做出决策。一种直观的方法是使用**决策树**。决策树通过一系列是/否问题来引导决策过程，最终得出一个行动方案。

![](img/f15e898e5a854caacbc93b2800ef3929_15.png)

例如，在一个简单的球拍游戏中，AI需要决定如何移动球拍来接球。其决策逻辑可以表示如下：

![](img/f15e898e5a854caacbc93b2800ef3929_17.png)

![](img/f15e898e5a854caacbc93b2800ef3929_19.png)

```
如果 球在球拍左侧:
    将球拍向左移动
否则 如果 球在球拍右侧:
    将球拍向右移动
否则:
    不移动球拍
```

以下是构建决策树的关键步骤：
1.  **识别关键问题**：确定影响决策的核心因素（例如，“球在左侧吗？”）。
2.  **定义行动**：为每个问题的答案指定相应的行动。
3.  **转化为代码**：将树状结构转化为`if-else`语句。

![](img/f15e898e5a854caacbc93b2800ef3929_21.png)

然而，对于更复杂的游戏（如井字棋），手动编写所有决策规则是不现实的。我们需要更强大的算法。

## ⚔️ 对抗性搜索与Minimax算法

![](img/f15e898e5a854caacbc93b2800ef3929_31.png)

在双方对抗的游戏中，AI需要预测对手的行动并做出最优反应。**Minimax算法**就是为此设计的。它将游戏结果量化为分数：己方获胜为`1`，平局为`0`，对手获胜为`-1`。

算法的核心思想是：
*   **最大化玩家（Max）**：选择能使最终分数**最大化**的走法。
*   **最小化玩家（Min）**：选择能使最终分数**最小化**的走法。

AI通过递归地模拟未来所有可能的走法序列，并为当前棋盘状态计算一个分数。伪代码如下：

```
function minimax(board, player):
    if 游戏结束(board):
        return 该局面的分数（1， 0， 或 -1）

    if player == 最大化玩家:
        最佳分数 = -∞
        for each 可能的走法 in board:
            新棋盘 = 执行走法(board, 走法)
            分数 = minimax(新棋盘, 最小化玩家)
            最佳分数 = max(最佳分数, 分数)
        return 最佳分数
    else:
        最佳分数 = +∞
        for each 可能的走法 in board:
            新棋盘 = 执行走法(board, 走法)
            分数 = minimax(新棋盘, 最大化玩家)
            最佳分数 = min(最佳分数, 分数)
        return 最佳分数
```

![](img/f15e898e5a854caacbc93b2800ef3929_33.png)

对于井字棋这类简单游戏，Minimax可以穷举所有可能并实现完美对战。但对于国际象棋等复杂游戏，可能走法数量巨大（例如前四步就有约2880亿种），无法穷举。

为了解决这个问题，我们引入了**深度限制Minimax**。算法不会搜索到游戏结束，而是在一定深度后停止，并使用一个**评估函数**来估算当前局面的优劣（例如，通过计算棋子价值和位置）。

## 🧭 搜索算法：寻找路径

除了游戏，搜索算法也用于解决路径查找问题，如地图导航。我们从最简单的算法开始。

![](img/f15e898e5a854caacbc93b2800ef3929_43.png)

![](img/f15e898e5a854caacbc93b2800ef3929_45.png)

### 深度优先搜索（DFS）

![](img/f15e898e5a854caacbc93b2800ef3929_47.png)

**深度优先搜索**的策略是沿着一条路径一直走到底，如果遇到死胡同就回溯到上一个岔路口尝试另一条路。

![](img/f15e898e5a854caacbc93b2800ef3929_49.png)

![](img/f15e898e5a854caacbc93b2800ef3929_51.png)

```
从起点A开始。
循环：
    如果到达终点B：成功。
    如果当前是死胡同：回溯到上一个选择点。
    否则：随机选择一个方向前进。
```

![](img/f15e898e5a854caacbc93b2800ef3929_53.png)

![](img/f15e898e5a854caacbc93b2800ef3929_55.png)

![](img/f15e898e5a854caacbc93b2800ef3929_57.png)

DFS能保证找到路径（如果存在），但找到的路径很可能不是最短的，且可能探索大量无关区域。

![](img/f15e898e5a854caacbc93b2800ef3929_59.png)

![](img/f15e898e5a854caacbc93b2800ef3929_61.png)

### 广度优先搜索（BFS）

![](img/f15e898e5a854caacbc93b2800ef3929_63.png)

![](img/f15e898e5a854caacbc93b2800ef3929_65.png)

**广度优先搜索**的策略是同时探索所有可能的方向，一层一层地向外扩展。

![](img/f15e898e5a854caacbc93b2800ef3929_67.png)

```
从起点A开始，将其放入队列。
循环：
    从队列中取出一个位置。
    如果该位置是终点B：成功。
    否则：将该位置所有未访问的相邻位置加入队列。
```

![](img/f15e898e5a854caacbc93b2800ef3929_69.png)

![](img/f15e898e5a854caacbc93b2800ef3929_71.png)

BFS总能找到从起点到终点的**最短路径**。然而，它需要探索大量节点，效率可能不高。

![](img/f15e898e5a854caacbc93b2800ef3929_73.png)

![](img/f15e898e5a854caacbc93b2800ef3929_75.png)

### 启发式搜索：利用额外知识

![](img/f15e898e5a854caacbc93b2800ef3929_77.png)

![](img/f15e898e5a854caacbc93b2800ef3929_79.png)

以上两种都是**无信息搜索**。我们可以通过引入**启发式**（一种估算某个状态好坏的方法）来改进搜索效率。例如，在迷宫中，可以使用**曼哈顿距离**（忽略障碍，只计算网格上的横向和纵向移动步数）来估算某个格子到目标的距离。

![](img/f15e898e5a854caacbc93b2800ef3929_81.png)

![](img/f15e898e5a854caacbc93b2800ef3929_83.png)

#### 贪婪最佳优先搜索

![](img/f15e898e5a854caacbc93b2800ef3929_85.png)

![](img/f15e898e5a854caacbc93b2800ef3929_87.png)

该算法总是选择**启发式值最小**（即看起来离目标最近）的节点进行扩展。

![](img/f15e898e5a854caacbc93b2800ef3929_89.png)

```
从起点A开始。
循环：
    如果到达终点B：成功。
    否则：从所有可选项中，选择启发式值最小的节点前进。
```

![](img/f15e898e5a854caacbc93b2800ef3929_91.png)

它通常比DFS和BFS更快，但启发式估计可能误导搜索，导致找不到最短路径甚至陷入死循环。

![](img/f15e898e5a854caacbc93b2800ef3929_93.png)

#### A*搜索

![](img/f15e898e5a854caacbc93b2800ef3929_95.png)

**A*搜索**结合了BFS的最优性保证和启发式搜索的效率。它评估一个节点的优先级时，同时考虑：
1.  `g(n)`：从起点到该节点的实际代价。
2.  `h(n)`：从该节点到终点的启发式估计代价。

优先级计算公式为：`f(n) = g(n) + h(n)`。算法总是优先扩展`f(n)`值最小的节点。

```
从起点A开始，计算其f值并入队。
循环：
    从队列中取出f值最小的节点。
    如果该节点是终点B：成功。
    否则：计算其所有邻居的f值，并入队。
```

![](img/f15e898e5a854caacbc93b2800ef3929_97.png)

![](img/f15e898e5a854caacbc93b2800ef3929_99.png)

![](img/f15e898e5a854caacbc93b2800ef3929_101.png)

![](img/f15e898e5a854caacbc93b2800ef3929_103.png)

只要启发式函数`h(n)`满足一定条件（如“可采纳性”，即不高估实际代价），A*搜索就能保证找到最短路径，并且通常比BFS搜索更少的节点。

![](img/f15e898e5a854caacbc93b2800ef3929_105.png)

## 🎯 机器学习：从经验中学习

上一节我们介绍了让AI“思考”的算法，本节中我们来看看让AI“学习”的技术。机器学习使计算机能够从数据或经验中改进性能。

### 强化学习

![](img/f15e898e5a854caacbc93b2800ef3929_113.png)

在**强化学习**中，AI代理通过与环境互动来学习。它执行行动，然后获得**奖励**（正面）或**惩罚**（负面），从而学会哪些行为能带来好结果。

![](img/f15e898e5a854caacbc93b2800ef3929_115.png)

例如，训练一个AI玩简单游戏：
1.  AI随机移动。
2.  如果碰到障碍（失败），获得负面反馈。
3.  如果到达目标（成功），获得正面反馈。
4.  经过多次尝试，AI学会避免导致失败的移动，重复导致成功的移动。

![](img/f15e898e5a854caacbc93b2800ef3929_117.png)

一个关键挑战是**探索与利用的权衡**：是应该利用已知的最佳行动，还是探索可能更好的新行动？**Epsilon贪婪策略**是一个常见解决方案：以概率`ε`随机探索，以概率`1-ε`利用已知最佳行动。

### 遗传算法

**遗传算法**模仿自然进化过程。我们不是直接编程一个智能体，而是创建一群随机生成的“候选解”，然后让它们“进化”。

以下是其基本流程：
1.  **初始化**：随机生成第一代候选解（例如，不同的驾驶策略）。
2.  **评估**：用**适应度函数**评估每个候选解的表现（例如，汽车行驶的距离）。
3.  **选择**：保留适应度最高的候选解。
4.  **繁殖**：通过复制、交叉（组合）和变异（随机改动）选中的候选解，产生新一代。
5.  **迭代**：重复步骤2-4，直到得到令人满意的解。

经过多代进化，种群的整体适应度会不断提高。

### 神经网络与深度学习

**神经网络**的灵感来源于人脑。它由大量互连的**人工神经元**（或单元）组成。每个连接都有一个**权重**，网络通过调整这些权重来学习输入到输出的映射关系。

![](img/f15e898e5a854caacbc93b2800ef3929_119.png)

![](img/f15e898e5a854caacbc93b2800ef3929_121.png)

一个简单的神经网络前向传播公式可以表示为：
`输出 = 激活函数( ∑ (输入_i * 权重_i) + 偏置 )`

![](img/f15e898e5a854caacbc93b2800ef3929_123.png)

![](img/f15e898e5a854caacbc93b2800ef3929_125.png)

![](img/f15e898e5a854caacbc93b2800ef3929_127.png)

**深度学习**指的是具有许多隐藏层的神经网络。通过向网络提供大量带标签的数据（例如，成千上万张标有数字的手写图片），并使用**反向传播**等算法调整权重，网络可以学会完成复杂任务，如图像识别、机器翻译和图像生成。

![](img/f15e898e5a854caacbc93b2800ef3929_129.png)

![](img/f15e898e5a854caacbc93b2800ef3929_131.png)

![](img/f15e898e5a854caacbc93b2800ef3929_132.png)

生成逼真人脸图像的AI，正是通过训练神经网络学习真实人脸图片的分布特征，然后从随机噪声开始，一步步“细化”出完整图像。

## 📚 总结

![](img/f15e898e5a854caacbc93b2800ef3929_138.png)

本节课中我们一起学习了人工智能的多个核心领域：
1.  **决策制定**：使用决策树将复杂决策分解为简单问题。
2.  **对抗性搜索**：使用Minimax算法在游戏中做出最优决策，并通过深度限制和评估函数处理复杂游戏。
3.  **路径搜索**：了解了DFS、BFS等无信息搜索，以及利用启发式的贪婪最佳优先搜索和A*搜索。
4.  **机器学习**：探讨了让计算机从经验中学习的三种范式：
    *   **强化学习**：通过奖励和惩罚信号学习行为策略。
    *   **遗传算法**：通过模拟自然选择的过程进化出解决方案。
    *   **神经网络**：通过模拟人脑结构，从大量数据中学习复杂的输入-输出关系。
