数字几何处理是数字信号处理在几何形状领域的延伸。与处理图像、音频和视频类似，我们希望对几何信号（即形状）进行上采样、下采样、重采样和滤波等操作。几何处理和网格算法是计算机图形学许多领域的基础构建模块，无论是渲染、动画还是模拟，都需要几何表示。

## 几何处理流程 🔄

![](img/4e9b8a9555c831aa01f264f90b511c35_21.png)

![](img/4e9b8a9555c831aa01f264f90b511c35_23.png)

![](img/4e9b8a9555c831aa01f264f90b511c35_27.png)

几何处理流程不像光栅化管线那样精确，但大致遵循以下步骤：首先从现实世界获取物理对象，通过扫描将其采样并转化为数字表示；然后在计算机中通过算法进行处理、编辑、变换和分析；最后，当我们对修改结果满意时，可以将其打印出来，赋予其新的功能。

以下是处理流程中可能涉及的一些核心任务：

![](img/4e9b8a9555c831aa01f264f90b511c35_29.png)

![](img/4e9b8a9555c831aa01f264f90b511c35_31.png)

*   **重建**：从点云等样本数据中重建出表面。
*   **上采样**：提高几何形状的分辨率，例如通过细分技术。
*   **下采样**：降低几何形状的分辨率，简化模型。
*   **重采样**：改变网格元素（如多边形）的分布和质量，而不一定改变分辨率。
*   **滤波**：去除噪声或增强特定特征。
*   **压缩**：减少存储大小，消除冗余数据。
*   **形状分析**：对模型进行分割、识别或匹配等高级分析。

## 什么是好的网格？ ✅

![](img/4e9b8a9555c831aa01f264f90b511c35_37.png)

在开始处理之前，我们需要明确什么是“好”的网格。一个好的网格应该能很好地近似原始形状。但这不仅仅是顶点位置接近表面那么简单。

![](img/4e9b8a9555c831aa01f264f90b511c35_39.png)

*   **法线近似**：网格的法线也应接近光滑表面的法线。如果法线近似良好，许多其他几何属性（如曲率）也会得到良好保持。
*   **元素形状**：对于许多算法（特别是数值模拟），三角形的形状至关重要。通常，接近等边的“胖”三角形比又长又瘦的“瘦长”三角形表现更好。一个有用的质量标准是 **Delaunay** 条件：在平面三角化中，每个三角形的外接圆内不包含其他顶点。Delaunay三角化能最大化最小角，并提供最平滑的插值。
*   **顶点度数**：在三角形网格中，规则的顶点度数（通常为6）也是理想特性。它有助于获得更规则的三角形形状，并且在细分等操作中能产生更平滑的结果。然而，在曲面上无法让所有顶点都达到规则度数，但我们可以尽量减少不规则顶点的数量。

![](img/4e9b8a9555c831aa01f264f90b511c35_41.png)

## 上采样：细分 🆙

![](img/4e9b8a9555c831aa01f264f90b511c35_43.png)

![](img/4e9b8a9555c831aa01f264f90b511c35_45.png)

上采样网格的基本工具是细分。其基本思想是：反复将每个元素分割成更小的部分（改变连接关系），然后根据相邻顶点位置的加权平均来更新顶点位置（改变几何形状）。

![](img/4e9b8a9555c831aa01f264f90b511c35_47.png)

细分方案有多种，主要考虑因素包括：
*   **插值 vs 近似**：极限曲面是精确穿过原始顶点，还是仅仅接近它们？
*   **连续性**：极限曲面具有多少阶可微性（光滑程度）？

![](img/4e9b8a9555c831aa01f264f90b511c35_49.png)

![](img/4e9b8a9555c831aa01f264f90b511c35_51.png)

以下是两种常见的细分方案：

![](img/4e9b8a9555c831aa01f264f90b511c35_53.png)

*   **Catmull-Clark细分（适用于四边形网格）**：通过插入面点和边点来细分，所有面最终都会变成四边形。新顶点位置的更新规则涉及面点、边点和原始顶点的加权组合。公式为：`新顶点坐标 = (Q + 2R + (n-3)S) / n`，其中 `n` 是顶点度数，`Q` 是周围面点的平均，`R` 是周围边点的平均，`S` 是原始顶点位置。
*   **Loop细分（适用于三角形网格）**：将每个三角形分割为四个。新边点位置是相邻四个顶点的加权平均（端点权重为3/8，对点权重为1/8）。旧顶点的新位置是其自身与其所有邻接顶点的加权平均。对于度数为 `n` 的顶点，权重 `u = 3/(8n)`（当 `n=3` 时 `u=3/16`），新位置 = `(1 - n*u) * 原始顶点 + u * 所有邻接顶点之和`。

![](img/4e9b8a9555c831aa01f264f90b511c35_55.png)

![](img/4e9b8a9555c831aa01f264f90b511c35_57.png)

细分可以通过基本的边操作（如分裂和翻转）来实现。例如，Loop细分可以先分裂所有边，然后翻转连接新顶点和旧顶点的边，最后更新顶点位置。

![](img/4e9b8a9555c831aa01f264f90b511c35_59.png)

## 下采样：简化 🔽

![](img/4e9b8a9555c831aa01f264f90b511c35_61.png)

下采样网格的一种流行方法是基于边折叠的迭代简化。这是一种贪心算法：为每条边分配一个折叠“代价”，然后反复折叠代价最小的边，直到达到目标网格元素数量。

一个非常有效的代价度量是 **二次误差度量**。其核心思想是：为网格的每个区域（如顶点）维护一个二次函数（用一个4x4矩阵 `K` 表示），该函数衡量任意点到该区域所有三角形所在平面的平方距离之和。

![](img/4e9b8a9555c831aa01f264f90b511c35_63.png)

**距离计算**：点 `x` 到法向量为 `n`、经过点 `p` 的平面的平方距离，在齐次坐标下可以表示为二次型 `u^T K u`，其中 `u = [x, y, z, 1]^T`，`K = v v^T`，`v = [a, b, c, d]^T`（`[a,b,c]` 是单位法向量 `n`，`d = -n·p`）。

![](img/4e9b8a9555c831aa01f264f90b511c35_65.png)

![](img/4e9b8a9555c831aa01f264f90b511c35_67.png)

**二次误差度量**：对于一组平面，总的平方距离就是它们对应矩阵 `K_i` 的和。因此，我们可以通过累加矩阵来高效地表示到复杂区域的误差。

**边折叠算法步骤**：
1.  **初始化**：为每个三角形计算其平面的 `K` 矩阵。为每个顶点 `i` 计算其二次误差 `Q_i`，即其所属所有三角形 `K` 矩阵之和。为每条边 `(i,j)` 计算其二次误差 `Q_ij = Q_i + Q_j`。
2.  **计算代价**：对于每条边，找到使 `Q_ij(x)` 最小的新顶点位置 `x*`（通过求解 `∇Q_ij = 0` 得到）。该最小值 `Q_ij(x*)` 即为折叠该边的代价。
3.  **迭代折叠**：
    *   找到当前代价最小的边 `(i,j)`。
    *   将其折叠到最优位置 `x*`。
    *   新顶点的二次误差设为 `Q_new = Q_i + Q_j`。
    *   更新所有与新顶点相连的边的代价。
4.  **注意事项**：折叠后需检查是否导致三角形法向翻转，若是则放弃此次折叠。

![](img/4e9b8a9555c831aa01f264f90b511c35_69.png)

## 重采样：改善网格质量 ✨

![](img/4e9b8a9555c831aa01f264f90b511c35_71.png)

![](img/4e9b8a9555c831aa01f264f90b511c35_73.png)

有时我们不想改变分辨率，而是想改善网格元素的质量（如形状、顶点度数）。以下是一些简单有效的方法：

![](img/4e9b8a9555c831aa01f264f90b511c35_75.png)

*   **Delaunay化（平面）**：对于平面三角化，可以通过贪婪地翻转边来使其成为Delaunay三角化。判断条件是：对于每条边，如果其对两个三角形的对角之和大于180度（π），则翻转该边。
*   **改善顶点度数**：同样可以使用边翻转操作。计算翻转前后相关顶点度数与理想度数（如6）的偏差总和，如果翻转后总偏差减小，则执行翻转。
*   **改善三角形形状**：可以通过将每个顶点移动到其所有邻接顶点的平均位置（拉普拉斯平滑）来使三角形更接近等边。在曲面上操作时，为了保持几何形状，应只沿切向移动顶点，即从移动向量中减去法向分量。

![](img/4e9b8a9555c831aa01f264f90b511c35_77.png)

一个结合了多种操作的重采样算法可以产生形状均匀、质量高的网格：
1.  **分裂** 过长的边（如长度 > 4/3 * 目标长度）。
2.  **折叠** 过短的边（如长度 < 4/5 * 目标长度）。
3.  **翻转** 边以优化顶点度数。
4.  **切向平滑** 顶点位置。
重复以上步骤直到收敛。

![](img/4e9b8a9555c831aa01f264f90b511c35_79.png)

## 重采样的风险与几何查询 ⚠️

![](img/4e9b8a9555c831aa01f264f90b511c35_81.png)

反复对信号（包括几何）进行重采样会导致质量下降，因为信息在每次采样和重建中都会丢失。一个有趣的想法是：在处理后，将新网格的顶点投影回原始网格表面。但这引出了一个核心问题：**如何高效地找到空间中一点到一个网格表面的最近点？**

![](img/4e9b8a9555c831aa01f264f90b511c35_83.png)

![](img/4e9b8a9555c831aa01f264f90b511c35_85.png)

![](img/4e9b8a9555c831aa01f264f90b511c35_87.png)

这涉及到一系列几何查询问题：点是否在表面内/外？两个三角形是否相交？我们学过的隐式/显式表示或半边数据结构是否能简化这些任务？如何加速对大网格的查询？这些将是下一节课要探讨的有趣话题。

![](img/4e9b8a9555c831aa01f264f90b511c35_89.png)
