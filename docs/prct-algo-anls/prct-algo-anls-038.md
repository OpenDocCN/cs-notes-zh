# 038：生日问题 🎂

![](img/873b25ab7274a1eb57a76e0465e59741_0.png)

在本节课中，我们将学习一个经典的组合问题——生日问题。我们将使用解析组合学的方法来分析它，并计算出在随机情况下，平均需要询问多少人才能找到两个生日相同的人。

---

## 生日问题概述

生日问题是一个著名的概率论问题。假设有一群人，我们逐一询问每个人的生日。问题是：平均需要询问多少人，才能找到两个生日相同的人？直觉上，可能需要很多人，但通过数学分析，我们会发现这个数字比想象中小得多。

## 建模为“球与瓮”问题

我们可以将这个问题建模为一个“球与瓮”模型。假设有 `M` 个瓮（对应一年中的 `365` 天），每个“球”（对应一个人）被随机放入一个瓮中（对应其生日）。问题是：平均需要放入多少个球，才能出现某个瓮中有两个球的情况？

## 使用符号方法分析

为了分析这个问题，我们首先需要定义“生日序列”。

### 生日序列的定义

一个生日序列是一个长度为 `n` 的单词（或序列），其中每个字符（对应一个瓮）最多出现一次。换句话说，在序列中，没有重复的字符。

### 生日序列的构造

构造生日序列很简单：我们有 `M` 个瓮，每个瓮要么是空的，要么恰好有一个球。因此，整个序列就是这 `M` 种状态的一个排列。

使用符号方法，这直接转化为指数生成函数（EGF）。每个瓮的状态（空或有球）的EGF是 `(1 + z)`。由于有 `M` 个独立的瓮，整个生日序列类的EGF就是每个瓮EGF的乘积。

因此，**生日序列的指数生成函数公式**为：
```
B(z) = (1 + z)^M
```

### 计数序列

从生成函数中，我们可以提取系数来计数长度为 `n` 的生日序列数量。`z^n` 项的系数乘以 `n!` 即为我们所需的数量。

具体计算如下：
- `(1+z)^M` 中 `z^n` 项的系数是 `C(M, n)`，即组合数 `M choose n`。
- 因此，长度为 `n` 的生日序列数量是 `n! * C(M, n)`。
- 这可以简化为 `M! / (M-n)!`，或者等价地，`M * (M-1) * ... * (M-n+1)`。

**公式**：长度为 `n` 且无重复字符的序列数量为：
```
P(M, n) = M! / (M-n)!
```

## 解决生日问题

有了无重复序列的计数，我们就可以解决原始的生日问题了。

### 无重复的概率

在随机情况下，所有可能的 `M^n` 个长度为 `n` 的序列是等概率的。因此，前 `n` 个人生日各不相同的概率，等于无重复序列的数量除以总序列数。

**公式**：前 `n` 次询问无重复生日的概率为：
```
P(无重复) = [M! / (M-n)!] / M^n
```

另一种理解是：这个概率也等于“第一次出现重复生日的询问位置 `T` 大于 `n`”的概率，即 `P(T > n)`。

### 计算首次重复的期望位置

我们关心的是首次出现重复的期望询问次数 `E[T]`。根据期望值的定义，它可以表示为所有 `P(T > n)` 的和（`n` 从 `0` 到 `M-1`，因为最晚在第 `M` 次一定会出现重复）。

**公式**：首次重复位置的期望值为：
```
E[T] = Σ_{n=0}^{M-1} P(T > n) = Σ_{n=0}^{M-1} [M! / ((M-n)! * M^n)]
```

这个求和式与我们在课程中讨论过的 Ramanujan Q 函数有关。通过渐近分析（或使用积分近似），可以得到一个简洁的结果。

**核心结果公式**：当 `M` 较大时，首次重复的期望位置近似为：
```
E[T] ≈ √(πM / 2)
```

## 应用到实际生日问题

现在，我们将这个结果应用到最初的生日问题上。一年有 `M = 365` 天。

计算过程如下：
```
E[T] ≈ √(π * 365 / 2) ≈ √(573.25) ≈ 23.94
```

因此，**平均只需要询问大约 24 个人**，就能找到两个生日相同的人。这个数字远小于通常的直觉（183 或 365）。

## 总结

![](img/873b25ab7274a1eb57a76e0465e59741_2.png)

本节课中，我们一起学习了生日问题及其解析组合学解法。

1.  **问题建模**：我们将寻找相同生日的问题，转化为“球与瓮”模型中寻找首次碰撞的问题。
2.  **符号方法**：我们定义了“生日序列”，并利用符号方法推导出其指数生成函数 `B(z) = (1+z)^M`。
3.  **概率计算**：我们计算了前 `n` 次询问无重复生日的概率 `P(T > n) = M! / ((M-n)! * M^n)`。
4.  **期望值**：通过对所有无重复概率求和，并利用渐近分析，我们得到了首次出现重复的期望位置公式 `E[T] ≈ √(πM / 2)`。
5.  **实际应用**：代入 `M=365`，我们得出平均只需询问约 **24 人** 就能找到生日相同的两人。
