# 算法分析：11：用生成函数计数 🧮

![](img/c0c6e97f19bae804fb50105ec7581f1d_0.png)

在本节课中，我们将学习如何直接使用生成函数进行计数。这是引导我们进入分析组合学中生成函数核心作用的第一步。我们将从一个更组合、更形式化的视角来看待生成函数，并通过具体例子展示其强大功能。

## 概述

我们将定义一个组合对象类，每个对象都有一个关联的“大小”函数。生成函数将是该类中所有对象的和。我们的目标是得到一个生成函数必须满足的方程，然后进行分析。通过这种对应关系，得到方程的过程将更加自然和基础。

## 从组合结构到生成函数方程

让我们看一个具体例子：二叉树。

设 **T** 是所有二叉树的集合。我们定义一个大小函数 `|t|`，对于任何二叉树 `t`，其大小等于该树内部节点的数量。

我们感兴趣的是计数序列 `T_n`，即大小为 `n`（具有 `n` 个内部节点）的二叉树数量。

我们定义生成函数为：
`T(z) = Σ_{t ∈ T} z^{|t|}`

这个定义意味着，每个树 `t` 单独为生成函数贡献一项 `z^{|t|}`。由于所有大小为 `n` 的树都贡献给 `z^n` 的系数，因此 `z^n` 的系数正是 `T_n`。这与我们之前按大小收集项的观点是等价的。

这种方法的有用之处在于，我们可以通过定义组合结构的方式来得到我们想要的生成函数方程。

对于二叉树，其定义是：一个二叉树要么是空树，要么是一个根节点，其左子树和右子树都是二叉树。

设左子树有 `T_L` 个节点，右子树有 `T_R` 个节点。那么整个树的大小是 `|T_L| + |T_R| + 1`。

根据这个递归分解，生成函数 `T(z)` 必须满足：
`T(z) = 1 + z * T(z) * T(z) = 1 + z T(z)^2`

这里的 `1` 对应空树。这个方程直接从二叉树的定义翻译而来，比单纯从计数角度推导更直接。

另一种理解方式是，生成函数 `T(z)` 中的每一项对应一棵具体的树。当我们计算 `T(z)^2` 时，相当于取两棵树进行组合，这正好对应了二叉树的构造过程。因此，方程 `T(z) = 1 + z T(z)^2` 在符号层面也成立。

## 引入成本参数

除了大小，我们通常还关心结构的其他属性，即“成本”。生成函数方法可以统一地研究这些参数。

假设我们有一个组合对象类 **P**，每个对象 `p ∈ P` 有一个大小 `|p|` 和一个成本 `cost(p)`。

我们已知计数函数 `P_n`，即大小为 `n` 的对象数量。

我们感兴趣的是平均成本，即大小为 `n` 的对象的期望成本。根据定义：
`期望成本 = (Σ_{所有大小为n的对象} cost(p)) / P_n`

我们称分子为“累积成本” `C_n`。

那么，如何用生成函数计算累积成本呢？

我们定义累积成本生成函数为：
`C(z) = Σ_{p ∈ P} cost(p) * z^{|p|}`

通过代数运算，`z^n` 在 `C(z)` 中的系数正是所有大小为 `n` 的对象的成本之和，即 `C_n`。

因此，要计算期望成本，我们需要解决两个生成函数计数问题：
1.  计数生成函数 `P(z)`，以得到 `P_n`。
2.  累积成本生成函数 `C(z)`，以得到 `C_n`。

然后计算 `C_n / P_n` 即可。

## 示例一：随机比特串中的1比特数

让我们用一个简单例子热身：随机比特串中1比特的期望数量。

*   **对象类 B**：所有比特串。
*   **大小函数**：比特串的长度。
*   **成本函数**：比特串中1比特的数量 `ones(b)`。
*   **已知**：大小为 `n` 的比特串数量 `B_n = 2^n`。

首先，计数生成函数为：
`B(z) = Σ_{b ∈ B} z^{|b|} = Σ_{n≥0} 2^n z^n = 1 / (1 - 2z)`

接下来，我们推导累积成本生成函数 `C(z)`。我们利用比特串的递归定义：一个比特串要么是空串，要么是 `0` 或 `1` 后面跟着一个比特串。

对于累积成本，我们有：
`C(z) = Σ_{b ∈ B} ones(b) * z^{|b|}`
`= Σ_{b‘ ∈ B} [ ones(”0“ + b’) * z^{1+|b‘|} + ones(”1“ + b’) * z^{1+|b‘|} ]`
`= Σ_{b‘ ∈ B} [ ones(b’) * z^{1+|b‘|} + (1 + ones(b’)) * z^{1+|b‘|} ]`
`= z * C(z) + z * B(z) + z * C(z)`
`= z * B(z) + 2z * C(z)`

整理方程：
`C(z) - 2z C(z) = z B(z)`
`C(z) (1 - 2z) = z / (1 - 2z)`
`C(z) = z / (1 - 2z)^2`

现在，我们从 `B(z)` 和 `C(z)` 中提取系数：
*   `B_n = 2^n`
*   `C_n` 是 `z / (1 - 2z)^2` 中 `z^n` 的系数。我们知道 `1/(1-2z)^2 = Σ_{n≥0} (n+1) 2^n z^n`，所以 `C_n = n * 2^{n-1}`。

因此，期望成本为：
`C_n / B_n = (n * 2^{n-1}) / 2^n = n/2`

正如我们所料，随机比特串中1比特的期望数量是长度的一半。

## 示例二：二叉树中的叶子节点数

现在来看一个更复杂、也更具实际意义的问题：随机二叉树中叶子节点（两个子节点都是外部的内部节点）的期望数量。这在计算机表示中关乎空间节省。

*   **对象类 T**：所有二叉树（按内部节点计大小）。
*   **大小函数**：内部节点数。
*   **成本函数**：树中的叶子节点数 `leaves(t)`。
*   **已知**：大小为 `n` 的二叉树数量是卡特兰数 `T_n = (1/(n+1)) * C(2n, n)`。

计数生成函数 `T(z)` 满足：
`T(z) = 1 + z T(z)^2`

我们的目标是找到累积成本生成函数 `C(z) = Σ_{t ∈ T} leaves(t) * z^{|t|}`。

我们再次利用二叉树的递归分解来建立 `C(z)` 的方程。
一棵二叉树要么是单个叶子节点（一个内部节点，无子节点），要么是一个根节点连接着左子树和右子树。

*   **情况1（单个叶子）**：贡献为 `1（叶子数） * z^1（大小） = z`。
*   **情况2（根节点+左右子树）**：对于左子树 `t_L` 和右子树 `t_R`，整棵树的叶子数等于 `leaves(t_L) + leaves(t_R)`。树的大小为 `|t_L| + |t_R| + 1`。

因此，`C(z)` 可以写为：
`C(z) = z + Σ_{t_L, t_R ∈ T} [leaves(t_L) + leaves(t_R)] * z^{|t_L|+|t_R|+1}`

由于对 `t_L` 和 `t_R` 的求和是独立的，我们可以拆分：
`= z + z * [ Σ_{t_L} leaves(t_L) z^{|t_L|} * Σ_{t_R} z^{|t_R|} + Σ_{t_L} z^{|t_L|} * Σ_{t_R} leaves(t_R) z^{|t_R|} ]`
`= z + z * [ C(z) * T(z) + T(z) * C(z) ]`
`= z + 2z T(z) C(z)`

于是我们得到方程：
`C(z) = z + 2z T(z) C(z)`

我们知道 `T(z)` 满足 `T(z) = 1 + z T(z)^2`，由此可以解出 `T(z) = (1 - sqrt(1-4z)) / (2z)`。代入 `C(z)` 的方程求解：

`C(z) - 2z T(z) C(z) = z`
`C(z) [1 - 2z T(z)] = z`
`C(z) = z / [1 - 2z T(z)]`

将 `T(z)` 的表达式代入分母：
`1 - 2z T(z) = 1 - 2z * (1 - sqrt(1-4z))/(2z) = 1 - (1 - sqrt(1-4z)) = sqrt(1-4z)`

因此：
`C(z) = z / sqrt(1-4z)`

现在，我们提取系数。`1 / sqrt(1-4z)` 是广义二项式定理的标准形式：
`1 / sqrt(1-4z) = Σ_{n≥0} C(2n, n) z^n`

所以：
`C(z) = z * Σ_{n≥0} C(2n, n) z^n = Σ_{n≥1} C(2(n-1), n-1) z^n`

因此，累积成本 `C_n = C(2n-2, n-1)`。

大小为 `n` 的二叉树总数 `T_n` 是卡特兰数 `C(2n, n) / (n+1)`。

期望叶子数为：
`C_n / T_n = [C(2n-2, n-1)] / [C(2n, n) / (n+1)] = (n+1) * C(2n-2, n-1) / C(2n, n)`

利用组合数恒等式化简：
`C(2n, n) = (2n)/(n) * C(2n-1, n-1) = 2 * C(2n-1, n-1)`，但更直接的计算可得：
`C_n / T_n = n(n+1) / (2(2n-1))`

当 `n` 很大时，其渐近值为 `~ n/4`。

**结论**：在一个随机的 `n` 节点二叉树中，叶子节点的平均数量约为 `n/4`。这是一个通过累积生成函数求解参数平均值的典型例子。

## 总结

本节课中，我们一起学习了如何直接使用生成函数进行组合计数。
1.  我们首先展示了如何从组合对象的递归定义直接推导出其生成函数方程，以二叉树为例。
2.  接着，我们引入了“成本”参数的概念，并定义了累积成本生成函数 `C(z)` 来计算所有对象的成本总和。
3.  通过“随机比特串中1的个数”和“二叉树中叶子节点数”两个例子，我们演示了如何利用递归分解同时得到计数生成函数 `P(z)` 和累积成本生成函数 `C(z)` 的方程，并通过求解和提取系数，最终计算出参数的期望值。
