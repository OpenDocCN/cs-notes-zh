在本节课中，我们将继续学习排列参数的例子，以加深对基本方法的理解，并进一步熟悉排列的性质。我们将分析“单循环数”和“逆序数”这两个参数，并了解它们在算法分析中的应用。

## 单循环数

上一节我们讨论了循环数和左右最小值，本节我们来看看排列中的“单循环数”。一个“单循环”是指一个元素映射到自身的循环，例如在排列 `(1)(2 3)` 中，元素 `1` 构成了一个单循环。

我们的目标是计算一个大小为 `n` 的随机排列中，单循环的平均数量。通过观察小例子（n=2, 3, 4），我们发现平均值似乎总是 `1`。接下来，我们将用生成函数法来证明这一点。

我们使用与计算循环数时相同的构造方法：对于一个大小为 `p` 的排列，通过将新元素 `p+1` 插入到每个可能位置，生成 `p+1` 个大小为 `p+1` 的新排列。

分析的关键在于计算新排列集合中的单循环总数。设原排列 `P` 有 `cycle1(P)` 个单循环。在构造出的新排列中：
*   每个原排列的单循环在新排列中依然存在，贡献了 `(p+1) * cycle1(P)` 个单循环。
*   当我们把新元素 `p+1` 单独作为一个单循环插入时，会为每个新排列增加一个单循环，总共增加 `p+1` 个。
*   但是，如果原排列中某个位置 `i` 本身是单循环，当我们把 `p+1` 插入到 `i` 之前时，`i` 就不再是单循环了（它和 `p+1` 形成了一个二元循环）。这种情况在原排列的每个单循环上都会发生一次，因此总共需要减去 `cycle1(P)` 个单循环。

因此，新排列集合中的单循环总数为：`(p+1) * cycle1(P) + (p+1) - cycle1(P) = p * cycle1(P) + (p+1)`。

设 `B(z)` 是单循环数的指数生成函数（EGF）。应用上述构造，我们得到以下方程：
`B(z) = z * B'(z) + 1/(1-z)`
这里，对 `z^(p+1)/(p+1)!` 求导消去了分母中的 `(p+1)`，得到了 `z^p/p!`，从而引入了导数项 `B'(z)`。

解这个微分方程：
`B'(z) = 1/(1-z)^2`
积分得：
`B(z) = 1/(1-z)`
`1/(1-z)` 展开后 `z^n` 的系数是 `1`。因此，**大小为 `n` 的随机排列中，单循环的平均数量为 1**。

一个直观的应用是“学生与房间”问题：如果每个人随机进入一个房间，平均恰好有一个人进入自己原来的房间。

为了巩固对此方法的理解，你可以尝试计算随机排列中二元循环（或更一般的 r-元循环）的平均数量，答案分别是 `1/2` 和 `1/r`。

## 逆序数

现在，我们转向另一个重要参数：**逆序数**。一个逆序是指排列中一个“顺序错误”的数对，更准确地说，对于每个位置，我们计算其左侧比它大的元素个数，所有这些计数的总和就是逆序数。

例如，在排列 `[1, 2, 4, 3]` 中，只有 `(4,3)` 是逆序，所以逆序数为 `1`。在 `[2, 1, 4, 3]` 中，逆序对有 `(2,1)` 和 `(4,3)`，逆序数为 `2`。

计算小例子（n=3, 4）的逆序总数并求平均，我们发现平均值似乎在增长。逆序数的一个直接应用是分析**插入排序**算法。

插入排序的工作方式如下：它遍历数组，对于每个位置 `i`，确保 `i` 之前的元素已排序。当处理一个新元素时，它不断与左侧更大的元素交换，直到到达正确位置。**交换的次数正好等于排列的逆序数**。因此，了解随机排列的逆序数对于分析插入排序的性能至关重要。在实践中，当递归排序算法（如快速排序、归并排序）处理小规模子数组时，通常会切换到更高效的插入排序，这需要我们对逆序数有准确的估计。

为了分析逆序数，我们使用一种新的构造方法：从一个大小为 `p` 的排列出发，通过将新元素 `p+1`（这里我们使用 `n`）插入到所有 `p+1` 个可能的位置（从最右到最左），来生成大小为 `p+1` 的新排列。

这个构造的关键观察是：当我们将新元素从右向左移动插入时，每向左移动一个位置，就会为所有位于它右侧的元素增加一个逆序（因为新元素现在位于它们左侧且更大）。因此，第一个插入位置（最右）不增加新逆序，第二个位置增加1个新逆序，第三个增加2个，以此类推，直到最左位置增加 `p` 个新逆序。

设原排列 `P` 的逆序数为 `inv(P)`。那么，新排列集合中的逆序总数为：
*   所有原排列的逆序数被复制了 `p+1` 次：`(p+1) * inv(P)`。
*   额外增加的逆序数总和为：`0 + 1 + 2 + ... + p = p(p+1)/2`。

因此，总数为：`(p+1) * inv(P) + p(p+1)/2`。

设 `B(z)` 是逆序数的指数生成函数。应用构造并整理，我们得到方程：
`B(z) = z * B(z) + (1/2) * z^2 / (1-z)^2`
这里，`z * B(z)` 对应第一项，`(1/2) * z^2 / (1-z)^2` 对应第二项（求和 `k*z^k` 的生成函数是 `z/(1-z)^2`，再乘以 `z/2`）。

解这个方程：
`B(z) = (1/2) * z^2 / ((1-z)^3)`
我们知道 `1/(1-z)^3` 的展开式中 `z^n` 的系数是 `C(n+2, 2) = (n+2)(n+1)/2`。因此，`B(z)` 中 `z^n` 的系数（即逆序数的总和）为 `(1/2) * (n-1)n/2 = n(n-1)/4`。由于有 `n!` 个排列，**平均逆序数**为：
`平均逆序数 = [n(n-1)/4] / n! * n! = n(n-1)/4`

所以，大小为 `n` 的随机排列中，平均逆序数为 `n(n-1)/4`。这验证了我们从 n=3 和 n=4 观察到的情况。

## 总结

![](img/4f699116cb1c4bc976bd896436638cf6_2.png)

本节课中我们一起学习了排列的两个重要参数：单循环数和逆序数。
*   我们证明了**单循环的平均数量恒为 1**，这与直观的“学生房间”问题相符。
*   我们推导出**逆序数的平均值为 n(n-1)/4**，并解释了它在分析**插入排序**算法性能中的关键作用。
*   我们再次运用了生成函数和组合构造（插入新元素）这一强大方法，展示了如何处理不同的参数和递归关系。
