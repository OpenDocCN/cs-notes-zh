在本节课中，我们将学习各种不同类型的递归关系。这些类型比我们之前看到的线性一阶递归更为复杂。了解这些类型，有助于我们应对在算法分析中可能遇到的各种递归问题。

## 线性一阶递归

上一节我们介绍了最简单的递归类型。本节中，我们来看看更复杂的类型。

最简单的类型是**线性一阶递归**。
*   “线性”意味着方程中不涉及未知项（如 `a_n` 和 `a_{n-1}`）之间的乘法或除法，仅使用不依赖于未知项的系数。
*   “一阶”意味着方程右侧只包含一个前项，例如 `a_{n-1}`。

## 非线性一阶递归

接下来，我们看看非线性一阶递归。这类递归在数值计算中很常见。

一个著名的例子是用于计算函数根的**牛顿法**。例如，计算 `√2` 的牛顿法定义如下：
1.  从一个初始估计值开始，例如 `a_0 = 2`。
2.  使用公式 `a_n = (a_{n-1} + 2 / a_{n-1}) / 2` 计算下一个估计值。

这个公式定义了一个序列。如果当前值接近 `√2`，那么下一个值会更接近。实际上，牛顿法具有二次收敛性，每次迭代的有效数字位数大约会翻倍。虽然分析这类递归算法本身也是算法分析，但其递归性质与本课程主要讨论的类型有较大不同。

以下是计算 `√2` 的示例代码：
```python
def sqrt_newton(two, initial_estimate=2.0, iterations=10):
    a = initial_estimate
    for i in range(iterations):
        a = (a + two / a) / 2.0
        print(a)
    return a
```

## 高阶线性递归

现在，我们转向高阶线性递归。这类递归在算法分析中经常出现，我们将在后续课程中系统性地学习如何使用生成函数等方法求解。

这是一个**二阶线性常系数递归**的例子，类似于斐波那契数列：
```
a_n = 5 * a_{n-1} - 6 * a_{n-2}, 其中 a_0 = 0, a_1 = 1
```
直接使用“望远镜法”求解这类递归会很困难。一种求解思路是假设解的形式为 `a_n = x^n`。

将 `a_n = x^n` 代入原递归式，得到：
```
x^n = 5 * x^{n-1} - 6 * x^{n-2}
```
两边同时除以 `x^{n-2}`，得到一个二次方程：
```
x^2 = 5x - 6  =>  x^2 - 5x + 6 = 0
```
解这个二次方程，得到两个根：`x = 2` 和 `x = 3`。因此，`2^n` 和 `3^n` 都满足递归关系。通解是这两个特解的线性组合：
```
a_n = C0 * 3^n + C1 * 2^n
```
最后，利用初始条件 `a_0 = 0` 和 `a_1 = 1` 求解常数 `C0` 和 `C1`：
```
a_0 = C0 + C1 = 0
a_1 = 3*C0 + 2*C1 = 1
```
解方程组得 `C0 = 1`, `C1 = -1`。因此，该递归的闭合形式解为：
```
a_n = 3^n - 2^n
```
这种方法可以系统性地应用于任何二阶线性常系数递归。

## 斐波那契数列示例

斐波那契数列是另一个经典的二阶线性常系数递归：
```
F_n = F_{n-1} + F_{n-2}, 其中 F_0 = 0, F_1 = 1
```
同样，假设 `F_n = x^n`，代入得到特征方程：
```
x^2 = x + 1  =>  x^2 - x - 1 = 0
```
解这个方程，得到两个根：
```
φ = (1 + √5) / 2 （黄金比例）
ψ = (1 - √5) / 2
```
因此，通解为：
```
F_n = C0 * φ^n + C1 * ψ^n
```
利用初始条件 `F_0 = 0` 和 `F_1 = 1` 求解常数：
```
F_0 = C0 + C1 = 0
F_1 = C0*φ + C1*ψ = 1
```
解得 `C0 = 1/√5`, `C1 = -1/√5`。因此，斐波那契数列的闭合形式解（比内公式）为：
```
F_n = (φ^n - ψ^n) / √5
```

## 其他递归类型

除了上述类型，递归关系还有其他重要分类：

以下是其他几种重要的递归类型：
*   **非线性高阶递归**：方程右侧包含未知项的非线性组合（如乘法、平方根等）。这类递归容易描述和编程计算，但数学分析可能极具挑战性。
*   **变系数递归**：系数是 `n` 的函数，而非常数。这类递归在算法分析中大量出现。
*   **高阶递归**：项依赖于两个以上前项。快速排序的平均情况分析就是一个“全历史”递归的例子，序列中的每一项都依赖于之前的所有项。
*   **分治递归**：这是算法分析中特别常见的一类，`a_n` 依赖于序列中大约中间位置的项。我们将在本课程后面详细讨论。

对于高阶常系数线性递归，现代通常使用符号数学软件（如 Sage、Mathematica、Maple）来计算特征根并求解。此外，需要注意特征根可能出现重根或复数的情况，这些将在后续关于生成函数的讨论中详细处理。
