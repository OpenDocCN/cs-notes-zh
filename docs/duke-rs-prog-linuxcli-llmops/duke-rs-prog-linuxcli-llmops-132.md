# 132：运行PyTorch Hello World 🚀

![](img/76b35eca3a0eb7dfc0e6ecdb11d1ed56_0.png)

在本节课中，我们将学习如何在Rust项目中使用PyTorch的Rust绑定。我们将从一个简单的“Hello World”示例开始，了解如何创建项目、添加依赖以及执行基本的张量操作。

---

## 概述

PyTorch的C++ API的Rust绑定非常令人兴奋，因为它允许你使用Rust这样优雅的语言直接进入PyTorch的低层部分。让我们开始看看具体如何操作。

首先，这里有一个“Hello World”示例。但在运行它之前，我们先深入了解一下必要的准备工作。

---

## 项目初始化与依赖配置

要开始使用，首先需要使用`cargo new`命令创建一个新的项目骨架结构。

以下是创建项目所需的步骤：

1.  **创建新项目**：使用`cargo new`命令。
2.  **配置依赖**：在`Cargo.toml`文件中添加`tch` crate作为依赖。Cargo的一个优点是能轻松检查最新版本。我们可以确认已使用最新版本。
3.  **编写主函数**：在`main.rs`文件中，我们可以创建“Hello World”张量并开始使用它。

实际上，只需几行代码就能开始。

---

## 代码解析与运行

让我们逐步解析代码。首先，我们声明一个类型为`Tensor`的变量`t`。

```rust
let t = ...;
```

![](img/76b35eca3a0eb7dfc0e6ecdb11d1ed56_2.png)

接着，我们创建一个切片，并将切片内的所有元素乘以2。

```rust
// 假设的切片操作和乘法
let result = t.slice(/* 参数 */).mul(2);
```

得到结果后，我们将其打印出来。

```rust
println!("{:?}", result);
```

要执行以上所有操作，我们只需在终端中输入`cargo run`。你会看到程序成功运行并开始计算。

---

## 扩展功能：创建辅助函数

如果你想为其添加更多功能，一个好方法是构建另一个函数。例如，我们可以创建一个辅助函数。

以下是创建辅助函数的步骤：

1.  **定义辅助函数**：例如，可以命名为`helper_function`。
2.  **封装逻辑**：将部分张量操作逻辑放入这个辅助函数中。
3.  **调用函数**：在主函数中调用这个辅助函数，例如`hello_world_pytorch`。

修改后，再次运行`cargo run`。Cargo会重新编译，程序将再次执行计算。

---

## 进一步修改与操作

如果我想进一步修改，可以添加另一个操作。例如，再次乘以2并打印结果。

以下是添加新操作的步骤：

1.  **添加新操作**：在现有逻辑后添加另一个乘法操作。
2.  **打印新结果**：将新操作的结果打印出来。
3.  **运行程序**：使用`cargo run`命令执行修改后的代码。

使用Rust绑定开始使用PyTorch非常简单直接。
