## 下载

### Docker

```
docker pull apachecn0/sicp-py-zh
docker run -tid -p <port>:80 apachecn0/sicp-py-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g sicp-py-zh
sicp-py-zh <port>
# 访问 http://localhost:{port} 查看文档
```
