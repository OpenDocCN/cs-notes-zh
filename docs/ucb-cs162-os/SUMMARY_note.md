# 📚 UCB CS162 操作系统课程笔记索引

在本教程中，我们将一起浏览加州大学伯克利分校CS162操作系统课程的系列讲座笔记。这些笔记涵盖了从操作系统基础概念到高级主题的完整知识体系，旨在帮助初学者系统性地理解操作系统的核心原理与设计。

## 📖 课程内容概览

以下是本系列课程的全部讲座笔记列表，按逻辑顺序排列，方便你循序渐进地学习。

*   **P1：讲座 1 什么是操作系统** - 介绍操作系统的定义、目标与核心作用。
*   **P2：讲座 2：四大操作系统基本概念** - 讲解构成操作系统基础的四个关键抽象。
*   **P3：讲座 3：进程、系统调用与 Fork** - 深入探讨进程的概念及其创建机制。
*   **P4：讲座 4：Fork（续）、I/O 简介（一切皆文件！）** - 继续讨论进程创建，并引入Unix哲学中的重要I/O概念。
*   **P5：讲座 5：套接字与 IPC（已完成）、并发进程与线程** - 介绍进程间通信和并发编程的初步概念。
*   **P6：讲座 6：同步 1 并发与互斥** - 开始探讨多线程/进程环境下的同步问题与互斥需求。
*   **P7：讲座 7：同步 2 并发（续）、锁的实现、原子** - 深入讲解如何通过锁和原子操作实现互斥。
*   **P8：讲座 8：同步 3 锁、信号量和监视器** - 介绍更高级的同步原语：信号量和监视器。
*   **P9：讲座 9：信号量（续）、监视器和读写者** - 继续讨论信号量的应用，并分析经典的“读者-写者”问题。
*   **P10：讲座 10：调度 1 概念与经典策略** - 开启调度主题，讲解调度器的基本目标和经典算法。
*   **P11：讲座 11：调度 2 案例研究，实时与前进进程** - 分析实际系统中的调度案例，并介绍实时调度概念。
*   **P12：讲座 12：调度 3 饥饿（完成），死锁** - 讨论调度中的饥饿问题，并引入系统性的资源竞争问题——死锁。
*   **P13：讲座 13：内存 1 地址转换与虚拟内存** - 开始内存管理部分，讲解地址转换和虚拟内存的基本思想。
*   **P14：讲座 14：内存 2 虚拟内存（续），缓存与 TLB** - 深入虚拟内存机制，并介绍加速地址转换的关键硬件：TLB。
*   **P15：讲座 15：内存 3 缓存与 TLB（续），分页** - 继续讨论缓存和TLB，并详细讲解分页机制。
*   **P16：讲座 16：内存 4 按需分页策略** - 探讨虚拟内存中按需调页的各种替换策略。
*   **P17：讲座 17：按需分页（完成），通用 I/O，存储设备** - 完成内存主题，转向I/O系统，介绍通用I/O模型和存储设备。
*   **P18：讲座 18：通用 I/O（续），存储设备，性能** - 继续讨论I/O，并分析存储设备的性能特性。
*   **P19：讲座 19：文件系统 1 性能（续），排队理论，文件系统** - 引入排队理论分析I/O性能，并开始讲解文件系统设计。
*   **P20：讲座 20：文件系统 2 文件系统设计（续），文件系统案例研究** - 深入文件系统结构，并分析实际文件系统案例。
*   **P21：讲座 21：文件系统 3 案例研究（续），缓冲，可靠性等** - 继续案例研究，并讨论文件系统的缓冲、一致性和可靠性机制。
*   **P22：讲座 22：事务（续）、端到端论证、分布式 D** - 介绍事务处理、端到端原则，并开启分布式系统主题。
*   **P23：讲座 23：分布式决策（续）、网络与 TCP I** - 讨论分布式一致性，并开始讲解网络基础与TCP协议。
*   **P24：讲座 24：网络与 TCP/IP（续）、RPC、分布式文件系统** - 继续网络主题，介绍远程过程调用（RPC）和分布式文件系统概念。
*   **P25：讲座 25：RPC、NFS 和 AFS** - 深入分析RPC以及两个经典的分布式文件系统：NFS和AFS。

## 🎯 学习总结

本节课中，我们一起梳理了UCB CS162操作系统课程的全套讲座笔记索引。这个系列从**什么是操作系统**这一根本问题出发，逐步深入到**进程管理**、**内存管理**、**文件系统**和**分布式系统**等核心模块。通过跟随这个索引顺序学习，你将能够建立起对操作系统工作原理全面而扎实的理解。每一讲笔记都对应着课程中的一个关键知识点，建议你按顺序阅读，并动手实践其中的代码示例，以巩固学习效果。
