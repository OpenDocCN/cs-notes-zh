# 001：课程介绍 🖥️

![](img/9e0628f8f8345944bd1ddbbcd3954d61_0.png)

在本节课中，我们将了解这门高级计算机体系结构课程的整体情况、课程目标、所需教材以及课程的组织方式。

欢迎来到计算机体系结构课程的Encore版本。我是David Wentzlaff。我是普林斯顿大学电气工程系的教授。我的背景是在学术界和工业界构建众核与多核微处理器。

例如，这是我在工业界时构建的一款微处理器。这实际上是一个64核的多核微处理器。它被安装在这样的电路板上。你可以看到，芯片就在这个散热片下面。在本课程中，你将学习如何构建与此复杂度相当的处理器。

本课程并非旨在教你如何构建基础处理器。我们会概述基础处理器和基础顺序流水线处理器。相反，在本课程中，我们将更深入地探讨如何构建乱序执行处理器。你将学习如何构建更高级的组件，例如高级缓存。我们将探讨如何构建向量处理器。我们将研究如何利用不同类型的并行性，例如指令级并行、线程级并行、进程级并行、向量并行等。我们还将详细讨论如何构建缓存一致性系统。在课程的最后，我们将讨论如何构建多核和众核微处理器。

需要指出的是，本课程是一门高级计算机体系结构课程。因此，我们将深入细节。我们将学习如何构建内部工作组件。例如，构建这些先进现代微处理器所需的所有不同结构和算法。我们不会花太多时间研究如何构建二三十年前的处理器。

在这个Encore版本中，我们将扩展首次开课的内容。在首次开课时，我们有超过10万名学生注册。这非常成功，我们深受鼓舞。我们希望这次开课同样成功。因此，请告诉你的朋友。如果你正在观看这个视频，并且知道有朋友可能从这门课程中受益，请让他们知道。如果他们对该领域感兴趣，请告知他们。此外，如果你正在其他大学学习计算机体系结构课程，并希望使用这些补充材料，请注册并告诉你的朋友也来注册。我们努力使这门课程对正在其他大学学习相关课程的人也有用。

现在，让我们停下来看看本课程的理念。在本课程中，我们力求提供与普林斯顿大学同等质量的高水平课程。这门课程将是严谨的。我们将使用优秀的教材，布置有挑战性的习题，并设置高难度的期中考试和期末考试。这里的权衡是，由于课程如此严谨，你将学到很多知识，并且会非常有收获。但与其他可能只是概论级别的课程不同，本课程将深入细节。相应的好处是，你将能够在你未来的生活中应用这些知识。

在讨论课程的严谨性和构成时，我想谈谈本课程推荐的两本教材。

以下是第一本推荐教材。它被称为Patterson和Hennessy的著作，书名是《计算机体系结构：量化方法》。我们将使用该书的第五版。我强烈推荐第五版。但对于那些无法获取第五版或觉得价格太贵的同学，你也可以尝试获取第四版。如果第四版中有相应内容，我们也会发布第四版的等效页码。有些主题只在第五版中涵盖，第四版中没有。此外，我想说明，如果我们指定了教材中的某个问题，我们将在课程网站上发布该问题的完整内容或进行转述。我们已经获得了出版商的许可。

本课程将使用的第二本书名为《现代处理器设计：超标量处理器基础》。我们将提及这本书，并称之为Shen和Lapoti的著作，因为这是该教材的两位作者。需要指出的是，这本蓝色封面、精装版的书籍最近已由出版商McGraw-Hill绝版，但它正由Waltham Press以更便宜的平装版重新发行。平装版将是白色封面，并将很快发布。我们会在课程网站上发布更多详细信息。

对于那些在课程中途加入的同学，我们鼓励你直接参与。期中考试和期末考试将开放较长时间供大家参加和评分。但如果你在课程中途加入，比如在期中考试前一周或之后加入，我们会发布这些考试的答案。然而，我们无法提供无限次数的、带有完整评分的期中考试和期末考试。你完全可以加入课程，将其用于自学。课程结束后，我们也会将课程内容保持开放，供自学使用。例如，如果你错过了期中考试，但赶上了期末考试，欢迎参加期末考试。

虽然这是课程的Encore版本，但我想谈谈自首次开课以来我们如何改进了这门课程。我们重新录制了部分讲座，并修正了幻灯片中的许多错误。谈到错误，我想鼓励大家，如果你在课程材料中发现任何错误，请登录并使用课程论坛，前往课程材料错误板块查看，确保该错误尚未被报告。如果没有，请告诉我们。我们真的希望尽可能完善这门课程。请在那里发布问题，我们会尽快修复。

在讨论论坛的话题时，我还想谈谈语言和地区特定的聚会小组或学习小组。我们在论坛上有一个区域，人们可以在那里聚会，如果他们来自特定地区或使用特定语言，可以在那里发帖。遗憾的是，我只说英语，因此无法用其他语言回答你的问题。但你的同学或许可以提供帮助。如果你能用另一种语言讨论课程中的不同问题，这可能对你有所帮助。请查看网站和论坛的那个特定板块，尝试组建聚会小组。在首次开课时，我们实际上在几个不同的国家有实体聚会小组，并且有关于本课程问题的深入的语言特定讨论。据我们了解，这确实帮助了来自世界各地的学生。

此外，我想谈的另一件事是调查问卷。这与我们首次开课有所不同。这次，我们将定期向你发送调查问卷。例如，在课程开始时、期中考试前、期中考试后以及课程结束时会有几次。这些当然是可选的。你不必填写。但我们试图了解选修本课程的学生来自哪里，包括地理位置、背景和准备情况。我们这样做的一个原因是为了在未来更好地调整课程，使其变得更好。同时，我们也在积极研究如何在未来更好地教授计算机体系结构，以及如何在大规模开放在线课程环境中进行更有效的教学。

最后，我想谈谈普林斯顿大学电气工程系以及所有在计算机工程和计算机体系结构领域非常强大的教职员工。如果你正在学习这门课程，并且非常享受、正在学习，并说“我希望未来继续从事这方面的工作”，例如，如果你是一名本科高年级学生，请申请普林斯顿大学电气工程系。来学习并探索如何在未来构建更好的微处理器。我们目前在电气工程系和计算机科学系有五名专注于计算机体系结构的教职员工。我们有数十名研究生，以及如前所述的五名计算机体系结构教职员工。计算机工程专业设在电气工程系，因此请申请电气工程系。我们在计算机科学系也有电路教授和许多软件教授。我们是一所顶尖的常春藤盟校。申请截止日期是12月15日。
